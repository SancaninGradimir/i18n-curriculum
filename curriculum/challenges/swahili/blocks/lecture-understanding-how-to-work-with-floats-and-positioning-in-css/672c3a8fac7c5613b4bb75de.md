# --interactive--

Apsolutno pozicioniranje vam omogućava da izvučete element iz normalnog protoka dokumenta, čineći ga tako da se ponaša nezavisno od drugih elemenata. Kada je element apsolutno pozicioniran, postavlja se u svoj sloj, potpuno odvojen od svega ostalog u rasporedu (layoutu). To ga čini korisnim za kreiranje plutajućih UI funkcija kao što su modali, alati za napomene ili padajući meniji, koji mogu preklapati druge elemente na stranici.

Po podrazumevanju, elementi apsolutno pozicionirani se postavljaju u odnosu na najbližeg pozicioniranog predaka. Ako se ne pronađe nijedan pozicionirani predak, element će biti pozicioniran u odnosu na početni kontejner blok, koji je obično pregledačov vidokrug (viewport). Možete pomeriti apsolutno pozicionirani element koristeći svojstva `top`, `bottom`, `left` i `right` kako biste specificirali koliko daleko treba da bude od ivica svog kontejner elementa.

Na primer, postavljanje `top: 30px` i `left: 30px` pomeraće element `30px` dalje od gornjih i levih ivica sadržajnog bloka. Evo primera kako primeniti apsolutno pozicioniranje na element u CSS-u:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

Kada se ovaj kod primeni, element će biti uklonjen iz normalnog protoka dokumenta i postavljen `30px` udaljen od vrha i leve strane svog kontejner bloka.

Apsolutno pozicioniranje može biti posebno korisno za kreiranje elemenata koji su namenjeni da plivaju preko ostatka sadržaja, kao što su preklapanja ili modalni prozori. Međutim, pošto je element uklonjen iz protoka dokumenta, može takođe stvoriti praznine ili uzrokovati da se drugi elementi sruše zajedno ako nije pravilno obrađen.
