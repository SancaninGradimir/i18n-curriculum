# --interactive--

Metoda `map` je moćna i široko korišćena funkcija u JavaScriptu koja operiše na nizovima. Dizajnirana je da kreira novi niz primenom zadane funkcije na svaki element originalnog niza.

Ovaj metod ne modifikuje originalni niz, već vraća novi niz koji sadrži rezultate funkcije primenjene na svaki element.

Evo primera korišćenja metode `map` na nizu brojeva:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

Da bismo kreirali novi niz gde je svaki broj udvostučen, koristimo metod ``map``. Metod ``map`` prima funkciju povratne pozive gde se funkcija poziva na svakom pojedinačnom elementu u nizu.

U ovom slučaju, svaki broj u nizu će biti pomnožen sa `2`. Rezultat će biti novi niz brojeva `2,4,6,8,10`.

Funkcija povratnog poziva može da prihvati do tri argumenta.

Prvi argument je trenutni element koji se obrađuje.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

Drugi argument je indeks trenutnog elementa koji se obrađuje.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

Treći argument je niz na kojem se `map` poziva.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

Razumevanje i efikasno korišćenje metode ``map`` može značajno poboljšati vašu sposobnost rada sa nizovima u JavaScript-u. U budućim lekcijama, detaljnije ćemo istražiti naprednije upotrebe ``map`` i istražiti kako to može biti moćan alat za izradu dinamičnih i efikasnih programa.
