# --interactive--

U prethodnoj lekciji ste naučili kako da bacate izuzetke u svojim programima. U ovoj lekciji ćemo pogledati kako pravilno rukovati ovim greškama u bloku `try…catch…finally`.

Blok ``try`` se koristi za obmotavanje koda koji bi mogao da izbaci grešku. Deluje kao sigurno mesto za pokušaj nečega što bi moglo da propadne.

Blok `catch` hvata i obrađuje greške koje nastaju u bloku `try`. Možete koristiti objekat `Error` unutar `catch` da biste pregledali šta je pošlo naopako.

Blok ``finally`` se izvršava nakon blokova ``try`` i ``catch``, bez obzira da li je došlo do greške. Često se koristi za zadatke čišćenja, kao što su zatvaranje fajlova ili oslobađanje resursa.

Ovo je primer korišćenja bloka `try…catch`:

:::interactive_editor

```js
function processInput(input) {
  if (typeof input !== "string") {
    throw new TypeError("Input must be a string.");
  }

  return input.toUpperCase();
}

try {
  console.log("Starting to process input...");
  const result = processInput(9);
  console.log("Processed result:", result);
} catch (error) {
  console.error("Error occurred:", error.message);
} 
```

:::

U ovom primeru, imamo funkciju nazvanu `processInput` koja prvo proverava da li je `input` nije tipa `string`. Ako je to slučaj, onda bacamo grešku. Inače, vraćamo rezultat korišćenjem metode `toUpperCase` na `input`.

Pozivamo funkciju unutar bloka `try`. Pošto poziv funkcije baca grešku, onda će biti uhvaćen unutar bloka `catch` i poruka o grešci će biti prikazana u konzoli.

Greška prosleđena u blok ``catch`` je objekat ``Error`` koji sadrži informacije o toj grešci. U ovom slučaju, koristimo svojstvo ``message`` koje prikazuje čitljivo za čoveka informacije korisniku.

Koristimo `console.error` jer je dizajniran posebno za logovanje grešaka. U mnogim modernim pregledačima, izlaz `console.error()` se pojavljuje u crvenoj boji u konzoli.

Izjava ``finally`` se izvršava bez obzira na to da li je izbačena izuzetna situacija ili ne.

```js
try {
  // Code that might throw an error
} catch (error) {
  // Code to handle the error
} finally {
  // Code that runs regardless of whether an error occurred or not
}
```

Dobar slučaj upotrebe za izjavu `finally` je ako radite sa fajlovima. U JavaScript-u, možete otvoriti fajl, koristiti blok `try` za pisanje podataka u fajl. Ako postoji bilo kakve greške, možete koristiti `catch` da uhvatite te greške. Zatim koristite izjavu `finally` za zatvaranje fajla.
