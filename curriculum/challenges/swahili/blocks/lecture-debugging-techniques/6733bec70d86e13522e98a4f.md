# --description--

Izjava `throw` u JavaScript-u se koristi za bacanje izuzetka definisanog od strane korisnika. Izuzetak u programiranju je situacija kada se dogodi neočekivani događaj i poremeti normalan tok programa.

Kao programeri, važno je obraditi ove izuzetke kako bi vaši programi neočekivano ne srušili kada dođe do grešaka. Evo osnovne sintakse za izjavu `throw`:

```js
throw expression;
```

Izraz (`expression`) u ovom slučaju bi bio objekat ili vrednost koja predstavlja izuzetak koji želite baciti. Primeri ovoga su ugrađene klase izuzetaka kao što su `Error`, `TypeError` ili klasa `RangeError`. Evo primera korišćenja izjave `throw` za bacanje `TypeError`:

```js
function validateNumber(input) {
  if (typeof input !== "number") {
    throw new TypeError("Expected a number, but received " + typeof input);
  }
  return input * 2;
}
```

U ovom primeru, proveravamo da li je tip `input`-a različit od tipa `number`. Ako nije, bacamo `TypeError` sa prilagođenom porukom. Inače, funkcija će vratiti rezultat množenja `input`-a sa `2`.

Ako želite baciti opštiju poruku o grešci, možete se na konstruktor `Error` referisati ovako:

```js
function divide(numerator, denominator) {
  if (denominator === 0) {
    throw new Error("Cannot divide by zero");
  }
  return numerator / denominator;
}
```

Evo primera funkcije koja će proveriti da li je imenilac `0`. Ako je to slučaj, onda će baciti prilagođenu poruku o grešci koja kaže `Cannot divide by zero` (Ne može se deliti sa nulom).

U sledećoj lekciji, osvrnuti ćemo se na način bacanja poruka o greškama unutar konteksta bloka `try`/`catch`, koji se koristi za elegantnu obradu izuzetaka u JavaScript-u.
