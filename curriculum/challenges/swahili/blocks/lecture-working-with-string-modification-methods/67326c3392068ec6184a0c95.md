# --interactive--

Kada radite sa JavaScript-om, možete naići na situacije u kojima trebate ponoviti string specifičan broj puta.

Bez obzira da li generišete ponavljajuće obrasce ili samo duplirate tekst, metoda `repeat()` pruža jednostavan i efikasan način za postizanje toga.

Metoda ``repeat()`` je ugrađena funkcija u JavaScriptu koja vam omogućava da ponovite niz određeni broj puta. Evo osnovne sintakse:

```js
string.repeat(count);
```

`string` je niz koji želite da ponovite, a `count` je broj puta koliko želite da se niz ponovi. Evo primera:

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

U ovom slučaju, string `Hello!` je ponovljen tri puta, što rezultira u `Hello!Hello!Hello!`.

Iako je metoda ``repeat()`` korisna, postoje neki izuzeci i ograničenja o kojima treba zapamtiti.

Parametar `count` mora biti neujem broj. Ako prosledite negativan broj, JavaScript će baciti `RangeError`.

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

Ovaj `count` mora biti konačan broj. Ako pokušate da ponovite niz beskonačno mnogo puta ili koristite `Infinity` kao brojač, tako ćete dobiti i `RangeError`.

U JavaScriptu, `Infinity` je posebna vrednost koja predstavlja beskonačnu količinu. Koristi se za označavanje brojeva koji su veći od bilo kog konačnog broja.

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

Ako broj nije ceo broj (kao što je decimalni broj poput `2.5`), metoda `repeat()` će ga zaokružiti nadole na najbliži ceo broj.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

Ako prosledite ``0`` kao brojač, metoda ``repeat()`` će vratiti prazan string.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

Metoda `repeat()` može pojednostaviti zadatke koji uključuju dupliranje niza karaktera, čineći vaš kod sažetijim i čitljivijim.

Bilo da generišete ponavljajuće tekstualne obrasce ili popunjavate prostor znakovima, ``repeat()`` vam može uštedeti od pisanja petlji ili složenijeg koda.

Niste ograničeni na prosleđivanje broja direktno u metod ``repeat()``. Takođe možete proslediti varijablu koja čuva numeričku vrednost.

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

U ovom primeru, varijabla `count` skladišti broj ponavljanja. Ovo može biti korisno kada zavisi broj ponavljanja od unosa korisnika ili drugih dinamičkih vrednosti u vašem programu.
