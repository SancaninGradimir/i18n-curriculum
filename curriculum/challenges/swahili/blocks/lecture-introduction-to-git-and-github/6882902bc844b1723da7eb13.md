# --description--

Pre nego što zaista počnete da koristite kontrolu verzija, i sajtove poput GitHub-a, morate imati instaliran i spreman za upotrebu Git.

Da proverite da li je Git već instaliran na vašoj mašini, možete pokrenuti sledevu komandu u terminalu:

```sh
git --version
```

Ako vidite broj verzije, to znači da je Git instaliran. Ako ne, onda ćete morati da ga instalirate.

Za Linux sisteme, Git često dolazi predinstaliran sa većinom distribucija. Ako nemate Git predinstaliran, trebalo bi da ga instalirate komandama vašeg menadžera paketa, kao što su `sudo apt-get install git` ili `sudo pacman -S git`.

Za korisnike Mac-a, možete instalirati Git preko Homebrew-a sa `brew install git`, ili možete preuzeti izvršni instalater sa sajta Gita.

Za Windows, možete preuzeti izvršni instalater sa sajta Gita. Ili, ako ste postavili Chocolatey, možete pokrenuti `choco install git.install` u PowerShell-u. Napomena da na Windows-u možda želite da preuzmete GitBash kako biste imali dostupan Unix-sličan okruženje (shell).

Da biste bili sigurni da je instalacija uspela, ponovo pokrenite komandu `git --version` u terminalu.

Nakon što instalirate Git, moraćete da izvršite nekoliko promena na konfiguraciji. Trebalo bi da napravite ove promene samo jednom na svom računaru i nećete ih morati stalno menjati kada vršite nadogradnje (upgrade).

`git config` se koristi za postavljanje varijabli konfiguracije koje su odgovorne za način na koji Git radi na vašoj mašini. Da biste videli vaše trenutne varijable podešavanja i gde su sačuvane na vašem sistemu, možete pokrenuti sledevu komandu:

```sh
git config --list --show-origin
```

Trenutno bi trebalo da vidite samo sistemska podešavanja konfiguracije ako ste tek prvi put instalirali Git. Ako ste već instalirali Git pre ovog kursa, možda ćete videti korisničke konfiguracije za korisničko ime i e-mail adresu.

Da biste postavili vaše korisničko ime, možete pokrenuti sledevu komandu:

```sh
git config --global user.name "Jane Doe"
```

Flaga `--global` se ovde koristi za postavljanje korisničkog imena za sve projekte na vašem sistemu koji koriste Git. Ako morate da preuzmete (override) korisničko ime za određeni projekat, onda možete pokrenuti komandu u tom određenom direktorijumu projekta bez flage `--global`.

Da biste postavili korisnički e-mail adres, možete pokrenuti sledevu komandu:

```sh
git config --global user.email janedoe@example.com
```

Konfigurisati vaše korisničko ime i e-mail je važno jer će Git koristiti ove informacije za svaki commit koji napravite u projektu. Vi ćete saznati više o commitima u budućem kursu.

Druga konfiguracija koju možete postaviti je preferirani editor koji želite da koristi Git. Evo primera kako da postavite vaš preferirani editor na Emacs:

```sh
git config --global core.editor emacs
```

Ostale opcije uključuju Vim, Nano, VS Code i više. Na primer, ako koristite VS Code za svoj editor, onda možete konfigurisati podešavanja Gita ovako:

```sh
git config --global core.editor "code --wait"
```

Ako odlučite da ne postavljate preferirani editor, Git će podrazumevano koristiti standardni editor vašeg sistema.

Ako koristite Windows mašinu i želite da promenite svog editora, moraćete da navedete pun put do izvršne datoteke ovako:

```sh
git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe' -multiInst -notabbar -nosession -noPlugin"
```

Da biste videli listu svojih konfiguracija, možete pokrenuti sledevu komandu:

```sh
git config --list
```

Sada kada imate instaliran i podešen Git, spremni ste da počnete sa dodavanjem kontrole verzija svojim projektima.
