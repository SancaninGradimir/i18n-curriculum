# --description--

Naučili ste o API-jima `HTMLMediaElement` i MediaStreams, ali postoji još nekoliko API-ja koji mogu biti veoma korisni pri radu sa video i audio sadržajem.

Prvi je Screen Capture API. Kako ime može da sugeriše, ovaj API vam omogućava da snimite ekran korisnika. Ovaj API je dostupan pozivanjem metode ``getDisplayMedia()`` objekta ``mediaDevices`` i konzumiranjem vraćenog medijskog streama.

Zatim postoji MediaStream Recording API. Ovaj API radi u tandemu sa MediaStreams API-jima, omogućavajući vam da snimite MediaStream (ili čak direktno `HTMLMediaElement`). Zatim emituje `dataavailable` događaje sa Blob učitateljima koje možete zapisati u lokalno skladište fajlova.

Osnova sve ove tehnologije jeste Media Source Extensions API. Media Source Extensions API vam omogućava da direktno prosledite korisnikovu feed sa web kamere na video element sa svojstvom `srcObject`, na primer.

A konačno, Web Audio API koji pokreće sve što je čujno na vebu. Ovaj API uključuje važne objekte kao što su `AudioBuffer` (koji predstavlja Buffer sa audio podacima) ili `AudioContext`.

I time završavamo naše lekcije o zvuku i videu!
