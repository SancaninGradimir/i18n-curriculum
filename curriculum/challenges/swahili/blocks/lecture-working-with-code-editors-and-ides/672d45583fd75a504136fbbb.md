# --description--

Da biste instalirali VS Code, trebaće vam da posetite njihov sajt kako biste dobili odgovarajući instalater za vaš operativni sistem. Uverite se da posetićete sajt VS Code-a, a ne sajt Visual Studija.

Za Windows, možete preuzeti EXE instalater. Nakon što ga preuzmete, pokrenite fajl i čarobnjak za podešavanje će vas voditi kroz proces instalacije.

Za Mac, možete preuzeti zip arhivu koja sadrži ``.app`` koji možete pokrenuti. Takođe možete instalirati VS Code preko homebrewa.

Otvorite aplikaciju Terminal na vašem Mac-u i pokrenite sledeću komandu:

```sh
brew install --cask visual-studio-code
```

Na Linuksu, možete preuzeti fajl `.deb` ili `.rpm` direktno. Kada ga instalirate, možete koristiti odgovarajući izvršni program, ili CLI komandu (obično `code`), da pokrenete aplikaciju. I sada ste spremni za kodiranje.
