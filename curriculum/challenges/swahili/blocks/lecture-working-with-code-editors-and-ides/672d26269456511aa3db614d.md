# --description--

Možda mislite da su ovo iste stvari, ali nisu sasvim. Editor koda je bilo koja aplikacija koja vam omogućava da uređujete fajlove sa kodom. IDE, ili Integrisano razvojno okruženje (Integrated Development Environment), je puna aplikacija koja vam omogućava da kompajlirate, pokrećete i debagujete svoj kod dok ga uređujete.

Možda bi neki primeri bili korisni.

Prvo, razmotrimo nekoliko popularnih IDE-ova. Visual Studio je integrisano razvojno okruženje kompanije Microsoft koje pruža sveobuhvatni set alata za izradu, debagovanje i implementaciju aplikacija na različitim platformama.

Još jedan IDE je XCode. Xcode je integrisano razvojno okruženje kompanije Apple dizajnirano za kreiranje, kodiranje i debagovanje aplikacija za macOS, iOS, watchOS i tvOS.

I još jedan IDE bi bio Android Studio. Android Studio je integrisano razvojno okruženje kompanije Google specifično dizajnirano za izradu, debagovanje i testiranje Android aplikacija.

To su samo nekoliko primera IDE-ova. Sada, hajde da pogledamo nekoliko popularnih editora koda.

Visual Studio Code je lagan, open-source editor koda od Microsofta koji podržava širok spektar programskih jezika i pruža funkcije kao što su debagovanje, isticanje sintakse (syntax highlighting) i kontrola verzija putem ekstenzija.

Drugi popularni editor bi bio Sublime Text. Sublime Text je brzi, svestrani tekstualni editor poznat po svom elegantnom interfejsu, moćnim funkcijama i podršci za širok spektar programskih jezika putem prilagodljivog isticanja sintakse i dodataka (plugin-ova).

I još jedan je Notepad++. Ovo je besplatan, open-source tekstualni i editor izvornog koda za Windows koji nudi isticanje sintakse, skupljanje koda (code folding) i niz dodataka za povećanje produktivnosti i prilagodljivosti.

Možda ste primetili kako se editori koda fokusiraju prvenstveno na tekstualni sadržaj fajla, dok IDE-ovi izlažu različite alate za upravljanje vašim kodom.

Svi ovi primeri su lokalne programe koje možete pokrenuti na svom računaru, ali postoje i cloud-bazirani editori koje možete koristiti.

Cloud-bazirani editor je online alat koji korisnicima omogućava da pišu, uređuju i upravljaju kodom ili tekstom direktno putem web pregledača bez potrebe za lokalnom instalacijom softvera.

Hajde da pogledamo nekoliko cloud-baziranih editora.

Replit je online platforma koja pruža kolaborativno okruženje za kodiranje, omogućavajući korisnicima da pišu, pokreću i dele kod u različitim programskim jezicima direktno iz web pregledača.

Drugi popularni cloud-bazirani editor je GitHub Codespaces. Ovo je cloud razvojno okruženje koje pruža trenutni pristup potpuno konfigurisanim editoru koda i alatima za razvoj direktno sa GitHub-a, omogućavajući besprekorno kodiranje i saradnju.

I još jedan je Ona. Ona je cloud razvojno okruženje koje se integriše sa GitHub-om i GitLab-om, nudeći trenutne, prilagodljive radne površine za kodiranje, izradu i debagovanje direktno iz vašeg pregledača.

I postoji mnogo više opcija. Neke opcije, kao što je Visual Studio Code, veoma su proširive i mogu raditi sa više različitih tipova projekata i jezika. Ostale opcije mogu biti specifično prilagođene malom podskupu jezika ili tipova projekata.

Aplikacija koju koristite može biti drugačija za određene projekte. Trebalo bi da istražite opcije kako biste videli šta će najbolje odgovarati vašim potrebama.
