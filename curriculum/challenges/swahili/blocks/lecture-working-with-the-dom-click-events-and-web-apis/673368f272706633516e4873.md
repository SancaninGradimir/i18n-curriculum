# --interactive--

Možda ste upoznati sa radom sa atributima u svojim HTML i CSS projektima. Ali znate li da možete dodati atribute vašim HTML elementima u svoj JavaScript fajl?

U ovoj lekciji, naučićete kako da radite sa metodom `setAttribute()` za dodavanje atributa vašim HTML elementima.

Evo osnovne sintakse:

```js
setAttribute(attribute, value);
```

Pogledajmo nekoliko primera kako bismo bolje razumeli kako koristiti metod `setAttribute()`.

In this first example, we have a `p` element inside the HTML:

```html
<p id="para">I am a paragraph</p>
```

Da bismo dodali atribut `class`, prvo moramo pristupiti tom elementu `p` koristeći metodu `getElementById()`. Zatim možemo koristiti metodu `setAttribute()` da dodamo atribut `class` i postavimo vrednost na `my-class`.

Koristimo ``outerHTML`` za logovanje celog HTML elementa, uključujući njegove atribute.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

Ako imate postojeći atribut na HTML elementu, možete ažurirati njegovu vrednost koristeći metod `setAttribute()`.

U ovom primeru imamo element `div` sa atributom `class` čija je vrednost postavljena na `my-class`:

```html
<div class="my-class"></div>
```

Da bismo ažurirali vrednost atributa `class` na primer, možemo ponovo koristiti metodu `setAttribute()`:

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

Dakle, koji su neki primeri iz stvarnog sveta kada koristiti metodu `setAttribute()`?

Ako gradite dinamičnu galeriju slika, možda ćete morati da ažurirate atribut `src` elementa slike kada korisnik klikne na minijaturu.

Drugi primer bi bio ako se bavite validacijom formulara i morate dodati određene atribute kao što su `required` ili `minlength` na `input` element.
