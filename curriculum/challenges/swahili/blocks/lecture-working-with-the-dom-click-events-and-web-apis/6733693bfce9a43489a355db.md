# --interactive--

Događaj `DOMContentLoaded` se aktivira kada je sve u HTML dokumentu učitano i parsirano. Ako imate eksterne stilove ili slike, događaj `DOMContentLoaded` neće čekati da se oni učitaju. Čekaće samo da se HTML učita.

Ovo se razlikuje od `load` događaja, koji čeka da se sve učita, uključujući eksterne stilove, slike i slično.

Evo primer sintaksa za korišćenje događaja `DOMContentLoaded`:

```js
document.addEventListener("DOMContentLoaded", function () {
  console.log("DOM is loaded.");
});
```

Nakon što se DOM učita, funkcija će biti izvršena i poruka `DOM is loaded.` će biti logovana u konzolu.

Sada, hajde da pogledamo još jedan primer koristeći događaj `DOMContentLoaded`. U ovom primeru, imamo sliku unutar HTML-a. Da bismo ažurirali sliku, možemo kreirati funkciju koja menja atribute `src` i `alt` za sliku:

:::interactive_editor

```html
<h1>Image Change on DOM Loaded</h1>
<img
  id="example-img"
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
  alt="Cat relaxing"
/>

<script src="index.js"></script>
```

```js
function changeImg() {
  const img = document.getElementById("example-img");
  img.src =
    "https://cdn.freecodecamp.org/curriculum/responsive-web-design-principles/FCCStickers-CamperBot200x200.jpg";
  img.alt = "CamperBot sticker";
  console.log("image was just changed");
}

changeImg();
```

:::

Zatim možemo proveriti da li je DOM još uvek učitavanje i dodati slušalicu događaja za `DOMContentLoaded` događaj. Ako je `DOMContentLoaded` događaj već isplivao, možemo direktno pozvati funkciju `changeImg`:

```js
if (document.readyState === "loading") {
  document.addEventListener("DOMContentLoaded", changeImg);
} else {
  console.log("DOMContentLoaded has already fired");
  changeImg();
}
```

Ako pokrenete ovaj kod i osvježite stranicu, videćete da se slika promijeni u novu sliku nakon što je DOM učitan. To se dešava prilično brzo, ali ćete vidjeti blago treperenje dok se slika mijenja.

Događaj `DOMContentLoaded` je koristan kada želite da pokrenete neki JavaScript kod čim se DOM učita, ali pre nego što se učitaju sve eksterne resurse kao što su slike i stilovi.

Kako se događaj ``DOMContentLoaded`` razlikuje od događaja ``load``?
## --answers--

`DOMContentLoaded` čeka da se slike učitaju, dok učitavanje ne čeka.
### --feedback--

---

`DOMContentLoaded` fires before all resources are loaded, while load waits for everything.

---

`DOMContentLoaded` can only be used with external scripts.
