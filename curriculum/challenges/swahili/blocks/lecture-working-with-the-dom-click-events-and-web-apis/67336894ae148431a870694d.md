# --interactive--

Naučićemo se o DOM-u i zašto je toliko važan za razvoj veb aplikacija. DOM znači Document Object Model. To je programski interfejs koji nam omogućava da interagujemo sa HTML dokumentima.

Sa POM-om, možete dodati, modifikovati ili obrisati elemente na veb stranici. Čak možete učiniti da je vaš sajt interaktivan tako što ćete elementima omogućiti da slušaju i reaguju na događaje.

U DOM-u, HTML dokument je predstavljen kao drvo čvorova. Svaki elementni čvor predstavlja HTML element iz HTML dokumenta:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

Ovo je dijagram koji predstavlja osnovnu DOM strukturu našeg primera:

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

Naravno, ovo može biti detaljnije i složenije na osnovu strukture HTML oznaka web stranice.

HTML dokument je koreni čvor u DOM hijerarhiji. Ima jedan dete čvor, element `html`. Ovo je koreni element HTML dokumenta, pošto sve ostale čvorove potiču od njega.

Element `html` ima dvoje dece: `head` i `body`.

Element `head` sadrži metapodatke o dokumentu. Pruža esencijalne informacije o web stranici.

Element ``body`` sadrži vidljiv sadržaj web stranice.

Možete pristupiti ovim elementima pomoću JavaScript-a, raditi sa njima u svom kodu i čak ih dinamički modifikovati. To je moć DOM-a i web API-ja.

Da biste pristupili ovim elementima u JavaScriptu, možete koristiti metode `getElementById()` i `querySelector()`. Ove metode su web API-ji jer pružaju standardizovane načine za interakciju sa DOM-om korišćenjem JavaScripta.

Sa `getElementById()`, možete dobiti objekat koji predstavlja HTML element sa navedenim `id`. Zapamtite da `id` moraju biti jedinstveni u svakom HTML dokumentu, pa će ova metoda vratiti samo jedan `Element` objekat. Ovde možete videti primer:

```js
const container = document.getElementById("container");
```

Ova linija JavaScript koda dobija element sa vrednošću `id` od `container` i dodeljuje taj objekat JavaScript konstanti. Morate proslediti `id` unutar navodnika kao argument. Ako ovaj objekat logujete u konzolu, videćete ga u izlazu:

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` je širi od `getElementById()`. Sa `querySelector()`, možete dobiti prvi element u HTML dokumentu koji odgovara CSS selektoru prosleđenom kao argument.

U ovom primeru, dobićete element koji odgovara selektoru `section`:

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

Ako želite da izaberete element po njegovom imenu klase koristeći `querySelector()`, morate dodati tačku (dot) ispred imena klase (`.`). Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

Takođe imate druge metode za podudaranje više elemenata, kao što su `getElementsByClassName()` i `querySelectorAll()`. Vi ćete saznati više o njima u budućim lekcijama.

Razumevanjem kako da se koriste web API-ji za efikasnu manipulaciju DOM-om, možete kreirati moćne i interaktivne veb aplikacije.
