# --interactive--

API `Canvas` je moćan alat koji vam omogućava manipulisanje grafikom direktno unutar vašeg JavaScript fajla. Sve počinje sa `canvas` elementom u HTML-u. Ovaj element služi kao površina za crtanje koju možete manipulisati koristeći instancijske metode i svojstva API-ja `Canvas`.

API pruža sve što vam je potrebno da kreirate neverovatne vizuale, uključujući oblike, tekst, animacije i čak kompleksne igre. Ima interfejse kao što su `HTMLCanvasElement`, `CanvasRenderingContext2D`, `CanvasGradient`, `CanvasPattern`, `TextMetrics` koji pružaju metode i svojstva koja možete koristiti za kreiranje grafike u vašoj JavaScript datoteci.

Pogledajmo kako radi `Canvas` API tako da možete početi kreirati grafiku sa JavaScript-om.

Prvo, morate kreirati `canvas` element u vašoj HTML datoteci:

```html
<canvas id="my-canvas"></canvas>
```

Element `canvas` je predstavljen interfejsom `HTMLCanvasElement`, koji pruža metode i svojstva za manipulisanje njime. Pored toga, možete koristiti metode i svojstva iz drugih interfejsa u `Canvas` API-ju.

Možete dati vašem `canvas` u `width` i `height` unutar HTML-a:

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

Ili možete koristiti svojstva ``width`` i ``height`` interfejsa ``HTMLCanvasElement``:

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

Trenutno ne možete ništa videti na ekranu. Nakon kreiranja vašeg elementa `canvas`, sledeća stvar je da dobijete pristup kontekstu crtanja platna pomoću metode interfejsa `getContext()` od `HTMLCanvasElement`.

Najčešći kontekst je ``2d``, koji vam omogućava crtanje u dve dimenzije:

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

Ako logujete varijablu `ctx` u konzolu, videćete metode i svojstva `CanvasRenderingContext2D` koja možete koristiti za kreiranje oblika, boja, linija i više, zajedno sa njihovim podrazumevanim vrednostima:

```js
console.log(ctx);
```

Kada imate 2D kontekst, možete početi da crtate na kanvasu.

API `Canvas` pruža nekoliko metoda i svojstava za crtanje oblika, linija i teksta. Jedno od njih je svojstvo `fillStyle`, koje možete kombinovati sa metodom `fillRect()` za crtanje pravougaonika ili kvadrata:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

`fillRect` uzima 4 numeričke vrednosti koje predstavljaju x osu, y osu, širinu i visinu, respektivno.

Sada je nešto na ekranu. Takođe možete crtati tekst ili čak kreirati animaciju. Evo platna za predstavljanje teksta:

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

Da biste konačno nacrtali tekst, prosledite tekst u metod ``fillText()`` kao prvi argument, praćen vrednostima za x i y osu:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

Rezultat u pretraživaču će biti crveni tekst `Hello HTML Canvas!`.

Ovime možete učiniti mnogo više sa API-jem `Canvas`. Na primer, možete ga kombinovati sa `requestAnimationFrame` da kreirate prilagođene animacije, vizualizacije, igre i još mnogo toga.
