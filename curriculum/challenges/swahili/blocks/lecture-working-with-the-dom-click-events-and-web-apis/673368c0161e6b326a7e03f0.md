# --interactive--

Pogledaćemo kako možete kreirati čvorove sa `innerHTML` i `createElement()`.

Počećemo sa `innerHTML`.

`innerHTML` je svojstvo objekata `Element` koje možete koristiti za postavljanje njihovog HTML oznakovanja. Sa `innerHTML`, možete postaviti HTML strukturu postojećeg elementa sa stringom, stvarajući sve potrebne čvorove.

Ovo je primer. Imamo prazan `div` i dodaćemo neke elemente unutar njega koristeći svojstvo `innerHTML`:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

First, we need to select the element by its ID:

```js
const container = document.getElementById("container");
```

Zatim, postavljamo svojstvo ``innerHTML`` kontejnera na niz znakova. Ovaj niz znakova mora da sadrži sav označni jezik neophodan za predstavljanje čvorova i strukture koje želite kreirati. Možete to zamisliti kao pisanje HTML-a unutar niza znakova.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

Postavićemo ``innerHTML`` ovog elementa na neuređenu listu sastojaka za picu (sir i paradajz).

Takođe možete napisati niz na više linija ako to čini oznaku lakšom za razumevanje. Važno je da oznaka tačno predstavlja HTML strukturu koju želite kreirati.

Nakon pokretanja ovog koda, videćete sledeću HTML strukturu ako pregledate svoj ispis.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

Novi čvorovi su kreirani i dinamički dodati u DOM nakon što je niz parsiran.

``innerHTML`` može biti veoma korisno za određene scenarije. Međutim, ima nekih sigurnosnih rizika sa kojima biste trebali da budete upoznati. Ne bi trebalo da ga koristite ako nećete imati potpunu kontrolu nad nizom.

Na primer, ako će korisnik uneti niz karaktera, ne bi trebalo da koristite `innerHTML` jer korisnik može da ubaci zlonamerni sadržaj na vaš sajt. Zbog toga, obično se preporučuje korišćenje `textContent` umesto njega, za unos običnog teksta.

Drugi način za kreiranje novog čvora je korišćenje metode ``createElement()``. Ovim novim metodom, možete kreirati novi element specifikovanjem njegovog imena oznake.

Na primer, ako želite da kreirate element slike, prosledili biste oznaku `img` kao string prilikom pozivanja ove metode:

```js
document.createElement("img");
```

A možete dodeliti ovaj novi objekat promenljivoj:

```js
const img = document.createElement("img");
```

Metoda `createElement()` vraća novi objekat `HTMLElement` ako je dokument `HTMLDocument`. Inače, vraća objekat `Element`.

Nakon što imate ovaj novi element spreman, možete ga dodati u DOM kao dete drugog postojećeg elementa koristeći metod `appendChild()`, ili ga možete umetnuti na određenu lokaciju koristeći druge metode. Izaberite onaj koji najbolje odgovara vašim potrebama.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

Kreiranje čvorova sa `innerHTML` i `createElement()` omogućava vam da dinamički manipulišete strukturom i sadržajem vaših veb-sajtova. Kombinovanjem ovih tehnika, možete izgraditi interaktivne veb aplikacije.
