# --interactive--

Biće trenutaka kada ćete morati da manipulišete stilovima HTML elementa unutar svog JavaScript fajla. Primer ovoga je kada trebate sakriti ili prikazati meni kada korisnik klikne na dugme.

U ovoj lekciji, naučićete kako da manipulišete stilovima koristeći svojstva `Element.style` i `Element.classList`.

Svojstvo ``Element.style`` je svojstvo samo za čitanje koje predstavlja ugrađeni stil elementa. Možete koristiti ovo svojstvo da dobijete ili postavite stil elementa.

Ovo je primer paragrafa elementa sa ID-om `para`:

```html
<p id="para">This is a paragraph</p>
```

Možete koristiti svojstvo ``style`` da promenite boju paragrafa na crvenu:

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

Svojstvo `style` se može koristiti za postavljanje mnogih CSS svojstava, kao što su `color`, `background-color`, `font-size`, `font-weight`, i tako dalje.

Drugi način za manipulisanje stilovima je korišćenjem svojstva ``Element.classList``. Svojstvo ``classList`` je svojstvo samo za čitanje (read-only) koje se može koristiti za dodavanje, uklanjanje ili prebacivanje klasa na element. Pogledajmo nekoliko primera.

U ovom primeru, dodati ćete klasu nazvanu `highlight` paragraf elementu sa ID-om `para`. U CSS-u, postaviti ćemo klasu `highlight` da promeni pozadinsku boju paragraf elementa u žutu. U JavaScript-u, dodaćemo klasu `highlight` paragraf elementu koristeći metod `classList.add()`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

Takođe možete dodati više klasama elementu prosleđivanjem ih kao argumenata metodi `classList.add()`:

```js
classList.add("class1", "class2", "class3");
```

Ako trebate ukloniti klasu sa elementa, možete koristiti metod `classList.remove()` na ovaj način:

```js
classList.remove("highlight");
```

Da biste prebacivali klasu na elementu, možete koristiti metodu `classList.toggle()`. U ovom primeru, imamo element `button` sa ID-om `toggle-btn` i element `nav` sa ID-om `menu`. Unutar CSS-a, dodali smo neke stilove za meni i klasu nazvanu `show`, koja će promeniti svojstvo `display` elementa `.menu` na `block`. Unutar JavaScript-a, pristupamo elementu menija i elementu `button` koristeći metodu `getElementById()`. Zatim, dodajemo slušalicu događaja (event listener) na element `button`, koja će prebacivati klasu `show` na element `.menu` kada se klikne na dugme:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

Kada korisnik klikne na dugme, klasa `show` će biti dodata elementu `.menu`, i meni će biti prikazan. Ako korisnik ponovo klikne na dugme, klasa `show` će biti uklonjena sa elementa `.menu`, i meni će biti sakriven.

Koristeći metode i svojstva kao što su `style`, `add()`, `remove()`, i `toggle()`, možete lako manipulisati stilovima elementa pomoću JavaScript-a, kreirajući dinamične i interaktivne veb stranice.
