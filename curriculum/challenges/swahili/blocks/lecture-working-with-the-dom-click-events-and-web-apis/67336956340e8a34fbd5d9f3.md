# --interactive--

Kreiranje glatkih animacija na veb stranici može biti komplikovano, posebno ako niste sigurni kako pravilno da rukujete vremenom (timingom). Velika vest je da ``requestAnimationFrame()`` API to olakšava.

`requestAnimationFrame()` je metoda koja vam omogućava da zakazujete sledeći korak vaše animacije pre sledećeg ponovnog crtanja ekrana, što rezultira tečnim i vizuelno privlačnim iskustvom.

Sledeći repaint ekrana odnosi se na trenutak kada pretraživač osveži vizuelni prikaz web stranice. Ovo se dešava više puta u sekundi, obično oko 60 puta (ili 60 frejmova po sekundi) na većini ekrana.

Da biste koristili metodu `requestAnimationFrame()`, sve što treba da uradite je da ga pozovete i prosledite mu funkciju povratne pozive:

```js
requestAnimationFrame(callback);
```

Pozivanje `requestAnimationFrame()` mora se prvo desiti unutar funkcije koja obrađuje animaciju, kao što je `animate()`, zajedno sa funkcijom za ažuriranje animacije, tradicionalno nazvanom `update()`:

```js
function animate() {
 // Update the animation...
 // for example, move an element, change a style, and more.
 update();
 // Request the next frame
 requestAnimationFrame(animate);
}
```

Funkcija ``update()`` je mesto gde se čudo dešava. Unutar nje, možete promeniti sve što želite da animirate. Na primer, ažuriranje stila ili menjanje pozicije elementa:

```js
function update() {
 element.style.transform = `translateX(${position}px)`;
 position += 2;
}
```

Šta konačno pokreće animaciju je pozivanje `requestAnimationFrame()` i prosleđivanje funkcije `animate`, ovog puta izvan funkcije `animate`:

```js
requestAnimationFrame(animate);
```

Petlja će se nastaviti dok je ne zaustavite.

Sada, hajde da pogledamo još jedan primer. HTML za ovaj primer je element `div` sa tekstom `freeCodeCamp is Awesome`. CSS čini `div` pravougaonikom i skriva sve što izlazi iz vidokruga sa leve ili desne strane. JavaScript kod pomera pravougaonik `2px` udesno pri svakom pozivu `requestAnimationFrame(animate)` sa funkcijom `animate` kao povratnom funkcijom:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div id="rect" class="rect">freeCodeCamp is Awesome</div>
<script src="index.js"></script>
```

```css
body {
  overflow-x: hidden;
}

.rect {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 400px;
  height: 250px;
  border-radius: 5px;
  background-color: #1b1b32;
  color: #f5f6f7;
  font-size: 2rem;
  position: absolute;
}
```

```js
const rect = document.getElementById("rect");

let position = 0;

function update() {
  // Move the rectangle 2px to the right
  rect.style.left = position + "px";
  position += 2;

  if (position > window.innerWidth) {
    // Move the rectangle just outside the left side of the screen
    position = -rect.offsetWidth;
  }
}

function animate() {
  update();

  //request the next frame
  requestAnimationFrame(animate);
}

// Start the animation
requestAnimationFrame(animate);
```

:::

Rezultat u pretraživaču će biti animirana titlovka koja pluta preko ekrana.
