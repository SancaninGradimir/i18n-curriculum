# --description--

Naučićemo o DOM čvorovima i njihovim odnosima u DOM stablu.

Kao što stvarni drvo ima velike i male grane povezane u hijerarhijsku strukturu, DOM čvorovi takođe imaju direktne i indirektne odnose jedan sa drugim. Koristićemo ovaj primer da ilustrujemo ove odnose:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>DOM Tree Example</title>
  </head>
  <body>
    <h1>Heading 1</h1>
    <p>Paragraph 1</p>
    <ul>
      <li>List item 1</li>
      <li>List item 2</li>
    </ul>
  </body>
</html>
```

Počnimo od vrha. Koren DOM stabla je element ``html``. To je kontejner na najvišem nivou za sav sadržaj HTML dokumenta. Svi ostali čvorovi su potomci ovog korenovog čvora.

Zatim, ispod korenovog čvora, nalazimo druge čvorove u hijerarhiji. Roditeljski čvor je element koji sadrži druge elemente. Detiljajući čvor je element koji je sadržan unutar drugog elementa.

U ovom primeru, element `body` je roditelj elementa paragraf, dok je element paragraf dete elementa `body`. Ovi elementi su predstavljeni kao čvorovi u DOM stablu sa istim odnosima.

Kao što imamo roditeljske čvorove i detijske čvorove, imamo i braćanske čvorove. Braćanski čvorovi su elementi koji dele istog roditelja. U našem primeru, dva elementa liste stavki dele isti roditeljski element neuređene liste, pa su braća.

Slično, elementi `h1` i paragraf su braća jer dele istog roditelja, element `body`.

Takođe imamo indirektne odnose kroz različite nivoe u hijerarhiji. Potonici su elementi koji su sadržani unutar drugog elementa, bilo direktno ili indirektno. Element se smatra potonikom drugog ako ga je moguće doći uzlazeći kroz DOM stablje nizvodno od njegovog predaka.

U našem primeru, elementi stavke liste `li` su potomci elementa `body`, jer su indirektno sadržani u njemu.

Ancestorski čvor je element koji je viši u hijerarhiji DOM stabla od drugog elementa. U našem primeru, element ``body`` je predak elemenata stavke liste ``li``.

Razumevanje ovih odnosa je esencijalno za manipulisanje i navigaciju kroz DOM stabljiku korišćenjem JavaScripta.
