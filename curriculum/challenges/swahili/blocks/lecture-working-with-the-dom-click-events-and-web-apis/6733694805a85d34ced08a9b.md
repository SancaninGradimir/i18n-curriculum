# --interactive--

Kada gradite svoje programe, često ćete želeti da se nešto dogodi nakon odlaganja, ili da se ponovi u redovnim intervalima. Tamo su metode `setTimeout()` i `setInterval()` korisne. Oba metoda prima dva parametra: funkciju i odlaganje.

Pogledaćemo svaki od metoda detaljnije kako biste počeli da ih koristite u svojim veb projektima.

`setTimeout()` omogućava vam da odložite akciju na određeno vreme. Evo osnovne sintakse metode `setTimeout()`:

```js
setTimeout(functionToRun, delay);
```

`functionToRun` je kod koji želite izvršiti nakon određenog kašnjenja, a `delay` je vreme (u milisekundama) čekanja pre pokretanja funkcije. Evo primera:

:::interactive_editor

```js
setTimeout(function () {
 console.log("This runs after 3 seconds");
}, 3000);
```

:::

Takođe možete koristiti streličnu funkciju za povratni poziv, kao što je ovo:

:::interactive_editor

```js
setTimeout(() => {
 console.log("This runs after 3 seconds"); 
}, 3000);
```

:::

`setInterval()` ponavlja funkciju u zadatom intervalu. To je kao imati alarm koji zvoni svakih nekoliko sekundi.<0xC2><0xA0>`setInterval()` je savršeno kada želite da se nešto dogodi ponovo i ponovo, poput osvežavanja podataka ili kreiranja animacije koja se kontinuirano kreće.

Kao što je `setTimeout()`, uzima funkciju i odlaganje:

```js
setInterval(functionToRun, delay);
```

`functionToRun` je kod koji želite ponavljano pokrenuti, dok je `delay` vreme u milisekundama između svakog izvršavanja te funkcije. Evo primera:

:::interactive_editor

```js
setInterval(() => {
 console.log("This runs every 2 seconds");
}, 2000);

// Result:
// This runs every 2 seconds
// This runs every 2 seconds
// This runs every 2 seconds
// ...
```

:::

Pošto `setInterval()` nastavlja da izvršava navršenu funkciju u zadatom intervalu, možda želite da ga zaustavite. Da biste to uradili, morate koristiti metodu `clearInterval()`.

Da bi se obavio posao, `clearInterval()` uzima ID za `setInterval()` koji želite da zaustavite. Ovo može biti varijabla kojoj dodeljujete interval. Jedan način za zaustavljanje intervala je unutar `setTimeout()`, jer će to zaustaviti interval nakon određenog vremena:

:::interactive_editor

```js
const intervalID = setInterval(() => {
 console.log("This will stop after 5 seconds");
}, 1000);

setTimeout(() => {
 clearInterval(intervalID);
}, 5000);
```

:::

Takođe možete zaustaviti timeout pomoću metode `clearTimeout()`:

:::interactive_editor

```js
let timeoutID = setTimeout(() => {
 console.log("This will not run");
}, 5000);

clearTimeout(timeoutID);
```

:::

Sa ovim kodom, `setTimeout()` neće uopšte raditi. Dakle, elegantniji način da se to uradi je preko DOM-a, dodavanjem slušaoca događaja za klik na dugme kako bi se zaustavio tajmer:

:::interactive_editor

```html
<h1>Cancel Timeout Example</h1>
<button id="cancelButton">Cancel Timeout</button>
<script src="index.js"></script>
```

```js
let timeoutID = setTimeout(() => {
  console.log("This will run if not canceled");
}, 5000);

document
.querySelector("#cancelButton")
.addEventListener("click", () => {
  clearTimeout(timeoutID);
  console.log("Timeout canceled!");
});
```
