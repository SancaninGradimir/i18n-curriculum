# --interactive--

API `Web Animations` (WAAPI) omogućava vam da kreirate i kontrolišete animacije direktno unutar JavaScript-a. Sa WAAPI-jem, možete raditi sa animacijama dinamičnije, što olakšava njihovu manipulaciju.

Pogledajmo WAAPI u većoj detaljnosti, tako da možete početi da radite sa animacijama direktno unutar vašeg JavaScript koda. U srži WAAPI-ja je konstruktor `Animation`, koji pruža nekoliko instancijalnih metoda i svojstava koja vam omogućavaju da dinamički animirate elemente.

Značajna metoda u konstruktoru `Animation` jeste `animate()`. Omogućava vam kreiranje animacije specifikovanjem ključnih kadrova i opcija kao što su trajanje, smer, olakšavanje (easing) i iteracije.

Evo osnovne sintakse metode `animate()`:

```js
element.animate(keyframes, options);
```

Pogledajmo primer.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  align-items: center;
  justify-content: center;
  height: 100vh;
}

.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
}
```

```js
const square = document.querySelector("#square");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(100px)" }],
  {
    duration: 2000, // makes animation lasts 2 seconds
    iterations: Infinity, // loops indefinitely
    direction: "alternate", // moves back and forth
    easing: "ease-in-out" // smooth easing
  }
);
```

:::

Rezultat u pretraživaču će biti plavi kvadrat koji se kreće napred i nazad horizontalno.

Instancijske metode konstruktora `Animation` uključuju:

```js
play()
pause()
reverse()
finish()
cancel()
```

I svojstva instanci uključuju:

```md

effect
timeline
playState

```

Let's modify the example to use the `play()`, `pause()` methods and the `onfinish` property. Here is new version:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>

<button id="playBtn">Play</button>
<button id="pauseBtn">Pause</button>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  height: 100vh;
}
.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
  margin-bottom: 20px;
}
button {
  margin: 10px;
  padding: 10px 20px;
  font-size: 16px;
}
```

```js
const square = document.querySelector("#square");
const playBtn = document.querySelector("#playBtn");
const pauseBtn = document.querySelector("#pauseBtn");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(200px)" }],
  {
    duration: 5000, // Animation lasts 5 seconds
    // iterations: Infinity, // Loops indefinitely
    direction: "alternate", // Moves back and forth
    easing: "ease-in-out" // Smooth easing function
  }
);

// Set the onfinish property to log a message when the animation ends
animation.onfinish = () => {
  console.log("Animation finished!");
};

// Play the animation when the "Play" button is clicked
playBtn.addEventListener("click", () => {
  animation.play();
  console.log("You start the animation");
});

// Pause the animation when the "Pause" button is clicked
pauseBtn.addEventListener("click", () => {
  animation.pause();
  console.log("You pause the animation");
});
```

:::

Rezultat u pretraživaču će prikazati plavu kutiju koja se kreće iz leve u desno kada se klikne na dugme `play`.

WAAPI proširuje mogućnosti CSS animacija pružanjem dinamičnije kontrole nad animacijama direktno unutar JavaScript-a.

Sa CSS animacijama, definišete animacije deklarativno koristeći svojstva kao što su `animation-name`, `animation-duration`, i `animation-timing-function`. Takođe imate priliku da uradite isto koristeći metod `animate()` u WAAPI-ju.

Razlika je u tome što možete direktnije i dinamičnije kontrolisati animacije koje kreirate sa metodom ``animate()``, ali sa CSS animacijama, morate učiniti mnogo više definišući prilagođene stilove i pokrećući ih unutar vašeg JavaScript fajla.

CSS animacija je idealna za jednostavne i deklarativne animacije koje se automatski pokreću. One uključuju efekte prelaska mišem (hover), tranzicije ili animacije koje ne zahtevaju mnogo interakcije nakon aktiviranja. Ako vaša animacija mora da reaguje na interakcije korisnika kao što su klikovi i skrolovanje, ili želite da korisnik može dinamički pauzirati, obrnuti ili promeniti brzinu, WAAPI je bolji izbor.

Možete razmotriti kombinovanje i WAAPI i CSS animacija kada želite jednostavnost CSS-a za osnovno podešavanje, ali vam je potrebno da kontrolišete animacije u vreme rada.
