# --interactive--

Kada korisnik klikne na dugme ili dođe do promene u formularu, to se naziva događaj. U vašim programima, biće vam potreban način da slušate ove događaje i reagujete na njih.

Metoda ``addEventListener()`` se koristi za slušanje događaja. Prihvata dva argumenta: događaj na koji želite da slušate i funkciju koja će biti pozvana kada dođe do događaja.

Evo osnovne sintakse:

```js
element.addEventListener("event", listener);
```

Element je HTML element za praćenje događaja, a `event` specificira tip događaja na koji treba slušati, kao što je ``"click"``.

Listener je objekat koji će primiti obaveštenje kada se dogodi događaj. Najčešće će ovo biti funkcija koju definišete da biste obradili događaj.

Evo primera:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

Takođe možete izabrati da prosledite referencu funkcije na ovaj način:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

Ponekad želite da kreirate zasebnu funkciju za rukovanje događajem. To može učiniti vaš kod lakšim za čitanje i održavanje.

Argument slušaoca takođe može biti `null`, ili može biti objekat koji koristi interfejs `EventListener`. Interfejs `EventListener` definiše jednu metodu nazvanu `handleEvent()`. Ova metoda se automatski poziva svaki put kada dođe događaj za koji slušate. Korišćenjem ovog interfejsa omogućeno je istom objektu da rukuje sa više događaja, ako je potrebno.

U većini slučajeva, nećete morati da koristite ovaj interfejs, i umesto toga možete proslediti funkciju koja će biti pozvana kada se dogodi događaj.

Sada, hajde da pogledamo primer kako ovo funkcioniše:

U ovom primeru, imamo `button` element sa ID-om `btn`:

```html
<button id="btn">Show alert</button>
```

Želimo da slušamo događaj ``"click"`` na ovom dugmetu i prikažemo upozorenje kada se dugme klikne.

Prvo moramo pristupiti tom `button` elementu u našem JavaScript kodu. Zatim moramo dodati slušalicu događaja da sluša klik događaj i prikaže upozorenje kada se dugme klikne:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

Sada svaki put kada se klikne na dugme, korisnik će videti poruku upozorenja prikazanu na ekranu ovako:

```md
You clicked the button
```

Drugi tip događaja na koji možete slušati je događaj `"input"`. Ovaj događaj se aktivira kada vrednost elementa za unos promeni.

Evo primera:

```html
<input type="text" id="input" placeholder="Type something" />
```

Želimo da slušamo događaj unosa na ovom elementu za unos i logujemo vrednost unosa u konzolu svaki put kada korisnik nešto ukuca.

Prvo moramo pristupiti tom elementu ``input`` u našem JavaScript kodu. Zatim moramo dodati slušalicu događaja (event listener) da prati događaj unosa (input event) i loguje vrednost unosa u konzolu svaki put kada korisnik nešto ukuca:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

Postoji mnogo više događaja na koje možete slušati koristeći metod ``addEventListener()``. Neki uobičajeni događaji uključuju ``mouseover``, ``mouseout``, ``keydown``, ``keyup`` i ``submit``.

U budućim časovima, pokrićemo više događaja i kako koristiti metod ``addEventListener()`` za slušanje njih.
