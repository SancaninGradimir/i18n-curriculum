# --description--

Naučićmo se o metodi `querySelectorAll()`. Možete koristiti ovu metodu da dobijete listu svih DOM elemenata koji odgovaraju određenom CSS selektoru.

Ovim putem možete pozvati metodu `querySelectorAll()`:

```js
document.querySelectorAll(selectors);
```

Pozovite je na objektu `document` i prosledite string sa CSS selektorima kao argument. Argument mora biti validan string CSS selektora. U suprotnom, će biti izbačena izuzetak (exception) tipa `SyntaxError`.

`querySelectorAll()` vraća objekat `NodeList`, kolekciju čvorova koji odgovaraju navedenom CSS selektoru.

Lista će sadržati jedan objekat `Element` za svaki element koji odgovara CSS selektorima. Ako se ne pronađu podudarnosti, lista će biti prazna.

Elementi će biti u redu kojim se pojavljuju u HTML dokumentu. Na primer, možete pronaći sve elemente određenog tipa:

```js
document.querySelectorAll("div");
```

Svi elementi sa specifičnom klasom:

```js
document.querySelectorAll(".rounded");
```

Svi elementi sa specifičnim ID-em:

```js
document.querySelectorAll("#logo");
```

Ili sve elemente sa specifičnim atributom, kao što su sve veze (linkovi) koje korisnike vode na određenu URL adresu:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

Takođe možete koristiti složenije CSS selektore, kao ovaj, koji pronalazi sve stavke liste unutar elemenata neuređene liste sa klasom `ingredients`:

```js
document.querySelectorAll("ul.ingredients li");
```

Pošto će metoda vratiti kolekciju čvorova, možete dodeliti ovu kolekciju promenljivoj i koristiti je u svojim programima za rad sa pojedinačnim elementima, kao što vidite u ovom primeru sa opštim sintaksom:

```js
const matches = document.querySelectorAll(selectors); 
```

Na primer, ako imate ovu neuređenu listu sa tri stavke liste:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

Možete pozvati metodu `querySelectorAll()` da pronađete sve stavke liste unutar neuređene liste sa klasom `ingredients` i dodeliti vraćenu vrednost konstanti `matches`:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

Ovo će vratiti kolekciju od tri čvora. Svaki čvor predstavlja element stavke liste:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

Možete raditi sa ovom kolekcijom tačno kao što biste radili sa bilo kojim drugim JavaScript nizom (array).

Na primer, možete ga prikazati u konzoli pomoću `console.log()`:

```js
console.log(matches);
```

Takođe možete proveriti dužinu kolekcije korišćenjem svojstva `length`:

```js
console.log(matches.length);
```

Izlaz će biti ceo broj koji predstavlja broj čvorova u kolekciji:

```md
3
```

Takođe možete pristupiti pojedinačnim elementima kolekcije njihovim odgovarajućim indeksima. Prvi element će biti na indeksu `0`:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

Ovo je izlaz: 

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

Elementi se štampaju jedan po jedan u konzoli.

Takođe možete iterirati kroz elemente pomoću petlje `for`:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

Izlaz će biti potpuno isti i u ovom slučaju. Ali ovo može biti veoma moćno za rad sa svim elementima kolekcije.

Metoda `querySelectorAll()` u JavaScriptu je moćan alat za selektovanje više elemenata na osnovu njihovih CSS selektora. Ovom metodom, možete izabrati specifične elemente unutar svojih web stranica i manipulisati njima kako vam je potrebno.
