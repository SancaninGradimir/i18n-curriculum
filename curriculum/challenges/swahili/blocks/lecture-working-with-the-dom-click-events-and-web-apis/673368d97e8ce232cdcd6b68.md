# --interactive--

Biće vremena kada ćete morati dodavati ili uklanjati čvorove iz DOM-a i postoji nekoliko Web API-ja koje možete koristiti.

U ovoj lekciji ćemo obraditi metode `appendChild()` i `removeChild()`.

Metoda `appendChild()` se koristi za dodavanje čvora na kraj liste dece navedenog roditeljskog čvora.

Evo osnovne sintakse za metod `appendChild()`:

```js
parentNode.appendChild(newNode);
```

Pogledajmo primer kako bismo bolje razumeli korišćenje metode `appendChild()`.

U ovom primeru, imamo neuređenu listu elemenata sa ID-om `desserts` unutar HTML-a. Ova lista sadrži dva stavke liste od `Cake` i `Pie`:

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

Da bismo pristupili toj listi unutar JavaScript fajla, možemo koristiti metodu `getElementById()`:

```js
const dessertsList = document.getElementById("desserts");
```

Imamo varijablu nazvanu `dessertsList` koja čuva referencu na element `ul` sa ID-om `desserts`.

Zatim moramo kreirati novi element stavke liste koristeći metod `createElement()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

Sledeći kod će kreirati novi element stavke liste i sačuvati ga u varijablu nazvanu `listItem`.

Da bismo dodali taj čvor na kraj liste dece elementa `ul`, možemo koristiti metodu `appendChild()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

Ako pokrenete ovaj kod, videćete da je novi element stavke liste dodat na kraj liste dece elementa ``ul``.

```html

  <li>Cake</li>
  <li>Pie</li>
</ul>

```

```js

```

:::

Problem je što novi element liste stavke je prazan. Da biste dodali tekstualni sadržaj novom elementu liste stavke, možete koristiti svojstvo `textContent`:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

Sada lista će prikazati `Cake`, `Pie`, i `Cookies`.

Da biste uklonili čvor iz DOM-a, možete koristiti metodu ``removeChild()``.

Evo osnovne sintakse za metod `removeChild()`:

```js
parentNode.removeChild(childNode);
```

Pogledajmo primer gde želimo ukloniti taj poslednji paragrafski element iz ovog elementa `section`:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

Možemo pristupiti elementu ``section`` unutar JavaScript fajla koristeći metodu ``getElementById()``:

```js
const sectionEl = document.getElementById("example-section");
```

Kada imamo referencu na `section` element, možemo zatim pristupiti tom poslednjem pasusu koristeći `querySelector()` metod:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

Koristimo pseudo-klasu ``:last-of-type`` za selekciju poslednjeg paragraf elementa unutar elementa ``section``.

Sada kada imamo roditeljske i detilske čvorove, možemo ukloniti poslednji paragraf element iz elementa `section` koristeći metodu `removeChild()`:

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::
