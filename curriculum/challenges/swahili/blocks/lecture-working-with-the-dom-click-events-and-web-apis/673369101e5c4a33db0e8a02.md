# --interactive--

U prethodnoj lekciji ste naučili kako da radite sa `"click"` i `"input"` događajima koristeći `addEventListener()` metodu. U ovoj lekciji, naučićete kako da uklonite slušače događaja koristeći `removeEventListener()` metodu.

Metoda `removeEventListener()` se koristi za uklanjanje slušaoca događaja koji je prethodno dodan na element korišćenjem metode `addEventListener()`. Ovo je korisno kada želite da prestanete sa slušanjem određenog događaja na nekom elementu.

Evo osnovne sintakse za metod `removeEventListener()`:

```js
element.removeEventListener("event", listener);
```

Kao i kod metode `addEventListener()`, metoda `removeEventListener()` prima dva argumenta: događaj koji želite ukloniti i slušalicu funkciju koja je prethodno dodata.

Takođe postoji dodatni opcioni treći argument koji se može proslediti metodi `removeEventListener()`. Ovaj argument može biti ili `options` ili `useCapture`.

Argument `options` je objekat koji određuje opcije za slušaoca događaja (event listener), kao što je da li bi slušalac događaja trebalo da bude pasivan ili samo jednom.

Argument ``useCapture`` je booleanska vrednost koja određuje da li događaj treba da bude uhvaćen tokom faze propagacije događaja.

U većini slučajeva, samo ćete morati da prosledite argumente događaja i slušaoca metodi `removeEventListener()`.

Ako želimo da prebacujemo boju pozadine između sive i plave, onda možemo koristiti slušalicu događaja za to.



Prvo moramo pristupiti elementu `button` zajedno sa elementom `body`:

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

Zatim moramo kreirati funkciju koja je odgovorna za prebacivanje između sive i plave boje:

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

Koristimo booleanski promenljiva `isBgColorGrey` da pratimo trenutnu pozadinsku boju. Ako je pozadinska boja siva, onda je menjamo u plavu, i obrnuto.

Zatim moramo dodati slušalicu događaja na element `button` kako bismo pozvali funkciju `toggleBgColor` kada se dugme klikne:

```js
btn.addEventListener("click", toggleBgColor);
```

Svakog puta kada se klikne na dugme, pozadinska boja tela će se menjati između sive i plave.

Možemo ažurirati HTML da uključimo paragraf element koji će se koristiti za uklanjanje slušaoca događaja kada se pređe mišem preko njega:

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

Da bismo uklonili slušalicu događaja kada se pređe mišem preko paragrafa, možemo dodati slušalicu događaja na element paragrafa. Koristimo `mouseover` događaj i prosleđujemo funkciju koja uklanja slušalicu događaja sa `button` elementa:

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

Kada se pređe mišem preko paragrafa, slušalica događaja za klik na dugme je uklonjena, i pozadina više neće menjati boju kada se klikne na dugme.
