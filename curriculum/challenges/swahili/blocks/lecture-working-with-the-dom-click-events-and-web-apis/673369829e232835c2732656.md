# --interactive--

Dijalozi vam omogućavaju prikaz važnih informacija ili akcija korisnicima. Sa ugrađenim elementom `dialog` u HTML-u, možete lako kreirati ove dijaloge (i modalne i ne-modalne dijaloge) u svojim veb aplikacijama.

Modal dijalog je vrsta dijaloga koja prisiljava korisnika da sa njim interaguje pre nego što pristupi ostatku aplikacije ili web stranice. On efektivno blokira interakciju sa drugim sadržajem sve dok korisnik ne izvrši akciju, kao što je zatvaranje dijaloga ili slanje obrasca.

Za razliku od toga, ne-modalni dijalog korisniku omogućava da nastavi sa interakcijom sa drugim delovima stranice ili aplikacije čak i kada je dijalog otvoren. Ne sprečava pristup ostatku sadržaja.

U ovoj lekciji, naučićete kako da otvorite i zatvorite modale koristeći metode `showModal()` i `close()`.

Kada želite da osigurate da se korisnik fokusira na određenu akciju ili poruku modala, možete otvoriti dijalog modala koristeći metodu ``showModal()``. Ovo će dodati pozadinu ostalim elementima na stranici i onemogućiti ih. Ovo je idealno za modale koji prikazuju forme, potvrde i kritične informacije koje zahtevaju akciju korisnika.

Evo HTML-a za modalni dijalog:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

Za sada, ne možete ništa videti na stranici jer je modal zatvoren pri inicijalnom renderovanju. Možete automatski otvoriti modal koristeći metod `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

Rezultat u pretraživaču će prikazati modal sa tekstom `This is a modal dialog.`

Najbolje je dati kontrolu korisniku. Da biste to postigli, možete dodati slušalicu za događaj klik na dugme i koristiti metodu ``showModal()``:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

Ako je potrebno prikazati dijalog a istovremeno omogućiti interakciju sa sadržajem van dijaloga, možete koristiti metod `show()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

Da biste zatvorili modal, možete dodati dugme u modal unutar elementa `dialog` i koristiti metodu `close()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::
