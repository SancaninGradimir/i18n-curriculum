# --description--

U prethodnoj lekciji kreirali ste projektnu fasciklu i instalirali Express:

``````bash
mkdir my-app
cd my-app
npm init -y
npm install express
```
```

Kao podsetnik, ove komande kreiraju novu projektnu fasciklu, ulaze u tu projektnu fasciklu (`cd`), inicijalizuju projekat i kreiraju osnovnu datoteku `package.json`, te instaliraju Express.

Zatim otvorite editor koda, kreirajte fajl pod nazivom `index.js` i napišite svoj prvi Express server:

``````javascript
const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```
```

Vaš kod možete pokrenuti sledećom komandom u terminalu:

``````bash
node index.js
```
```

Otvorite vaš pregledač na [http://localhost:3000](http://localhost:3000) i videćete poruku **"Hello World!"**

Tako brzo je početi sa Express-om! Sada prođemo kod korak po korak.

U prvoj liniji koda, učitavate biblioteku Express i dodeljujete je promenljivoj nazvanoj `express`:

``````javascript
const express = require("express");
```
```

Promenljiva `express` je fabrička funkcija. Kada je pozovete, kreira i vraća novu instancu Express aplikacije:

``````javascript
const app = express();
```
```

Sada promenljiva `app` ima pristup svim metodama koje pruža Express, kao što su `.get()`, `.post()`, `.listen()` i slično. Ovo je vaš glavni objekat servera.

Sledeća linija definiše promenljivu `port`. Port je mrežna tačka na kojoj vaša aplikacija sluša (slušuje), a `3000` je često korišćen port za lokalni razvoj:

``````javascript
const port = 3000;
```
```

Sada, pogledajmo sledeći deo ovde:

``````javascript
app.get("/", (req, res) => {
  res.send("Hello World!");
});
```
```

Ovaj blok koda definiše rutu i govori serveru kako da odgovori na zahteve. `app.get()` registruje rukovatelja (handler) za HTTP GET zahteve. GET zahtev je često korišćena HTTP metoda za preuzimanje podataka.

Argument `"/"` unutar `app.get()` predstavlja putanju rute. U ovom slučaju, ukazuje na koreni URL aplikacije.

Deo `(req, res) => { ... }` je povratna funkcija (callback function) koja se izvršava kada zahtev odgovara ovoj ruti. Objekat `req` sadrži informacije o dolaznom zahtevu, a objekat `res` se koristi za slanje odgovora nazad klijentu.

Na kraju, `res.send("Hello World!")` šalje odgovor klijentu. U ovom primeru, odgovor je običan tekst `"Hello World!"`.

Sada, pogledajmo poslednji deo koda ovde:

``````javascript
app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```
```

Metoda `app.listen()` pokreće Express aplikaciju i govori Node-u da sluša dolazne HTTP zahteve na navedenom portu, u ovom slučaju, `3000`.

Nakon što se server pokrene, povratna funkcija se izvršava i poruka `Example app listening on port 3000` je logovana u konzolu, potvrđujući da je Express aplikacija uspešno počela i spremna za prijem zahteva.
