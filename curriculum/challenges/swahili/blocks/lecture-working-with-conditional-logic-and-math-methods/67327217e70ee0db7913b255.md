# --interactive--

Kada uđete u JavaScript, brzo ćete otkriti da je obavljanje matematičkih operacija uobičaj zadatak. Iako osnovni aritmetički operateri mogu da obrade jednostavne proračune, JavaScript nudi ugrađeni `Math` objekat za rešavanje složenijih matematičkih izazova.

Ovaj praktičan alat pruža raznovrsne metode koje olakšavaju obavljanje naprednih kalkulacija i manipulisanje brojevima. Ispitajmo ove metode i vidimo kako mogu da pojednostave vaše iskustvo kodiranja.

Metoda `Math.random()` generiše nasumičan broj sa plutajućom zarezom između `0` (uključivo) i `1` (isključivo). To znači da mogući izlaz može biti `0`, ali nikada zaista neće dostići `1`. Evo primera koji radi sa metodom `Math.random()`:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` i `Math.max()` oba uzimaju skup brojeva i vraćaju minimalnu i maksimalnu vrednost, respektivno. Evo primera kako da se koriste oba ta metoda:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

Prvi `console.log()` će logovati broj `1`, jer je `1` najmanji u toj listi brojeva. A drugi `console.log()` će logovati broj `9`, jer je `9` najveći broj u toj listi.

Ako želite zaokružiti brojeve naviše ili nadole na najbliži ceo broj, možete koristiti metode `Math.ceil()` i `Math.floor()`. Evo primera rada sa `Math.ceil()`:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` će zaokružiti `4.3` navam pa do najbližeg celog broja, što je u ovom slučaju `5`. Sada, hajde da pogledamo zaokruživanje broja nadole:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` će zaokružiti `4.7` nadole na najbliži ceo broj, što je u ovom slučaju `4`. `Math.round()` je hibrid od `Math.ceil()` i `Math.floor()`. Zaokružuje broj na najbliži ceo broj, uzimajući u obzir decimalnu tačku:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

Dakle, ako je decimalna tačka manja od ``5``, broj se zaokružuje nadole. A ako je decimalna tačka ``5`` ili veća, broj se zaokružuje naviše. Praktična primena ``Math.floor()`` i ``Math.random()`` je generisanje nasumičnog broja između dva cela broja. Evo sintakse za to:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

Generisanje nasumičnog broja između `20` i `1` bi izgledalo ovako:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

Još jedna korisna `Math` metoda bi bila `Math.trunc()` metoda. `Math.trunc()` uklanja decimalni deo broja, vraćajući samo ceo deo, bez zaokruživanja:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

Ako vam je potrebno da dobijete kvadratni ili kubni koren broja, možete koristiti metode `Math.sqrt()` i `Math.cbrt()`, respektivno:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

Prvi izjava za logovanje će logovati ``9`` jer je kvadratni koren od ``81`` jednak ``9``, dok će druga izjava za logovanje logovati ``3`` jer je kubni koren od ``27`` jednak ``3``. Ako trebate dobiti apsolutnu vrednost broja, možete koristiti metod ``Math.abs()``:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` vraća apsolutnu vrednost broja, pretvarajući negativne u pozitivne. Posledna metoda koju ćemo pregledati je `Math.pow()` metoda:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` uzima dva broja i podiže prvi na stepen drugog. Postoji mnogo više metoda koje pripadaju objektu `Math`, koje možete istražiti sami. Međutim, ovo je samo mali deo onih koji se najčešće koriste u JavaScript bazama koda.
