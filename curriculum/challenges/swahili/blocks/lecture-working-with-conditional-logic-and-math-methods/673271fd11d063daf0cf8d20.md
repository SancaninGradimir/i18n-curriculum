# --interactive--

Uslovne izjave vam omogućavaju da donosite odluke u vašem JavaScript kodu. One dozvoljavaju vašem programu da teče na određeni način na osnovu određenih uslova. Pogledajmo kako `if`, `else if`, `else` i ternarni operator rade da vam omoguće kontrolu protoka vašeg koda.

An `if` statement takes a condition and runs a block of code if that condition is truthy. Truthy values are any values that result in `true` when evaluated in a Boolean context like an `if` statement. Here are examples of truthy values:

- ne-prazni stringovi, na primer, `hello`

- bilo koji broj osim `0` i `-0`, na primer, `4`, `-5`, i drugi

- nizovi

- objekti

- booleanski `true`

S druge strane, falsy vrednosti su vrednosti koje se procenjuju na `false` u boolovskom kontekstu. JavaScript ima malo falsy vrednosti, što ih čini lako pamtljivima. Evo nekoliko falsy vrednosti:

- booleanski `false`

- `0` (nula)

- `""` (prazan niz znakova)

- `null`

- `undefined`

- `NaN` (Nije broj)

Sada, pošto imamo osnovno razumevanje istinitih i lažnih vrednosti, hajde da vidimo kako to radi sa ``if`` izjava. U ovom prvom primeru, koristimo nekoliko ``if`` izjava za proveru istinitih i lažnih vrednosti:

:::interactive_editor

```js
if (null) {
  console.log("This will not run.");
}

if ("freeCodeCamp") {
  console.log("This will run.");
}
```

:::

Pošto je `null` lažno vrednost, poruka unutar bloka nikada neće biti logovana u konzolu. Ali za drugu izjavu `if`, niz `freeCodeCamp` je istinit vrednost i biće smatran `true` u ovom boolovskom kontekstu izjave `if`. Kao rezultat, poruka `This will run.` će biti logovana u konzolu.

Pogledajmo još nekoliko primera kako `if` naredbe rade sa različitim operatorima poređenja. Evo primera korišćenja `if` naredbe za proveru da li je korisnik u mogućnosti da glasa:

:::interactive_editor

```js
const age = 22;

if (age >= 18) {
 console.log("You're eligible to vote"); // You're eligible to vote
}
```

:::

U ovom primeru, pošto ``age`` je trenutno ``22``, ovo znači da će uslov biti evaluiran na ``true`` jer ``22`` je veće ili jednako ``18``. Dakle, poruka ``You're eligible to vote`` će biti logovana u konzolu. Ako promenimo primer tako da ``age`` sada bude ``15``, onda će uslov biti evaluiran na ``false`` i poruka neće biti logovana u konzolu:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote"); // Code not running because age is less than 18
}
```

:::

Kada je uslov `false`, onda možete koristiti klauzulu `else`:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote");
} else {
 console.log("You're not eligible to vote"); // You're not eligible to vote
}
```

:::

U ovom primeru, `15` nije veće ili jednako `18`, pa bi uslov bio `false`. Kod unutar `else` bloka će se izvršiti u ovom slučaju.

Ako želite da proverite više uslova, možete koristiti blok `else if`. Ovo omogućava vašem programu da izabere između više od dve putanje.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

Pošto je ``score`` trenutno ``87``, onda bi poruka ``You got a B`` bila logovana u konzolu.

Ternarni operator je kompaktan način za pisanje jednostavnih `if/else` izjava. Ima tri dela: uslov, rezultat ako je uslov tačan i rezultat ako nije tačan. Evo osnovne sintakse:

```js
condition ? expressionIfTrue : expressionIfFalse;
```

TEKST:

Evo primera koji se bavi vremenskim temperaturama u Celzijusima:

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`);
```

:::

Ako je `temperature` veće od `25`, kod iznad loguje `It's a sunny day!`. Ako je `temperature` ikada manje ili jednako `25`, loguje `It's a cool day!`.

Pa, šta biste trebali koristiti između `if` izjave i ternarnog operatora? Koristite ternarni operator kada radite sa jednim uslovom ili pojedinačnim izrazima, ili kada želite kompaktnu sintaksu za jednostavnu logiku. Koristite ``if/else`` izjave kada radite sa kompleksnim uslovima i više izjava, jer stvari postaju nečitljive ako ugnježđujete ternarne operatore.
