# --interactive--

Prezentacioni HTML se fokusira na izgled i stil sadržaja. U ranim danima HTML-a, programeri su koristili elemente poput elemenata `center`, `big` i `font`. Ali u modernom web razvoju ne bi trebalo da koristite ovakve tipove elemenata, zbog njihovih ograničenja i negativnog uticaja na pristupačnost i održivost.

Mnogi prezentacioni HTML elementi su zastareli/deprecirani, što znači da su zaostali i više se ne preporučuju. Postoje bolji načini postizanja istih rezultata. Međutim, korisno je znati da postoje, pa ćemo pogledati neke primere.

Element ``font`` je zastareli element koji se koristi za postavljanje veličine i boje fonta teksta. Evo primera ``font`` elementa.

Omogući interaktivni urednik i promeni veličinu sa 5 na 7 da bi video kako se povećava font.

:::interactive_editor

```html
<font size="5" color="blue">This text is blue and large.</font>
```

:::

Ovaj primer postavlja boju teksta na `blue` i veličinu na vrednost `5`. Opseg za atribut `size` je od `1` do `7`, pri čemu `1` je najmanji, a `7` je najveći. Podrazumevana vrednost je `3`, ako ne postavite vrednost eksplicitno.

Iako ovaj element još uvek radi, ne biste ga trebali koristiti jer veličina fonta i boja bi trebalo uvek da se postavljaju u CSS-u, a ne u HTML-u.

Element ``center`` je još jedan zastareli element koji se koristi za horizontalno centriranje sadržaja unutar svog kontejnera. Evo primera elementa ``center`` koji sadrži tekst i element paragrafa.

Omogući interaktivni uredu i dodaj ``center`` oznake oko ``<p>Another example text.</p>`` da biste ga videli centriranog na stranici.

:::interactive_editor

```html
<center>
  This text is centered.
  <p>HTML is awesome.</p>
</center>

<p>Another example text.</p>
```

:::

I sledeći, imamo element `big`. Ovo je još jedan zastareli prezentacioni HTML element koji čini zatvoreni tekst za jedan nivo veći od okolnog teksta. Ovde imamo primer koji definiše pasus sa dve dele.

Omogući interaktivni uresnik i dodaj ``big`` oznake oko ``Some other text`` i vidi promene u prozoru za pregled.

:::interactive_editor

```html
<p>
  This text has a normal font size.
  <big>This text is larger.</big>
  Some other text.
</p>
```

:::

Zapamtite da se veličina fonta uvek treba postavljati pomoću CSS-a, pa ne biste trebali koristiti ovaj element u modernom HTML-u.

Ovo su bili primeri elementa za prezentaciju u HTML-u. Ali svi oni su zastareli i više se ne preporučuju. Pa šta biste trebali da koristite umesto toga? Hajde da vidimo.

Semantički HTML je sada preporučena praksa. On opisuje sadržaj elemenata, pa je mnogo lakše za čitanje, razumevanje i održavanje.

Pretraživači mogu lako razumeti vaš sajt kada koristite semantički HTML. Takođe je korisno za svrhe pristupačnosti, jer čitači ekrana zahtevaju semantičke informacije da bi opisali šta se nalazi na veb stranici.

Primeri semantičkih HTML elemenata uključuju:

- Element `header` za definisanje zaglavlja dokumenta ili sekcije.
- Element navigacione sekcije, `nav`, za sekcije sa navigacionim linkovima.
- Element `section` za grupisovanje povezanih informacija.
- Element `figure` za ilustracije i dijagrame.

Ovo je primer `header` elementa koji sadrži element sekcije navigacije.

Omogući interaktivni uresnik i dodaj ``<a href="#">Products</a>`` unutar ``nav`` i vidi promene u prozoru za pregled.

:::interactive_editor

```html
<header>
  <nav>
    <a href="#">Home</a>
    <a href="#">About</a>
    <!--Add the products link here-->

    <a href="#">Contact</a>
  </nav>
</header>
```

:::

Semantički elementi jasno pokazuju svoju svrhu unutar HTML strukture. Postoji mnogo različitih semantičkih HTML elemenata. Sigurno ćete pronaći jedan koji odgovara potrebama vašeg projekta.

Odličan posao. Sada znate razliku između prezentacionog i semantičkog HTML-a: semantički HTML opisuje sadržaj, dok se prezentacioni HTML fokusira na izgled.
