## Python Klase i Objekti

- **Definicija klase**: Klasa je šema za kreiranje objekata. Ona definiše ponašanje koje će objekat imati kroz svoje atribute i metode. Evo osnovnog primera definicije klase u Pythonu:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **Kreiranje objekata**: Objekti su instance klase. Kreiraju se pozivanjem klase sa neophodnim argumentima.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **Pozivanje Metoda na Objektima**: Možete pozivati metode na objektima za izvršavanje akcija ili dohvaćanje informacija.

```python
object_name1.method_name()
object_name2.method_name()
```

- **Razlika između klase i objekta**: Klasa je ponovno upotrebljiv šablon, dok je objekat specifična instanca te klase sa stvarnim podacima.

## Atributi

- **Atributi instance**: Definisan u ``__init__()`` korišćenjem ``self``, i jedinstven za svaki objekat.
- **Atributi klase**: Definisan direktno unutar klase i zajednički svim instancama.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## Metode

- **Metode**: Funkcije definisane unutar klase koje operišu na atributima objekta.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **Pristup metodama**: Pozovite metode na objektima koristeći tačku notaciju (dot notation). Evo primera pozivanja metode `describe` na dva različita objekta automobila:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Dunder (Čarobne) Metode

- **Definicija**: Specijalne metode koje počinju i završavaju se dvostrukim donjom crtom (npr., `__init__`, `__len__`, `__str__`, `__eq__`). Python ih koristi interno za ugrađene operacije.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Indirektno pozivanje dunder metoda**: Ne morate direktno pozivati dunder metode. Umesto toga, Python ih automatski poziva kada se izvrše određene operacije. Ove operacije uključuju:

  - **aritmetičke operacije kao što su sabiranje, oduzimanje, množenje, deljenje i druge**. Pored toga, poziva se `__add__()` za sabiranje, `__sub__()` za oduzimanje, `__mul__()` za množenje i `__truediv__()` za deljenje.
        
  - **operacije sa stringovima kao što su konkatenacija, ponavljanje, formatiranje i konverzija u tekst**. Poziva se `__add__()` za konkatenaciju, `__mul__()` za ponavljanje, `__format__()` za formatiranje, `__str__()` i `__repr__()` za konverziju teksta, i tako dalje.
        
  - **operacije poređenja kao što su jednakost, manje od, veće od i druge**. Poziva se `__eq__()` za provere jednakosti, `__lt__()` za manje od, `__gt__()` za veće od, i tako dalje.
        
  - **operacije iteracije kao što je činjenje objekta iterabilnim i napredovanje kroz stavke**. Poziva se `__iter__()` da bi se vratio iterator i<0xC2><0xA0> `__next__()` da bi se dohvatila sledeća stavka.

## Primer iz stvarnog sveta: Korpa za kupovinu

- **Klasa Korpe sa Dunder metodima**: Omogućava dodavanje, uklanjanje, iteriranje i proveru sadržaja uz ugrađeno ponašanje.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
