# --description--

Aplikacioni middlever je middlever koji je vezan za instancu Express aplikacije. Obično se koristi za rukovanje zahtevima koji zahtevaju obradu preko više ruta. Može se primeniti globalno ili na specifične rute.

Ovaj middlever se registruje korišćenjem `app.use()` ili specifičnih metoda rute kao što su `app.get()`, `app.post()`, itd. Izvršava se za svaki dolazni zahtev, osim ako nije ograničen na specifične rute.

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`: Ovo je aplikacioni middlever. Loguje svaki zahtev poslat serveru, bez obzira na rutu.
* Middlever radi za svaki dolazni zahtev, pružajući serveru detalje kao što je trenutni vremenski pečat.

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

Aplikacioni middlever je koristan za zadatke koji treba da se primene preko mnogo ili svih ruta u aplikaciji. Ovo može uključivati logovanje, autentifikaciju i rukovanje greškama.
