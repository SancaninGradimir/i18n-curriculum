# --description--

Middleware na nivou rutera sličan je middleware-u na nivou aplikacije, ali je vezan za instancu `express.Router()` umesto glavne instance aplikacije. Može se koristiti za modularizovanje vaše Express aplikacije kreiranjem izolovanih rutnih modula.

Middleware na nivou rutera radi na isti način kao middleware na nivou aplikacije, ali se primenjuje na specifične instance rutera. Ovo olakšava primenu middleware-a samo na skup ruti, umesto globalno.

```js
const express = require('express')
const app = express()
const router = express.Router()

// Router-level middleware to log requests to '/menu' routes
router.use((req, res, next) => {
  console.log('Request made to /menu route')
  next() // Pass control to the next handler
})

// Define routes
router.get('/drinks', (req, res) => {
  res.send('Welcome to the drinks menu!')
})

app.use('/menu', router) // Apply the router

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `router.use()`: Ovo je middleware na nivou rutera, koji se primenjuje samo na rute koje obrađuje router.
* `app.use('/menu', router)`: Middleware se primenjuje samo na rute koje počinju sa `/menu` (npr., `/menu/drinks`).

Middleware na nivou rutera omogućava da razbijete svoju aplikaciju na modularne komponente i selektivno primenite middleware na grupu ruti. Ovo pomaže u organizaciji i jasnoći koda.
