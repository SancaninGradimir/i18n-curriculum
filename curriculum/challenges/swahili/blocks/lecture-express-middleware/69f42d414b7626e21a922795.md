# --description--

Express pruža niz ugrađenih middleware funkcija koje pomažu pri obavljanju uobičajenih zadataka, kao što su serviranje statičkih fajlova, parsiranje JSON tela zahteva i još mnogo toga. Takođe možete koristiti middleware trećih strana za dodavanje funkcionalnosti.

**Evo nekih primera ugrađenog middlewarea**:

1. `express.json()`: Ugrađeni middleware koji parsira dolazne JSON opterećenja (payloads).

```js
app.use(express.json())
```

1. `express.static()`: Middleware za serviranje statičkih fajlova kao što su slike, CSS i JavaScript fajlovi.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: Middleware koji parsira dolazne zahteve sa URL-enkodiranim opterećenjima (payloads), formatom koji je tipično poslat od strane HTML formulara. Čini parsirane podatke dostupnim na `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**Sada, evo nekih primera middlewarea trećih strana:**

1. `cors`: Popularni middleware treće strane za omogućavanje deljenja resursa iz različitih izvora (Cross-Origin Resource Sharing).

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: Middleware za logovanje HTTP zahteva.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

Sada, pogledajmo primer korišćenja kako ugrađenog tako i middlewarea trećih strana:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* Ugrađeni middleware `express.json()` parsira dolazne JSON opterećenja.
* Middleware treće strane `cors` omogućava zahteve iz različitih izvora (cross-origin), a `morgan` loguje dolazne HTTP zahteve.

Važno je napomenuti da se funkcije middlewarea dodaju u redosledu kojim su pozvane sa `app.use()`.

Express pruža esencijalni ugrađeni middleware za obradu uobičajenih zadataka kao što su parsiranje JSON-a i serviranje statičkih fajlova. Takođe možete proširiti funkcionalnost sa middlewareom trećih strana za dodatne mogućnosti.
