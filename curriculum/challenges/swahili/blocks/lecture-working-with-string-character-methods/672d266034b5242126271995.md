# --interactive--

U programiranju, razumevanje kako su karakteri predstavljeni kao brojevi je fundamentalno. Tu dolazimo do ASCII-ja. ASCII, skraćenica za American Standard Code for Information Interchange, jeste standard za kodiranje karaktera koji se koristi u računarima za predstavljanje teksta. On dodeljuje numeričku vrednost svakom karakteru, što je univerzalno prepoznato od strane mašina.

U ovoj lekciji ćemo istražiti šta je ASCII, kako funkcioniše i kako se JavaScript metode poput `charCodeAt()` i `fromCharCode()` odnose na kodiranje karaktera. Iako JavaScript nizovi koriste Unicode (UTF-16) interno, ASCII vrednosti odgovaraju prvih 128 Unicode karaktera, zbog čega primeri zasnovani na ASCII funkcionišu u JavaScriptu.

ASCII je sistem za kodiranje znakova kao što su slova, cifre i simboli u numeričke vrednosti. Svaki karakter je mapiran na određeni broj.

Na primer, veliko slovo `A` je predstavljeno brojem `65` u ASCII-ju, dok je malo slovo `a` predstavljeno brojem `97`. Ovo kodiranje omogućava računarima da skladište i manipulišu tekstom.

ASCII standard pokriva 128 znakova, uključujući:

- Velika i mala engleska slova (A-Z, a-z).
- Brojevi (0-9).  
- Uobičajeni interpunkcijski znaci i simboli (!, @, #, itd.).
- Kontrolni znakovi (kao što su novi red i tabulator).

U JavaScriptu, možete pristupiti numeričkom kodu karaktera koristeći metod `charCodeAt()`. Ovaj metod vraća UTF-16 kodnu jedinicu karaktera na navedenom indeksu. Za prvih 128 karaktera, ova vrednost odgovara ASCII kodu.

Pogledajmo primer:

:::interactive_editor

```js
let letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

U ovom primeru, `A` je prvi karakter niza, a pozivanje `charCodeAt(0)` vraća njegov numerički kod (koji odgovara njegovoj ASCII vrednosti za osnovne latinske karaktere), `65`.

Takođe možete koristiti ovu metodu sa drugim znakovima da pronađete njihove numeričke kod vrednosti:

:::interactive_editor

```js
let symbol = "!";
console.log(symbol.charCodeAt(0));  // 33
```

:::

Ovde, numerički kod za uzvičnik `!` je vraćen kao `33` (što odgovara njegovoj ASCII vrednosti).

Dok `charCodeAt()` pomaže da povučete numerički kod karaktera, metoda `fromCharCode()` omogućava vam da uradite suprotno: konvertujete UTF-16 kod jedinicu (koja odgovara ASCII za osnovne karaktere) u njen odgovarajući karakter.

Hajde da vidimo ovo u akciji:

:::interactive_editor

```js
let char = String.fromCharCode(65);
console.log(char);  //  A
```

:::

U ovom primeru, `fromCharCode(65)` konvertuje numerički kod `65` (koji odgovara ASCII vrednosti za `A`) nazad u karakter `A`.

Još jedan primer bi bilo konvertovanje broja `97` u odgovarajuće mala slovo:

:::interactive_editor

```js
let char = String.fromCharCode(97);
console.log(char);  // a
```

:::

Ovi metodi su posebno korisni kada je potrebno manipulisati ili uporediti znakove na osnovu njihovih numeričkih kodnih vrednosti.

Na primer, možete koristiti ``charCodeAt()`` da proverite da li je karakter veliko slovo, malo slovo ili cifra poređenjem njegove ASCII vrednosti.

S druge strane, `fromCharCode()` može se koristiti za dinamičko generisanje karaktera iz njihovih ASCII kodova.
