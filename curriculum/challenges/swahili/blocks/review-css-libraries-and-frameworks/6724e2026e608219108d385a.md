## CSS Okviri

- **CSS frameworks**: CSS frameworkovi mogu ubrzati vaš radni proces, stvoriti uniforman vizuelni stil na celoj veb stranici, osigurati da je dizajn konzistentan u više pregledača i održati vaš CSS kod organizovanijim.
- **Popular CSS frameworks**: Neki od popularnih CSS frameworkova su Tailwind CSS, Bootstrap, Materialize i Foundation.
- **Potencijalne nedostatke**:
  - CSS koji pruža framework može kolidirati sa vašim prilagođenim (custom) CSS-om.
  - Vaša veb stranica može izgledati slično drugim veb stranicama koje koriste isti framework.
  - Veliki frameworkovi mogu uzrokovati probleme sa performansama.

## Dve vrste CSS framework-ova

- **Utilitetni CSS frameworkovi**: Ovi frameworkovi imaju male klase sa specifičnim namenama, kao što je postavljanje margine, paddinga ili boje pozadine. Ove male klase možete direktno dodeliti na HTML elemente po potrebi. Tailwind CSS je klasifikovan kao utilitetni CSS framework.

Ovo je primer korišćenja Tailwind CSS za stilizovanje dugmeta.

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **Komponentno zasnovani CSS okviri**: Ovi okviri imaju predizgrađene komponente sa predodređenim stilovima koje možete dodati na svoj veb sajt. Komponente su dostupne u zvaničnoj dokumentaciji CSS okvira, a vi ih možete kopirati i zalepiti u svoj projekat. Bootstrap je kategorisan kao komponentno zasnovan CSS okvir.

Ovo je primer korišćenja Bootstrapa za kreiranje liste grupa. Umesto primene malih klasa na vaše HTML elemente, dodati ćete ceo komponent, uključujući i HTML strukturu.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

Tailwind je CSS framework koji koristi pristup zasnovan na utilitetima. Umesto da pišete prilagođena CSS pravila, kreirate dizajn kombinovanjem malih utilitarnih klasa direktno u vašem HTML-u.

### Utiliteti za responsivan dizajn

Tailwind koristi prefikse kao što su `sm:`, `md:`, i `lg:` za primenu stilova na različitim veličinama ekrana.

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### Utiliteti za Flexbox

Klasama poput `flex`, `flex-col`, `justify-around` i `items-center` olakšava kreiranje fleksibilnih rasporeda.

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### Funkcije za Mreže

Tailwind uključuje utilitete za CSS Grid, kao što su `grid`, `grid-cols-1`, i `md:grid-cols-3`.

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### Alati za Razmake

Alati poput `mt-8`, `mx-auto`, `p-4`, i `gap-4` pomažu u kreiranju konzistentnog razmaka bez pisanja CSS-a.

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### Tipografski Utiliteti

Alati poput ``uppercase``, ``font-bold``, ``font-semibold``, i ``text-4xl`` kontrolišu izgled teksta.

Možete postaviti veličine fonta koje se prilagođavaju na tačkama prekida, kao što su `text-3xl` `md:text-5xl`.

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### Boje i stanja prelaska mišem

Tailwind pruža široku paletu boja, kao što su `text-red-700`, `bg-indigo-600`, i `bg-gray-100`.

Klase poput `hover:bg-pink-600` čine interaktivne efekte jednostavnim.

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### Granice, Prstenovi i Efekti

- **Granice**: `border-2 border-red-300` dodaje granice sa specificiranom debljinom i bojama.
- **Prstenje**: `ring-1 ring-gray-300` kreira efekte slične konturi, često korišćene za fokus ili kartice.
- **Zaobljeni uglovi i skaliranje**: Klase poput `rounded-md`, `rounded-xl`, i `scale-105` dodaju završni dodir.

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### Gradijenti

Tailwind podržava utilitete za gradijente kao što su `bg-gradient-to-r from-fuchsia-500 to-indigo-600`.

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## CSS Preprocesori

- **CSS preprocessor**: CSS predprocesor je alat koji proširuje standardni CSS. Kompajlira kod sa proširenom sintaksom u nativni CSS fajl. Može biti koristan za pisanje čistijeg, ponovno upotrebljivijeg, manje repetitivnog i skalabilnog CSS-a za složene projekte.
- **Features**: Neke od funkcionalnosti koje mogu pružiti CSS predprocesori su varijable, mixini, ugnježđivanje (nesting) i nasleđivanje selektora (selector inheritance).
- **Popular CSS preprocessors**: Neki popularni CSS predprocesori su Sass, Less i Stylus.  
- **Potential disadvantages**:
  - Kompajliranje CSS pravila u standardni CSS može uzrokovati dodatno opterećenje (overhead).
  - Kompajlirani kod može biti težak za debugovanje.

## Sass

- **Sass**: To je jedan od najpopularnijih CSS predprocesora. Sass skraćeno je iz fraze "Syntactically Awesome Style Sheets."
- **Funkcionalnosti koje podržava Sass**: Sass podržava funkcionalnosti kao što su varijable, ugnježdena CSS pravila, moduli, mixini, nasleđivanje i operateri za osnovne matematičke operacije

## Dve sintakse koje Sass podržava

- **SCSS sintaksa**: SCSS (Sassy CSS) proširuje osnovnu sintaksu CSS-a. To je najčešće korišćena sintaksa za Sass. SCSS fajlovi imaju ekstenziju `.scss`.

Ovo je primer definisanja i korišćenja varijable u SCSS-u.

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

**Indentirana sintaksa**: Indentirana sintaksa je bila originalna sintaksa Sass-a i poznata je takođe kao „Sass sintaksa“.

Ovo je primer definisanja i korišćenja varijable u uvlačenoj sintaksi.

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### Miksini

- **Mixins**: Mixini vam omogućavaju da grupišete više CSS svojstava i njihove vrednosti pod imenom, a zatim ponovo koristite taj blok CSS koda u celom vašem stilskom listu.

Ovo je primer definisanja miksina u SCSS sintaksi. U ovom slučaju, miksin se zove `center-flex`. Ima tri CSS svojstva za centriranje elemenata korišćenjem flexbox-a.

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

Ovo je primer korišćenja miksina koji ste definisali.

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
