# --description--

Do ovog trenutka, koristili ste React za izradu korisničkih interfejsa. Ako vam su bile potrebne dodatne funkcije kao što je rutiranje, morali ste uvesti biblioteku treće strane poput React Routera kako biste mogli prebacivati između različitih prikaza.

Ali šta se dešava kada vam je potrebno izgraditi punostak web aplikaciju? Pa, mogli biste koristiti React za prednji kraj (front-end) i koristiti Node i Express za logiku zadnjeg kraja (back-end), ako samo želite da ostanete sa JavaScript-om. Ili možete koristiti druge jezike kao što su Go, Python ili Java za vaš zadnji kraj (back-end).

Iako su sve ovo izvodljive opcije, postoje situacije kada biste možda želeli da umesto toga koristite React okvir. React okviri pružaju funkcije kao što su rutiranje (routing), optimizacija slika, dohvaćanje podataka (data fetching), autentifikacija i još mnogo toga. To znači da možda nećete morati da postavite zasebne front-end i back-end aplikacije za određene slučajeve upotrebe.

Pogledajmo bliže Next.js, koji je popularan React okvir. Jedna od glavnih karakteristika za Next.js je ruter zasnovan na sistemu datoteka. Ovaj sistem rutiranja uključuje podršku za dinamičke rute, paralelne rute, rukovače ruta, preusmeravanja, internacionalizaciju i više.

Evo primera kreiranja prilagođenog rukovaoca zahteva:

```js
export async function GET() {
  const res = await fetch("https://example-api.com");
  const data = await res.json();

  return Response.json({ data });
}
```

Možete definisati rukovače rute, kao što su GET ili POST zahtevi, u fajlu pod nazivom `route.js` unutar direktorijuma `app/api`.

Još jedna karakteristika Next.js-a su automatske optimizacije slika i fontova.

Evo primera rada sa komponentom `Image` unutar fajla `page.js`:

```jsx
import Image from "next/image";

export default function Page() {
  return (
    <Image src="link-to-image-goes-here" alt="descriptive-title-goes-here" />
  );
}
```

Komponenta `Image` proširuje na izvorni (native) HTML `img` element i omogućava brže učitavanje stranica i optimizacije veličine. Ovo znači da će slike biti učitane samo kada uđu u vidokrug (viewport), a komponenta `Image` će automatski servirati slike ispravne veličine za svaki uređaj.

Iako su to samo nekoliko funkcija Next.js-a, postoji mnogo više funkcija koje možete koristiti za izgradnju robusnih full-stack veb aplikacija. Postoje i drugi React framework-ovi poput Remixa koji će pružiti istu sposobnost za izgradnju modernih full-stack veb aplikacija.

Iako se pejzaž biblioteka i okvira za JavaScript stalno menja, važno je biti svestan dostupnih opcija za vas i saznati prednosti i mane svakog okvira.
