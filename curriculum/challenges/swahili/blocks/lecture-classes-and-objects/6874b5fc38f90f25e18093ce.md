# --description--

U poslednjoj lekciji, naučili ste o klasama i kako deluju kao nacrti za kreiranje objekata.

Ovde, detaljnije ćemo istražiti atribute i metode.

Pogledajmo bliže atributima prvo, a zatim i metodama.

Atributi su varijable koje pripadaju objektu, pa čuvaju podatke. Postoje dve vrste atributa: instancijski atributi i klasni atributi.

Atributi instanci su jedinstveni za svaki objekat kreiran iz klase, a obično ih postavljate pomoću metode ``__init__``. Klasni atributi, s druge strane, pripadaju samoj klasi i dele se sa svim instancama te klase.

Da biste pristupili atributu, koristite tačkastu notaciju.

Evo primera kako se koriste i instancijalni i klasni atributi, i kako ih pristupiti iz objekata:

```python
class Dog:
    species = "French Bulldog" # Class attribute

    def __init__(self, name):
        self.name = name # Instance attribute

print(Dog.species) # French Bulldog

dog1 = Dog("Jack")
print(dog1.name)    # Jack
print(dog1.species) # French Bulldog

dog2 = Dog("Tom")
print(dog2.name)    # Tom
print(dog2.species) # French Bulldog
```

Napomena da možete pristupiti atributima klase direktno iz same klase, ali morate prvo kreirati objekat i proslediti mu podatke pre nego što možete pristupiti instancijalnim atributima.

Automobili su još jedan dobar primer, jer svi automobili imaju model i boju:

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.model) # Toyota Corolla
print(car_2.model) # Lamborghini Revuelto

print(car_1.color) # red
print(car_2.color) # green
```

Metode su funkcije definisane unutar klase. Njima, bilo koji objekat definisan iz klase može izvršiti akcije koje operišu na ili modifikuju sopstvene podatke. Metodu takođe pristupate sa tačkastom notacijom.

Na primer, psi mogu da laju. Dakle, možemo imati `bark` metodu u klasi `Dog` kao što ste videli u prethodnoj lekciji:

```python
class Dog:
   species = "French Bulldog"

   def __init__(self, name):
     self.name = name

   def bark(self):
       return f"{self.name} says woof woof!"

jack = Dog("Jack")
jill = Dog("Jill")

print(jack.bark()) # Jack says woof woof!
print(jill.bark()) # Jill says woof woof!
```

Klasa `Car` takođe može imati metod `describe`:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  # Instance attribute
        self.model = model  # Instance attribute

    def describe(self):
        return f"This car is a {self.color} {self.model}"

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.describe()) # This car is a red Toyota Corolla
print(car_2.describe()) # This car is a green Lamborghini Revuelto
```
