# --interactive--

Unarni operateri deluju na jedan operand da izvrše operacije kao što su konverzija tipa, manipulacija vrednostima ili proveravanje određenih uslova. Pogledajmo neke uobičajene unarne operatore i kako funkcionišu.

Unarni plus operater konvertuje svoj operand u broj. Ako je operand već broj, ostaje nepromenjen.

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

Unarni plus je koristan kada želite da budete sigurni da radite sa numeričkom vrednošću. Kao što možete da pogodite, postoji unarni negacioni operator. Negira vrednost operanda. Radi slično kao unarni plus, osim što menja znak.

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

Logički NOT operater, predstavljen uzvičnikom (`!`), je još jedan unarni operater. On menja booleansku vrednost svog operanda. Dakle, ako je operand `true`, postaje `false`, a ako je `false`, postaje `true`.

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

Operater bitnog NOT je jednosložni operater koji se ređe koristi. Predstavljen tildom, `~`, on invertuje binarnu reprezentaciju broja. Računari skladište brojeve u binarnom formatu (jedinice i nule). Operater `~` prevrće svaki bit, što znači da sve jedinice menja u nule i sve nule u jedinice. Vi ćete saznati više o binarnom sistemu i bitovima u budućoj lekciji.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

U ovom primeru, `5` je postao `-6` jer primenjivanjem operatera `~` na `5`, dobijate `- (5 + 1)`, koji je jednak `-6` zbog reprezentacije dopune dva. Dopuna dva je način na koji računari predstavljaju negativne brojeve u binarnom sistemu. Verovatno nećete često koristiti bitwise NOT osim ako ne radite sa zadacima niskonivošnjeg programiranja, kao što je direktna manipulacija bitovima.

Ključna reč ``void`` je unarni operator koji procenjuje izraz i vraća ``undefined``.

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` takođe se često koristi u hiperlinkovima kako bi sprečio navigaciju:

```js
<a href="javascript:void(0);">Click Me</a>
```

Konačno, postoji operater ``typeof`` o kojem ste naučili u prethodnim lekcijama. Ovo vraća tip svog operanda kao niz znakova (string).

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
