# --interactive--
## Značaj semantičkog HTML-a

- **Strukturna hijerarhija za elemente zaglavlja**: Važno je koristiti odgovarajući element zaglavlja kako bi se održala strukturalna hijerarhija sadržaja. Element `h1` predstavlja najviši nivo zaglavlja, a element `h6` predstavlja najniži nivo zaglavlja.
- **HTML elementi za prezentaciju**: Elementi koji definišu izgled sadržaja. Primer: zastareli elementi `center`, `big` i `font`.
- **Semantički HTML elementi**: Elementi koji nose značenje i strukturu. Primer: `header`, `nav` i `figure`.

## Semantički HTML elementi

- **Element zaglavlja**: koristi se za definisanje zaglavlja dokumenta ili sekcije.

:::interactive_editor

  __CODE_BLOCK_0__

:::

- **Glavni element**: Koristi se za kontejnerovanje glavnog sadržaja veb stranice.

:::interactive_editor

  __CODE_BLOCK_1__

:::

- **Element sekcije**: koristi se za razdvajanje sadržaja na manje sekcije.

:::interactive_editor

  __CODE_BLOCK_2__

:::

- **Navigaciona sekcija (`nav`) element**: predstavlja sekciju sa navigacionim linkovima.

:::interactive_editor

  __CODE_BLOCK_3__

:::

- **Element figure**: koristi se za sadržaj ilustracija i dijagrama.

:::interactive_editor

  __CODE_BLOCK_4__

:::

- **Emphasis (`em`) element**: označava tekst koji je naglašen.

:::interactive_editor

  __CODE_BLOCK_5__

:::

- **Idiomatic Text (`i`) element**: koristi se za isticanje alternativnog glasa ili tona, idiomatskih termina iz drugog jezika, tehničkih termina i misli.

:::interactive_editor

  __CODE_BLOCK_6__

:::

Atribut ``lang`` unutar otvorenog taga ``i`` koristi se za određivanje jezika sadržaja. U ovom slučaju, jezik bi bio francuski. Element ``i`` ne ukazuje da li je tekst važan ili ne; on samo pokazuje da je na neki način drugačiji od okolnog teksta.

- **Element visoke važnosti (`strong`) element**: označava tekst koji ima veliku važnost.

:::interactive_editor

  __CODE_BLOCK_7__

:::

- **Bring Attention To (`b`) element**: koristi se za privlačenje pažnje na tekst koji nije važan za značenje sadržaja. Često se koristi za isticanje ključnih reči u sažecima ili imena proizvoda u recenzijama.

:::interactive_editor

  __CODE_BLOCK_8__

:::

- **Description List (`dl`) element**: koristi se za predstavljanje liste grupa pojm-opis.
- **Description Term (`dt`) element**: koristi se za predstavljanje pojma koji se definiše.
- **Description Details (`dd`) element**: koristi se za predstavljanje opisa pojma.

:::interactive_editor

  __CODE_BLOCK_9__

:::

- **Block Quotation (`blockquote`) element**: koristi se za predstavljanje sekcije koja je citirana iz drugog izvora. Ovaj element ima atribut `cite`. Vrednost atributa `cite` je URL izvora.

:::interactive_editor

  __CODE_BLOCK_10__

:::

- **Citation (`cite`) element**: Koristi se za vizuelno pripisivanje izvora referenciranog rada. Označava naslov reference.

:::interactive_editor

  __CODE_BLOCK_11__

:::

- **Element ugrađene citacije (`q`)**: koristi se za predstavljanje kratke ugrađene citacije.

:::interactive_editor

  __CODE_BLOCK_12__

:::

- **Skraćeni element (`abbr`)**: koristi se za predstavljanje skraćenice ili akronima. Da biste korisnicima pomogli da razumeju šta je ta skraćenica ili akronim, možete prikazati njegov puni oblik, čitljiv opis, koristeći atribut `title`.

:::interactive_editor

  __CODE_BLOCK_13__

:::

- **Adresa za kontakt (`address`) element**: koristi se za predstavljanje informacija o kontaktu.
- **Vreme (Datum) (`time`) element**: koristi se za predstavljanje datuma i/ili vremena. Atribut `datetime` se koristi za prevođenje datuma i vremena u format koji je čitljiv mašinama.

:::interactive_editor

  __CODE_BLOCK_14__

:::

- **ISO 8601 Date (`datetime`) attribute**: Koristi se za predstavljanje datuma i vremena u formatu koji je čitljiv mašinama. Standardni format je `YYYY-MM-DDThh:mm:ss`, pri čemu veliko slovo `T` služi kao razdvojnik između datuma i vremena.
- **Superscript (`sup`) element**: Koristi se za predstavljanje teksta sa nadpisom (superscript). Uobičajene upotrebe za element `sup` uključuju eksponente, superiorno slovo i redne brojeve.

:::interactive_editor

  __CODE_BLOCK_15__

:::

- **Subscript (`sub`) element**: koristi se za predstavljanje teksta sa podskriptom. Uobičajene upotrebe ovog elementa uključuju hemijske formule, podnožja i podskripte varijabli.

:::interactive_editor

  __CODE_BLOCK_16__

:::

- **Inline Code (`code`) element**: koristi se za predstavljanje fragmenta računarskog koda.
- **Preformatted Text (`pre`) element**: predstavlja preformatirani tekst

:::interactive_editor

  __CODE_BLOCK_17__

:::

- **Unarticulated Annotation (`u`) element**: koristi se za predstavljanje raspona ugrađenog teksta koji bi trebalo da bude renderovan na način koji ukazuje da poseduje ne-tekstualnu anotaciju.

:::interactive_editor

  __CODE_BLOCK_18__

:::

- **Ruby Annotation (`ruby`) element**: koristi se za anotiranje teksta sa izlaganjem izgovora ili značenja. Primer upotrebe je u istočnoj azijskoj tipografiji.
- **Ruby fallback parenthesis (`rp`) element**: koristi se kao rezervna opcija (fallback) za pretraživače koji nemaju podršku za prikaz ruby anotacija.
- **Ruby text (`rt`) element**: koristi se za označavanje teksta namenjenog ruby anotaciji. Obično se koristi za izgovor ili detalje prevoda u istočnoj azijskoj tipografiji.

:::interactive_editor

  __CODE_BLOCK_19__

:::

- **Element za prekriženo (Strikethrough) (`s`)**: koristi se za predstavljanje sadržaja koji više nije tačan ili relevantan.

:::interactive_editor

  __CODE_BLOCK_20__

:::

- **Interni linkovi**: Koriste se za povezivanje na drugi deo stranice korišćenjem ``href="#id"`` na elementu ``a`` i dajući odredišnom elementu isti ``id``. Ovo se često koristi za linkove za preskakanje (skip links), sadržaj stranice ili dugačke stranice sa više sekcija.

:::interactive_editor
