# --description--

U JavaScriptu, modul je samostalna jedinica koda koja enkapsulira povezane funkcije, klase ili varijable.

Zamislite modul kao građevni blok za vašu aplikaciju, slično poglavlju u knjizi. Svaki modul se fokusira na specifičnu funkcionalnost, čime je vaš kod organizovaniji, održiviji i ponovno upotrebljiv. Moduli pomažu u sprečavanju sukoba imena i omogućavaju vam da strukturirate svoju aplikaciju u odvojene, međusobno povezane delove.

Koncept modula u JavaScriptu razvijao se tokom vremena, ali najšire korišćeni i podržani pristup je sistem modula ES6 (ECMAScript 2015). Ovaj sistem pruža standardizovan način za definisanje i korišćenje modula u različitim JavaScript okruženjima.

Da biste kreirali modul, pišete svoj JavaScript kod u zasebnoj datoteci. Sve varijable, funkcije ili klase koje želite da učinite dostupnim drugim delovima vaše aplikacije moraju biti eksplicitno izveštane (exportovane). To možete uraditi koristeći ključnu reč `export`.

Na primer, recimo da imate fajl nazvan `math.js` sa nekim matematičkim funkcijama:

```javascript
// math.js
export const PI = 3.14159;

export function add(a, b) {
  return a + b;
}

export function subtract(a, b) {
  return a - b;
}
```

U ovom primeru, eksportujemo dve funkcije (`add` i `subtract`) i konstantu (`PI`). Možete eksportovati koliko god stavki je potrebno iz jednog modula.

Da biste koristili ove eksportovane stavke u drugom delu vaše aplikacije, morate ih uvesti (importovati). To se radi koristeći ključnu reč `import`.

Recimo da želite da koristite ove matematičke funkcije u fajlu nazvanom `app.js`:

```javascript
// app.js
import { add, subtract } from './math.js';

console.log(add(5, 3)); // Izlaz: 8
console.log(subtract(10, 2)); // Izlaz: 8
```

Ovde uvozimo specifične funkcije i konstantu koje nam trebaju iz modula `math.js`. Deo `'./math.js'` govori JavaScriptu gde da pronađe modul fajl u odnosu na trenutni fajl.

Ponekad, možda želite da uvezete sve što modul eksportuje. To možete uraditi koristeći sintaksu sa zvezdastim znakom (`*`):

```javascript
// app.js
import * as Math from './math.js';

console.log(Math.PI);
console.log(Math.add(5, 3));
```

U ovom slučaju, svi eksporti iz `math.js` uvoze se kao svojstva objekta nazvanog `Math`.

Drugi uobičajeni obrazac je imati podrazumevani eksport (default export) u modulu. Ovo se obično koristi kada modul primarno eksportuje jednu funkciju. Možete imati samo jedan podrazumevani eksport po modulu.

Evo kako to izgleda:

```javascript
// math.js
export default function calculateArea(radius) {
  return Math.PI * radius * radius;
}
```

Primećujte da prilikom uvoza podrazumevanog eksporta, ne morate koristiti vitičaste zagrade, i možete nazvati uvoz kako god želite.

Važno je napomenuti da za korišćenje ES6 modula u pretraživaču, morate specificirati `type` kao `module` u vašem `script` tagu:

```html
<script type="module" src="app.js"></script>
```

Moduli pružaju moćan način za organizovanje i strukturiranje vašeg JavaScript koda. Omogućavaju vam da razbijete svoju aplikaciju na manje, upravljive delove, podstiču ponovnu upotrebu koda i pomažu u održavanju čiste razdvajanja briga.
