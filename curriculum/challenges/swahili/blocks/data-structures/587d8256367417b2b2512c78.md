# --description--

Drugi način za predstavljanje grafa je stavljanje u <dfn>matricu susjednosti</dfn>. <dfn>Matrica susjednosti</dfn> je dvodimenzionalni niz podataka (2D) gdje svaki unutrašnji niz ima isti broj elemenata kao i glavni niz. Drugim riječima, to je matrica ili mreža brojeva, gdje ti brojevi predstavljaju ivice.

**Napomena**: Brojevi na vrhu i lijevo matrice su samo oznake čvorova. Unutar matrice, jedinica znači da postoji ivica između rednih i kolonskih brojeva koji predstavljaju čvorove. Naprotiv, nula označava da nema ivice ili veze.

<pre>
    1 2 3
  \------
1 | 0 1 1
2 | 1 0 0
3 | 1 0 0
</pre>

Izgore je vrlo jednostavan nedirektni graf sa tri čvora, gdje je prvi čvor povezan sa drugim i trećim. Ispod je JavaScript implementacija istog koncepta.

```js
var adjMat = [
  [0, 1, 1],
  [1, 0, 0],
  [1, 0, 0]
];
```

Za razliku od liste susjednosti, svaki "red" matrice mora imati isti broj elemenata kao i čvorova u grafu. Ovde imamo matricu tri na tri, što znači da imamo tri čvora u našem grafu. Direktni graf bi imao sličan izgled. Ispod je graf gdje prvi čvor ima ivicu usmjerenu ka drugom čvoru, a drugi čvor ima ivicu usmjerenu ka trećem čvoru.

```js
var adjMatDirected = [
  [0, 1, 0],
  [0, 0, 1],
  [0, 0, 0]
];
```

Grafovi također mogu imati <dfn>težinu</dfn> na svojim ivicama. Do sada smo imali <dfn>netežinske</dfn> ivice gdje je postojanje ili nepostojanje ivice binarno (`0` ili `1`). Možete imati različite težine ovisno o vašoj aplikaciji.

# --instructions--

Kreirajte matricu susjednosti nedirektnog grafa sa pet čvorova. Ova matrica treba da bude u dvodimenzionalnom nizu podataka. Ovi pet čvorova imaju vezu između prvog i četvrtog, prvog i trećeg, trećeg i petog, te četvrtog i petog čvora. Težina svih ivica je jedan.
