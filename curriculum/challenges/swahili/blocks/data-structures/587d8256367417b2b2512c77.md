# --description--

Grafovi se mogu predstaviti na različite načine. Ovde objašnjavamo jedan način, poznat kao <dfn>lista susjednosti</dfn>. Lista susjednosti je listu sa oznakama gdje je lijeva strana čvor, a desna lista sve ostale čvorove povezanih s njim. Ispod se nalazi predstavljanje liste susjednosti.

<blockquote>Nodi1: Nodi2, Nodi3<br>Nodi2: Nodi1<br>Nodi3: Nodi1</blockquote>

Izgore je graf bez smjera jer je `Node1` povezan sa `Node2` i `Node3`, a ta informacija odgovara konekciji prikazanoj između `Node2` i `Node3`. Lista susjednosti za usmjeren graf znači da svaka linija liste pokazuje smjer. Ako bi gore bio usmjeren, onda bi `Node2: Node1` značilo da postoji smjer toka iz `Node2` ka `Node1`. Graf bez smjera prikazan gore možemo predstaviti kao listu susjednosti stavivanjem u JavaScript objekat.

```js
var undirectedG = {
  Node1: ["Node2", "Node3"],
  Node2: ["Node1"],
  Node3: ["Node1"]
};
```

Ovo se također može predstaviti jednostavnije kao niz podataka gdje čvorovi imaju brojeve umjesto sekvencijalnih slovnih oznaka.

```js
var undirectedGArr = [
  [1, 2], // Node1
  [0],    // Node2
  [0]     // Node3
];
```
