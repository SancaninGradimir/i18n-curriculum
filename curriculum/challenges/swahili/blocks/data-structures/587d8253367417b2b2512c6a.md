# --description--

Nizovi podataka su JavaScript entitete koji mogu držati različite vrste elemenata.

```js
var complexArr = [1, 5, "2", "Word", {"name": "James"}];
```

Po osnovu, ono što se dešava u pozadini je to da vaš pregledač dodeljuje odgovarajuću memorijsku lokaciju za taj niz. Ta lokacija se takođe menja kada dodate ili uklonite podatke.

Međutim, u svetu visokih performansi i različitih tipova elemenata, ponekad je potrebno biti precizniji o količini memorije dodeljene nizu.

<dfn>Typed Array</dfn> je rešenje ovog problema. Sada možete odrediti koliko memorije želite dodeliti nizu. Ispod je pregled različitih tipova nizova dostupnih i veličine svakog elementa u bajtovima.

Postoje dva načina za kreiranje ovih tipova nizova. Jedan način je direktno kreiranje. Evo kako se kreira `Int16Array` dužine 3.

```js
var i8 = new Int16Array(3);
console.log(i8);
// Returns [0, 0, 0]
```

Takođe možete kreirati <dfn>ArrayBuffer</dfn> da odredite količinu podataka (u bajtovima) koju niz treba da zauzme. **Napomena:**  
Da biste kreirali Typed Array koristeći ArrayBuffer, morate postaviti broj bajtova kao višestruki broj bajtova navedenih gore.

```js
// Create same Int16Array array differently
var byteSize = 6; // Needs to be multiple of 2
var buffer = new ArrayBuffer(byteSize);
var i8View = new Int16Array(buffer);
buffer.byteLength; // Returns 6
i8View.byteLength; // Returns 6
console.log(i8View); // Returns [0, 0, 0]
```

<dfn>ArrayBuffer</dfn> je opšta struktura koja nosi samo podatke. Ne možete ga dobiti direktno. Da biste mu pristupili, prvo morate kreirati <dfn>View</dfn>.

```js
i8View[0] = 42;
console.log(i8View); // Returns [42, 0, 0]
```

**Napomena:**  
Typed Array nema neke metode koje obični nizovi imaju, kao što su `.pop()` ili `.push()`. Takođe ne prolaze `Array.isArray()` koja proverava da li je nešto niz podataka. Iako je to jednostavnije, ovo može biti prednost za JavaScript motore sa ograničenim kapacitetom izvršavanja ovih nizova.

```js
assert(buffer.byteLength === 64);
```

```js
assert(i32View.byteLength === 64);
```

```js
assert(i32View.length === 16);
```
