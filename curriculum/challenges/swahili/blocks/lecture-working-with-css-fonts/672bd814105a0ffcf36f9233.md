# --interactive--

Font porodica je grupa fontova koji dele zajednički dizajn. Svi fontovi koji pripadaju istoj porodici zasnovani su na istom osnovnom tipografskom izgledu, ali imaju i varijacije u stilu, težini i širini. Možete ih zamisliti kao braću i sestre koji dele slične karakteristike, ali takođe imaju neke razlike. Na primer, Arial je font porodica koja uključuje varijacije poput Arial Bold i Arial Italic.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-font-families-and-how-do-they-work-1.png" alt="Diagram showing 'Arial' at the top with two blue arrows pointing down to 'Arial Bold' on the left and 'Arial Italic' on the right, illustrating font family variations" />

U CSS-u, možete postaviti porodične fontove sa svojstvom `font-family`. Evo primera postavljanja fonta za pasus na `Arial`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="arial-font">Example paragraph using Arial font.</p>
<p>Paragraph not using Arial font.</p>
```

```css
#arial-font {
  font-family: Arial;
}
```

:::

Ali šta ako se porodica fonta ne pronađe? Možete navesti više porodičnih fontova u redosledu prioriteta, od najvišeg do najnižeg, razdvajanjem zarezima. Ovi alternativni fontovi će delovati kao opcije za rezervnu upotrebu. U ovom primeru, Arial je primarni font, a Lato je alternativni font:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato;
}
```

:::

Pretraživač će prikazati prvi font (Arial) ako je pronađen na uređaju. Ako ne, biće prikazan drugi font (Lato).

Takođe biste trebali da znate kako ovaj proces funkcioniše iza kulisa. Proces odabira ne prestaje ako je prvi font dostupan. Font porodica se bira po jedan karakter, pa ako font nema određeni karakter, pretraživač ga traži u fontovima nižeg prioritet. Zanimljivo, zar ne?

U kontekstu web razvoja, takođe ćete pronaći generičke porodice fontova. One su veoma važne. Generička porodična grupa fontova je podrazumevani font koji će veb pregledači prikazati kada nisu dostupni fontovi višeg prioriteta. Da bi se osiguralo da je sadržaj i dalje čitljiv, pregledač zamenjuje originalni font najprikladnijim pronađenim fontom, na osnovu navedene generičke porodične grupe fontova. Neke od najčešće korišćenih su:

- serif
- sans-serif
- monospace
- kurzivni (ili kursiv)
- fantazija

Ali postoji više opcija među kojima se može birati. U kodu ispod, imamo tri alternativne porodice fontova:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato, sans-serif;
}
```

:::

Arial ima najviši prioritet. Ako se Arial ne pronađe, pretraživač će pokušati da prikaže Lato. Ako se nijedan od njih ne pronađe, pretraživač će koristiti generičku sans-serif porodicu fontova, birajući font sa ovim karakteristikama iz onih instaliranih na korisnikovom sistemu.

Uvek biste trebali da uključite generičku porodicu fontova na kraju liste porodica fontova. Imajte na umu da generički font može izgledati drugačije od fonta koji ste prvobitno zamislili u svom dizajnu, pa je uvek korisno proveriti kako ovi rezervni (fallback) fontovi izgledaju na različitim pretraživačima.

Da bi se korisničko iskustvo učinilo što konzistentnijim, preporučuje se korišćenje fontova sigurni za web. Ove font porodice su obično instalirane na većini uređaja, pa je vrlo verovatno da će biti pronađene i iscrtane ispravno za većinu korisnika. Vi ćete više o njima naučiti u sledećim lekcijama.

Porodice fontova su esencijalne u svetu web dizajna. Korišćenjem svojstva `font-family`, možete trenutno transformisati izgled i osećaj vašeg teksta.
