# --description--

Pre nego što uđemo u detalje, trebalo bi da znate šta je at-pravilo u CSS-u. At-pravila su izjave koje pružaju instrukcije pretraživaču. Možete ih koristiti za definisanje različitih aspekata stilskog lista, kao što su medijski upiti (media queries), keyframe-ovi, fontovi i još mnogo toga. Vi ćete saznati više o konceptima poput medijskih upita i keyframe-ova u budućim lekcijama. Ovog puta, naučićete se o at-pravilu `@font-face`.

Sa `@font-face`, možete definisati prilagođeni font navođenjem fajla fonta, formata i drugih važnih svojstava, kao što su težina i stil. Ovo je osnovna sintaksa. Možete videti `@font-face` praćen blokom zatvorenim uglastim zagradama:

```css
@font-face {
  /* Descriptors */
}
```

Unutar vitičastih zagrada, moraćete da uključite deskriptore za prilagođavanje izgleda vašeg fonta. Pogledajmo neke od najčešće korišćenih. Deskriptor font-family navodi ime koje ćete koristiti širom stilskog lista za referenciranje tog fonta. Na primer, recimo da definišete ovo `@font-face rule`. Ima definisan deskriptor `font-family` i njegova vrednost je `MyCustomFont`:

```css
@font-face {
  font-family: "MyCustomFont";
}
```

U tom slučaju, morali biste koristiti to ime u vašem stilskom listu gde god želite dodeliti tu prilagođenu porodicu fonta svim elementima koje odgovara CSS selektor. Na primer, koristili biste ga kao vrednost za `font-family`:

```css
body { 
  font-family: "MyCustomFont"; 
}
```

Ali da bi `@font-face` deklaracija bila važeća, morate navesti i `src`. Ovo sadrži reference na font resurse. U osnovi je lista eksternih referenci ili lokalno instaliranih imena fonta, razdvojenih zarezima. Takođe može da sadrži naznake o formatu i tehnologiji font resursa.

U sledećem kodu, `src` je definisan. Vrednost je lista URL-ova razdvojenih zarezima i postavljena na zasebnim redovima radi poboljšanja čitljivosti:

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
    url("path/to/font.woff"),
    url("path/to/font.otf");
}
```

Možete pozvati funkciju ``url()`` da uključite fajl u svoj stilski list. U ovom slučaju, uključujemo font fajlove. Moraćete da napišete putanju do fajla unutar zagrada i navodnika, uključujući ekstenziju fajla. Da biste poboljšali čitljivost, možete svaki resurs napisati na novoj liniji, ali poslednji mora da se završi tačkom-zarezom.

Za svaki font resurs, možete navesti i format. Ovo je opcionalno. To je podsetnik za pretraživač o formatu fonta. Ako se format izostavi, resurs će biti preuzet i format će biti detektovan nakon što bude preuzet. Ako je format nevažeći, resurs neće biti preuzet.

Mogući formati fonta uključuju `collection`, `embedded-opentype`, `opentype`, `svg`, `truetype`, `woff`, i `woff2`.

Evo primera sa font formatima. Primećujte kako pišemo specifičan format unutar zagrada i navodnika:

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype"),
    url("path/to/font.woff") format("woff");
}
```

U ovom primeru, možete videti da specifikujemo WOFF2 format, OpenType format i WOFF format.

WOFF znači za "Web Open Font Format". Razlika između WOFF-a i WOFF2 je algoritam koji se koristi za kompresovanje podataka. OpenType je format za skalabilne računarske fontove razvijene od strane Microsofta i Adobe-a koji korisnicima omogućava pristup dodatnim funkcijama u fontu. Široko se koristi na glavnim operativnim sistemima.

Pored određivanja formata, možete odrediti i tehnologiju font resursa. Ovo je takođe opcionalno. Evo primera gde određujemo tehnologiju drugog font resursa.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype") tech(color-COLRv1),
    url("path/to/font.woff") format("woff");
}
```

Ovo su osnove pravila `@font-face`. Ovim at-pravilom, možete specificirati fajl fonta da definišete prilagođene fontove za vaše jedinstvene dizajne.
