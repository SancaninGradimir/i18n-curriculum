# --interactive--

CSS podrazumevano ne primenjuje nijedive senke na tekst. Ovo je primer paragrafa bez ikakvih senki. Ali ako zaista treba da dodate senke, možete koristiti svojstvo `text-shadow`.

```css
text-shadow: /* Values */
```

U CSS-u, možete opisati senku kombinacijom njenog ``X`` pomaka, ``Y`` pomaka, radijusa zamućenja i boje. Prvo, morate navesti ``X`` i ``Y`` pomak, koji predstavljaju horizontalnu i vertikalnu udaljenost senke od teksta, respektivno. Ove vrednosti su neophodne.

Evo primera kako primeniti `X` i `Y` pomak na senku. Primjenjujemo svojstvo `text-shadow` sa x-pomakom od `3px` i y-pomakom od `2px`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px;
}
```

:::

U pretraživaču, tekst i senka će izgledati slično, ali možemo prilagoditi i boju senke navođenjem vrednosti pre ili posle pomaka. Postavićemo boju senke. Ovde ćemo koristiti heksadecimalnu boju, ali možete koristiti bilo koji validan format boje.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

Pišemo boju nakon pomaka, ali možete napisati i boju pre pomaka ako želite. Ekvivalentno je:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: #00ffc3 3px 2px;
}
```

:::

U pretraživaču, senka će imati specifičnu boju, tako da je možemo vrlo lako razlikovati od teksta. Sada kada možemo da razlikujemo senku od teksta, važno je i videti kako pozitivne i negativne vrednosti utiču na pomeranje senke (shadow offset). Pozitivne vrednosti pomeranja ``X`` i pomeranja ``Y`` će pomeriti senku desno i dole, respektivno, dok će negativne vrednosti pomeriti senku levo i gore. Evo primera koji umesto toga koristi negativne vrednosti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: -3px -2px #00ffc3;
}
```

:::

Sada će senka pomerati se ulevo i gore u odnosu na tekst. Odlično. Ali senka ne izgleda baš lepo, jer izgleda tačno kao originalni tekst, ali postavljen iza njega. Da bi izgledalo lepše, moramo dodati treću vrednost, radijus zamućenja (blur radius). Ova vrednost je opcionalna, ali čini da senka izgleda mnogo glatkoje i subtilnije. Podrazumevana vrednost radijusa zamućenja je nula. Što je vrednost veća, to je zamućenje veće, što znači da senka postaje svetlja.

Ovde postavljamo radijus zamućenja na `3px` i vratili smo se pozitivnim vrednostima za pomeraj:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

Sada počinje da izgleda kao senka. Ono je zamućeno i suptilnije, pa možemo se fokusirati na glavni tekst dok senka dodaje neki stil u pozadini. Odavde, možemo malo podesiti vrednosti dok ne pronađemo kombinaciju pomaka (offset), boje i radijusa zamućenja (blur radius) koja odgovara našim potrebama.

Takođe je korisno znati da tekst može imati više od jedne senke. Samo treba da ih napišete u istoj ``text-shadow`` svojstvu, razdvojeno zarezima. Oni će biti primenjeni u slojevima, od prednjeg ka zadnjem, sa prvom senkom na vrhu.

Ovo je primer pasusa sa tri senke primenjene u slojevima. Primećujte da su sve ove senke definisane u istoj `text-shadow` svojstvu i razdvojene zarezima:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 
    3px 2px 3px #00ffc3, 
    -3px -2px 3px #0077ff, 
    5px 4px 3px #dee7e5;
}
```

:::

`text-shadow` je moćno CSS svojstvo za isticanje vašeg teksta. Takođe može poboljšati ukupni dizajn vaše veb aplikacije. Biranjem pomaka, zamućenja i boje, možete kreirati širok spektar efekata senke.
