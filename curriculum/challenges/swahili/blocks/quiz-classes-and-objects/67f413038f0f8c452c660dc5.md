### --question--

#### --text--
Koji od sledećih je ispravan način za definisanje klase?
#### --distractors--

```py
def className(class):
    pass
```

---

```py
self className:
    pass
```

---

```py
class className
    pass
```

#### --answer--

```py
class className:
    pass
```

### --question--

#### --text--
Koja je razlika između atributa instance-e i atributa klase?
#### --distractors--
Atributi instanci definišu metode koje se mogu pristupiti iz objekta; atributi klase definišu metode koje se mogu pristupiti iz klase.

---

Atributi instance se postavljaju u metodi ``__init__``; atributi klase se postavljaju u metodi ``__class__``.

---

Atributima instance-a se može direktno pristupiti iz same klase; atributi klase se mogu pristupiti samo preko stvarne instance te klase.
#### --answer--
Atributi instance pripadaju specifičnom objektu; atributi klase pripadaju klasi.
