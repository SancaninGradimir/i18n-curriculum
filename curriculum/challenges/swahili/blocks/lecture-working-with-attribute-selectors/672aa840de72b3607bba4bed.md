# --interactive--

Selektor atributa u CSS-u je moćan alat koji vam omogućava da ciljate HTML elemente na osnovu njihovih atributa. To znači da možete primeniti stilove na elemente koji imaju specifične atribute ili čak određene vrednosti za te atribute.

Ovo je posebno korisno kada želite dinamički stilizovati elemente ili kada imena klasa sama po sebi ne pružaju dovoljno specifičnosti.

Na primer, možete koristiti atributni selektor da biste ciljali sve linkove koji imaju atribut `href`. Ovo je posebno korisno kada želite primeniti ujednačen stil na sve linkove na stranici.

Evo kako možete koristiti selektor atributa da ciljate linkove sa atributom `href`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

Ovo CSS pravilo će primeniti plavu boju i podvlačenje na svaki link koji sadrži atribut ``href``, osiguravajući da su svi klikabilni linkovi dosledno stilizovani.

Ali selektori atributa mogu biti specifičniji. Pretpostavimo da želite da ciljate samo linkove koji imaju atribut ``title``. Atribut ``title`` često pruža dodatne informacije o linku, i možda želite da stilizujete ove linkove drugačije kako biste ukazali na to da imaju dostupne dodatne informacije. Evo kako to možete uraditi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

Ovo pravilo primenjuje podebljano pisanje i uklanja podvlačenje sa svakog linka koji poseduje atribut ``title``. To je odličan način da se vizuelno razlikuju ovi linkovi od drugih na stranici.

Takođe možete kombinovati selektore atributa da biste postigli još veću preciznost. Na primer, ako želite da stilizujete linkove koji imaju oba atributa `href` i `title`, možete to učiniti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

U ovom slučaju, samo linkovi koji sadrže i `href` i `title` atribute biće stilizovani zelenim tekstom. Ovaj nivo kontrole je ono što čini selektore atributa tako moćnim u CSS-u.

Još jedan primer za selektor atributa je poklapati se sa jednom vrednošću unutar liste vrednosti razdvojenih razmakom u atributu.

Evo primera ankornog elementa sa više klasa:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

Da biste ciljali ovaj specifični anker element, možete koristiti sledeći selektor:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

Sintaksa `[attr~=value]` se ovde koristi za ciljanje svih anker elemenata gde atribut klase sadrži reč `"primary"`.

Ako treba da ciljate element gde je vrednost atributa prefiksirana specifičnom vrednošću, onda možete koristiti sintaksu `[attr^=value]`. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

U ovom primeru, selektor `a[href^="https://"]` će ciljati sve elemente ankeri gde vrednost atributa `href` počinje sa `"https://"`.

Da biste ciljali elemente čija vrednost atributa završava sa određenom vrednošću, možete koristiti sintaksu ``[attr$=value]``. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

U ovom primeru, selektor `a[href$=".com"]` će ciljati sve elemente ankeri gde vrednost atributa `href` se završava sa `.com`.

Korišćenjem atributnih selektora ne samo da poboljšava stilizovanje vaše web stranice, već i poboljšava pristupačnost tako što čini interaktivne elemente kao što su linkovi vidljivijim na osnovu njihovih atributa.
