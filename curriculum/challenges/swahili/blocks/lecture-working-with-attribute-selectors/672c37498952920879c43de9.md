# --interactive--

Kada gradite višejezične veb-sajtove ili obrađujete prilagođene podatkovne atribute, često morate stilizovati elemente na osnovu jezika koji sadrže ili specifičnih vrednosti podataka.

Atributi `lang` i `data-lang` se često koriste za ove svrhe, a atributni selektor u CSS-u vam omogućava da efikasno primenite stilove na osnovu ovih atributa.

Atribut `lang` se koristi u HTML-u za određivanje jezika sadržaja unutar elementa. Možda želite da stilizujete elemente drugačije na osnovu jezika na kojem su napisani, posebno na višejezičnom veb-sajtu.

Evo primera kako možete koristiti selektor atributa da ciljate elemente sa specifičnim atributom `lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

Ova CSS pravila primenjuje italik stilizovanje na bilo koji paragraf element gde je atribut `lang` podešen na engleski (`en`). Ovo bi moglo biti korisno za naglašavanje engleskog teksta u dokumentu koji uključuje više jezika.

Slično tome, možete koristiti selektor atributa da ciljate elemente sa atributom `data-lang`.

Prilagođeni atributi podataka, poput `data-lang`, često se koriste za skladištenje dodatnih informacija u elementima, kao što je navođenje jezika korišćenog unutar određenog dela teksta.

Evo kako možete koristiti CSS da ciljate i stilizujete elemente na osnovu atributa `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

U ovom slučaju, svaki `p` element sa atributom `data-lang` podešenim na francuski (`fr`) imaće svoju boju teksta promenjenu u plavu. Ovo vam omogućava da brzo identifikujete i stilizujete sekcije sadržaja na osnovu informacija o jeziku sačuvanih u atributu `data-lang`.

Selektori atributa poput ovih pružaju moćan način za primenu uslovnog stilizovanja na osnovu metapodataka ugrađenih u vaš HTML, čineći vaše veb stranice dinamičnijim i svesnijim kontekstu.
