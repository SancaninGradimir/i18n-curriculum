# --interactive--

Kada radite sa porednim listama u HTML-u, atribut `type` vam omogućava da odredite stil brojanja koji se koristi, kao što su numerički, alfabetski ili rimski brojevi.

Atribut ``type`` se može postaviti na element ``ol`` (lista sa rednim brojevima) za kontrolisanje formata numerisanja:

- `1` za numeričke liste (kao što su 1, 2, 3, ...),

- `A` za velika slova alfabetske liste (kao što su A, B, C, ...),

- `a` za male slova alfabetske liste (kao što su a, b, c, ...),

- `I` za velika rimska broja (kao što su I, II, III, ...),

- `i` za mala rimska brojeva (kao što su i, ii, iii, ...).

Evo kako možete koristiti selektor atributa da odredite listu sa rednim brojevima koja ima specifičan `type` atribut:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

U ovom primeru, CSS pravilo će primeniti ljubičasti tekst i podebljani težinu fonta na bilo koji poredjeni spisak koji koristi velika slova alfabetsko brojanje, što je specificirano sa `type="A"`.

Takođe, možete ciljati numerisane liste koje koriste mala rimska slova izabravanjem onih sa `type="i"`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

Ovo pravilo će promeniti boju teksta u zelenu za bilo koji numerisani spisak koji koristi mala rimska brojeva.

Sposobnost stilizovanja numerisanih lista na osnovu njihovog atributa ``type`` daje vam veću kontrolu nad izgledom lista u vašem dokumentu, omogućavajući prilagođavanje na osnovu toga kako je sadržaj strukturiran.
