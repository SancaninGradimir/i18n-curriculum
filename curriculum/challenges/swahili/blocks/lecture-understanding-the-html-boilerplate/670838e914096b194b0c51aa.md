# --description--

Naučimo o HTML šablonu.

Šta je HTML boilerplate, pitate? To je kao gotov šablon za vaše web stranice. Zamislite ga kao temelj kuće. Boilerplate uključuje osnovnu strukturu i esencijalne elemente koje svaki HTML dokument zahteva. Štedi vam vreme i pomaže da se osigura da su vaše stranice pravilno postavljene. Evo primera:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

Detaljno ćemo pogledati ključne delove ovog šablona. Prvo, tu je deklaracija `DOCTYPE`:

```html
<!DOCTYPE html>
```

Kaže pretraživačima koju verziju HTML-a koristite. Sledeći je tag ``html``:

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

Ovo obuhvata sav vaš sadržaj i može da navede jezik vaše stranice. Unutar taga `html`, pronaći ćete dva glavna dela - `head` i `body`:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

Sekcija `head` sadrži važne informacije iz pozadine:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Metadata vašeg sajta, sadržaj koji je smešten u elementima `meta`, sadrži detalje o stvarima kao što su kodiranje karaktera i kako sajtovi poput Twitera treba da pregledaju (preview) link vaše stranice. Naslov vašeg sajta, koji se nalazi u elementu `title`, određuje tekst koji se pojavljuje u kartici ili prozoru pretraživača. Na kraju, obično ćete povezati eksterne stilove (stylesheets) vaše stranice u sekciji `head` koristeći elemente `link`.

Sekcija `body` je mesto gde ide sav vaš sadržaj:

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

Sada, zašto je šablon važan? Osigurava da su vaše stranice pravilno strukturirane i dobro rade na različitim pregledačima. Korišćenje šablona pomaže vam da izbegnete uobičajene greške i pratite najbolje prakse. To je odlična početna tačka za svaki veb projekat. Zapamtite, možete prilagoditi sopstveni šablon svojim potrebama. Kako steknete iskustvo, možda ćete dodati svoje preferirane elemente ili `meta` tagove. Kako nastavite da unapređujete svoj lični šablon, shvatićete da vam štedi vreme kada započinjete nove projekte.

Sledeći put kada kreirate novi HTML fajl, razmislite o korišćenju šablona. To će vam definitivno dati čvrste osnove na kojima možete graditi.
