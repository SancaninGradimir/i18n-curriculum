# --description--

Naučićemo se o elementu ``link``, i kako ga koristiti za povezivanje na eksterne listove stilova.

Element ``link`` se koristi za povezivanje na eksterne resurse kao što su stilovi i ikone sa sajta. Evo osnovne sintakse za korišćenje elementa ``link`` za eksternu CSS datoteku:

```html
<link rel="stylesheet" href="./styles.css" />
```

Atribut `rel` se koristi za određivanje odnosa između povezanog resursa i HTML dokumenta. U ovoj situaciji, moramo da navedimo da je ovaj povezani resurs `stylesheet`.

Smatra se da je najbolje razdvojiti vaš HTML i CSS u različite fajlove. Programeri će koristiti element ``link`` za svoj eksterni CSS fajl umesto da sve pišu u HTML dokument.

Atribut `href` se koristi za određivanje lokacije URL-a za eksterni resurs.

`dot``, praćen kose crte u primeru, govori računaru da potraži fajl "`styles.css`" u trenutnom folderu ili direktorijumu.

Element ``link`` treba da bude postavljen unutar elementa ``head``, kao što je prikazano u sledećem primeru:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Često ćete videti više elemenata ``link`` unutar profesionalnog koda koji se povezuju na različite stilove, fontove i ikone. Evo primera korišćenja elementa ``link`` za povezivanje na eksterni Google font pod nazivom *Playwright Cuba*:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fontovi su skup besplatnih i otvorenog koda prilagođenih fontova koje možete koristiti u bilo kojem projektu. Možete izabrati koje fontove želite da koristite, a Google će vam obezbediti neophodan HTML i CSS kod. U ovom primeru, vrednost `preconnect` za atribut `rel` govori pretraživaču da kreira rani konekciono na vrednost navedenu u atributu `href`. Ovo se radi kako bi se ubrzalo vreme učitavanja ovih eksternih resursa.

Još jedan uobičajen slučaj upotrebe za element ``link`` je povezivanje na ikone. Evo primera povezivanja na favikon:

```html
<link rel="icon" href="favicon.ico" />
```

Favikon, koji je skraćenica od "favorite icon" (ikona omiljenog), mala ikonica koja se obično prikazuje u karti pretraživača pored naslova sajta. Mnogi sajtovi će koristiti favikon da prikažu svoju brendsku ikonicu.
