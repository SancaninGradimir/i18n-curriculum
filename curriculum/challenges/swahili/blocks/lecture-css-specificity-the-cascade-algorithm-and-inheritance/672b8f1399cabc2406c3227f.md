# --interactive--

Ključna reč ``!important`` u CSS-u koristi se za davanje najvišeg prioriteta pravilu stila, omogućavajući mu da nadjačava bilo koje druge deklaracije za svojstvo.

Kada se koristi, on prisiljava pretraživač da primeni navedeni stil, bez obzira na specifičnost drugih selektora.

Recimo da imate HTML paragraf element sa ugrađenim stilovima kao što je ovo:

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

U ovom primeru, paragraf element ima pozadinsku boju postavljenu na `lightblue` i tekstualnu boju postavljenu na `black`.

CSS datoteka primjenjuje sljedeće stilove na paragraf element:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

Pošto inline stilovi imaju viši prioritet od selektora klase, ID-a i tipa, crna pozadinska boja i bela boja teksta neće biti primenjene na taj paragraf element.

Da biste nadjačali te ugrađene stilove, možete koristiti ključnu reč `!important` na ovaj način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

Ključna reč ``!important`` se koristi nakon CSS vrednosti i pre tačke-zadesa.

Sada će paragrafski element imati primenjene te crno-bele boje jer se ugrađeni stilovi nadjačavaju korišćenjem ključne reči ``!important``.

Ključna reč ``!important`` u CSS-u koristi se za davanje stilskoj pravilnici najvišeg prioritet, efektivno nadjačavajući druge deklaracije, uključujući one sa većom specifičnošću i inline stilove.

Međutim, ključna reč `!important` ne menja specifičnost samog CSS selektora. Ono jednostavno osigurava da se pravilo sa `!important` primeni, čak i ako postoje druga protivrečna pravila sa većom specifičnošću.

Još jedan odgovarajući slučaj upotrebe za ključnu reč `!important` je nadjačavanje stilova iz biblioteka ili okvira trećih strana kada nemate kontrolu nad originalnim CSS-om.

Međutim, prekomerno korišćenje ključne reči `!important` može dovesti do poteškoća u održavanju i debagiranju vašeg CSS-a, jer narušava prirodno kaskadiranje stilova i može dovesti do neželjenih posledica.

Dakle, najbolje je da se ključna reč `!important` koristi štedljivo.
