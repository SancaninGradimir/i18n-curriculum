# --interactive--

Specifičnost CSS-a je fundamentalni koncept koji određuje koji stilovi se primenjuju na element kada više pravila može da se primeni.

Razumevanje specifičnosti pomaže programerima da reše konflikte između različitih CSS pravila i osigurava da su željeni stilovi dosledno primenjeni.

Specifičnost CSS-a se računa na osnovu tipa korišćenih selektora.

Najviša specifičnost pripada ugrađenim stilovima (inline styles), koji se primenjuju direktno na element putem atributa ``style``.

U ovom primeru, prvi pasus će biti crven dok će drugi `p` elementi biti plavi. Ovo je zato što ugrađeni stilovi imaju veću specifičnost od tip selektora kao što je `p` selektor prikazan u `styles.css` fajlu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p style="color: red;">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p {
  color: blue;
}
```

:::

Nakon ovoga, selektori ID-a imaju visok nivo specifičnosti.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Sledeći, selektori klasa, selektori atributa i pseudokrase stupaju u igru.

Primeri uključuju selektore klasa kao što su ``.container`` i ``.button``, selektori atributa poput ``[type="text"]``, i pseudokrase kao što su ``:hover``. Ovo ima umeren nivo specifičnosti.

**NAPOMENA**: U budućim lekcijama ćete saznati više o pseudo-klasama.

U ovom primeru, prvi pasus će biti crven jer selektor `id` ima višu specifičnost od klasa i tip selektora. Elementi `.example-para` će imati zelenu boju dok preostali pasus elementi će imati plavu boju.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p class="example-para">Other paragraph</p>
<p class="example-para">Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

.example-para {
  color: green;
}

p {
  color: blue;
}
```

:::

Selektori tipa, kao što su `div` i `h1`, i pseudo-elementi poput `::before` i `::after`, imaju nižu specifičnost u poređenju sa prethodnim grupama.

**NAPOMENA**: Naučićete više o pseudoelementima u budućim časovima.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Konačno, univerzalni selektor, predstavljen zvezdicom `*`, ima najnižu specifičnost od svih.

Evo primera postavljanja boje za sve elemente na crvenu koristeći selektor `*`. Međutim, nećete videti nijedan crveni element jer postoje selektori tipa ID i tip koji nadjačavaju te stilove, što ističe nisku specifičnost selektora `*`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
* {
  color: red;
}

#para {
  color: green; 
}

p {
  color: blue;
}
```

:::

Vrednosti specifičnosti se izračunavaju kao četvorostrukost vrednost `(a, b, c, d)`:

- `a`: Inline stilovi (`1` ili `0`).
- `b`: Broj selektora za ID.
- `c`: Broj klasnih selektora, atributnih selektora i pseudoklasa.
- `d`: Broj tipskih selektora, pseudoelementa i univerzalnih selektora.

Svaki deo vrednosti specifičnosti nosi različitu težinu:

- **Stilovi u liniji** (`a`) imaju najveću težinu, doprinoseći vrednost od `1` prvom delu specifičnosti.
    
- **Selektori ID-a** (`b`) nose sledeću najveću težinu, sa svakim ID-om koji doprinosi `1` drugom delu specifičnosti.
    
- **Selektori klasa, selektori atributa i pseudokrase** (`c`) nose umerenu težinu, sa svakim koji doprinosi `1` trećem delu specifičnosti.
    
- **Selektori tipa i pseudoelementi** (`d`) imaju najnižu težinu, sa svakim koji doprinosi `1` četvrtom delu specifičnosti.
    
- **Univerzalni selektor** (`*`): Univerzalni selektor doprinosi `0` računanju specifičnosti i ne utiče na specifičnost. Njegovo uključivanje u selektor ne menja vrednost specifičnosti.

Inline CSS ima najvišu specifičnost jer se direktno primenjuje na element. Nadjačava svaki interni ili eksterni CSS. Vrednost specifičnosti za inline stilove je `(1, 0, 0, 0)`.

Evo još jednog primera ugrađenog stila:

:::interactive_editor

```html
<p style="color: red;">This text is red.</p>
```

:::

Interni CSS je definisan unutar elementa `style` u sekciji `head` HTML dokumenta. Ima nižu specifičnost od inline stilova. I interni i eksterni CSS dele isti nivo specifičnosti — specifičnost je određena korišćenim selektorima, a ne mestom gde je CSS definisan. Kada dve pravila dele jednaku specifičnost, pobeđuje ono koje se pojavi kasnije u dokumentu. Dakle, ako je eksterni fajl za stilove povezan nakon internog bloka `style`, eksterni pravila imaju prednost. Ako je povezan pre, interni pravila pobeđuju.

Vrednost specifičnosti za interne stilove određena je korišćenim selektorima. Na primer, ID selektor unutar internog CSS-a ima vrednost specifičnosti od `(0, 1, 0, 0)`.

:::interactive_editor

```html
<head>
  <style>
    #text {
      color: blue;
    }
  </style>
</head>
<body>
  <div id="text">This text is blue.</div>
</body>
```

:::

U ovom primeru, tekst će biti plav osim ako se ne primeni ugrađeni stil ili specifičniji selektor.

Eksterni CSS je povezan putem elementa `link` u sekciji `head` i napisan je u zasebnim fajlovima `.css`. Kao interni CSS, specifičnost eksternih stilova određena je korišćenim selektorima. Eksterni CSS pruža najbolju održivost za veće projekte.

Vrednost specifičnosti za eksterne stilove je takođe određena korišćenim selektorima. Na primer, selektor klase unutar eksternog CSS-a ima vrednost specifičnosti od `(0, 0, 1, 0)`.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="text">This text's color is defined in an external CSS file.</div>
</body>
```

```css
.text {
  color: purple;
}
```

:::

U ovom primeru, boja teksta je definisana u fajlu `styles.css` i biće primenjena osim ako nije nadjačana specifičnijim selektorom ili inline stilom.
