# --interactive--

ID selektori su među najmoćnijim selektorima u CSS-u, omogućavajući programerima da primene stilove na specifične elemente sa jedinstvenim identifikatorima.

Ovo ih čini veoma efikasnim za ciljanje pojedinačnih elemenata koji zahtevaju jedinstveno stilizovanje.

Selektori za ID se definišu znakom heš (#) (`#`) praćenim imenom ID-a. Trebalo bi da budu jedinstveni unutar HTML dokumenta, što znači da se dva elementa ne bi trebalo da dele isti ID.

TEKST:

Evo primera korišćenjem selektora ID-a:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

U ovom primeru, element sa ID-om `unique` će imati svoju boju teksta postavljenu na `purple`.

ID selektori imaju veoma visoku specifičnost, višu od tipovnih selektora i klasnih selektora, ali nižu od ugrađenih stilova. Vrednost specifičnosti za ID selektor je `(0, 1, 0, 0)`.

Ovo znači da selektori za ID-jeve mogu da nadjačaju selektore klasa i selektore tipova, ali mogu biti nadjačani inline stilovima.
