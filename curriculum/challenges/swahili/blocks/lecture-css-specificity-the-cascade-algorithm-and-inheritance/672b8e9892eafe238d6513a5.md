# --interactive--

Selektori klasa su ključni deo CSS-a, omogućavajući programerima da ciljaju više elemenata sa istim atributom klase i primene dosledno stilizovanje.

Ovo ih čini veoma svestranim i efikasnim za primenu stilova širom veb stranice.

Selektori klasa su definisani tačkom (`.`) praćenom imenom klase. Mogu se primeniti na bilo koji element u HTML dokumentu.

Evo primera korišćenjem selektora klase:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

U ovom primeru, bilo koji element sa klasom `highlight` će imati boju teksta postavljenu na `green`.

Selektori klasa imaju veću specifičnost od selektora tipova, ali manju od selektora ID-a i ugrađenih stilova.

Vrednost specifičnosti za selektor klase je `(0, 0, 1, 0)`. To znači da selektori klase mogu nadjačati tip selektore, ali ih može nadjačati selektori ID-a i ugrađeni stilovi.

Selektori klasa se mogu kombinovati sa drugim selektorima kako bi se kreirale specifičnije pravila.

Evo primera koji kombinuje selektor tipa paragrafa sa selektorom klase:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

Ovo pravilo važi samo za `p` elemente koji takođe imaju `bold-text` klasu, čime se njihov tekst čini podebljan.
