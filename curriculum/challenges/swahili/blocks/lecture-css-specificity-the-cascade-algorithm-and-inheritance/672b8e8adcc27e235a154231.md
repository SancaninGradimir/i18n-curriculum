# --interactive--

Selektori tipa, poznati i kao elementi selektori, ciljaju elemente na osnovu njihovog imena taga.

Ovi selektori su fundamentalni u CSS-u i omogućavaju vam da primenite stilove na sve instance određenog HTML elementa.

Tip selektori su jednostavni za korišćenje i pišu se jednostavno kao naziv taga elementa koji želite stilizovati.

Ovo je primer tip selektora koji cilja sve paragraf elemente na stranici:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

U ovom primeru, svi `p` elementi će imati boju teksta postavljenu na `blue`.

Selektori tipova imaju relativno nisku specifičnost u poređenju sa drugim selektorima. Vrednost specifičnosti za selektor tipa je `(0, 0, 0, 1)`.

To znači da tip selektori će biti nadjačani od strane klasa selektora, ID selektora i ugrađenih stilova, ali mogu i dalje primeniti stilove osim ako te pravila sa većom specifičnošću ne postoje.

Pogledajmo primer gde klasni selektori će nadjačati stilove tipovnog selektora.

Evo primera sa dva paragrafa elementa:

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

Oba elementa paragrafa imaju klasu nazvanu `para`.

Unutar CSS fajla, tip selektor cilja pasuse, a klasa selektor cilja elemente sa `para` klasom.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

Svi pasusi na stranici sa klasom `para` će imati boju teksta postavljenu na `red` umesto `blue` jer selektori klasa imaju veću specifičnost od selektora tipova.
