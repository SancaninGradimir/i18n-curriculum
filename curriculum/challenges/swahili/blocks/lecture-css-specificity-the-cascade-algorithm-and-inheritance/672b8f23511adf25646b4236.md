# --interactive--

Cascade Algoritam je proces koji pretraživač koristi da odluči koje CSS pravila primeniti kada postoji više stilova koji ciljaju isti element. Osigurava da se koriste najprikladniji stilovi, na osnovu skupa dobro definisanih pravila.

Proces počinje sa **relevantnošću**. Pregledač prvo filtrira sve CSS pravila kako bi pronašao ona koja zapravo važe za element u pitanju. To uključuje podudaranje selektora i razmatranje upita medija koji možda važe.

Media upit (media query) je CSS tehnika koja se koristi za primenu stilova na osnovu karakteristika uređaja ili vidokruga, kao što su njegova širina, visina ili orijentacija.

Zatim, algoritam uzima u obzir **poreklo i važnost**. CSS može doći iz različitih izvora: podrazumevani stilovi pregledača (user-agent), stilovi postavljeni od strane korisnika, i stilovi napisani od strane autora (vas).

Nakon razmatranja porekla, algoritam zatim procenjuje važnost svakog pravila, dajući prioritet pravilima označenim sa `!important`, koja nadjačavaju druga pravila bez obzira na njihovo poreklo.

Nakon filtriranja po poreklu i važnosti, algoritam gleda na **specifičnost**. Kada se dva pravila iz istog porekla i nivoa važnosti primene, primenjeno će biti pravilo sa većom specifičnošću.

Specifičnost je mera koliko je selektor ciljan, pri čemu specifičniji selektori imaju prioritet nad opštijima.

Konačno, ako je sve ostalo jednako, u igru ulazi **redos pojavljivanja**. Kada dva pravila imaju istu specifičnost, primenjeno će biti ono koje se pojavi poslednje u CSS-u.

Ovo je razlog zašto redosled kojim pišete svoje stilove ponekad može uticati na rezultat.

Pogledajmo primer.

U HTML fajlu, postoji jedan paragraf element. Zatim unutar CSS-a, imamo dva pravila, svako ciljajući paragraf element.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>example paragraph</p>
```

```css
p {
  color: blue;
}

p {
  color: green; 
}
```

:::

Prvo pravilo postavlja sve pasus elemente na tekstualnu boju plavu, dok drugo pravilo postavlja sve pasus elemente na tekstualnu boju zelenu.

Pa, koja boja će biti primenjena? Boja zelena će biti primenjena na paragraf elemente.

Uzimanjem u obzir relevantnost, poreklo i važnost, specifičnost i red pojavljivanja, Kaskadni algoritam osigurava da vaš CSS se ponaša predvidivo, omogućavajući vam da dizajnirate složenije i nijansiranije veb stranice.
