# --interactive--

Univerzalni selektor (`*`) je specijalni tip CSS selektora koji odgovara bilo kom elementu u dokumentu.

Često se koristi za primenu stila na sve elemente na stranici, što može biti korisno za resetovanje ili normalizovanje stilova u različitim pregledačima.

Univerzalni selektor se može koristiti za selektovanje svih elemenata unutar specifičnog konteksta ili globalno širom celog dokumenta.

Evo primera korišćenja univerzalnog selektora za postavljanje `margin` i `padding` za ceo HTML dokument:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

U ovom kod primeru, selektor `*` resetuje margine i padding svih elemenata na nulu, što je uobičajena tehnika korišćena u CSS resetovima.

Univerzalni selektor ima najnižu vrednost specifičnosti od bilo kog selektora. On doprinosi sa 0 svim delovima vrednosti specifičnosti `(0, 0, 0, 0)`.

Ovo znači da bilo koji drugi selektor, uključujući tipove selektore, selektore klasa, selektore ID-ja i ugrađene stilove, će nadjačati stilove postavljene univerzalnim selektorom.

Pogledajmo sledeći primer HTML i CSS-a:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

Univerzalni `*` selektor postavlja boju teksta na `blue` za sve elemente.

Međutim, tip selektor `p` nadjačava ovo postavljanjem boje teksta na `red` posebno za elemente `p`.

Ako element ima klasu ``highlight``, selektor klase ima prednost, menjajući boju teksta na ``green``.

Konačno, ID selektor `#unique`, koji ima najvišu specifičnost, nadjačati će sve ostale i postaviti boju teksta na `purple`.
