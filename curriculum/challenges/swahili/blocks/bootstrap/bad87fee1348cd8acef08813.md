# --description--

Bootstrap dolazi sa nekoliko unapred definisanih boja za dugmad. Klasa `btn-info` koristi se da skrene paznju na opcione akcije koje korisnik moze da izabere.

Napravi novo Bootstrap block dugme ispod `Like` dugmeta sa tekstom `Info`, i dodaj mu Bootstrap klasu `btn-info`.

Zapamti da ovoj dugmadi i dalje trebaju klase `btn` i `btn-block`.

# --hints--

Treba da napravis novi `button` element sa tekstom `Info`.

```js
const infoButton = document.querySelectorAll('button')?.[1]; 
assert.match(infoButton.textContent,/info/gi);
```

Oba Bootstrap dugmeta treba da imaju klase `btn` i `btn-block`.

```js
assert.lengthOf(document.querySelectorAll('button.btn-block.btn'),2);
```

Novo dugme treba da ima klasu `btn-info`.

```js
const infoButton = [...document.querySelectorAll('button')].at(1);
assert.isTrue(infoButton?.classList?.contains('btn-info'));
```

Svi `button` elementi treba da imaju zatvarajuce tagove.

```js
assert.match(code,/<\/button>/g);
assert.match(code,/<button/g);

assert.equal(code.match(/<\/button>/g)?.length,code.match(/<button/g)?.length);
```
