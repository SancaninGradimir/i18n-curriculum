# --interactive--

Događaj `change` je poseban događaj koji se aktivira kada korisnik promeni vrednost određenih elemenata unosa. Konkretnije:

- Kada je prokupljnik čekiran ili nečekiran.

- Kada je odabrano radio dugme.

- Kada korisnik napravi izbor iz nečega poput birača datuma ili padajućeg menija.

- Kada unos izgubi fokus (korisnik prebacuje fokus na sledevo polje ili klikne van forme) nakon što je korisnik promenio vrednost.

- Kada korisnik drugačije potvrdi vrednost, na primer pritiskom tastera Enter nakon kucanja nekog teksta.

Napomena da se događaj ``change`` NE pokreće kada korisnik unese tekst u polje za unos. Događaj ``change`` će se pokrenuti samo nakon što fokus prebace na drugi element.

Evo primera korišćenja događaja `change` sa padajućim menijumom:

:::interactive_editor

```html
<select id="select-menu">
  <option value="option1">Option 1</option>
  <option value="option2">Option 2</option>
  <option value="option3">Option 3</option>
</select>
<script src="index.js"></script>
```

```js
const selectMenu = document.getElementById("select-menu");
selectMenu.addEventListener("change", (event) => {
  console.log(`You selected: ${event.target.value}`);
});
```

:::

Svaki put kada korisnik izabere različitu opciju sa padajućeg menija, događaj `change` će se aktivirati, a odabrana vrednost će biti logovana u konzolu.

Događaj `change` i dalje generiše objekat `Event`, ali, za razliku od većine drugih događaja, ne generiše prilagođenu implementaciju – jedine svojstva i metode kojima ćete imati pristup su ona na osnovnom objektu `Event`.

Ovo se razlikuje od `input` događaja, koji generiše poseban `InputEvent` objekat. Događaj promene takođe se razlikuje na nekoliko načina. Događaj `input` će se aktivirati kada korisnik unese sadržaj u polje, na primer.

Ove razlike su važne da se zapamte, jer vas može zbuniti vremensko određivanje (timing) ovih događaja.
