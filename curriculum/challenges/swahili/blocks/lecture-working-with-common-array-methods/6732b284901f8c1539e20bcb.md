# --interactive--

Metoda ``splice()`` u JavaScriptu je moćan način za modifikovanje nizova. Omogućava vam da dodate ili uklonite elemente sa bilo koje pozicije u nizu, uključujući i sredinu. Vraćena vrednost za metodu ``splice()`` biće niz predmeta uklonjenih iz niza. Ako ništa nije uklonjeno, vraćaće se prazan niz.

Važno je napomenuti da će ova metoda mutirati originalni niz, modifikujući ga na mestu umesto kreiranja novog niza. Ovo je nešto o čemu treba biti svestan kada radite sa `splice()`. Evo osnovne sintakse:

```js
array.splice(startIndex, itemsToRemove, item1, item2)
```

`startIndex` navodi indeks od kojeg treba početi sa modifikovanjem niza, dok je `itemsToRemove` opcioni parametar koji ukazuje koliko elemenata treba ukloniti. Ako se `itemsToRemove` izostavi, `splice()` će ukloniti sve elemente od `startIndex` do kraja niza. Nakonvi parametri (`item1`, `item2` i tako dalje) su elementi koji se dodaju nizu, počevši od početnog indeksa.

Počajmo sa primerom uklanjanja elemenata iz sredine niza:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

:::

U ovom primeru, `splice(2, 2)` počinje na indeksu `2` i uklanja `2` elementa. Modifikovani niz će sada sastojati samo od `apple`, `banana`, i `kiwi`. Sada, pogledajmo kako dodati elemente u sredinu niza:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

Ovde, `splice(1, 0, "yellow", "purple")` počinje na indeksu `1`, uklanja `0` elementa, i dodaje `yellow` i `purple`. Drugi parametar (`0`) znači da se nijedan element ne uklanja pre ubacivanja. Takođe možete koristiti `splice()` za istovremeno uklanjanje i dodavanje elemenata:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

:::

U ovom slučaju, `splice(1, 2, 6, 7, 8)` počinje na indeksu `1`, uklanja `2` elemente (`2` i `3`), i ubacuje `6`, `7`, i `8`. Ako želite da originalni niz ostane nepromenjen, trebalo bi da napravite kopiju pre korišćenja `splice()`:

:::interactive_editor

```js
let original = [1, 2, 3, 4, 5];
let copy = [...original];
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5]
console.log(copy);     // [1, 2, 6, 4, 5]
```

:::

U ovom primeru, da bismo kreirali kopiju niza `original` bez menjanja ga, koristimo spread operator (`...`). Spread operator će kreirati plitku kopiju elemenata niza `original` u novi niz. Vi ćete više o ovome naučiti u budućim lekcijama.

Kada koristimo `copy.splice(2, 1, 6)`, ono modifikuje niz kopija uklanjanjem elementa na indeksu `2` (koji je `3`) i ubacivanjem novog elementa `6` na tu poziciju.

Jedna uobičajena upotreba za `splice()` je uklanjanje jednog elementa iz niza kada znate njegov indeks:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");
if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}

console.log(fruits); // ["apple", "banana", "mango"]
```

:::

U ovom primeru, prvo koristimo metod `indexOf()` da pronađemo indeks elementa `orange` u nizu `fruits`. Metod `indexOf()` vraća indeks prvog pojavljivanja datih elementa ili `-1` ako element nije pronađen u nizu.

Zatim upoređujemo `indexToRemove` sa `-1` kako bismo osigurali da element postoji u nizu pre nego što pokušamo da ga uklonimo. Ako `indexToRemove` nije jednako `-1` (što znači da je element pronađen), koristimo `splice()` za uklanjanje jednog elementa počevši od pozicije `indexToRemove`.

Takođe možete koristiti ``splice()`` za brisanje niza uklanjanjem svih elemenata:

:::interactive_editor

```js
let array = [1, 2, 3, 4, 5];
array.splice(0);

console.log(array); // []
```

:::

Iako je `splice()` moćan, vredno je napomenuti da za veoma velike nizove može biti manje efikasan od drugih metoda, posebno kada se modifikuje početak niza. To je zato što `splice()` možda mora da pomeri sve naknadne elemente. U takvim slučajevima, ako dodajete ili uklanjate elemente samo na kraju niza, metode poput `push()`, `pop()`, `unshift()` i `shift()` mogu biti prikladnije.

Zaključno, metoda `splice()` je svestran način za modifikovanje nizova u JavaScriptu. Omogućava preciznu kontrolu nad dodavanjem i uklanjanjem elemenata iz bilo koje pozicije u nizu. Razumevanje kako efikasno koristiti `splice()` može značajno poboljšati vašu sposobnost manipulisanja nizovima u vašem JavaScript kodu.
