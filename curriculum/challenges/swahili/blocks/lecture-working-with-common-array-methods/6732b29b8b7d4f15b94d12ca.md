# --interactive--

Plitka kopija niza je novi niz koji ima iste elemente kao i originalni. Ako niz sadrži samo primitivne vrednosti poput brojeva ili stringova, novi niz je potpuno odvojen. Ali ako niz sadrži druge nizove unutar sebe, i originalni i kopija imaju reference na iste interne nizove. To znači da ako promenite nešto unutar zajedničkog internog niza, videćete tu promenu u oba niza.

Plitke kopije su korisne kada je potrebno modifikovati strukturu na najvišem nivou, kao što je dodavanje, uklanjanje ili ponovno uređenje elemenata, bez modifikovanja originalnog niza ili unutrašnjeg niza.

Postoji nekoliko metoda za kreiranje plitkih kopija nizova, i istražićemo neke od najčešćih: ``concat()``, ``slice()`` i operater širenja (spread operator).

Počinjemo sa metodom ``concat()``. Ova metoda kreira novi niz spajanjem dva ili više nizova. Kada se koristi sa jednim nizom, efektivno stvara plitku kopiju. Evo primera:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [].concat(originalArray);

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

U ovom primeru, koristimo metodu `concat()` da bi konkatenidirali prazan niz sa `originalArray`. Ovo će kreirati novi niz koji je plitka kopija od `originalArray`.

`copyArray` sadrži iste elemente kao `originalArray`, ali je to drugi niz objekat, zbog čega provera stroge jednakosti (`===`) vraća `false`.

Drugi metod za kreiranje plitke kopije je `slice()` metoda. Kada se pozove bez argumenata, `slice()` vraća plitku kopiju celog niza. Evo kako to radi:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = originalArray.slice();

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

U ovom slučaju, `originalArray.slice()` kreira novi niz koji je plitka kopija od `originalArray`. Opet, `copyArray` sadrži iste elemente, ali je drugačiji objekat niza.

Spread operater (`...`), uveden u ES6, pruža još jedan koncizan način za kreiranje plitkih kopija nizova. Evo primera:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

Operater raspršivanja (`...`) raspršuje elemente `originalArray` u novi niz, efektivno stvarajući plitku kopiju. Važno je napomenuti da sve ove metode kreiraju nove niz objekte, što znači da možete modifikovati kopiju bez uticaja na originalni niz. Na primer:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

copyArray.push(4);
console.log(originalArray); // [1, 2, 3]
console.log(copyArray);     // [1, 2, 3, 4]
```

:::

U ovom primeru, dodavanje elementa u `copyArray` ne utiče na `originalArray`.

Ukratko, plitke kopije nizova mogu se lako kreirati korišćenjem metoda kao što su `concat()`, `slice()`, ili operater raspršivanja. Ove metode su korisne za kreiranje novih nizova koji se mogu manipulisati nezavisno od originalnog niza.
