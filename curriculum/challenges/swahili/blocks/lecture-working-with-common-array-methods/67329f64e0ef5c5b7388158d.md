# --interactive--

U JavaScriptu, metoda `indexOf()` je korisna za pronalaženje prvog indeksa specifičnog elementa unutar niza. Ako se element ne može pronaći, onda će vratiti `-1`. Evo osnovne sintakse:

```js
array.indexOf(element, fromIndex)
```

`element` predstavlja vrednost koju želite da pretražite unutar niza, a parametar `fromIndex` je pozicija od koje bi pretraga trebalo da počne. Parametar `fromIndex` je opcionalan. Ako nije dostavljen `fromIndex`, pretraga počinje od početka niza. Pogledajmo primer:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

U ovom primeru, imamo niz `fruits` , koji sadrži različita imena voća. Koristimo metodu `indexOf()` da pronađemo indeks niza `banana` unutar niza `fruits`. Pošto `banana` je prisutan na indeksu `1`, metoda vraća `1`, što se čuva u promenljivoj `index` i loguje se u konzolu.

Ako element koji tražite nije pronađen u nizu, `indexOf()` vraća `-1`. Primer:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

Ovde tražimo string `grape` u nizu voća koristeći `indexOf()`. Pošto `grape` nije prisutan u nizu, metoda vraća `-1`, što se skladišti u varijabli `index` i loguje na konzolu.

Ako želite da počnete da tražite stavku nakon određenog broja indeksa, onda možete proslediti drugi argument kao u ovom primeru:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

U ovom primeru, pretraga ne počinje od početka niza, već počinje od broja indeksa `3` koji je `yellow` i dobija izlaz od `4`.
