# --instructions--

Napravite funkciju nazvanu `calculate()` u `mean_var_std.py` koja koristi Numpy da izračuna srednju vrednost, varijansu, standardnu devijaciju, maksimum, minimum i sumu redova, kolona i elemenata u matrici 3x3.

Argument funkcije treba da bude lista od 9 brojeva. Funkcija bi trebalo da konvertuje tu listu u Numpy niz podataka dimenzija 3x3, a zatim da vrati rečnik koji sadrži srednju vrednost, varijansu, standardnu devijaciju, maksimum, minimum i sumu za oba poprečna preseka i za rotiranu matricu.

Vraćeni rečnik bi trebalo da prati ovaj format:

```py
{
  'mean': [axis1, axis2, flattened],
  'variance': [axis1, axis2, flattened],
  'standard deviation': [axis1, axis2, flattened],
  'max': [axis1, axis2, flattened],
  'min': [axis1, axis2, flattened],
  'sum': [axis1, axis2, flattened]
}
```

Ako se funkcija pozove sa listom elemenata manjom od 9, treba da izbaci `ValueError` grešku sa porukom: "List must contain nine numbers." Vrednosti u vraćenom rečniku bi trebalo da budu liste, a ne Numpy nizovi.

Na primer, `calculate([0,1,2,3,4,5,6,7,8])` bi trebalo da vrati:

```py
{
  'mean': [[3.0, 4.0, 5.0], [1.0, 4.0, 7.0], 4.0],
  'variance': [[6.0, 6.0, 6.0], [0.6666666666666666, 0.6666666666666666, 0.6666666666666666], [6.666666666666667]],
  'standard deviation': [[2.449489742783178, 2.449489742783178, 2.449489742783178], [0.816496580927726, 0.816496580927726, 0.816496580927726], 2.581988897471611],
  'max': [[6, 7, 8], [2, 5, 8], 8],
  'min': [[0, 1, 2], [0, 3, 6], 0],
  'sum': [[9, 12, 15], [3, 12, 21], 36]
}
```
