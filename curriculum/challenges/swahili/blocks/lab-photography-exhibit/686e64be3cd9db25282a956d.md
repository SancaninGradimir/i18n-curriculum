# --description--

U ovom laboratorijumu, kreiraćete raspored izložbe fotografija koristeći Tailwind CSS utilitarne klase.

Za slike, možete koristiti sledeće URL adrese ako želite:

- https://cdn.freecodecamp.org/curriculum/labs/colosseo.jpg
- https://cdn.freecodecamp.org/curriculum/labs/alps.jpg
- https://cdn.freecodecamp.org/curriculum/labs/sea.jpg

**Cilj:** Ispunite priče korisnika ispod i učinite da svi testovi prođu kako biste završili laboratorijum.

**Priče korisnika:**

1. Trebalo bi da imate element sa klasom nazvanom `main-container`.
2. Vaš element `.main-container` takođe treba da ima sledeće Tailwind CSS utilitarne klase:
   - ispravnu utilitarnu klasu za kreiranje CSS Grid kontejnera.
   - utilitarnu klasu za definisanje broja kolona unutar mreže. Trebalo bi da koristite utilitarnu klasu koja koristi fiksni broj kolona. Primer: `grid-cols-<number>`.
   - utilitarnu klasu za podešavanje razmaka redova i kolona za elemente mreže.
3. Unutar vašeg elementa `.main-container`, trebalo bi da ima najmanje tri elementa, svaki sa klasom nazvanom `card`.
4. Svaki vaš element `.card` takođe treba da ima sledeće Tailwind CSS utilitarne klase:
   - utilitarnu klasu za podešavanje zaobljenog ivica po izboru.
   - utilitarnu klasu za podešavanje paddinga po izboru. Trebalo bi da koristite ovaj format `p-<number>`.
5. Unutar svakog vašeg elementa `.card`, trebalo bi da imate sledeće elemente:
   - element slike sa atributima `src` i `alt`.
   - element sa klasom nazvanom `subheading`.
   - element sa klasom nazvanom `description`.
6. Svaki vaš element slike treba da ima utilitarnu klasu za podešavanje zaobljenog ivica po izboru.
7. Svaki vaš element `.subheading` treba da koristi utilitarne klase za podešavanje težine i veličine fonta po izboru.
8. Svaki vaš element `.description` treba da koristi utilitarnu klasu za podešavanje veličine fonta po izboru.
