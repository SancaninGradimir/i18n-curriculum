# --interactive--

Svojstva flexa su svojstva koja se mogu primeniti na fleks kontejnerima radi određivanja raspodele elemenata dece. Pokrićemo neke od najčešće korišćenih: `flex-wrap`, `justify-content`, i `align-items`.

Počinjemo sa `flex-wrap`. Ova svojstvo određuje kako se fleksibilni elementi pakuju unutar fleks kontejnera da bi odgovarali dostupnom prostoru. `flex-wrap` može imati tri moguće vrednosti: `nowrap`, `wrap`, i `wrap-reverse`. `nowrap` je podrazumevana vrednost—fleksibilni elementi se neće pakovati na novi red, čak ni ako njihova širina premaši širinu kontejnera.

U kodu ispod, imamo tri `div` elementa. Fokusiraćemo se na `width`. Kontejner `main`, obrubljen crvenom bojom ima `width` od `200px`, dok zajedno njegova tri elementa deteta `div` imaju `width` od `240px` (`80px` svaki):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


Širina elemenata ``div`` prelazi širinu njihovog kontejnera, ali podrazumevano će biti smanjeni da odgovaraju dostupnom prostoru. Ako želite da ih umotate kada pređu širinu svog kontejnera, možete postaviti ``flex-wrap: wrap`` na flex kontejner:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Elementi `div` će biti ponovo raspoređeni u redove kada premaše širinu svog kontejnera. Možete obaviti umotavanje fleksibilnih elemenata u obrnutom redosledu pomoću `flex-wrap: wrap-reverse`. Svojstvo `flex-flow` je skraćeno svojstvo za `flex-direction` i `flex-wrap`. U ovom primeru, postavili smo `flex-direction` na `column` i `flex-wrap` na `wrap-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Odlično. Sada ćemo pričati o `justify-content`. `justify-content` poravnava elemente dece duž glavne ose fleks kontejnera. Ako dodelite vrednost `flex-start` na `justify-content`, fleks elementi će biti poravnati sa početak glavne ose. Ovo može biti horizontalno ili vertikalno:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

```css
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

```css
}

#third-div {
```

`justify-content: space-around` ravnomerno raspoređuje fleksibilne elemente duž glavne ose, dodajući razmak pre prvog i posle poslednjeg elementa. Ovaj dodatni razmak je pola prostora između svake pare susednih elemenata. Ako postoji samo jedan element za raspoređivanje, on će biti centriran.

:::interactive_editor

```html

<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>

```

```css
main {

  justify-content: space-around;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;

#third-div {
  background-color: #4da3b2;
```

Odlično. Sada znate kako da rasporedite fleksibilne elemente duž glavne ose. Ali možda želite da ih rasporedite i duž poprečne ose. Zapamtite da je poprečna osa perpendikularna na glavnu osu. Ovo možete uraditi sa svojstvom `align-items`. Da biste centrirali elemente duž poprečne ose, samo morate dodati `align-items: center` na fleks kontejner:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

U ovom primeru, fleksibilni elementi su centrirani duž poprečne ose, koja je podrazumevano vertikalna. Ako je poprečna osa horizontalna, oni će umesto toga biti centrirani horizontalno. Za razliku od toga, `align-items: flex-start` poravnava elemente na početak poprečne ose:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Suprotno je `align-items: flex-end`. Ovo će poravnati fleks elemente na kraj poprečne ose, vertikalno ili horizontalno.

:::interactive_editor

```html

  <div id="third-div"></div>
```

```css

#third-div {
  background-color: #4da3b2;

```

:::

Da biste protegnuli fleksibilne elemente duž poprečne ose, možete koristiti ``align-items: stretch``. Ovo takođe radi sa elementima koji su automatski definisani, kao što su oni bez definisanih ``width`` ili ``height``, ili sa samo minimalnim ``width`` ili ``height``. Fleksibilni elementi će se protezati kako bi popunili kontejner u smeru poprečne ose.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

I na kraju, možete koristiti svojstvo `align-self` da dodelite drugačije poravnanje na poprečnu osu pojedinačnom fleks elementu. Na primer, možete ga istegnuti sa `align-self: stretch`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css

  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: stretch; 
  background-color: lightcoral;
}
```
