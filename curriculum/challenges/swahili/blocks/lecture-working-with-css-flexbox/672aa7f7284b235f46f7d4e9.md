# --interactive--

CSS flexbox je jednodimenzionalni model rasporeda koji vam omogućava da rasporedite elemente u redovima i kolonama unutar kontejnera. Takođe možete kontrolisati njihov redosled i orijentaciju. Web programeri ga koriste za kreiranje responzivnih veb sajtova i web aplikacija koje se prilagođavaju različitim veličinama ekrana i orijentacijama. Flexbox nazivamo jednodimenzionalnim modelom rasporeda jer se fokusira na raspoređivanje elemenata duž jedne ose u jednom trenutku. Osa može biti ili horizontalna ili vertikalna.

Postoje dva ključna koncepta o kojima biste trebali znati pre nego što počnete da radite sa flexbox-om: fleks kontejner i fleks element.

Flex kontejner je HTML element sa flex rasporedom. Možete organizovati i poravnati elemente na različite načine unutar flex kontejnera. Da biste učinili HTML elementom flex kontejnerom, morate dodati `display: flex` u njegove CSS stilove.

Stavke fleksa su direktno dete fleks kontejnera. Ovi elementi se mogu organizovati i poravnati unutar fleks kontejnera na osnovu njegovih svojstava. Takođe se mogu smanjiti ili proširiti kako bi odgovarali dostupnom prostoru.

Ovo je primer sa `main` kontejnerom i tri deteta `div` elementa:

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

Ako postavite samo `width`, `height`, i `background-color` ovih `div` elemenata u CSS stilskom listu, svaki element deteta će biti postavljen na svoj red jer kontejner podrazumevano nije fleks.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Ali ako dodate `display: flex` u `main` kontejner, elementi `div` će biti preuređeni da stanu na istom redu i skupljaće se ako je potrebno (da biste videli predglednike, omogućite interaktivni uresnik):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Po podrazumevanju, flex kontejner će biti blok-nivo element, pa će sam kontejner biti na svom redu u odnosu na druge elemente i kontejnerima.

Sada kada znate više o fleks kontejnerima i fleks elementima, trebalo bi da znate i o fleks svojstvima. Ova svojstva određuju kako će se fleks elementi rasporediti, promeniti veličinu i distribuirati unutar fleks kontejnera. Neki od najčešće korišćenih su `flex-direction`, `justify-content`, `align-items`, i `flex-wrap`.

Odlično. Sada ćemo malo pričati o fleks modelu. Ovaj model određuje kako su fleksi elementi raspoređeni unutar fleks kontejnera. Svaki fleks kontejner ima dve ose:

- Glavna osa.
- Poprečna osa.

Orientacija ovih osa određuje kako će različita svojstva uticati na raspored i distribuciju fleksibilnih elemenata. Podrazumevano, glavna osa fleks kontejnera je horizontalna, a poprečna osa je vertikalna. Fleksibilni elementi su raspoređeni u smeru glavne ose. Poprečna osa je perpendikularna na glavnu osu.

Svojstvo `flex-direction` određuje smer glavne ose. Podrazumevana vrednost za `flex-direction` je `row`, što postavlja sve fleksibilne elemente na istom redu, u smeru podrazumevanog jezika vašeg pregledača (od leve ka desnoj ili od desne ka levoj):

```css
flex-direction: row; /* Default */
```

Da biste obrnuli stavke u redu, možete koristiti `flex-direction: row-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Ovo će obrnuti redosled fleks elemenata. Ako želite umesto toga da poravnate fleks elemente vertikalno, samo morate postaviti ``flex-direction`` na ``column`` u fleks kontejneru. Ovo će promeniti smer glavne ose:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Sada će elementi `div` biti vertikalno poravnati jer će glavna osa biti vertikalna, a poprečna osa horizontalna.

Takođe možete da obrnete redosled fleks elemenata vertikalno sa `flex-direction: column-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexbox je moćan model rasporeda koji pruža fleksibilan i efikasan način za aranžiranje elemenata unutar kontejnera. Razumevanjem ključnih koncepata fleksibilnih kontejnera, fleksibilnih elemenata i raznih fleksibilnih svojstava, možete kreirati dinamične i responzivne veb-sajtove koji se prilagođavaju različitim veličinama ekrana i orijentacijama.
