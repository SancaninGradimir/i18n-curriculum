# --description--
U ovom laboratorijumu, ćete izgraditi aplikaciju koja pruža informacije o vremenu iz različitih gradova.

Koristiće se vremenski API. Izlazni podaci imaju sledeći format:

__CODE_BLOCK_0__

**Cilj:** Ispunite korisničke priče ispod i osigurajte da svi testovi prođu kako biste završili laboratorijski zadatak.

**Korisničke priče:**

1. Trebalo bi da imate ``button`` element sa ``id`` od ``get-weather-btn``.

1. Trebalo bi da imate `select` element sa sedam `option` elemenata ugnježdeno unutar njega. Prva opcija bi trebalo da ima prazan string kao tekst i `value` atribut. Preostali bi trebalo da imaju sledeće za svoj tekst i vrednosti (sa vrednošću u malim slovima):

- Njujork
- Los Anđeles
- Hilbijari (ili Šikago)
- Pariz
- Tokio
- London

1. Ukoliko nije odabran grad, pritiskanje dugmeta ne bi trebalo ništa da učini.

1. Ako je odabran grad, elementi za prikaz vremena bi trebalo da se pojave:

   - Trebalo bi da imate element ``img`` sa ID-om ``weather-icon`` za prikaz ikone vremena.

   - Trebalo bi da imate element sa id-om `main-temperature` za prikaz glavne temperature.

- Trebalo bi da imate element sa ID-om ``feels-like`` za prikazivanje kako se oseća temperatura.

- Trebalo bi da imate element sa ID-om ``humidity`` za prikazivanje količine vlažnosti u vazduhu.

- Trebalo bi da imate element sa ID-om ``wind`` za prikaz brzine vetra.

- Trebalo bi da imate element sa ID-om ``wind-gust`` za prikaz jačine udara vetra.

- Trebalo bi da imate element sa ID-om ``weather-main`` za prikaz glavnog tipa vremena.

- Trebalo bi da imate element sa ID-om ``location`` za prikaz trenutne lokacije.

Trebalo bi da imate asinhronu funkciju pod nazivom ``getWeather`` koja dohvaća informacije o vremenu sa API-ja ``https://weather-proxy.freecodecamp.rocks/api/city/<CITY>`` i vraća ih. Napomena je da ovaj API vraća podatke koristeći metrički sistem, što znači m/s za brzinu vetra i Celzijus za temperaturu.

1. Asinhrona funkcija ``getWeather`` treba da prihvati grad kao svoj argument.

1. Trebalo bi da obradite sve greške koje se dešavaju unutar funkcije `getWeather` i logujete ih u konzolu.

1. Trebalo bi da imate asinhkronnu `showWeather` funkciju koja prima grad kao parametar.

Funkcija ``showWeather`` treba da pozove funkciju ``getWeather`` radi dohvaćanja podataka o vremenu za odabrani grad iz padajućeg menija.

1. Ako funkcija ``getWeather`` ima grešku, aplikacija bi trebalo da prikaže samo upozorenje koje kaže ``Something went wrong, please try again later``.

1. Ako su podaci iz `getWeather` upotrebljivi, funkcija `showWeather` bi trebalo da prikaže podatke o vremenu u odgovarajućim elementima. Ako je određena vrednost iz API-ja `undefined`, treba da unesete `N/A` u odgovarajući element.

NAPOMENA: Testovi će zahtevati vremena za završetak. Sve dok vidite ``// running tests`` u konzoli, oni se izvršavaju.

const getWeatherBtn = document.getElementById('get-weather-btn');
const selectEl = document.getElementById('location-selector');

const weatherInfoEl = document.getElementById('weather-info');
const iconEl = document.getElementById('weather-icon');
const tempEl = document.getElementById('main-temperature');
const feelEl = document.getElementById('feels-like');
const humidityEl = document.getElementById('humidity');
const windEl = document.getElementById('wind');
const gustEl = document.getElementById('wind-gust');
const mainEl = document.getElementById('weather-main');
const locationEl = document.getElementById('location');
const arrowEl = document.getElementById('compass-arrow');

getWeatherBtn.addEventListener(
  'click',
  () => selectEl.value && showWeather(selectEl.value)
);

async function showWeather(city) {
  const json = await getWeather(city);
  const {
    weather,
    main: { temp, feels_like, humidity },
    wind: { speed, gust, deg = 0 },
    name
  } = json;

  const { main, icon } = weather[0];

  weatherInfoEl.style.display = 'block';

  iconEl.src = icon || '';
  tempEl.innerHTML = temp ? `${temp}&deg; C` : 'N/A';
  feelEl.innerHTML = `Feels Like: ${feels_like ? `${feels_like}&deg; C` : 'N/A'}`;
  humidityEl.innerHTML = `Humidity: ${humidity ? `${humidity}%` : 'N/A'}`;
  windEl.innerHTML = `Wind: ${speed ? `${speed} m/s` : 'N/A'}`;
  gustEl.innerHTML = `Gusts: ${gust ? `${gust} m/s` : 'N/A'}`;

  mainEl.innerHTML = main || 'N/A';
  locationEl.innerHTML = name || 'N/A';
  arrowEl.style.transform = `rotate(${deg}deg)`;
}

async function getWeather(city) {
  try {
    const response = await fetch(
      `https://weather-proxy.freecodecamp.rocks/api/city/${city}`
    );

    if (!response.ok) {
      alert('Something went wrong, please try again later');
    }

    const json = await response.json();
    return json;
  } catch (error) {
    console.error(error.message);
  }
}
