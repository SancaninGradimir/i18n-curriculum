# --interactive--

Pravilo ``@property`` je moćna CSS karakteristika koja programerima omogućava definisanje prilagođenih svojstava sa većom kontrolom nad njihovim ponašanjem, uključujući kako se animiraju i njihove početne vrednosti.

Ovo pravilo pruža način za poboljšanje funkcionalnosti prilagođenih svojstava CSS-a i nudi veću fleksibilnost u njihovoj primeni.

Osnovna sintaksa pravila `@property` je sledeća:

```css
@property --property-name {
  syntax: '<type>';
  inherits: true | false;
  initial-value: <value>;
}
```

`--property-name` je naziv prilagođenog svojstva koje definišete. Kao i sva prilagođena svojstva, mora početi sa dve crte.

`syntax` definiše tip svojstva, koji mogu biti stvari poput `<color>`, `<length>`, `<number>`, `<percentage>`, ili složeniji tipovi.

`inherits` određuje da li svojstvo treba da nasledi svoju vrednost od svog roditeljskog elementa.

`initial-value` postavlja podrazumevanu vrednost svojstva.

Evo praktičnog primera korišćenja pravila `@property`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
@property --main-color {
  syntax: '<color>';
  inherits: false;
  initial-value: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

U ovom primeru, definišemo prilagođeno svojstvo `--main-color` kao vrednost `color`, postavljajući ga da ne nasleđuje, i dajući mu početnu vrednost `#3498db`.

Jedna od ključnih prednosti pravila ``@property`` je što omogućava animaciju prilagođenih svojstava, što nije bilo moguće ranije:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="gradient-box"></div>
```

```css
@property --gradient-angle {
  syntax: '<angle>';
  inherits: false;
  initial-value: 0deg;
}

.gradient-box {
  width: 100px;
  height: 100px;
  background: linear-gradient(var(--gradient-angle), red, blue);
  transition: --gradient-angle 0.5s;
}

.gradient-box:hover {
  --gradient-angle: 90deg;
}
```

:::

Gornji kod kreira gradijent koji se glatko animira kada se pređe mišem preko elementa, nešto što nije bilo ostvarivo sa standardnim prilagođenim svojstvima.

Sada, hajde da razmotrimo kako pravilo `@property` funkcioniše sa fallbackovima.

Fallback-ovi su ključni u CSS-u kako bi se osiguralo da stili graciozno degradiraju u pregledačima koji ne podržavaju određene karakteristike. Sa `@property`, fallback-ovi rade na dva nivoa: za samu pravilo i za korišćenje prilagođenog svojstva.

Za pretraživače koji ne podržavaju pravilo `@property`, možete pružiti rezervnu opciju deklarisanjem prilagođenog svojstva na tradicionalan način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
```

```css
:root {
  --main-color: #3498db;
}

@property --main-color {
  syntax: '<color>';
  inherits: false;
  initial-value: #3498db;
}

body {
  background-color: var(--main-color);
}
```

:::

U ovom slučaju, pretraživači koji podržavaju `@property` koristiće strože definisanu verziju, dok će drugi koristiti standardnu deklaraciju prilagođenog svojstva.

Kada koristite prilagođeno svojstvo, možete pružiti rezervnu vrednost koristeći funkciju `var()`, baš kao što biste to uradili sa standardnim prilagođenim svojstvima:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
.button {
  background-color: var(--main-color, #3498db);
}
```

:::

Ovo osigurava da čak ni ako svojstvo `--main-color` nije definisano ili je nevažeće, dugme će i dalje imati boju pozadine.

Pravilo ``@property`` takođe omogućava složenije scenarije rezervnih opcija. Na primer, možete ga koristiti za pružanje tip-bezbednih rezervnih opcija:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box">Colored box</div>
```

```css
@property --padding {
  syntax: '<length>';
  inherits: false;
  initial-value: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkred;
  color: white;
  padding: var(--padding);
}
```

:::

U ovom slučaju, ako neko pokuša da postavi `--padding` na nevažeću vrednost, kao što je boja, pregledač će se vratiti na početnu vrednost `10px`, održavajući sigurnost tipa.

CSS pravilo `@property` se smatra široko dostupnim u svim glavnim modernim pretraživačkim motorima (kao što su Chrome, Edge, Opera, Firefox i Safari) od ranog 2024. godine. Ako želite da podržite starije pretraživače, morate pružiti rezervni CSS kao što je objašnjeno iznad.

Zaključno, pravilo `@property` predstavlja značajan napredak u načinu na koji radimo sa prilagođenim svojstvima u CSS-u. Pruža više kontrole i omogućava nove mogućnosti, kao što je animacija prilagođenih svojstava. Međutim, važno je da ga koristite promišljeno i uvek pružite odgovarajuće fallback rešenja kako biste osigurali dobro iskustvo u svim pregledačima. Kao i kod bilo koje najsavremenije web tehnologije, ključ je poboljšanje iskustva za pregledače koji ga podržavaju, uz osiguravanje osnovne funkcionalnosti za one koji to ne podržavaju.
