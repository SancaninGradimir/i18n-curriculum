# --interactive--

CSS prilagođene svojstva, poznati i kao CSS varijable, su entiteti definisani od strane autora CSS-a koji sadrže specifične vrednosti koje se mogu ponovo koristiti kroz ceo dokument. One su moćna karakteristika koja omogućava efikasnije, održivije i fleksibilnije listove stilova.

Sintaksa za deklarisanje prilagođenog svojstva je jednostavna. Počinje sa dve crte (`--`) praćeno imenom svojstva:

```css
:root {
  --main-color: #3498db;
}
```

U ovom primeru, deklarišemo prilagođenu svojstvo nazvano `--main-color` sa vrednošću `#3498db`. Pseudo-klasa `:root` se često koristi za deklarisanje globalnih prilagođenih svojstava, jer predstavlja roditelja najvišeg nivoa u DOM stablu.

Da biste koristili prilagođeno svojstvo, koristite funkciju `var()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

Ovo postavlja boju pozadine elemenata sa klasom `button` na vrednost sačuvanu u `--main-color`.

Jedna od ključnih karakteristika prilagođenih svojstava je što prate CSS kaskadu. To znači da ih možete ponovo definisati za specifične elemente ili kontekste:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

U ovom slučaju, elementi sa klasom `alert` će koristiti drugačiju vrednost `--main-color`, prekrivajući globalnu definiciju.

Prilagođena svojstva takođe podržavaju rezervne vrednosti. Ako prilagođeno svojstvo nije definisano ili je nevažeće, možete pružiti rezervnu vrednost:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

Ovde, ako `--text-color` nije definisan, `color` će podrazumevano biti `green`.

Prilagođena svojstva su posebno korisna za kreiranje temačnih dizajna. Možete definisati skup svojstava za različite teme:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

Promena tema postaje jednostavno kao dodavanje ili uklanjanje klase sa elementa `body`.

Prilagođena svojstva se takođe mogu koristiti sa medijskim upitima za kreiranje responsivnog dizajna:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

Ovaj pristup omogućava održivije responsivne rasporede, jer možete promeniti vrednosti na jednom mestu umesto kroz ceo vaš stilski list. Ovo može značajno smanjiti ponavljanje u vašim stilskim listovima, posebno za često korišćene, kompleksne vrednosti.

Prilagođena svojstva takođe mogu da referenciraju druga prilagođena svojstva:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

Ovo omogućava stvaranje odnosa između različitih stilskih aspekata, što olakšava održavanje doslednih tema širom vašeg sajta.

Iako prilagođena svojstva nude mnogo prednosti, važno je biti svestan njihove podrške u pretraživačima. Dobro su podržana u modernim pregledačima, ali stariji pregledači možda neće prepoznati. Uvek imajte plan B kada koristite najnovije funkcije.

Zaključno, CSS prilagođene svojstva pružaju moćan način za kreiranje dinamičnijih, fleksibilnijih i održivijih stilskih listova. Omogućavaju kreiranje temačnih dizajna, pojednostavljuju responsivne rasporede i omogućavaju manipulaciju stilovima tokom izvršavanja.

Kako se razvoj veb-a nastavlja da evoluira, prilagođeni svojstva postaju sve važniji alat u repertoaru developera, nudeći nove mogućnosti za kreiranje prilagodljivog i efikasnog CSS-a.
