# --interactive--

`switch` izjave i `if/else if/else` lance su oba kontrolne strukture protoka u programiranju koje nam omogućavaju izvršavanje različitih blokova koda na osnovu određenih uslova. Međutim, oni imaju različite karakteristike i slučajeve upotrebe.

Podešlje (`switch`) procenjuje izraz i upoređuje njegovu vrednost sa nizom `case` klauzula. Kada se pronađe podudarnost, izvršava se blok koda povezan sa tim `case`. Evo osnovne strukture (`switch`) podešlja:

```js
switch (expression) {
  case value1:
    // code to be executed if expression === value1
    break;
  case value2:
    // code to be executed if expression === value2
    break;
  default:
    // code to be executed if expression doesn't match any case
}
```

Izjava `break` na kraju svakog `case` je ključna. Ona govori programu da izađe iz bloka prebacivanja nakon izvršenja odgovarajućeg `case`. Bez nje, program bi nastavio sa izvršavanjem naknadnih slučajeva, ponašanje poznato kao "padak" ("fall-through").

``switch`` izjave se obično koriste kada poredite jednu varijablu sa više mogućih vrednosti. Posebno su korisne kada imate mnogo potencijalnih uslova za proveru u odnosu na jednu varijablu. Evo primera korišćenja ``switch`` izjave za dane u nedelji:

:::interactive_editor

```js
let dayOfWeek = 3; 

switch (dayOfWeek) {
    case 1:
        console.log("It's Monday! Time to start the week strong.");
        break;
    case 2:
        console.log("It's Tuesday! Keep the momentum going.");
        break;
    case 3:
        console.log("It's Wednesday! We're halfway there.");
        break;
    case 4:
        console.log("It's Thursday! Almost the weekend.");
        break;
    case 5:
        console.log("It's Friday! The weekend is near.");
        break;
    case 6:
        console.log("It's Saturday! Enjoy your weekend.");
        break;
    case 7:
        console.log("It's Sunday! Rest and recharge.");
        break;
    default:
        console.log("Invalid day! Please enter a number between 1 and 7.");
}
```

:::

`switch` izjave mogu biti čitljivije i sažetnije kada se radi sa mnogim mogućim vrednostima za jednu varijablu.

Izjave, međutim, su fleksibilnije. Mogu da procene složene uslove i različite varijable u svakoj klauzi. To ih čini pogodnim za širi spektar scenarija. Evo primera kada biste mogli da koristite izjavu `if/else` umesto izjave `switch`:

:::interactive_editor

```js
let creditScore = 720; 
let annualIncome = 60000; 
let loanAmount = 200000; 

let eligibilityStatus;

if (creditScore >= 750 && annualIncome >= 80000) {
    eligibilityStatus = "Eligible for premium loan rates.";
} else if (creditScore >= 700 && annualIncome >= 50000) {
    eligibilityStatus = "Eligible for standard loan rates.";
} else if (creditScore >= 650 && annualIncome >= 40000) {
    eligibilityStatus = "Eligible for subprime loan rates.";
} else if (creditScore < 650) {
    eligibilityStatus = "Not eligible due to low credit score.";
} else {
    eligibilityStatus = "Not eligible due to insufficient income.";
}

console.log(eligibilityStatus);
```

:::

U ovom primeru, imamo godišnji prihod i kreditni rejting osobe i proveravamo za koje vrste kredita bi kvalifikovali. Pošto se bavimo složenijim logičkim procenama i više promenljivih, bolje je koristiti ``if/else`` izjavu ovde u poređenju sa ``switch`` izjavom.

Vredi napomenuti da ``switch`` izjava u JavaScript-u koriste strogo poređenje (``===``), što znači da ne vrše konverziju tipa. Ovo može biti prednost u smislu predvidivosti i izbegavanja suptilnih grešaka.

Zaključno, iako i oba `switch` izjava i `if/else if` lanac omogućavaju logiku sa više grana u vašem kodu, oni imaju različite snage. `switch` izjave su izvrsne za obradu više mogućih vrednosti za jednu varijablu, dok `if/else if` lanci nude veću fleksibilnost za složene uslove. Izbor između njih često zavisi od specifičnih zahteva vašeg koda i ličnih ili timskih preferencija stila kodiranja.
