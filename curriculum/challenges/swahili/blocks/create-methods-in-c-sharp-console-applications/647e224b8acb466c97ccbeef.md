## --text--

S obzirom na potpis metode,

```csharp
void Print(string name, string number = "", bool member = false)
```

Koja od sledećih opcija pravilno koristi nazvane argumente i opcione argumente?

## --answers--

`Print("Tony", member: true);`

---

`Print(number: "555", member: false)`

---

`Print("Tony", false)`
