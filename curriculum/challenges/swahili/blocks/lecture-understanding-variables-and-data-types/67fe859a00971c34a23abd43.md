# --description--

Pre nego što počnete da radite sa Python varijablama, važno je razumeti tipove podataka. Tip podataka opisuje vrstu vrednosti koju varijabla sadrži. Na primer, broj, deo teksta ili lista stavki. Programski jezici koriste tipove podataka kako bi znali kako da skladište i rade sa različitim vrstama informacija.

Python je dinamički tipizovan jezik poput JavaScripta, što znači da ne morate eksplicitno deklarisati tipove za varijable. Jezik zna koji je tip podataka varijabli na osnovu toga šta joj dodelite.

Evo nekih primera:

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

Ovo je u kontrastu sa nekim statički tipiziranim jezicima kao što su C#, Java i C++, gde morate deklarisati tipove sa varijablama, ovako:

```csharp
string name = 'John Doe'
int age = 25
```

Dinamičko tipiziranje Pythona čini kodiranje veoma brzim i fleksibilnijim, ali može dovesti do neočekivanih grešaka jer se greške u tipovima detektuju tek kada program radi (pokreće), a ne kada se kompajlira.

Pošto Python određuje tipove podataka dok vaš program radi, greške povezane sa tipovima se otkrivaju tek u tom trenutku. Kada program radi, Python izvršava vaš kod liniju po liniju. Ako dođe do linije gde se očekuje da određeni objekat ponaša na način na koji nije u stanju, Python će prestati i prikazati grešku.

Nasuprot tome, neki jezici **kompajliraju** vaš program pre nego što se pokrene. Kompajliranje znači da računar proverava vaš kod unapred i priprema ga za izvršavanje. Tokom ovog koraka, ti jezici mogu uhvatiti greške tipa pre nego što program uopšte počne.

Ne morate još da znate te jezike. Važna ideja je jednostavno:

- **U Pythonu greške tipa mogu se otkriti tokom izvršavanja**, kada je program zapravo pokrenut i koristi vaš kod.

- **Kompilovani jezici hvataju greške tipa tokom koraka kompilacije**, pre nego što se programu dozvoli da radi.

Zbog toga, možda ne saznate za tipsku grešku u Pythonu sve dok program ne stigne do te specifične linije koda tokom izvršavanja.

Evo najčešćih tipova podataka koje ćete koristiti u Pythonu:

- Celi broj: Celobrojni broj bez decimala, na primer, `10` ili `-5`.

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- Float: Broj sa decimalnom tačkom, kao što su `4.41` ili `-0.4`.

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- Niz karaktera: Sekvenca karaktera zatvorena u jednostru ili dvostru navodnike kao što je `'Hello world!'`.

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- Booleov (tip): Tip koji je tačan ili netačan, napisan kao `True` ili `False`.

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- Skup: Neuređana kolekcija jedinstvenih elemenata, kao što je `{0.5, 4, 'apple'}`.
 
```python
my_set_var = {7, 'hello', 8.5}
print('Set:', my_set_var) # Set: {7, 'hello', 8.5}
```

- Rečnik: Kolekcija parova ključ-vrednost zatvorena u vitičnim zagradama, kao što je `{'name': 'John Doe', 'age': 28}`.

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- Tupel: Imutabilna usmerena kolekcija, zatvorena u zagrade, kao što je `('apple', 4.5, 7)`.

```python
my_tuple_var = (7, 'hello', 8.5)
print('Tuple:', my_tuple_var) # Tuple: (7, 'hello', 8.5)
```

- Opseg: Sekvenca brojeva, često korišćena u petljama, na primer, `range(5)`.

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- Lista: Uredovana kolekcija elemenata koja podržava različite tipove podataka.


```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None: Specijalna vrednost koja predstavlja odsustvo vrednosti.

```python
my_none_var = None
print('None:', my_none_var) # None: None
```
