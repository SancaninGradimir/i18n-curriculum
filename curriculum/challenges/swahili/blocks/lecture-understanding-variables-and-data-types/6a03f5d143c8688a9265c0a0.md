# --description--

U prethodnoj lekciji ste naučili o uobičajenim tipovima podataka u Pythonu, uključujući niz (string), ceo broj (integer) i decimalni broj (float). Kako gradite svoje programe, moraćete da naučite kako da vidite tip varijable.


Evo primjera varijable:

```py
developer = 'Devin'
```

Da biste videli kakav je tip `developer`, možete koristiti funkciju `type()` na ovaj način:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

Izlaz od `<class 'str'>` znači da je `developer` tip stringa.

Ukoliko ne pružite bilo koje argumente funkciji ``type()``, dobićete sledeću poruku o grešci:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

Evo svih tipova podataka koje ste do sada naučili zajedno sa njihovim tipovima u terminalu:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Postojeće će trenutke u vašem programu kada ćete morati da proverite da li je određena varijabla određen tip pre obavljanja operacija na njoj. Ovde je funkcija ``isinstance()`` korisna.

Here is an example variable with a string assigned to it:

```py
account_balance = '12'
```

Ako pokušate da izvršite matematičke izraze poput deljenja koristeći varijablu ``account_balance``, dobićete poruku o grešci.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

Da biste videli da li je `account_balance` ceo broj, možete proveriti koristeći funkciju `isinstance()` na ovaj način:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

Ugrađena funkcija `isinstance()` vam omogućava da proverite da li varijabla odgovara određenom tipu podataka. Prihvata objekat i tip na koji želite da ga proverite, zatim vraća boolean vrednost. U ovom slučaju, pošto je `account_balance` string, on će vratiti `False`.

Funkcija ``isinstance()`` takođe vam omogućava da proverite više tipova odjednom.

Evo primera koji proverava da li je `account_balance` `int` ili `float`:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

U ovom primeru, `account_balance` je ceo broj pa `isinstance()` vraća `True`. Ako bi `account_balance` bio `12.0`, `isinstance()` i dalje bi vratio/vraćalo `True` jer proveravate da li je ceo broj ili float.

U budućim radionicama i laboratorijama, koristićete funkcije `type()` i `isinstance()` kako biste osigurali da vaše varijable sadrže ispravne tipove podataka pre obavljanja operacija na njima.
