# --description--

Fetch API omogućava veb aplikacijama da vrše mrežne zahteve, obično radi preuzimanja ili slanja podataka na server. Ovaj API pruža metod `fetch()` koji možete koristiti za slanje ovih zahteva. Pogledajmo osnovni primer kako da se koristi `fetch`:

```js
fetch('https://api.example.com/data')
```

U ovom primeru, izvršavamo zahtev ``GET`` ka ``https://api.example.com/data``. Ovo vraća podatke koji moraju biti konvertovani u JSON format pre nego što se mogu koristiti.

Podrazumevano, Fetch API koristi metod `GET` za dohvaćanje podataka. Ovo će biti pokriveno u sledećoj lekciji, zajedno sa drugim uobičajenim HTTP zahtevima metoda.

Sada ćemo razmotriti neke uobičajene tipove resursa koji se dohvaćaju sa mreže.

U našim veb aplikacijama, potreban nam je neki zajednički podaci kao što su vremenski podaci, podaci o spisku profesija, spisak imena država, državni kod ili ikone/slike zastave države. Koristeći ove podatke, možemo učiniti našu aplikaciju informativnijom i interaktivnijom. Zahvaljujući Fetch API-ju, možemo dobiti ove resurse iz mreže.

Slike su neki često dohvaćeni resursi. Možete koristiti `fetch` za učitavanje slika statički ili dinamički na osnovu akcija korisnika, i prikazivanje ih u vašoj veb aplikaciji.

Tekst fajlovi su drugi tip resursa koji se često dohvaćaju. Ovo može uključivati konfiguracione fajlove, log fajlove, ili čak cele dokumente koje želite prikazati na svojoj veb stranici.

U nekim slučajevima, možda ćete dohvatiti audio ili video fajlove. Fetch API takođe može da obrađuje ovakve tipove resursa, omogućavajući vam da radite sa širokim spektrom tipova podataka.
