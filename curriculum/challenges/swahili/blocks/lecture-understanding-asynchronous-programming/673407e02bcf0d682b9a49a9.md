# --description--

JavaScript motor poseduje sposobnost da čita, razume i izvršava vaš kod. Radi kao konverter koji uzima vaš kod, pretvara ga u instrukcije koje računar može da razumi i izvrši.

Jedan od najpoznatijih JavaScript motora je V8, razvijen od strane Google-a, korišćen u Chromu i Node.js-u. JavaScript motor radi u nekoliko koraka. Prvo, parsira vaš kod, čitajući ga red po red kako bi se uverio da nema greške u JavaScript kodu. Zatim, konvertuje ovaj kod u bajtkod (bytecode), koji je jednostavnija, međuvremenska verzija vašeg koda koja je lakša za računar da razume i izvrši. Na kraju, pokreće ovaj bajtkod kako bi izvršio instrukcije vašeg programa. Evo primera JavaScript koda:

```js
const greeting = "Hello, World!";
console.log(greeting);
```

Kada pokrenete ovaj kod, JavaScript engine ga prvo parsira kako bi proverio da li ima bilo kakvih sintaksnih grešaka. Parsiranje znači da motor čita kod i razlaže ga na strukturu koju može razumeti, proveravajući greške tokom procesa.

Zatim kompajlira kod u međuformat (često bajtkod ili mašinski kod, zavisno od motora). Kompajliranje je proces pretvaranja kodova koji su čitljivi za ljude u efikasniji format koji računar može izvršiti brže.

Konačno, motor izvršava kod, štampajući `Hello, World!` na konzolu.

Sada, hajde da pričamo o JavaScript okruženju za izvršavanje (runtime). JavaScript okruženje za izvršavanje je okruženje u kojem se vaš JavaScript kod izvršava. Ono uključuje JavaScript motor (poput V8 u Chrome-u ili SpiderMonkey u Firefoxu), koji obrađuje i izvršava kod, kao i dodatne funkcije koje pruža okruženje (kao što su veb pregledač ili Node.js, o kojima ćete saznati više u budućim lekcijama).

Iako je osnovni JavaScript jezik za stvari kao što su varijable, petlje i funkcije, okruženje izvršavanja (runtime) pruža dodatne alate koji omogućavaju JavaScript-u da interaguje sa stvarima izvan samog jezika, kao što su DOM (za veb stranice) ili Fetch API (za slanje mrežnih zahteva).

Ukratko, runtime je ono što omogućava JavaScriptu da radi više od samo osnovnih programskih zadataka – kao što su interakcija sa veb stranicama ili rukovanje akcijama zasnovanim na vremenu – pružajući ove dodatne funkcije izvan samog jezika.

Iako ne morate znati svaki detalj motora i okruženja izvršavanja kako biste napisali JavaScript, imanje osnovnog razumevanja može vam pomoći da napišete efikasniji kod i efikasnije otklonite probleme.
