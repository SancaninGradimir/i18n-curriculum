# --interactive--

CSS animacije vam omogućavaju da kreirate dinamične, vizuelno privlačne efekte na veb stranicama bez potrebe za JavaScript-om ili kompleksnim programiranjem. One pružaju način da se elementi glatko prebacuju između različitih stilova tokom određene trajanja.

U suštini, CSS animacija se sastoji od dve glavne komponente: pravila `@keyframes` i svojstva animacije.

Pravilo ``@keyframes`` definiše faze i stilove animacije. Ono određuje koje stilove bi element trebalo da ima u različitim tačkama tokom animacije.

Evo primera:

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

Ovo pravilo `@keyframes`, nazvano `slide-in`, definiše animaciju koja pomera element od leve ka desnoj strani. Procenati predstavljaju napredak animacije, pri čemu je `0%` početak a `100%` kraj.

Funkcija `translateX` u vašoj `@keyframes` animaciji kontroliše horizontalnu poziciju elementa dok se animira u vidokrug.

Da biste primenili ovu animaciju na element, koristite svojstvo `animation`. Ovaj primer takođe ponavlja animaciju beskonačno kako biste je videli u akciji (moraćete da omogućite interaktivni editor da biste videli predgleđaje):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">Hello, I slide in!</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

Ovo primenjuje animaciju `slide-in` na element sa trajanja od 2 sekunde i funkciju vremena `ease-out`.

`animation` svojstvo je zapravo skraćeni oblik za više pojedinačnih svojstava:

`animation-name` koji određuje `@keyframes` pravilo za upotrebu.

`animation-duration` koji određuje koliko dugo animacija treba da potraje do završetka.

`animation-timing-function` što definiše kako animacija napreduje tokom vremena - kao što su `ease`, `linear`, `ease-in-out`.

`animation-delay` što određuje kašnjenje pre nego što animacija počne.

`animation-iteration-count` koji određuje koliko puta animacija treba da se ponovi.

`animation-direction` koji određuje da li animacija treba da se reprodukuje napred, unazad ili naizmenično.

`animation-fill-mode` što određuje kako bi se element trebalo stilizovati pre i posle animacije.

`animation-play-state` što vam omogućava da pauzirate i nastavišete animaciju.

Možete koristiti ova svojstva pojedinačno za precizniju kontrolu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">Watch my colors change!</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

Ovo stvara animaciju koja kontinuirano prebacuje pozadinsku boju elementa između crvene, plave i zelene.

CSS animacije se mogu pokrenuti različitim događajima, kao što je prebacivanje mišem preko elementa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Hover over me!</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

Iako ovaj primer koristi svojstvo ``transition``, koje je jednostavnije za osnovne efekte, demonstrira kako CSS može kreirati interaktivne, animirane elemente.

Važno je napomenuti da su CSS animacije moćne, ali bi trebalo ih koristiti umereno. Prekomerna upotreba animacija može dovesti do loših performansi i može biti odvraćajuća ili problematična za korisnike sa određenim potrebama pristupačnosti. Uvek razmislite o pružanju opcija za smanjenje ili onemogućavanje animacija za korisnike koji preferiraju manje pokreta.

CSS animacije nude način za kreiranje privlačnih, interaktivnih veb iskustava bez oslanjanja na JavaScript. Razumevanjem principa svojstava `@keyframes` i `animation`, možete oživeti svoje web dizajne na performantan i pristupačan način.
