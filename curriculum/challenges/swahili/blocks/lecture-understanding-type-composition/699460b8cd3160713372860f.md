# --description--

Ponekad će postojati situacije u kojima ćete znati više o nekom tipu nego što to radi TypeScript. Ovde nam dolaze u pomoć tvrdnje tipa (type assertions).

Evo primera korišćenjem metode `querySelector()` za pristup elementu sa `id` od `submit`:

```ts
const submitBtn = document.querySelector("#submit");
```

Trenutno, TypeScript zna samo da je ovo `submitBtn` neki tip od `Element` ili možda `null`. TypeScript neće pogledati odgovarajući HTML kako bi to utvrdio.

Ovo je dobro mesto za korišćenje tipne asercije da biste rekli TypeScript-u koji je ovo tip elementa. Evo primera:

```ts
const submitBtn = document.querySelector("#submit") as HTMLButtonElement;
```

Sada, TypeScript će ovo tretirati kao `button` element zbog te deklaracije tipa.

Postoje ograničenja u korišćenju ključne reči ``as``. Na primer, ne možete uraditi nešto ovako bez toga da TypeScript baci grešku:

```ts
// Conversion of type 'string' to type 'number' may be a mistake because neither type sufficiently overlaps with the other. If this was intentional, convert the expression to 'unknown' first.
const age = "awesome" as number;
```

U ovom primeru, nema mnogo smisla pokušavati da se tvrdi da bi string ``"awesome"`` trebalo da se tretira kao tip broja. TypeScript dozvoljava tipne tvrdnje samo između kompatibilnih ili preklapajućih tipova.

Jedan način da se ukloni ta greška je korišćenje dvostruke asertacije, kao što je ovo:

```ts
const age = "awesome" as unknown as number;
```

Iako ovo radi u TypeScriptu, i dalje nije preporučljivo jer logički nema mnogo smisla.

Drugi način za pisanje tipskih tvrdnji je korišćenje sintakse sa uglastim zagradama, kao što je ovo:

```ts
const submitBtn = <HTMLButtonElement>document.querySelector("#submit");
```

Morate biti oprezni sa ovom sintaksom kada koristite `tsx` fajlove jer će TypeScript misliti da pokušavate da renderujete komponentu i baciti grešku kao ovo:

```ts
// file: index.tsx
// This JSX tag requires 'React' to be in scope, but it could not be found.
// JSX element 'HTMLButtonElement' has no corresponding closing tag.
const submitBtn =  <HTMLButtonElement>document.querySelector("#submit");
```

U većini situacija, videćete da se prvi sintaks koji koristi ključnu reč `as` koristi umesto druge opcije.
