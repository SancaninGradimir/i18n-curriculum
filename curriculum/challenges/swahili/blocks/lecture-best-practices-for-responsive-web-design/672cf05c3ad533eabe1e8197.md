# --interactive--

Media upiti su fundamentalni komponent za responsivni web dizajn, omogućavajući programerima da primene različite stilove na osnovu karakteristika uređaja ili pretraživača korisnika. Oni pružaju način za prilagođavanje prezentacije sadržaja različitim uređajima bez menjanja samog sadržaja.

U svojoj srži, medijski upit sastoji se od tipa medija i jednog ili više izraza koji proveravaju specifične uslove. Ako su ovi uslovi istiniti, primenjuju se odgovarajući stilovi. Ovaj mehanizam omogućava kreiranje responsivnog dizajna koji se prilagođava različitim veličinama ekrana, rezolucijama i mogućnostima uređaja.

Osnovna sintaksa upita za medije u CSS-u izgleda ovako:

```css
@media mediatype and (feature: value) {
  /* CSS rules go here */
}
```

U ovoj strukturi, mediatype određuje tip medija na koji se upit primenjuje, a par 'feature: value' definiše uslov koji mora biti ispunjen da bi stilovi bili primenjeni.

Medijski tipovi opisuju opštu kategoriju uređaja. Razgovaraćemo o najčešće korišćenim medijskim tipovima:

- `all` je pogodan za sve uređaje. Ovo je podrazumevano ako nije naveden medijski tip.

- `print` je namenjen za materijal sa stranicama i dokumente gledane na ekranu u režimu pregleda za štampu.

- `screen` je namenjen pre svega za ekrane.

U prošlosti je bilo više tipova medija. kao što su `handheld` i `tv`, ali je većina njih zastarela u korist korišćenja funkcija za preciznije ciljanje uređaja.

Medijski atributi opisuju specifične karakteristike korisničkog agenta, izlaznog uređaja ili okruženja. Pričamo o nekim od najčešće korišćenih medijskih atributa. <code>`width`</code> i <code>`height`</code> se odnose na širinu i visinu vidokruga, i često se koriste sa prefiksima <code>`min-`</code> ili <code>`max-`</code> za upite opsega. Evo primera:

```css
@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
}
```

`aspect-ratio` opisuje odnos između širine i visine vidokruga. Evo primera:

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

Funkcija ``orientation`` ukazuje da li je uređaj u pejzažnom ili portretnom orijentaciji. Evo primera:

```css
@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
}
```

Funkcija `resolution` opisuje rezoluciju izlaznog uređaja u tačkama po inču (dpi) ili tačkama po centimetru (dpcm). Evo primera:

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

Funkcija `hover` testira da li primarni mehanizam unosa može da pređe mišem preko elemenata. Evo primera:

```css
@media (hover: hover) {
  /* Styles for devices that support hover */
}
```

TEKST:

Funkcija `prefers-color-scheme` prepoznaje da li je korisnik zahtevao svetlu ili tamnu boju temu. Evo primera:

```css
@media (prefers-color-scheme: dark) {
  /* Styles for dark mode */
}
```

Medijski upiti takođe mogu kombinovati više uslova korišćenjem logičkih operatora. Operator `and` se koristi za kombinovanje više medijskih karakteristika, dok se `not` i `only` mogu koristiti za negiranje ili izolovanje medijskih upita. Evo primera koji kombinuje više karakteristika:

```css
@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
}
```

Takođe je moguće ciljati više upita u listi odvojene zarezima, što funkcioniše kao operater „ili“:

```css
@media screen and (min-width: 768px), print {
  /* Styles for screens at least 768px wide OR for print */
}
```

Kada radite sa medijskim upitima, važno je razmotriti kaskadu. Medijski upiti ne povećavaju specifičnost – oni samo grupišu uslovna pravila. Normalna pravila CSS kaskade i dalje važe unutar svakog medijskog upita.

U praksi, medijski upiti se često koriste za kreiranje responsivnih raspoređaja. Uobičajen uzorak je definisanje osnovnog stila za mobilne uređaje, a zatim korišćenje medijskih upita za poboljšanje rasporeda za veće ekrane:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam.</p>
  <p>Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta.</p>
  <p>Mauris massa. Vestibulum lacinia arcu eget nulla. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 15px;
}

/* Styles for tablets */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* Styles for desktops */
@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```

:::

Ovaj pristup, poznat kao „mobilno-prvi“ responsivni dizajn, osigurava da osnovni stilovi odgovaraju mobilnim uređajima, sa poboljšanjima dodanim za veće ekrane.

Zaključno, media upiti su moćan alat u CSS-u koji omogućava kreiranje responsivnog, prilagodljivog dizajna veb stranica. Razumevanjem načina korišćenja različitih medij tipova i funkcija, programeri mogu kreirati veb sajtove koji pružaju optimalna korisnička iskustva na širokom spektru uređaja i preferencija. Kako se veb tehnologije nastavljaju razvijati, praćenje novih medij funkcija može pomoći u kreiranju nijansiranijeg i korisnički prijateljskog responsivnog dizajna.
