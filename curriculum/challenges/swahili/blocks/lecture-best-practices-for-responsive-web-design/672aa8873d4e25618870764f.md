# --interactive--

Odzivni web dizajn je pristup razvoju veb sajtova koji teži kreiranju sajtova koji pružaju optimalno iskustvo pregleda i interakcije na širokom spektru uređaja, od stolnih računara do mobilnih telefona.

Osnovni princip responsivnog dizajna je prilagodljivost – sposobnost veb-sajta da prilagodi svoj raspored i sadržaj na osnovu veličine ekrana i mogućnosti uređaja na kojem se gleda.

Responsivni dizajn se obično oslanja na tri glavne komponente - fluidske mreže, fleksibilne slike i medijske upite:

- Fleksibilne mreže koriste relativne jedinice kao što su procenati umesto fiksnih jedinica kao što su pikseli, omogućavajući sadržaju da se promeni veličina i preuredi na osnovu veličine ekrana.

- Fleksivne slike su podešene da se menjaju veličinu unutar svojih kontejnerskih elemenata, osiguravajući da ne premascuju svoje kontejnere na manjim ekranima.

- I, medijski upiti omogućavaju programerima da primene različite stilove na osnovu karakteristika uređaja, pre svega širine vidokruga (viewport). Vi ćete saznati više o medijskim upitima u budućim časovima.

Odnos između responsivnog web dizajna i alata kao što su CSS Grid i Flexbox je simbiotičan. Iako je responsivni dizajn koncept ili pristup, CSS Grid i Flexbox su praktični alati koji olakšavaju i učinjavaju implementaciju responsivnih dizajna mnogo lakšom i efikasnijom.

U prethodnim časovima ste naučili kako da radite sa Flexbox-om, a u budućim časovima ćete naučiti kako da radite sa CSS Grid-om. Ali za sada, evo kratkog uvođenja u CSS Grid.

CSS Grid je dvodimenzionalni sistem rasporeda koji omogućava složenije aranžmane. Odličan je za kreiranje ukupnih rasporeda stranica, kao i manjih komponentnih rasporeda.

Evo primera kako se CSS Grid može koristiti responzivno:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

:::

U ovom primeru, mreža počinje sa jednom kolonom na malim ekranima, prebacuje se na dve kolone na srednje veličine ekrane, a zatim i na tri kolone na većim ekranima. Ovo demonstrira kako može da kreira responsivne rasporede koji se prilagođavaju različitim veličinama ekrana.

Iako Flexbox i Grid nude značajne prednosti u odnosu na starije metode raspoređivanja poput plivajućih elemenata ili tabela, oni pružaju veću fleksibilnost i kontrolu, zahtevaju manje koda i dizajnirani su sa razmatranjem responzivnosti. Omogućavaju programerima da kreiraju složene, responzivne rasporede relativno lako, čineći ih neprocenjivim alatima u implementaciji responzivnog web dizajna.

Vredi napomenuti da se Flexbox i Grid često koriste zajedno u responsivnom dizajnu. Flexbox se obično koristi za komponente i jednodimenzionalne rasporede, dok se Grid koristi za ukupnu strukturu stranice i dvodimenzionalne rasporede. Izbor između njih često zavisi od specifičnih potreba dizajna rasporeda.

Pored Flexboxa i Grida, druge CSS karakteristike igraju važnu ulogu u responsivnom dizajnu.

Funkcija ``calc()``, na primer, omogućava mešanje jedinica i obavljanje proračuna, što može biti veoma korisno pri kreiranju fleksibilnih rasporeda.

Odzivne slike su još jedan ključni aspekt odzivnog web dizajna. Atribut ``srcset`` i element ``picture`` u HTML5 omogućavaju serviranje različitih fajlova slika na osnovu mogućnosti uređaja, osiguravajući da korisnici ne preuzimaju nepotrebno velike fajlove slika na uređajima sa manjim ekranima ili nižom rezolucijom.

Zaključno, responsivni web dizajn je pristup koji ima za cilj kreiranje veb sajtova koji dobro rade na bilo kom uređaju.

Iako je to dizajnerska filozofija, a ne specifična tehnologija, oslanja se na CSS funkcije kao što su medijski upiti (media queries) i značajno ga olakšavaju savremeni alati za raspored poput Flexboxa i Grid-a. Ovi alati pružaju fleksibilnost i kontrolu potrebne za kreiranje zaista responsivnog dizajna, omogućavajući sajtovima da se besprekorno prilagode sve rastućoj raznolikosti uređaja koji se koriste za pristup internetu.
