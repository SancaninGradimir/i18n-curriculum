# --description--

Aplikacije u React-u često zavise od eksternih API-ja i baza podataka za dinamički sadržaj. Da biste pristupili podacima iz tih API-ja i baza podataka, morate koristiti neke tehnike preuzimanja podataka (data fetching).

Pogledajmo kako funkcioniše preuzimanje podataka u React-u i različite opcije koje imate za preuzimanje podataka.

React nije ograničen po pitanju načina na koji preuzimate podatke; to znači da, na osnovnom nivou, možete koristiti ugrađeni Fetch API, koji podržavaju svi moderni pregledači.

Takođe možete koristiti Axios i SWR. Axios je biblioteka za HTTP zahtevima zasnovana na obećanjima (promise-based), napravljena na osnovu objekta XMLHttpRequest, a SWR je React hook za preuzimanje podataka koji je kreirao tim Vercel.

Počnimo sa primerom. Prvo morate uvesti hookove `useState` i `useEffect`:

```js
import { useState, useEffect } from "react";
```

Zatim ćete morati da kreirate tri promenljive stanja (state variables) nazvane `loading`, `data` i `error`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);
```

Varijabla `loading` će pratiti da li se podaci još uvek preuzimaju. Varijabla `data` predstavlja same podatke, a varijabla `error` će uhvatiti sve greške koje bi mogle nastupiti tokom procesa preuzimanja podataka.

Pošto je preuzimanje podataka nuspojavnost (side effect), najbolje je koristiti Fetch API unutar `useEffect` hook-a.

Evo primera:

```js
useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/posts")
    .then((res) => res.json())
    .then((data) => {
      setData(data);
      setLoading(false);
    })
    .catch((err) => {
      setError(err);
      setLoading(false);
    });
}, []);
```

Ovaj `useEffect` preuzima podatke pomoću Fetch API-ja i postavlja sva stanja. 

Možete stvari poboljšati korišćenjem `async`/`await` umesto sintakse `.then()`. To znači da morate imati zasebnu funkciju unutar `useEffect`, jer ne možete predložiti `useEffect` ključnom rečima `async`:

```js
useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
      if (!res.ok) {
        throw new Error("Network response was not ok");
      }

      const data = await res.json();
      setData(data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

Zatim možete koristiti sva ta stanja za renderovanje podataka iz API-ja.

Evo kompletnog koda:

```jsx
import { useState, useEffect } from "react";

const FetchPosts = () => {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
        if (!res.ok) {
          throw new Error("Network response was not ok");
        }

        const data = await res.json();
        setData(data);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, []);

  if (loading) {
    return <p>Loading...</p>;
  }

  if (error) {
    return <p>{error.message}</p>;
  }

  return (
    <ul>
      {data.map((post) => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
};

export default FetchPosts;
```

U korisničkom interfejsu (UI), videćete `Loading...` na ekranu dok se podaci preuzimaju, a zatim će biti prikazani podaci ili greška zavisno od toga da li je preuzimanje podataka bilo uspešno.

Zapamtite da smo razgovarali i o preuzimanju podataka sa Axios-om i SWR-om. Pogledajmo primer korišćenjem Axios-a.

Prvo ćete morati da instalirate Axios iz komandne linije ovako:

```sh
npm i axios
```

Zatim ćete morati da uvezete Axios ovako:

```js
import axios from "axios";
```

Nakon toga možete koristiti iste varijable stanja sa ranije i preuzeti podatke iz API-ja koristeći `axios.get`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);

useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await axios.get(
        "https://jsonplaceholder.typicode.com/users"
      );
      setData(res.data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

Možda ste primetili da nema linije `await res.json()` u ovom primeru. To je zato što Axios automatski parsira JSON, pa nije potrebno to.

Poslednji primer koji ćemo pogledati je korišćenje hook-a `useSWR` za preuzimanje podataka.

Kao i sa Axios-om, moraćete da instalirate SWR ovako:

```sh
npm install swr
```

Zatim ćete morati da uvezete `useSWR` hook u fajl ovako:

```js
import useSWR from "swr";
```

U poređenju sa prethodnim primerima, sintaksa SWR je mnogo kraća. Ono što treba da uradite jeste da kreirate funkciju za preuzimanje (fetcher function) i prosledite je `useSWR` hook-u kao drugi parametar (endpoint je prvi parametar).

Takođe dobijate mogućnost dekonstrukcije stanja podataka i greške iz `useSWR` hook-a, pa ne morate koristiti `useState` hook.

Evo sintakse:

```js
const fetcher = (url) => fetch(url).then((res) => res.json());
const { data, error } = useSWR(endpoint, fetcher);
```

Napomena da je naziv "fetcher" ovde samo konvencija, tako da ste slobodni da promenljivoj dodelite bilo koji željeni naziv.

Evo komponente koja preuzima to-do liste sa JSON Placeholder API-ja:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const FetchTodos = () => {
  const { data, error } = useSWR(
    "https://jsonplaceholder.typicode.com/todos",
    fetcher
  );

  if (!data) {
    return <h2>Loading...</h2>;
  }
  if (error) {
    return <h2>Error: {error.message}</h2>;
  }

  return (
    <>
      <h2>Todos</h2>
      <div>
        {data.map((todo) => (
          <h3 key={todo.id}>{todo.title}</h3>
        ))}
      </div>
    </>
  );
};

export default FetchTodos;
```

Kao što ste naučili u prethodnoj lekciji o prilagođenim hook-ovima (custom hooks), preuzimanje podataka je logika koju možete izvući u prilagođeni hook. Stoga, ako preuzimate podatke u više komponenti i stranica, najbolje je kreirati `useFetch` hook.

Evo `useFetch` hook-a koji koristi SWR za preuzimanje podataka:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const useFetch = (url) => {
  const { data, error } = useSWR(url, fetcher);

  return {
    data,
    loading: !data && !error,
    error,
  };
};

export default useFetch;
```

I evo kako koristiti `useFetch` hook da prepravimo prvi primer koji preuzima postove sa JSON Placeholder API-ja:

```jsx
import useFetch from "./useFetch";

const FetchPosts = () => {
  const { data, loading, error } = useFetch(
    "https://jsonplaceholder.typicode.com/posts"
  );

  if (loading) {
    return <h2>Loading...</h2>;
  }

  if (error) {
    return <h2>{error.message}</h2>;
  }

  return (
    <>
      <h2>Posts</h2>
      <ul>
        {data.map((post) => (
          <li key={post.id}>{post.title}</li>
        ))}
      </ul>
    </>
  );
};

export default FetchPosts;
```
