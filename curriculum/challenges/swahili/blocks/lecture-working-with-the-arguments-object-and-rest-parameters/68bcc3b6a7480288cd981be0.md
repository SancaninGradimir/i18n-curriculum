# --interactive--

Kao što se sećate iz ranijih lekcija i radionica, možete kreirati funkcije sa brojem parametara i pozvati tu funkciju sa argumentima.

Evo podsetnika:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

Ali šta ako imate funkciju koja je pozvana sa više argumenata nego što je definisana da primi?

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript neće baciti grešku u ovom slučaju. Umesto toga, ignoriše dodatni argument i samo sabira brojeve `3` i `4` zajedno. Funkcije koje prihvataju varijabilan broj argumenata poznate su kao <dfn>variadijske funkcije</dfn>.

Ako radiš sa variadičkim funkcijama, onda možeš da koristiš objekat ``arguments``. Ovaj objekat sličan nizu sadrži vrednosti argumenata prosleđenih u funkciju.

Evo primera:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

Pošto je objekat `arguments` sličan nizu, možete pristupiti argumentu na određenom indeksu ovako:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

Takođe možete koristiti svojstvo ``length`` na ovaj način da biste dobili broj argumenata sa kojima je funkcija pozvana:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

Iako objekat `arguments` izgleda da radi kao pravi niz, on nema ugrađene metode kao što su `Array` poput `includes` ili `push`. Da biste pristupili tim metodama, prvo biste trebali da konvertujete objekat `arguments` u pravi niz koristeći nešto poput `slice`, `Array.from()` ili operater raspršivanja:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

Iako je moguće raditi sa objektom ``arguments`` za variadike funkcije, moderne JavaScript aplikacije obično koriste sintaksu parametara ostatka (rest parameters). Vi ćete više o tome naučiti u budućoj lekciji.

Šta je objekat `arguments`?

Specijalni rečnik koji sadrži vrednosti argumenata prosleđenih u funkciju.
### --feedback--

An array-like object that contains the values of the arguments passed into a function.

---

A variable that contains only the first argument passed into a function.

## --text--

Zašto ne možete koristiti ugrađene metode `Array` kao što su `includes` ili `push` na objektu `arguments`?
## --answers--

Objekat `arguments` je uvek prazan osim ako nije eksplicitno inicijalizovan.
### --feedback--

Pogledajte na kraju lekcije za odgovor.
---

JavaScript automatically blocks method calls on the `arguments` object for performance reasons.

### --feedback--

Pogledajte na kraju lekcije za odgovor.
---

The `includes` or `push` methods unreliably work on the `arguments` object so those methods should be avoided.

### --feedback--

Pogledajte na kraju lekcije za odgovor.
---

The `arguments` object is not a real array so it doesn't have those built in methods.
