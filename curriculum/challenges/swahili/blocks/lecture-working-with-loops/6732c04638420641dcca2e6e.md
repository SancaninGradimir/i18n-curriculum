# --interactive--

Petlja ``for...of`` se koristi kada je potrebno prolaziti kroz vrednosti iz iterabilnog objekta. Primeri iterabilnih objekata bi bili nizovi i stringovi.

Evo osnovne sintakse za `for...of` petlju:

```js
for (variable of iterable) {
  // code block to be executed
}
```

Varijabla u primeru predstavlja trenutnu vrednost iterabilnog objekta preko kojeg se prolazi petljom.

Ako imate niz brojeva, varijabla bi bila trenutni broj u nizu. Ako imate string, varijabla bi bio trenutni karakter u tom stringu.

Pogledajmo nekoliko primera kako biste bolje razumeli kako radi petlja `for...of`.

U ovom prvom primeru imamo niz brojeva i želimo da iteriramo kroz svaki broj i logujemo ga u konzolu.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

Kreirali smo varijablu nazvanu `num` koja će predstavljati trenutni broj u nizu. Za iteraciju 1, `num` će biti `1`, za iteraciju 2, `num` će biti `2`, i tako dalje.

Unutar petlje, logujemo trenutni broj na konzolu.

Evo još jednog primera gde imamo string i želimo da iteriramo kroz svaki karakter i logujemo ga u konzolu.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

U ovom primeru, kreirali smo varijablu nazvanu `char` koja će predstavljati trenutni karakter u nizu.

Za svaku iteraciju, petlja će zapisati trenutni karakter u konzolu.

Važno je napomenuti da možete koristiti `let`, ili `const` prilikom deklarisanja varijable u `for...of` petlji.

Ako planirate da koristite ``const`` međutim, uverite se da vrednost varijable ne menja unutar petlje. Ako to učini, dobićete ćete grešku.

Evo primera korišćenja `const` što rezultira greškom:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

U ovom primeru, pokušavamo da promenimo vrednost `num` unutar petlje. Pošto smo deklarisali `num` sa `const`, dobići ćemo grešku. Dakle, ako morate da promenite vrednost varijable unutar petlje, koristite `let` umesto toga.

Pogledajmo još jedan zadnji primer koji radi sa nizom objekata.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

U ovom primeru, imamo niz objekata nazvan `people`. Svaki objekat ima svojstvo `name` i `age`.

Kada iteriramo kroz niz, kreiramo varijablu nazvanu `person` koja će predstavljati trenutni objekat u nizu.

U petlji, ispisujemo poruku na konzolu.

Prva poruka će biti `John is 30 years old`, druga poruka će biti `Jane is 25 years old`, a treća poruka će biti `Jim is 40 years old`.

`for...of` Petlje su zaista korisne kada vam je potrebno da prolazite kroz vrednosti iz iterabilnog objekta, kao što su niz ili string. Takođe su jednostavne za čitanje i mogu učiniti vaš kod konciznijim.
