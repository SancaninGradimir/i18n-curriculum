# --interactive--

U prethodnim lekcijama, naučili ste kako da radite sa `for` petljama, `for...in` petljama i `for...of` petljama. U ovoj lekciji, naučićete o `while` petlju i `do...while` petlju.

Petlja ``while`` će izvršiti blok koda sve dok je uslov istinit. Evo osnovne sintakse za petlju ``while``:

```js
while (condition) {
  // code block to be executed
}
```

Uslov se proverava pre nego što se blok koda izvrši. Ako uslov nije tačan, blok koda neće biti izvršen.

`while` petlje su korisne kada ne znate koliko puta morate pokrenuti blok koda. Evo primera korišćenja `while` petlji:

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

U ovom primeru, imamo varijablu nazvanu `counter` koja je inicijalizovana na `0`. Petlja `while` će nastaviti da radi sve dok je vrednost `counter` manja od `5`. Unutar petlje, logujemo vrednost `counter` na konzolu i zatim inkrementiramo `counter` sa `1`.

Druga petlja slična petlji `while` bila bi petlja `do...while`. Evo osnovne sintakse:

```js
do {
  // code block to be executed
} while (condition);
```

Jedna ključna razlika između `do...while` petlje i `while` petlje je u tome što će `do...while` petlja izvršiti blok koda barem jednom pre proveravanja uslova.

Ako je uslov tačan, blok koda će nastaviti da se izvršava. Ako je uslov netačan, blok koda će prestati sa izvršavanjem.

Evo primera korišćenja petlje `do...while`:

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

U ovom primeru, imamo varijablu nazvanu `counter` koja je inicijalizovana sa `0`. Petlja `do...while` će logovati vrednost `counter` na konzolu, a zatim inkrementirati `counter` sa `1`. Nakon izvršavanja bloka koda, proverava da li je vrednost `counter` manja od `5`. Ako jeste, petlja će nastaviti da radi. Ako nije, petlja će prestati.

U većini slučajeva, verovatno ćete koristiti petlju `while` češće nego petlju `do...while`. Međutim, dobro je znati oba tipa petlji i kada ih koristiti.

Koja je glavna razlika između petlje ``while`` i petlje ``do...while``?
## --answers--

Petlja ``while`` izvršava blok koda barem jednom, dok petlja ``do...while`` to ne radi.
