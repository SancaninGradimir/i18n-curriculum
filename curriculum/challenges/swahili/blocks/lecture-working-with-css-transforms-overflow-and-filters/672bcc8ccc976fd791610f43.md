# --interactive--

CSS svojstvo `transform` je moćan alat koji vam omogućava da modifikujete vizuelnu prezentaciju elemenata na vašoj veb stranici bez uticaja na raspored drugih elemenata. Omogućava vam da primenite različite transformacije na elemente kao što su rotiranje, skaliranje, naginjanje ili translacija (pomicanje) u 2D ili 3D prostoru.

Svojstvo `transform` radi primenom matematičke transformacije na koordinatni sistem elementa. To znači da možete manipulisati oblikom i pozicijom elementa, zadržavajući pritom njegovo originalno mesto i tok dokumenta netaknutima.

Istražićemo neke uobičajene transform funkcije. Evo primera kutijaste elementa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

Postavili smo da ``body`` ima čvrstu crnu ivicu kako biste videli element ``.box`` ugnježdjen unutar elementa ``body``.

Funkcija ``translate`` pomera element sa svoje trenutne pozicije. Evo ažuriranog primera korišćenjem funkcije ``translate``:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

Ovo CSS pravilo će pomeriti element sa klasom `box` za 50 piksela udesno i za 100 piksela dole od svoje originalne pozicije.

Funkcija ``rotate`` rotira element oko fiksne tačke, i ovo je primer korišćenja funkcije ``rotate`` za element ``.box`` iz ranije:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

Ovo će rotirati element za četrdeset pet stepeni u smeru kazaljke na satu.

Funkcija ``scale`` vam omogućava da promenite veličinu elementa. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

Ovo će učiniti elementem koji je pola i po puta širi i dvostruko viši od njegove originalne veličine.

Možete kombinovati više transformacija u jednoj deklaraciji:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

Ovo će pomeriti element za 50 piksela udesno i dole, rotirati ga za 45 stepeni, i skalirati ga da bude pola i po puta njegove originalne veličine.

Iako je svojstvo ``transform`` moćno za kreiranje vizuelno privlačnih dizajna, važno je razmotriti pristupačnost prilikom korišćenja. Evo nekoliko važnih pitanja pristupačnosti o kojima treba pamtiti.

Čitači ekrana možda neće tačno preneti transformisani sadržaj. Na primer, ako koristite ``transform`` za preuređivanje vizuelnog reda elemenata, čitači ekrana će i dalje čitati sadržaj u originalnom DOM redosledu. Ovo može dovesti do zbunjenosti korisnika koji se oslanjaju na čitače ekrana.

Kada koristite `scale` za skaliranje teksta, budite oprezni da ga ne učinite ni previše malim ni previše velikim. Ekstremno mali tekst može biti težak za čitanje, dok preveliki tekst može da prelije svoj kontejner i postane nečitljiv. Generalno je bolje koristiti odgovarajuće tehnike stilizovanja fonta za skaliranje teksta.

Ako koristite ``transform`` za animacione efekte, budite svesni korisnika koji su osetljivi na pokret. Prekomerne ili brze animacije mogu izazvati nelagodu ili čak pokrenuti konvulzije kod nekih ljudi. Razmislite o pružanju načina za korisnike da smanje ili isključe animacije. Kada koristite 3D transformacije, zapamtite da ne svi korisnici percipiraju dubinu na isti način. Osigurajte da su sve kritične informacije prenete kroz 3D efekte takođe dostupne u 2D formatu ili putem teksta.

Ako koristite `transform` za skrivanje ili otkrivanje sadržaja, uverite se da je sadržaj i dalje dostupan čitačima ekrana i navigaciji pomoću tastature. Skriveni sadržaj bi trebalo da bude zaista skriven, na primer korišćenjem `display: none;` ili `visibility: hidden;`, umesto samo vizuelnog pomeranja sa ekrana.

Kada primenite ``transform`` na interaktivne elemente kao što su dugmad ili linkovi, osigurajte da je klikabilna površina intuitivna i lako dostižljiva. Drastično transformisano dugme može biti vizuelno zbunjujuće ili teško za klikanje, posebno za korisnike sa motornim smetnjama.

Zaključno, iako je svojstvo CSS `transform` moćan alat za kreiranje vizuelno dinamičnih web dizajna, esencijalno je koristiti ga odgovorno uzimajući u obzir pristupačnost. Uvek testirajte svoje transformisane elemente sa raznim asistivnim tehnologijama i razmislite o pružanju alternativnih načina pristupa informacijama ili funkcionalnosti koje bi mogle biti pogođene transformacijama.
