# --interactive--

CSS svojstvo ``filter`` je moćan alat koji vam omogućava primenu grafičkih efekata na elemente na veb stranici. Posebno je korisno za podešavanje vizuelne prezentacije slika, pozadina i čak teksta bez modifikovanja originalnog resursa.

Svojstvo ``filter`` se može koristiti za kreiranje raznih efekata, kao što su zamućenje, pomeranje boja i podešavanje kontrasta. Razmotrićemo kako svojstvo `filter` funkcioniše i istražiti neke uobičajene primere. Osnovna sintaksa za svojstvo ``filter`` je jednostavna:

```css
selector {
  filter: function(amount);
}
```

Ovde, `function` predstavlja specifičan filter efekat koji želite primeniti, a `amount` je obično vrednost koja određuje intenzitet efekta. Sada hajde da pogledamo neke uobičajene filter funkcije i njihove upotrebe.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

Funkcija `blur` primenjuje Gaussijsku zamućenost (blur) na element, količina je specificirana u pikselima i predstavlja radijus zamućenja. Ova CSS pravila će primeniti 2-pikselnu zamućenost na sve slike na stranici. Efekat `blur` može biti koristan za stvaranje dubine u vašem dizajnu ili za zaklanjanje delova slike.

Funkcija `brightness` podešava svetlinu elementa. Vrednost od `0%` će učiniti element potpuno crnim, dok će vrednosti preko `100%` povećati osvetljenost.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Ovo CSS pravilo povećava svetlinu elementa slike sa `50%`. Podešavanja svetline mogu se koristiti za oživljavanje slika ili stvaranje efekta blago izbleđene boje.

Funkcija ``grayscale`` konvertuje element u nijansu sive. Iznos je definisan kao procenat, gde ``100%`` je potpuno nijansa sive i ``0%`` ostavlja sliku nepromenjenu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Ovo pravilo će pretvoriti element slike u potpunu nijansu sive boje. `grayscale` može se koristiti za kreiranje vintage izgleda ili smanjenje naglaska na određene elemente na stranici.

TEKST:

Funkcija `sepia` primjenjuje ton sepije na element. Kao i skala sivih boja, koristi procenat vrijednosti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Ovo pravilo primenjuje `80%` sepiju efekat na element slike. Sepija efekat je odličan za stvaranje vintage ili staromodnog izgleda.

Funkcija `hue-rotate` primenjuje rotaciju nijanse na element. Vrednost je definisana u stepenima i predstavlja rotaciju oko kruga boja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Ovo pravilo rotira nijansu elementa slike za `90` stepeni. Rotacija nijanse se može koristiti za kreiranje psihedeličkih efekata ili za prilagođavanje ukupne palete boja slike.

Jedan od najmoćnijih aspekata svojstva `filter` je mogućnost kombinovanja više efekata. Možete primeniti nekoliko filtera na isti element razdvajanjem ih razmakima:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Ovo pravilo primenjuje povećan kontrast, blago povećanu svetlinu i suptilan sepija efekat na element slike.

Kombinovanjem filtera, možete kreirati složene i jedinstvene vizuelne efekte prilagođene vašim dizajnerskim potrebama. Svojstvo CSS filter je svestran alat koji omogućava kreativnu vizuelnu manipulaciju web elementima.

Iako smo pokrili neke od najčešćih filter funkcija, dostupne su i druge, kao što su `contrast`, `invert`, i `saturate`. Kao i kod svake moćne funkcije, važno je biti pažljiv kako koristite filtere da poboljšate dizajn bez preopterećenja korisnika ili kompromitovanja pristupačnosti.
