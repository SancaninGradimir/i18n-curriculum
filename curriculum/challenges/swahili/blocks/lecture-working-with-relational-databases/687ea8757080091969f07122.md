# --description--

SQL znači Structured Query Language. Razvijen je u 1970-ima, kada je prvobitno poznat kao structured English query language (SEQUEL). Ovaj termin je kasnije skraćen u SQL.

SQL je programski jezik koji se koristi za skladištenje i upravljanje podacima u relacionalnim bazama podataka.

Kao što ste naučili u prethodnoj lekciji, relacione baze podataka organizuju podatke u tabele. Svaka tabela ima skup redova i kolona, gde su predstavljeni atributi podataka i njihovi odnosi. Svaki red predstavlja zapis (ili: rekord), dok svaki stub predstavlja specifičan atribut ili polje tog zapisa.

Sa SQL-om, možete dodati, promeniti, obrisati, pronaći i povući podatke iz relacione baze podataka. Pruža sveobuhvatni set komandi za upitovanje, filtriranje, sortiranje i agregovanje podataka.

SQL može pomoći u optimizaciji performansi baze podataka, upitujući samo potrebne podatke, a lako se može integrisati sa širokim spektrom programskih jezika, tako da ga možete koristiti za direktnu interakciju sa bazama podataka iz vaših aplikacija.

Takođe je usvojeno kao industrijski standard. Osigurava integritet podataka i uključuje mnoge sigurnosne funkcije, kao što je podrška za autentifikaciju korisnika i šifrovanje podataka.

Takođe je skalabilno, prenosivo i kompatibilno sa širokim rasponom sistema za upravljanje bazama podataka.

SQL je zasnovan na komandama, koje se često nazivaju SQL izjavama ili SQL upitima.

Kao razvijatelj, pišaćete ove izjave koristeći specifične elemente jezika ili ključne reči SQL-a. Oni vam omogućavaju da izvršite neophodne operacije na vašoj bazi podataka.

Pre nego što unesete SQL komande, morate otvoriti psql shell kako biste direktno interagovali sa PostgreSQL bazom podataka. U komandnoj liniji ili terminalu, unesite:

```bash
psql -U <username> -d <database_name>
```

Zameni `username` sa svojim korisničkim imenom i `database_name` sa bazom podataka kojoj želiš da se povežeš.

Ako još niste kreirali svoju bazu podataka, PostgreSQL automatski uključuje podrazumevanu bazu podataka nazvanu `postgres` kada je instaliran. Možete koristiti ovu bazu podataka `postgres` da pratite.

Nakon povezivanja, videćete da se podsticaj promeni u:

```bash
postgres=#
```

Prikazuje ime baze podataka kojoj ste povezani i čeka na SQL komande.

Pogledaćemo neke od najosnovnijih SQL komandi sa kojima biste trebali da se upoznate. Primeći ćete da se, po konvenciji, SQL komande završavaju tačkom-zarezom (`;`).

Prvo, možete koristiti ovu komandu da kreirate bazu podataka pod nazivom `my_database`:

```sql
CREATE DATABASE my_database;
```

Možete ukucati ``CREATE`` praćen sa ``DATABASE``, i naziv baze podataka koju želite da kreirate. Napomene se da su ključne reči SQL komande napisane velikim slovima.

Zatim, kada imate svoju bazu podataka, možete se na nju povezati.

Način na koji se povezujete na bazu podataka zaista zavisi od okruženja i alata koje koristite.

U komandnoj liniji ili terminalu, možete koristiti metodu prikazanu ranije. U interaktivnom psql šelpu, možete prebacivati baze podataka sa komandom prečica `\c` praćenom imenom baze podataka na ovaj način:

```bash
\c my_database
```

Napomena da je komanda `\c` i druge skraćene komande (one koje počinju kosom crtom) deo psql šele i nisu deo samog SQL jezika, pa ne zahtevaju razmaknicu za dovršavanje komande.

Sada kada ste povezani na vašu novu bazu podataka, videćete da se prompt promenio:

```bash
my_database=#
```

Nakon što kreirate i povežete se na bazu podataka, možete kreirati tabelu sa ključnim rečima: `CREATE TABLE`

```sql
CREATE TABLE products (
  id SERIAL,
  name VARCHAR(255)
);
```

U komandi, napišete ``CREATE TABLE``, praćen nazivom tabele koju želite kreirati. U ovom slučaju, kreiraće tabelu pod nazivom ``products``.

Zatim, u zagradi, pišete imena kolona koje tabela treba da ima i navedite tip podataka vrednosti koje će biti pohranjene u svaku kolonu.

U primeru, tabela će imati dve kolone, kolonu za ID proizvoda i drugu za naziv proizvoda.

Standardna konvencija imenovanja za nazive tabela i kolona je "snake case", što znači pisanje reči malim slovima i razdvajanje ih donjom crtom (`_`). Na primer: `delivery_orders`

Ovo su nekoliko ključnih SQL komandi koje bi trebalo da znate kako biste počeli.

Sa SQL-om, možete lako upitovati, manipulisati i analizirati podatke kako biste doneli informisane odluke i rešili stvarne probleme.
