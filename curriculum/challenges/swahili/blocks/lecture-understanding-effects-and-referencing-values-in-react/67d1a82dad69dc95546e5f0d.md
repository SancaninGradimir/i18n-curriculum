# --description--

U Reactu, može postojati situacija u kojoj vam je potreban direktan pristup DOM elementu. Zde su vam korisni "refs". Refs takođe mogu skladištiti mutabilne vrednosti, ali je stanje (state) bolji izbor za to.

U čistom JavaScriptu, koristili ste metode `getElementById()` i `querySelector()` za pristup elementima DOM-a. Ali u Reactu, koristite refove za pristup elementima u DOM-u.

Jedna od glavnih razlika je u tome što, sa refovima, ne trebate identifikatore kao što su ID-ovi i klase za referenciranje elemenata.

Pa, kako možete kreirati i koristiti reference? React pruža ``useRef()`` hook koji vam to omogućava.

Prvi korak je da se uveze kuka iz React-a:

```js
import { useRef } from "react";
```

Zatim, morate kreirati varijablu koja drži referencu sa početnom vrednošću reference unutar kuka `useRef`, recimo da je `sectionRef` inicijalizovan na `null`:

```js
const sectionRef = useRef(null);
```

Poslednja stvar koju treba uraditi je da povežete varijablu ref sa elementom u vašem JSX-u koristeći atribut `ref`:

```jsx
<section ref={sectionRef}>
  {/* Section content */}
</section>
```

Ako logujete referencu u konzolu, videćete da je objekat sa trenutnom vrednošću, u ovom slučaju, `null`:

```js
console.log(sectionRef); // { current: null }
```

Takođe možete trenutnu vrednost zapisati u konzolu sa svojstvom ``current`` kako biste direktno videli tu vrednost:

```js
console.log(sectionRef.current); // null
```

Naredne vrednosti ref-a zavise od životnog ciklusa komponente.

Na primer, početna vrednost `sectionRef` uvek će biti `null` jer je tako inicijalizovana. Nakon što se komponenta montira, vrednost referenc(e) će biti element `section` kojem je referenca vezana.

Ako je komponenta odmontirana, vrednost referencijalne vrednosti se vraća na početnu vrednost `null`.

Tipičan primer za prikazivanje referencije je fokusiranje elementa za unos pri renderovanju, ili klikom na dugme.

Evo kako to učiniti kada kliknete na dugme:

```jsx
import { useRef } from "react";

const Focus = () => {
  const inputRef = useRef(null);

  const handleFocus = () => {
    if (inputRef.current) {
      inputRef.current.focus();
    }
  };

  return (
    <div>
      <input ref={inputRef} type="text" placeholder="Enter text" />
      <button onClick={handleFocus}>Focus Input</button>
    </div>
  );
};

export default Focus;
```

U kodu iznad, ``inputRef`` je kreiran i priložen ka elementu ``input``. Postoji takođe dugme sa događajem ``onClick`` koje poziva funkciju ``handleFocus``.

Sve što `handleFocus` funkcija radi jeste pozivanje metode `focus()` na elementu `input`. Napomena da je, jer `input` ugrađena komponenta koja dolazi sa Reactom, stvarni `input` DOM element postavljen je na svojstvo `current` referencije. Dakle, pozivate metodu `focus()` sa `input.current.focus()`.

Evo nekih najboljih praksi o kojima biste trebalo da znate dok radite sa referencama:

- Koristite refove pre svega za interakciju sa DOM-om. Možete ih koristiti i za mutabilne podatke, ali je stanje bolji izbor za to.

- Ne koristite ref-ove za osnovno upravljanje stanjem – za to je `useState`.

- Proverite da li `ref.current` postoji pre pristupa njegovim svojstvima. Evo kako ponovo to učiniti:

```js
const handleFocus = () => {
  if (inputRef.current) {
    inputRef.current.focus();
  }
};
```

Ovo sprečava greške u slučaju da se referenca pristupi pre nego što je pričvršćena za DOM ili nakon što je uklonjena.
