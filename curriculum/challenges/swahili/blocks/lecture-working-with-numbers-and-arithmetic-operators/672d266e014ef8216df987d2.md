# --interactive--

Tip ``Number`` je jedan od najčešće korišćenih tipova podataka u JavaScriptu i drugim programskim jezicima. Brojevi mogu delovati jednostavno, ali postoji mnogo toga za istraživanje kada je reč o brojevima u JavaScriptu. Zato, hajde da pogledamo detaljnije. U JavaScriptu, tip podataka ``Number`` predstavlja numeričku vrednost.

Za razliku od mnogih drugih programskih jezika koji razdvajaju cele brojeve i brojeve sa decimalnim tačkama u različite tipove, JavaScript koristi jedan objedinjeni tip ``Number`` za brojeve. To znači da možete raditi sa celim brojevima, decimalama i čak posebnim numeričkim vrednostima sve pod šatorom istog tipa podataka ``Number``.

Evo osnovnog primera koji pokazuje da ceo brojevi, brojevi sa zaetlenom tačkom i negativni brojevi su svi tipa broj:

:::interactive_editor

```js
const wholeNumber = 50;
const decimalNumber = 4.5;
const negativeNumber = -7;

console.log(typeof wholeNumber); // number
console.log(typeof decimalNumber); // number
console.log(typeof negativeNumber); // number
```

:::

Tip `Number` u JavaScriptu uključuje različite vrste numeričkih vrednosti, koje se kreću od jednostavnih celobrojnih brojeva i brojeva sa fiksnim zarezom do posebnih slučajeva kao što su `Infinity` i `NaN`, ili "Not a Number". Razmotrićemo glavne tipove koje ćete naići. Celobrojni brojevi su celi brojevi bez bilo kakvog frakcionog ili decimalnog dela. Mogu biti pozitivni, negativni ili nula. Evo nekih primera:

:::interactive_editor

```js
const positiveInteger = 100;
const negativeInteger = -25;
const zero = 0;

console.log(typeof positiveInteger); // number
console.log(typeof negativeInteger); // number
console.log(typeof zero); // number
```

:::

Brojevi sa plutajućom zarezom su brojevi koji imaju decimalne tačke. Često ih JavaScript programeri nazivaju samo "float"-ovima. Floatovi su korisni kada vam je potrebna veća preciznost, kao što je slučaj kada radite sa merenjima ili valutama. Evo nekih primera:

:::interactive_editor

```js
const floatingPointNumber = 4.5;
const anotherFloat = 89.56;
const oneMoreFloat = 16.462;

console.log(typeof floatingPointNumber); // number
console.log(typeof anotherFloat); // number
console.log(typeof oneMoreFloat); // number
```

:::

JavaScript može predstavljati brojeve koji su izvan maksimalnog limita sa `Infinity`. Susretnete se ovim kada pokušate da podelite broj sa nulom ili u retkim slučajevima, pređete gornju granicu tipa `Number`. Evo primera:

:::interactive_editor

```js
const infiniteNumber = 1 / 0;
console.log(infiniteNumber); // Infinity
console.log(typeof infiniteNumber); // number
```

:::

TEKST:

Ponekad u JavaScript-u, neke matematičke operacije ne rezultiraju u validan broj. Na primer, ako pokušate da izvršite matematičku operaciju na nešto što nije broj, dobićete `NaN`, što znači "Not a Number":

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(notANumber); // NaN
```

:::

Iznenadno, tip `NaN` je takođe i `Number`:

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(typeof notANumber); // number
```

:::

Osim standardnog decimalnog sistema (osnova 10), JavaScript takođe podržava brojeve u različitim bazama kao što su binarni, oktalni i heksadecimalni. Binarni sistem je sistem sa osnovom 2 koji koristi samo cifre 1 i 0. Oktalni sistem je sistem sa osnovom 8 koji koristi samo cifre od 0 do 7. Heksadecimalni sistem je sistem sa osnovom 16 koji koristi cifre od 0 do 9 i slova od a do f, kao što vidite u CSS heksadecimalnim bojama.
