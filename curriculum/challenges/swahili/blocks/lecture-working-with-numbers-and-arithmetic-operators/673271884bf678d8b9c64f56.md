# --interactive--

JavaScript pruža alate za obavljanje osnovnih aritmetičkih operacija na brojevima, kao što su sabiranje, oduzimanje, množenje i deljenje. JavaScript takođe uključuje operatore za složenije aritmetičke operacije, kao što su ostatak i eksponencijacija.

Svi ovi alati nazivaju se aritmetički operateri. Pogledajmo ove operatore detaljno, kako ih koristiti i kako ih kombinovati.

Operator za sabiranje je znak plusa (`+`). Operator za sabiranje vam omogućava da pronađete zbir dve ili više brojeva. U operacijama sabiranja, redosled brojeva nije bitan:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

Operator za oduzimanje je znak minusa (`-`). To vam omogućava da pronađete razliku između dva broja. Koristite znak minusa kada želite oduzeti broj od drugog broja, obično manji od većeg:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

Ako manji broj dođe prvi, dobićete negativan rezultat:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

Takođe možete dodeliti brojeve varijable i izvršiti oduzimanje sa nazivima promenljivih:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

U JavaScriptu, operator za množenje je predstavljen zvezdikom (`*`) i koristi se za pronalaženje proizvoda dve ili više brojeva. Redosled brojeva koje množite nije bitan:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

U JavaScriptu, operator za deljenje je kosina (`/`), što se razlikuje od simbola za deljenje korišćenog u tradicionalnoj matematici (`÷`). Izvršavate operacije deljenja pomoću operatora za deljenje. Redosled brojeva koje delite je važan u ovom slučaju:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

Važno je napomenuti da ako pokušate da podelite sa nula, JavaScript će vratiti ``Infinity``:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

Obavezno izbegavajte takve vrste kalkulacija kako ne biste dobili neočekivane rezultate u vašem kodu.

Operator za ostatak, predstavljen znakom procenata (`%`), vraća ostatak deljenja. Ostatak u matematici je preostala vrednost nakon izvršenja deljenja:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

Operator potenciranja, predstavljen dvostrukim asteriskom (`**`), podiže jedan broj na stepen drugog:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

Moguće je kombinovati operatore u jednoj operaciji ili izrazu:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

Kada pomešate različite operatore u jednom izrazu, JavaScript interegirinski motor prati sistem koji se zove prioritet operatera kako bi utvrdio redosled operacija. Prioritet operatera određuje redosled u kojem se operacije izvršavaju u izrazima. Vi ćete saznati više o prioritetu operatera u budućim lekcijama.
