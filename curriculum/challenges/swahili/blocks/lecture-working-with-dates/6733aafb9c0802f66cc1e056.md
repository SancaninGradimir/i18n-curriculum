# --interactive--

Datumi i vremena u JavaScriptu nisu uvek bila laka za rad. U profesionalnom kodu, verovatno ćete koristiti biblioteku koja je rešila mnogo tih problema za vas.

Ali će biti vremena kada ćete morati da radite sa ugrađenim objektom JavaScript-a `Date`.

U ovoj lekciji, pregledaćemo osnove objekta `Date` i neke od najčešćih metoda koje ćete koristiti kada radite sa datumima i vremenima u JavaScriptu.

Objekat `Date` je ugrađeni objekat u JavaScriptu koji vam omogućava da radite sa datumima i vremenima. Evo osnovne sintakse za kreiranje novog objekta `Date`:

```js
const now = new Date();
```

Ključna reč ``new`` se koristi za kreiranje nove instance objekta ``Date``, a objekat ``Date`` se zatim dodeljuje promenljivoj ``now``. Ako biste zapisali vrednost ``now`` u konzolu, videli biste trenutni datum i vreme na osnovu sistemskog sata računara koji pokreće kod.

Evo je primer odgovora koji biste videli ako ste logovali vrednost `now` na konzolu:

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

Za vreme, koristi se vojni vremenski format, pa je `14:30` = `2:30 PM`. `GMT-0700` je pomak vremenske zone, a `Pacific Daylight Time` je naziv vremenske zone.

Takođe možete proslediti određeni datum i vreme objektu `Date` pružanjem vrednosti godine, meseca, dana, sata, minute, sekunde i milisekunde kao argumenata.

Ovo je primer kreiranja novog objekta `Date` sa određenim datumom i vremenom:

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

Ovo je korisno kada trebate raditi sa određenim datumom i vremenom, umesto trenutnog datuma i vremena. Ulaz uvek mora biti niz (string), a format bi trebalo da bude prepoznatljiv od strane objekta `Date`.

Da biste dobili trenutni datum i vreme, možete koristiti metod `Date.now()`, koji vraća broj milisekundi od `January 1, 1970, 00:00:00 UTC`. Ovo je poznato kao Unix epoha vreme.

Unix epoha vreme je uobičajen način za predstavljanje datuma i vremena u računarskim sistemima jer je ceo broj koji se lako može sačuvati i manipulisati. UTC znači Univerzalno koordinisano vreme, što je primarni vremenski standard kojim svet reguliše satove i vreme.

Ako trebate dobiti dan meseca na osnovu trenutnog datuma, možete koristiti metod `getDate`. Evo primera korišćenja metoda `getDate`:

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

U ovom primeru, kreiramo novu instancu datuma koristeći objekat `Date` i dodeljujemo je promenljivoj `now`. Zatim pozivamo metod `getDate` na objektu `now` da bismo dobili dan meseca i dodeljujemo ga promenljivoj `date`. Na kraju, logujemo vrednost `date` na konzolu, što će ispisati trenutni dan meseca.

`getDate` vraća celobrojnu vrednost između `1` i `31`, u zavisnosti od dana u mesecu. Ako je datum nevažeći, vraća `NaN` (Nije broj).

Da biste dobili mesec, možete koristiti metod `getMonth` na ovaj način:

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

Mesec je baziran na nuli, pa `January` jeste `0`, `February` jeste `1`, i tako dalje. Ako mesec nije validan, vratiće `NaN`.

Ako vam je potrebno da dobijete punu godinu, onda možete koristiti metodu `getFullYear` na ovaj način:

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

Ako je godina nevažeća, vratiće `NaN`.

Postoji mnogo više metoda dostupnih na objektu `Date` uključujući `getHours`, `getMinutes`, `getSeconds` i slično. Podstičem vas da sami istražite još mnogo toga kroz dokumentaciju Mozille ili druge resurse.
