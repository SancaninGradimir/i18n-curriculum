# --interactive--

Prethodne lekcije su vam predstavile `console.log()` , ali će ova lekcija detaljnije istražiti njegovu svrhu i upotrebu.

U JavaScriptu, ``console.log()`` je jednostavan, ali moćan alat koji se koristi za prikaz poruka ili izlaz informacija u konzolu pretraživača. Najčešće ga programeri koriste za debagovanje i inspekciju koda dok rade na svojim programima.

Možete koristiti ``console.log()`` za logovanje teksta ili promenljivih u konzolu i osigurati da vaš kod radi ispravno.

Da biste koristili ``console.log()``, pozivate metodu sa vrednošću ili porukom koju želite da izbacite unutar zagrada. Evo nekoliko primera:

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

Prvi primer ispisuje ``Hello, world!`` u konzoli pretraživača, dok drugi primer ispisuje vrednost ``5``.

Evo još jednog primera rada sa `console.log()`:

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

Takođe možete proslediti više vrednosti na ``console.log()`` razdvojene zarezima. Na primer:

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

Ovo je korisno za logovanje više delova informacija odjednom.

Metoda `console.log()` vam pomaže da pratite svoj kod dok radi, što olakšava uočavanje grešaka i razumevanje kako vaš program funkcioniše.
