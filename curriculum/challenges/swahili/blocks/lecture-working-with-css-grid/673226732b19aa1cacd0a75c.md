# --interactive--

U prethodnoj lekciji, predstavili smo vam CSS mrežu (grid) koja se može koristiti za kreiranje kompleksnih i fluidnih rasporeda na vašim veb stranicama. U ovoj lekciji, istraživaćemo kako kreirati fleksibilne mrežne rasporede koristeći jedinicu ``fr``.

Počnimo sa ovim HTML oznakama koje će predstavljati naš kontejner mreže:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

Unutar CSS-a, postavljamo svojstvo 'display' na 'grid' za kontejner.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` se koristi za postavljanje veličine za svaku kolonu. U ovom slučaju, veličina svake kolone biće `25%` kontejnera. Zatim se svojstvo 'gap' koristi za kreiranje prostora oko svake kolone.

Do sada smo koristili procenat za veličinu kolone, ali možemo koristiti i jedinicu ``fr``.

`fr` jedinica je frakciona jedinica koja predstavlja razlomak prostora za grid kontejner.

Evo kako će kod izgledati kada se refaktoriše da koristi jedinice `fr` umesto procenata.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

Svaka kolona će zauzeti jednu frakciju dostupnog prostora. Pošto ima četiri kolone, svaka kolona će imati jednak deo prostora dostupan u grid kontejneru.

Kada počnete da gradite svoje mrežne rasporede, shvatićete da želite češće koristiti `fr` jedinice jer pružaju fleksibilan, proporcionalan način za raspodelu prostora, omogućavajući vam da kreirate responsivne rasporede koji se prilagođavaju različitim veličinama ekrana bez potrebe za ručnim podešavanjem vrednosti piksela.
