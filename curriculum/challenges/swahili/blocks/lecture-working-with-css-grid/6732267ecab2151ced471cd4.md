# --interactive--

U prethodnim lekcijama, malo smo pričali o tome kako stvoriti razmak između elemenata u mrežici. Ali u ovoj lekciji, detaljnije ćemo se baviti kako koristiti svojstva `row-gap`, `column-gap` i `gap` u mrežnom rasporedu.

Ali prvo moramo da pregledamo šta je "track" u CSS mreži (gridu). Track je prostor između dve susedne linije mreže. Ove linije se automatski kreiraju kada koristite CSS Grid. U ovom kontekstu, track-ovi generalno se odnose na redove i kolone koje čine raspored mreže.

Da biste kreirali razmake između kolona u CSS mrežici, možete koristiti svojstvo `column-gap`. Prihvatljive vrednosti za ovo svojstvo uključuju piksele, jedinicu `em`, procenat ili ključnu reč `normal`.

Ako koristite vrednost `normal` za svojstvo `column-gap`, rezultat će biti `0` za mrežne rasporede i `1em` za višestubične rasporede.

Evo primera oznake za raspored mreže od četiri kolone:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

Za CSS, postavili smo display na grid i svojstvo `column-gap` na 10 piksela:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

Ako bismo hteli da promenimo primer tako da ima dva reda plavih kutija i stvorimo više prostora između redova, možemo koristiti svojstvo `row-gap`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

U ovom revidiranom primeru, postavljamo svojstvo `row-gap` na 30 piksela i menjamo `grid-template-columns` da koristi samo dve jedinice `1fr` umesto četiri kako bismo kreirali dva reda kutija.

Slično kao što je svojstvo ``column-gap``, prihvatljive vrednosti za svojstvo ``row-gap`` mogu uključivati procenata, ``em``, i piksele.

Ako želite da koristite skraćeni način za kreiranje razmaka između redova i kolona, možete koristiti svojstvo ``gap``. Evo osnovne sintakse:

```css
gap: row-value optional-column-value;
```

Ako navedete jednu vrednost za svojstvo ``gap``, ta će vrednost biti primenjena na redove i kolone. Ako navedete dve vrednosti, prva će ići u red, a druga će biti primenjena na kolonu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Prihvatljive vrednosti za `gap` skraćeni atribut uključuju procenate, piksele, `em` i čak `calc()` vrednosti. Ali ne možete koristiti `fr` jedinice ovde.

Svojstva `row-gap`, `column-gap` i `gap` pružaju fleksibilne načine za kontrolisanje razmaka između elemenata u CSS Grid rasporedu. Koristeći ova svojstva, lako možete kreirati vizuelno privlačne mreže sa doslednim i podesivim razmakima između redova i kolona.
