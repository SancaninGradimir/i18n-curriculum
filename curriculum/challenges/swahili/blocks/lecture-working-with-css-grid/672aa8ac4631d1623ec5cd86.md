# --interactive--

CSS Grid je moćan sistem rasporeda koji omogućava web programerima da sa lakoćom kreiraju složene i responzivne rasporede veb stranica.

Zamislite da raspoređujete nameštaj u sobi – CSS Grid je kao imati nevidljivu mrežu na podu koja vam pomaže da sve pozicionirate tačno tamo gde želite.

Kada gradimo veb sajtove, često nam je potrebno da rasporedimo različite elemente na stranici.

Pre CSS Grida, ovo je nekada bilo malo komplikovano, pogotovo za složene rasporede. CSS Grid pojednostavljuje ovaj proces deleći vašu veb stranicu na redove i kolone, stvarajući strukturu sličnu mrežici.

Zamislite da radite sa kontejnerom `div` sa nekoliko stavki ugnježdene unutra na ovaj način:

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

Ako želite da stilizujete te elemente u mrežnom formatu, možete postaviti `display` na `grid` i primeniti kolone ovako:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

U ovom kodu, kažemo pretraživaču da kreira mrežu sa tri kolone jednake širine, to je ono što znači `1fr 1fr 1fr`, i dodajemo razmak od 20 piksela između svakog elementa mreže.

Sada, možda se pitate: "Šta je sa Flexbox-om? Zar se to ne koristi i za rasporede?"

U pravu si! Flexbox je još jedan CSS model rasporeda, i takođe je prilično koristan. Ali postoje neke ključne razlike.

Flexbox je jednodimenzionalan, dok je Grid dvodimenzionalan. To znači da je Flexbox odličan za raspored elemenata u jednom redu ili koloni, dok se Grid ističe kreiranjem rasporeda koji imaju i redove i kolone.

Flexbox je zasnovan na sadržaju, što znači da prilagođava raspored na osnovu sadržaja. Grid, s druge strane, je zasnovan na rasporedu, omogućavajući vam da kreirate raspored, a zatim postavite stavke u njega. Grid vam daje precizniju kontrolu nad pozicioniranjem. Možete reći stavci tačno koji red i kolonu da zauzme.

Evo primera Flexbox-a za poređenje:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

Ovo stvara fleks kontejner gde su elementi ravnomerno raspoređeni duž glavne ose.

I Grid i Flexbox imaju svoje snage, a često najbolje raspoređivanje koristi kombinaciju oba. Možete koristiti Grid za ukupni raspored stranice, a zatim koristiti Flexbox za poravnavanje elemenata unutar svake mrežne oblasti.

Zaključno, CSS Grid je moćan alat koji omogućava precizne, dvodimenzionalne rasporede. Iako na prvi pogled može delovati složeno, vežbanjem postaje neprocenjiv alat za kreiranje responsivnih i kompleksnih veb rasporeda.

`fr` znači `fraction`, i radi se o podeli dostupnog prostora.
