# --interactive--

Svojstva `grid-column` i `grid-row` omogućavaju vam da specificirate horizontalno i vertikalno postavljanje elemenata mreže unutar rasporeda mrežice.

Drugim rečima, oba vam omogućavaju da kontrolišete gde gridi stavka počinje i gde se završava referenciranjem linija mreže. Ove linije mreže su granice koje razdvajaju redove i kolone koje ste već definisali koristeći svojstva `grid-template-rows` i `grid-template-columns`.

Pogledajmo sintakse oba `grid-row` i `grid-column`, kao i praktičan primer.

Evo sintakse za `grid-row` svojstvo:

```css
grid-row: <start-line> / <end-line>;
```

I od `grid-column`:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` je mrežna linija gde se stavak počinje i `<end-line>` je mrežna linija gde se stavak završava. Oboje su indeksirani od 1, što znači da počinjete sa brojanje od 1, a ne od 0.

Zapamtite, linije mreže za redove generišu se na osnovu broja redova navedenog u svojstvu `grid-template-rows`. Isto važi i za kolone sa svojstvom `grid-template-columns`.

Za praktični primer, počnimo sa redovima koje generiše pregledač kada koristite oba svojstva, `grid-template-rows` i `grid-template-columns`.

Evo primjera mreže sa 4 kolone i 3 reda:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

Pregledom grid kontejnera (klasa `grid`) pokazuje da svaki red i kolona su ograničeni dve linije – početna linija na početku reda ili kolone, i krajnja linija na kraju reda ili kolone.

Možete ciljati ove linije kako biste počeli da koristite svojstva `grid-row` i `grid-column` da utvrdite gde treba da postavite stavku.

Evo kako možete učiniti da prvi element mreže zauzme prve dve kolone:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;

  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;

  grid-column: 1 / 3;
}
```

:::

Ovim, kažete da prvi element mreže treba početi u koloni 1 i završiti se neposredno pre kolone 3.

Prvi stavak će sada zauzimati dve kolone, a četvrti stavak se pomera u drugi red. Super!
