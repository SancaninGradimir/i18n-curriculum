## Pregled grafova

Graf je skup čvorova (vrhova) povezanih ivicama (vezama). Svaki čvor može da se poveže na više drugih čvorova, formirajući mrežu. Različiti tipovi grafova uključuju:

- **Usmerene:** ivice imaju smer (od jednog čvora ka drugom), često predstavljene pravim linijama i strelicama.
- **Nesmerne:** ivice nemaju smer, predstavljene jednostavnim linijama.
- **Verteks (Čvor):** svakom čvoru je pridružena oznaka ili identifikator.
- **Cikličan:** sadrži cikluse (putanju koja počinje i završava se na istom čvoru).
- **Acikličan (DAG):** ne sadrži cikluse.
- **Ivica sa oznakom:** svaka ivica ima oznaku koja je obično crtana pored odgovarajuće ivice.
- **Težki/Sa težinama:** ivice imaju težine (vrednosti) povezane sa njima, koje se mogu koristiti za izvođenje aritmetičkih operacija.
- **Diskonektovan:** sadrži dva ili više čvorova koja nisu povezana nijednom ivicom.

Grafovi se koriste u raznim aplikacijama, kao što su mape, mreže, sistemi preporuka i rešavanje zavisnosti.

## Traversije grafova

Ovaj proces podrazumeva posetu svih čvorova u grafu. Dva glavna algoritma su:

- **Pretraga po širini (Breadth-First Search - BFS)**
  - Koristi red (queue).
  - Istražuje sloj po sloj.
  - Pronalazi najkraći put u neponderisanim grafovima.

- **Pretraživanje u dubinu (DFS)**
  - Koristi stogu (ili rekurziju).
  - Istražuje celu granu pre nego što izvrši povratak (backtracking).
  - Korisno za detekciju ciklusa i pronalaženje putanja.

## Predstavljanja Grafova

Grafovi se mogu predstaviti na dva glavna načina:

- **Lista susednosti**
  - Svaki čvor poseduje listu svojih suseda.
  - Prostorno je efikasan za razređene grafove.
  - Lako je iterirati kroz susede.

- **Matrica susednosti**
  - Dvodimenzionalni niz gde redovi i kolone predstavljaju čvorove.
  - Visoka potrošnja prostora za velike grafove.
  - Brzo proveriti da li postoji ivica između dva čvora.

## Drveće

Drvo je poseban tip grafa koji je bez ciklusa i spojen. Ključne karakteristike uključuju:

- Ne sadrže cikluse (putanju gde je početni i krajnji čvor isti).
- Moraju biti povezani/spojeni (iz svakog čvora se može doći do svakog drugog čvora).

### Uobičajeni tipovi stabala

Najčešći tipovi stabala su:

- Binarni stablovi
  - Svaki čvor ima najviše dva deteta: levo i desno dete.

- Binarna stabla pretrage (BST)
  - Binarno stablo u kojem je svaki levi potomak manji od svog roditelja, a svaki desni potomak veći od svog roditelja.

## Trijes

Takođe poznati kao prefiksni stablići, koriste se za skladištenje skupova stringova, gde svaki čvor predstavlja karakter.

Deljeni prefiksi se čuvaju samo jednom, što ih čini efikasnim za zadatke kao što su automatsko dovršavanje i proveravanje pravopisa.

Operacije pretraživanja i ubacivanja imaju vremensku složenost O(L), gde je L dužina niza.

## Kue sa prioritetom

Prioritetna cola je apstraktni tip podataka gde svaki element ima prioritet.

Redovi i stogovi razmatraju samo redosled unosa, dok prioritetni redovi razmatraju prioritet elemenata.

Standardni redovi prate princip FIFO (First In First Out – Prvi uđi, prvi izađe), a stogovi prate LIFO (Last In First Out – Poslednji uđi, prvi izađe). Međutim, u prioritetnom redu, elementi sa višim prioritetom obrađuju se pre elemenata sa nižim prioritetom, bez obzira na red njihovog ukrcavanja.

## Kupovi

To je specijalizovana struktura podataka bazirana na stablu koja ima veoma specifično svojstvo nazvano „heap property“.

Svojstvo heapa određuje odnos između roditeljskih i detiljih čvorova. Postoje dva tipa heapa:

- Max-Heap
  - Vrednost svakog roditeljskog čvora je veća ili jednaka vrednostima njegovih potomaka.
  - Najveći element se nalazi na korenu.

- Min-Heap
  - Vrednost svakog roditeljskog čvora je manja ili jednaka vrednostima njegovih potomaka.
  - Najmanji element se nalazi na korenu.

### Primer modula za Python `heapq`

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### Korišćenje prioriteta

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```
