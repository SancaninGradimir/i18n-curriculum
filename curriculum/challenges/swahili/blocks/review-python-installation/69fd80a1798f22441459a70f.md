## Python u Vašem Lokalnom Okruženju

- **Instalacija**: Najbolji način za instaliranje Pythona na Windows, Mac i Linux je preuzimanje instalatora sa zvaničnog Python sajta (`https://www.python.org/`).
- **Terminal**: Tekstualni interfejs koji vam omogućava da komunicirate sa računarom unosom komandi. Svaki operativni sistem dolazi sa podrazumevanom aplikacijom terminala. Na macOS, možete koristiti Terminal aplikaciju. Na Windows-u, možete koristiti Command Prompt ili PowerShell. Na Linux-u, svako desktop okruženje ima svoju podrazumevanu aplikaciju terminala, kao što su GNOME Terminal ili Konsole.
- **IDE**: IDE je skraćenica za Integrated Development Environment (Integrisovano razvojno okruženje) i ima funkcije koje uključuju editor koda, alate za testiranje, terminal i još mnogo toga.
- **Popularni editori koda i IDE-ovi za Python**: VS Code, PyCharm i Spyder
- **Pokretanje koda lokalno**: Da biste pokrenuli Python kod, imate nekoliko opcija. Prva opcija je da kliknete na dugme "run" (pokreni) u VS Code-u, koje izgleda kao dugme za reprodukciju i nalazi se u gornjem desnom uglu. To će otvoriti terminal i izvršiti vaš Python skript tamo.
