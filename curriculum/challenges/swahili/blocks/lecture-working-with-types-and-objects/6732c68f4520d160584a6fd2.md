# --interactive--

Metoda `toString()` je fundamentalna karakteristika u JavaScriptu koja konvertuje vrednost u njeno string predstavljanje. To je metoda koju možete koristiti za brojeve, boolean-e, nizove i objekte. Jedna od najčešćih upotreba `toString()` je konverzija broja u njegovo string predstavljanje. Evo primera:

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

Ova metoda prima opcionalni radiks koji je broj od `2` do `36`. Ovaj radiks predstavlja bazu, kao što su baza 2 za binarni ili baza 8 za oktalni sistem. Ako radiks nije navedeno, podrazumevano je baza 10, što je decimalni sistem. Evo primera prosleđivanja `2` kao argumenta metodi `toString()`:

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

Rezultat će biti `1010` što je binarna reprezentacija za decimalni broj `10`.

Takođe možete koristiti metodu `toString()` za konverziju nizova i objekata u stringove. Nizovi imaju prilagođenu implementaciju `toString()`, koja svaki element konvertuje u string i spaja ih zarezima. Evo primera:

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

U ovom primeru svi elementi niza su spojeni zajedno da bi formirali string ``1,2,3``.

Kada se metoda ``toString()`` koristi sa objektima, rezultat neće biti stringifikovana verzija svojstava objekta.

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

U ovom primeru, rezultat će biti podrazumevana string reprezentacija za objekat koji je `[object Object]`. Da biste dobili stringifikovanu verziju svojstava objekta `person`, moraćete da koristite `JSON.stringify()`, o čemu ćete saznati više u budućim lekcijama.

Zaključno, metoda `toString()` se koristi za konvertovanje vrednosti u stringove. Razumevanje kako radi sa različitim tipovima podataka i kako ga prilagoditi sopstvenim objektima može značajno unaprediti vašu sposobnost manipulisanja i prikazivanja podataka u vašim JavaScript aplikacijama.
