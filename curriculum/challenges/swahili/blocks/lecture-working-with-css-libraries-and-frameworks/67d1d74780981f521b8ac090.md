# --description--

Kako radite sa CSS-om, možete naići na određene izazove, kao što su održavanje konzistentnih i kompatibilnih stilova između pregledača, izbegavanje ponavljanja i upravljanje rastućom složenošću i brojem CSS fajlova u vašem projektu. CSS okviri (frameworks) veoma pomažu da se ovi problemi spreče od samog početka.

CSS okvir sastoji se od unapred napisanog CSS koda koji možete koristiti za stilizovanje svojih HTML elemenata.

Korišćenjem CSS okvira, možete ubrzati svoj radni tok, kreirati uniforman vizuelni stil širom veb-sajta, učiniti da vaš dizajn izgleda konzistentno u više pregledača i ukupno održavati svoj CSS kod organizovanijim.

Možete zamisliti CSS okvir kao alatnu kutiju gde možete brzo pronaći stil ili komponentu koja odgovara vašim potrebama i odmah je koristiti, bez potrebe da ga sami definišete korišćenjem CSS svojstava. Ovi okviri često uključuju unapred definisane stilove i komponente koje možete vrlo lako koristiti, uključujući stilove za kreiranje responsivnih rasporeda.

Neki od najpopularnijih CSS okvira su Tailwind CSS, Bootstrap, Materialize i Foundation.

Pričaćemo malo o dve vrste CSS okvira: utility-first (prvi po pitanju utilitarnosti) CSS okviri i komponentno zasnovani CSS okviri.

Tailwind CSS je utility-first CSS okvir. To znači da ima male klase sa veoma specifičnim namenama, kao što su postavljanje margine, paddinga ili pozadinske boje. Ove male klase obično možete direktno dodeliti HTML elementima.

Na primer:

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

Ovaj dugme će imati plavu pozadinu sa belim i podebljanim tekstom, vertikalni padding od 2 rem, horizontalni padding od 4 rem i zaobljene ivice. Klasa `hover:bg-blue-700` će promeniti pozadinsku boju u tamno plavu kada korisnik pređe mišem preko nje.

Kao što vidite, klase Tailwind CSS su veoma granularne, tako da se mogu primeniti na pojedinačne HTML elemente po potrebi.

Za razliku od toga, Bootstrap je komponentno zasnovani CSS okvir. Ima unapred napravljene komponente sa unapred definisanim stilovima koje možete lako dodati svom veb-sajtu. Ove komponente su obično dostupne u zvaničnoj dokumentaciji CSS okvira, gde ih možete brzo kopirati i zalepiti u svoj projekat.

Na primer, ako želite da kreirate grupu stavki liste (list group) u Bootstrapu, napisali biste nešto slično ovome u HTML-u:

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

Ove unapred definisane klase automatski će primeniti standardne stilove na karticu (card), grupu stavki liste i stavke grupe liste.

Primećujte da ovde, umesto dodavanja malih i opštih klasa, dodajemo celu komponentu, uključujući HTML strukturu. Zato je Bootstrap komponentno zasnovan.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

Iako CSS okviri imaju mnogo prednosti, imaju i određene nedostatke. Važno je da znate oba kako biste doneli informisanu odluku.

Pošto se CSS okviri oslanjaju na unapred definisane stilove, ako pokušate da dodate prilagođeni CSS preko njih, mogu nastati problemi sa specifičnošću (specificity). Selektori koje koristi okvir mogu imati višu specifičnost od selektora koje koristite u vašem prilagođenom CSS-u, pa možda neće biti prikazan ispravan stil. Da biste to rešili, moraćete da koristite specifičniji CSS selektor, što može predstavljati dodatne izazove.

Takođe mogu ograničiti jedinstvenost do određene mere jer su njihovi standardni stilovi široko korišćeni na mnogim veb-sajtovima. Međutim, ovo se može rešiti prilagođavanjem CSS-a sa vašim jedinstvenim stilovima.

I konačno, trebalo biste da razmotrite potencijalne probleme sa performansama koje CSS okviri mogu uvesti. Neki okviri su prilično veliki i mogu potrajati neko vreme za učitavanje.

Kao developer, odlučićete da li je korišćenje CSS okvira korisno za vaš projekat, da li je bolje koristiti potpuno prilagođeni CSS ili možda mešavinu oba.
