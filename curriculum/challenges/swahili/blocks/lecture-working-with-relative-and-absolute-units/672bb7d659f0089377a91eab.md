# --interactive--

Procenti u CSS-u su relativne jedinice koje vam omogućavaju da definišete veličine, dimenzije i druge atribute kao proporciju svog roditeljskog elementa. Kada koristite procenatnu vrednost, suštinski kažete: "Neka ovo bude X% njegovog kontejnera."

Na primer, ako postavite ``width: 50%`` na element, zauzimaće pola širine svog roditeljskog kontejnera. To čini da su procenti neverovatno korisni za kreiranje responsivnog dizajna koji se prilagođava različitim veličinama ekrana.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

Procenati su idealni za kreiranje fluidnih rasporeda koji se prilagođavaju različitim veličinama ekrana. Na primer, postavljanje kontejnera na `width: 80%` osigurava da zauzima 80% širine svog roditelja, bez obzira na uređaj.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

Korišćenje procenata za fleksibilne slike je još jedna uobičajena praksa. Primene ``max-width: 100%`` na slike omogućava vam da se smanje na manjim ekranima dok održavaju svoj aspekt odnos.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

Iako je manje uobičajeno, procenati se takođe mogu koristiti za veličine fonta kako bi se kreirala skalabilna tipografija. Na primer, ``font-size: 120%`` bi učinio da je tekst 20% veći od veličine fonta svog roditelja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

Procenati mogu biti posebno korisni za vertikalno centriranje. Evo primera kako biste mogli da koristite procenate sa svojstvom `transform` za vertikalno centriranje elementa.

Ovaj primer pozicionira element na 50% od vrha svog kontejnera, a zatim koristi `transform` da ga pomeri nazad gore za pola njegove sopstvene visine, efektivno centrirajući ga vertikalno.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

Naučićete više o tome kako funkcionišu apsolutno pozicioniranje i svojstva transformacije u većoj detaljnosti u budućim lekcijama.

Zapamtite, procenati su uvek relativni u odnosu na nešto. Za horizontalne svojstva poput `width`, oni su relativni u odnosu na širinu roditelja. Za vertikalna svojstva poput `height`, obično su relativni u odnosu na visinu roditelja (ako je navedeno).

Međutim, budite oprezni prilikom ugnježđivanja elemenata sa dimenzijama zasnovanim na procentima, jer to može dovesti do neočekivanih rezultata. Takođe, imajte na umu da visine zasnovane na procentima mogu biti komplikovane ako roditelj nema definisanu visinu.

Zaključno, procenati u CSS-u su moćni alati za kreiranje fleksibilnih, responsivnih dizajna. Koristite ih kada želite da elementi skaliraju proporcionalno u odnosu na svoje kontejnere ili kada vam je potreban raspored koji se besprekorno prilagođava različitim veličinama ekrana.
