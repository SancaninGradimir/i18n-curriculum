# --interactive--

Kada radite sa CSS-om kako biste stilizovali web stranicu, jedan od najčešćih metoda za definisanje boja je korišćenje heksadecimalnih (hex) vrednosti boja. Heksadecimalne vrednosti boja, ili hex kodovi, su koncizan način za predstavljanje boja u RGB modelu boja. U ovoj lekciji, istraživaćemo šta su hex kodovi, kako funkcionišu u CSS-u i kada ih koristiti.

Hex kod (skraćeno od heksadecimalnog koda) je šestokarakterni niz koji se koristi za predstavljanje boja u RGB modelu boja. „Hex“ se odnosi na sistem brojanja sa bazom 16, koji koristi cifre `0` do `9` i slova `A` do `F`.

U kontekstu boja, heks kodovi određuju količine crvene, zelene i plave (RGB) koje čine određenu boju. Svaki heks kod počinje sa znakom '#' (`#`), praćen šest karaktera koji definišu intenzitet crvene, zelene i plave u boji.

Na primer, šesnaestični kod za crnu je `#000000`, a šesnaestični kod za belu je `#FFFFFF`. Prva dva karaktera predstavljaju vrednost crvene boje, sledeća dva predstavljaju zelenu boju, a poslednja dva predstavljaju plavu boju.

U CSS-u, možete primeniti boje na elemente korišćenjem heksadecimalnih kodova. Evo osnovne sintakse:

```css
element {
  color: #RRGGBB;
}
```

Ovde, `#RRGGBB` je zamena za stvarni heks kod. Parovi `RR`, `GG` i `BB` svaki predstavljaju intenzitet crvenila, zelenila i plavog, respektivno.

Ovi parovi mogu varirati od `00` (najniži intenzitet) do `FF` (najviši intenzitet). Što je broj veći, to će biti prisutnije više te boje u konačnoj mešavini. Na primer, primenimo neke heks boje na nekoliko elemenata:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

U prvom slučaju, `#FF5733` je crvenkasto-narandžasta boja. Evo raščlambe:

- `FF` je najviša vrednost za crvenu boju (potpuno zasićena).
- `57` je umerena količina zelene boje.
- `33` je mala količina plave boje.

U drugom slučaju, `#4CAF50` je nijansa zelene:

- `4C` je niska vrednost za crvenu boju.
- `AF` je snažna vrednost za zelenu boju.
- `50` je umerena vrednost za plavu boju.

Šesadecimalni kodovi su popularni jer su precizni i laki za upotrebu. Omogućavaju vam da fino podesite svoje boje, što ih čini idealnim za dizajne gde je konzistentnost važna.

Većina softvera za dizajn, kao što su Adobe Photoshop ili Figma, pruža heks kodove za boje koje izaberete, što olakšava kopiranje i lepljenje ovih vrednosti direktno u vaše CSS fajlove.

U web razvoju, šesnastični kodovi se često koriste za definisanje boja teksta, pozadina, ivica i drugih vizuelnih elemenata. Na primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

Ovo osigurava da vaše boje ostanu dosledne na celom vašem veb-sajtu ili aplikaciji.

U nekim slučajevima, možete primetiti heks kodove napisane u skraćjenom formatu, koristeći samo tri karaktera umesto šest. To je moguće kada su oba karaktera u svakoj boji pari jednaka. Na primer, `#FF5733` ne može se skratiti, ali `#FFFFFF` (white) se može napisati kao `#FFF`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

Skraćenica `#FFF` je ekvivalentna `#FFFFFF`. Slično, `#000` bi predstavljalo crnu koja je `#000000` u punom obliku.

Hex kodovi su popularni zbog svoje preciznosti i kompatibilnosti, omogućavajući preko 16 miliona boja u svim pregledačima. Oni su idealni kada vam trebaju specifične boje brenda, želite da kontrolišete tačne nijanse koje nisu lako definisane imenovanom bojom ili HSL-om, ili kada kopirate boje iz dizajnerskog softvera koji podrazumevano pruža hex kodove.
