# --interactive--

U prethodnoj lekciji ste naučili kako da počnete da radite sa linearnim i radialnim gradijentima. U ovoj lekciji ćemo zaroniti dublje u to kako koristiti CSS svojstva za kreiranje vizuelno privlačnijih i kompleksnijih dizajna.

Gradijenti u CSS-u vam omogućavaju da kreirate glatke prelaze između dve ili više definisanih boja. Često se koriste za dodavanje dubine, teksture i vizuelnog interesa web dizajnama bez potrebe za fajlovima slika.

CSS podržava dva glavna tipa gradijenata: linearni gradijenti i radijalni gradijenti.

Linearni gradijenti stvaraju postepeno pomešanje između boja duž prave linije. Možete kontrolisati pravac ove linije i korišćene boje. Evo osnovnog primera linearnog gradijenta:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

Ovo stvara gradijent koji prelazi iz crvene na levoj strani u plavu na desnoj strani. Parametar `to right` određuje smer gradijenta. Možete koristiti druge smerove kao što su `to left`, `to top`, `to bottom`, ili čak navesti kutove kao `45deg`.

Takođe možete koristiti više tačaka boje u linearnom gradijentu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

Ovo stvara gradijent koji prelazi iz crvene u žutu, pa u plavu, ravnomerno raspoređen preko elementa.

Radijalni gradijenti, s druge strane, stvaraju kružne ili elipsne gradijente koji zrače iz centralne tačke. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

Ovo stvara kružni gradijent koji počinje sa crvenom u centru i prelazi u plavu na ivicama. Takođe možete specificirati oblik i veličinu radijalnih gradijenata:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

Ovo stvara eliptični gradijent koji počinje od gornjeg levog ugla elementa. I linearni i radijalni gradijenti omogućavaju vam da odredite tačke za boju na preciznim pozicijama. Na primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

Ovo stvara gradijent gde crvena zauzima prvi `20%`, zatim prelazi na žuto na `40%`, i napokon na plavo na `80%`.

Gradijenti se mogu koristiti gde god biste koristili sliku u CSS-u. To ih čini svestranima za kreiranje pozadina, dugmadi, pa čak i tekstualnih efekata kada se kombinuju sa drugim CSS svojstvima.

Zapamtite, iako gradijenti mogu poboljšati vizuelnu privlačnost vašeg veb-sajta, važno je osigurati dovoljan kontrast za pristupačnost, posebno kada koristite gradijente iza teksta.

## --text--

Koja je razlika između linearnih i radijalnih gradijenata u CSS-u?
## --answers--

Linearni gradijenti koriste samo dve boje, dok radijalni gradijenti koriste više boja.
