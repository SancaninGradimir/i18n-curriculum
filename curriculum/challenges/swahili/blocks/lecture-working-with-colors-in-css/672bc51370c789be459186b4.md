# --interactive--

Kada radite sa bojama u CSS-u, razumevanje RGB modela boja je esencijalno. RGB znači Crvena, Zelena i Plava — primarne boje svetlosti. Ove tri boje se kombinuju u različitim intenzitetima kako bi stvorile širok spektar boja.

Model boja RGB je aditivni model boja, što znači da se boje stvaraju kombinovanjem svetlosti različitih intenziteta.

Intenzitet svake boje može varirati od `0` (što znači nivo svetlosti) do `255` (što znači puni nivo svetlosti). Mešanjem različitih nivoa crvene, zelene i plave, možete proizvesti bilo koju boju koju vidite na ekranu. Na primer:

- Crna je predstavljena sa `rgb(0, 0, 0)`, što znači da nema svetlosti za nijednu od tri boje.
- Bela je predstavljena sa `rgb(255, 255, 255)`, što znači punu intenzitet svetlosti za sve tri boje.
- Crvena je predstavljena sa `rgb(255, 0, 0)`, sa punim intenzitetom za crvenu i bez intenziteta za zelenu ili plavu.

U CSS-u, funkcija `rgb()` vam omogućava da definišete boje koristeći RGB model boja. Funkcija prima tri vrednosti, svaka od kojih predstavlja intenzitet crvenog, zelenog i plavog svetla, respektivno. Evo osnovne sintakse:

```css
element {
  color: rgb(red, green, blue);
}
```

Vrednosti za crvenu, zelenu i plavu mogu varirati od `0` do `255`, gde `0` predstavlja odsustvo te boje, a `255` predstavlja punu zasićenost. Evo primera kako biste mogli da koristite funkciju `rgb()` u CSS-u:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

Ovaj kod bi promenio boju teksta paragrafa u crvenu jer je vrednost crvene boje postavljena na `255`, dok su zelena i plava postavljene na `0`.

CSS takođe pruža funkciju `rgba()`, koja dodaje četvrti vrednost —alfa— koja kontroliše transparentnost boje. Alfa vrednost se opseže od `0` (potpuno transparentno) do `1` (potpuno neprozirno). Evo primera korišćenja `rgba()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

Ovaj kod bi primenio poluprozirnu plavu pozadinu na element `div` jer su vrednosti crvene i zelene postavljene na `0`, vrednost plave je postavljena na `255`, a alfa vrednost je postavljena na `0.5` (50% prozirnosti).

Model boja RGB je posebno koristan za digitalne medije jer direktno odgovara načinu na koji ekrani prikazuju boje. Monitori i displeji koriste male crvene, zelene i plave piksele da bi stvorili boje koje vidite. Kontrolisanjem intenziteta ovih piksela putem RGB vrednosti, možete postići širok spektar boja za vaš web dizajn.

Takođe, RGB model je idealan kada radite sa dinamičkim dizajnom. Na primer, ako želite da kreirate animacije ili kolor efekte koji zahtevaju promene u realnom vremenu, fleksibilnost RGB funkcije omogućava vam da modifikujete boje programski.

Razumevanjem RGB kolor modela i načinom korišćenja funkcija `rgb()` i `rgba()` u CSS-u, dobićete veću kontrolu nad time kako su boje prikazane na vašim veb stranicama. Možete kreirati živopisne dizajne, transparentnost, pa čak i programski prilagoditi boje za dinamičke efekte.
