# --interactive--

Prilikom stilizovanja web stranica, jedan od najsveobođnijih i intuitivnijih načina za rad sa bojom je korišćenje HSL modela boja. HSL je skraćenica za Ton, Saturaciju i Svetlinu — tri ključne komponente koje definišu boju.

Model boja HSL predstavlja boje na način koji više odgovara tome kako ljudi percipiraju boju. U ovoj lekciji ćemo istražiti šta je model boja HSL i kako se funkcija `hsl()` koristi u CSS-u.

Ton je tip boje, predstavljen kao ugao na točkići boji, koji se opseže od `0` do `360` stepeni. Na primer, `0` stepeni predstavlja crvenu, `120` stepeni je zelena, i `240` stepeni je plava. Podesivanjem tona, prebacujete se kroz različite boje na spektru.

Saturaција se odnosi na intenzitet ili čistoću boje. Meri se kao procenat od `0%` (potpuno de-saturisana, sivkasta boja) do `100%` (najživlja forma te boje). Boja sa `100%` saturacijom je potpuno živahna, dok boja sa `0%` saturacijom izgleda kao nijansa sive.

Svetlina određuje koliko je boja svetla ili tamna, ponovo mereno u procentima. Vrednost svetline od `0%` će proizvesti crnu boju, `50%` će vam dati normalan ton nijanse, a `100%` će rezultirati belom.

U CSS-u, funkcija `hsl()` se koristi za definisanje boja korišćenjem HSL modela boja. Moderni sintaksa koristi razdvojene prostranstvom vrednosti:

```css
element {
  color: hsl(hue saturation lightness);
}
```

Možete takođe naići na stariju sintaksu sa razmakom zarezima u postojećem kodu, koja je i dalje podržana od strane pregledača:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Razložimo ovo na primeru:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

U ovom slučaju, nijansa je `120` stepeni, što odgovara zelenoj boji. Saturacija je `100%`, pa će zelena boja biti potpuno živahna. Svetlina je `50%`, tako da je na svom normalnom tonu — ni previše tamno, ni previše svetlo. Kao rezultat, tekstualna boja paragrafa biće jarko zelena.

Jedna od glavnih prednosti HSL modela boja je njegova intuitivna priroda. Omogućava lako podešavanje živosti ili svetline boje menjanjem vrednosti zasićenja i svetline bez potrebe za menjanjem osnovne boje (tona).

Na primer, ako želite da kreirate različite nijanse ili tonove iste boje, možete jednostavno podesiti vrednost svetline.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

Ovde, oba `div` elementa koriste isti ton (`240` stepeni, što je plavo), ali jedan ima svetlinu od `80%` (svetlija nijansa plave), a drugi ima svetlinu od `20%` (tamnija nijansa plave).

Slično RGB modelu, funkcija `hsl()` takođe podržava opcioni alfa vrednost za kontrolisanje transparentnosti. Dodajete ga nakon razdvojnog znaka `/` kao četvrti parametar. Evo osnovne sintakse:

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

Pogledajmo primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

Ovaj kod bi dao `div` semi-transparent crveni pozadinu, gde je nijansa postavljena na `0` stepeni (crveno), saturacija je `100%`, svetlina je `50%`, a alfa je `0.5` (50% opaciteta). Takođe možete navesti alfa kao procenat — na primer, `hsl(0 100% 50% / 50%)` je ekvivalentno `hsl(0 100% 50% / 0.5)`.

Takođe ćete često videti funkciju ``hsla()`` u postojećem CSS-u. Ovo je bio originalni način za uključivanje transparentnosti u HSL boje, koristeći zarezom razdvojene vrednosti:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Na primer, `hsla(0, 100%, 50%, 0.5)` predstavlja zastareli ekvivalent od `hsl(0 100% 50% / 0.5)`. Oboje su podržani od strane pregledača, ali je savremena `hsl()` sintaksa sa separatorom `/` sada preferirana.

HSL model boja je naročito koristan kada vam treba da kreirate palete boja i lako prilagodite nijanse ili tonove.

Pošto HSL razdvaja nijansu od zasićenosti i svetline, možete zadržati doslednu boju temu dok menjate svetlinu za senke ili svetle delove. Ovaj nivo fleksibilnosti čini HSL omiljenim kod dizajnera i programera jednako.

Pored toga, HSL čini vrednosti boja čitljivijim i razumljivijim u poređenju sa drugim metodama kao što su heks kodovi ili RGB. Možete jasnije videti nameru iza boje i prilagoditi se shodno tome, posebno kada eksperimentišete sa različitim dizajnima.

Sa HSL bojnim modelom, dobijate više fleksibilnosti u kontrolisanju tonova boja, zasićenosti i svetline, što ga čini moćnim alatom za kreiranje bogatih i dinamičnih dizajna u CSS-u.
