# --description--

Dobrodošli u svakodnevnu vežbu 250!

Dekripujte sekvencu šifrovanih slova koristeći dekriptivni ključ i vratite rezultat.

Da biste dobili ključ:

- Pogledajte sve svakodnevne izazove do danas čiji je broj izazova deo 25 (uključujući i ovaj).
- Uzmite prvo slovo iz naslova svakog od tih izazova i spojite ih u sekvencu slova. Ako naslov počinje sa ne-slovom, pronađite njegovo prvo slovo.

Da biste dešifrovali poruku, prođite kroz svako slovo u šifrovanoj poruci i:

  1. Pogledajte odgovarajuće slovo u ključu (ponovite ključ ako je poruka duža od ključa).
  2. Prebacite slovo iz ključa na odgovarajući broj: `"A"` = 1, `"B"` = 2, ..., `"Z"` = 26.
  3. Pomerite šifrovano slovo unazad u alfabet za taj broj.
  4. Ako pomeranje ide pre od slova `"A"`, zavojte nazad na `"Z"`.

Na primer, ako šifrovana poruka počinje sa `"Y"` a prvo slovo ključa je `"V"` (22), pomerite `"Y"` unazad za 22 mesta da biste dobili `"C"`. Ponovite ovaj proces za svako slovo kako biste dešifrovali celu poruku.

- Šifruju se samo slova; razmaka ostaju nepromenjeni.
- Sva izvlačena i vraćena slova su velika (kapitalna) slova.

function decode(message) {
  const key = "VLHCGMDLNH";
  let keyIndex = 0;
  return message.split('').map(char => {
    if (char === ' ') return char;
    const shift = key[keyIndex % key.length].charCodeAt(0) - 64;
    keyIndex++;
    return String.fromCharCode(((char.charCodeAt(0) - 65 - shift + 26) % 26) + 65);
  }).join('');
}
