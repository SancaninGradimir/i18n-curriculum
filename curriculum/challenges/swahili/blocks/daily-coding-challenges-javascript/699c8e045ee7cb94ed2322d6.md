# --description--

Kutolewa mfuatano wa herufi miwili unaowakilisha wakati uliweka gari lako maegesho na wakati uliuchukua, hesabu ada ya maegesho.

- Generisana sekvenca karaktera će biti u formatu `"HH:MM"` koristeći 24-satni sistem. Na primer, `"14:00"` je 2 ujutru.
- Prva sekvenca će biti vreme kada ste parkirali vozilo, a druga će biti vreme kada ste ga preuzeli.
- Ako je vreme preuzimanja ranije nego vreme dolaska, to znači da je parkiranje trajalo do nakon šest uveče do sledećeg dana.

Sheria za ada:

- Kila saa ya maegesho inagharimu $3.
- Saa zisizokamilika zinarundikwa hadi saa kamili inayofuata.
- Ikiwa maegesho yalidumu usiku kucha (baada ya saa sita usiku), ada ya ziada ya $10 ya usiku inatozwa.
- Kuna ada ya kiwango cha chini ya $5 (inatumika tu ikiwa jumla ingekuwa chini ya $5).

Vratite ukupne troškove u formatu `"$cost"`, `"$5"` na primer.

```js
function calculateParkingFee(parkTime, pickupTime) {
  function toMinutes(time) {
    const [hours, minutes] = time.split(":").map(Number);
    return hours * 60 + minutes;
  }

  const entryMinutes = toMinutes(parkTime);
  const exitMinutes = toMinutes(pickupTime);

  let totalMinutes;
  let overnight = false;

  if (exitMinutes < entryMinutes) {
    overnight = true;
    totalMinutes = (24 * 60 - entryMinutes) + exitMinutes;
  } else {
    totalMinutes = exitMinutes - entryMinutes;
  }

  const hours = Math.ceil(totalMinutes / 60);
  let cost = hours * 3;

  if (overnight) {
    cost += 10;
  }

  if (cost < 5) {
    cost = 5;
  }

  return `$${cost}`;
}
```
