# --interactive--

U JavaScriptu, template literali su moćan i fleksibilan način za rad sa stringovima. Za razliku od običnih stringova, koji koriste pojedinačne (`'`) ili dvostruke (`"`) navodnike, template literali su definisani sa backtick-ovima (`` ` ``).

Omogućavaju lakšu manipulaciju nizovima karaktera, uključujući ugradnju promenljivih direktno unutar niza karaktera, funkciju poznatu kao interpolacija nizova karaktera.

Template literali olakšavaju kreiranje nizova koji se protežu na više linija ili uključuju izraze (poput varijabli ili čak JavaScript koda) direktno unutar niza.

Evo primera šablona literala:

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

Obratite pažnju na upotrebu backtickova umesto pojedinačnih ili dvostrukih navodnika. Sintaksa ``${name}`` je primer interpolacije stringa, gde se vrednost varijable ``name`` direktno ubacuje u string.

Interpolacija stringova vam omogućava da ugradite varijable i izraze unutar stringa. Ovo je značajno poboljšanje u odnosu na stariji metod, gde biste konkatenirali stringove i varijable koristeći operater `+`.

Evo primera korišćenjem konkatenacije nizova karaktera i operatera plus (`+`):

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

I ovde je primer korišćenjem *template* literala i interpolacije nizova:

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

Kao što vidite, interpolacija stringova korišćenjem literalnih šablona je mnogo čistije i lakše za čitanje, posebno kada radite sa više promenljivih.

Još jedna sjajna karakteristika literala šablona je što podržavaju višelinijske nizove. Sa običnim nizovima, morali biste koristiti znakove za bježanje (escape characters) (``\n``) da biste kreirali nove linije. Sa literalima šablona, možete jednostavno napisati niz preko više linija, a formatiranje se čuva:

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

Još jedna karakteristika literala šablona je što vam omogućavaju da ugradite izraze JavaScript-a direktno unutar niza, kao što je u ovom primeru:

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

Šablonski literali su posebno korisni kada vam je potrebno uključiti varijable ili izraze u nizove, formatirati složene nizove ili raditi sa višelinijskim tekstom. Oni su kraći i čitljiviji u poređenju sa tradicionalnim spajanjem niza.
