# --interactive--

U JavaScriptu, varijable deluju kao kontejneri za skladištenje podataka kojima možete pristupiti i modifikovati u celom vašem programu.

Možete razmišljati o varijablama kao kutijama koje čuvaju vrednosti. Varijablama možete pratiti stvari poput brojeva ili teksta i pozivati se na ove vrednosti kad god vam zatrebaju u programu.

Jedan način za deklarisanje varijable u JavaScriptu je korišćenje ključne reči ``let``. Naučićete više o ključnoj reči ``let`` kao i o drugim načinima deklarisanja promenljivih u budućim lekcijama.

Evo primera korišćenja `let` za deklarisanje varijable nazvane `age`:

```js
let age;
```

Trenutno, varijabla `age` nije joj dodeljena vrednost. Ako pokušate da je koristite, vraćaće `undefined`, što znači da nema vrednost.

Evo primera.

**NAPOMENA**: `console.log()` je funkcija koja izbacuje informacije u konzolu, što je deo vašeg veb pregledača korišćen za debagovanje koda. Naučićete više o `console.log()` u budućim časovima. Takođe, simboli `//` se koriste za dodavanje komentara u vaš kod. Komentari su napomene za vas ili druge programere koje se ignorišu kada se kod izvrši.

:::interactive_editor
 
```js
let age;
console.log(age); // undefined
```

:::

Da biste dodelili vrednost promenljivoj, moraćete da koristite operater dodeljivanja na ovaj način:

```js
let age = 25;
```

Sada kada koristite varijablu ``age``, ona će vratiti vrednost ``25``.

:::interactive_editor

```js
let age = 25;
console.log(age); // 25
```

:::

Operater dodeljivanja izgleda kao znak jednakosti (`=`) ali ne proverava jednakost. Naučićete o ispravnim operatorima za proveru jednakosti u budućim lekcijama.

Operater dodeljivanja se koristi za dodeljivanje vrednosti varijabli. Ovaj proces dodeljivanja vrednosti varijabli poznat je kao inicijalizacija.

Jedna prednost korišćenja ključne reči `let` za deklarisanje promenljivih je što možete da im ponovo dodelite vrednosti. U programiranju, ponovno dodeljivanje znači davanje nove vrednosti promenljivoj koja već ima jednu.

Ovo je primer ponovnog dodeljivanja vrednosti za varijablu `age`.

:::interactive_editor

```js
let age = 25;
console.log(age); // 25
age = 30;
console.log(age); // 30
```

:::

Sada promenljiva `age` sadrži vrednost `30`. Napomena da nije bio potreban ponovo ključna reč `let` jer je promenljiva `age` već deklarisana, pa nema potrebe da je deklarišete drugi put.

Kada koristite ponovno dodeljivanje, potrebno je samo da navedete ime varijable. Ponovno dodeljivanje je korisno jer vam omogućava da ažurirate i promenite vrednost sačuvanu u varijabli dok vaš program radi. Dobar primer ovoga bi bilo ažuriranje poena u igri.

Imenovanje promenljivih može delovati jednostavno, ali postoje neka pravila i najbolje prakse kako biste osigurali da je vaš kod čitljiv i funkcionalan.

Vaša imena promenljivih bi trebalo da opisuju šta podaci predstavljaju. Na primer, umesto korišćenja imena kao što je `x`, opisnije ime kao što su `age` ili `points` čini vaš kod lakšim za razumevanje.

```js
// Bad variable names
let x = 10;
let y = "John";

// Good variable names
let age = 10;
```

Varijable u JavaScriptu moraju početi slovom, donjom crtom (`_`), ili znakom dolara (`Varijable u JavaScriptu moraju početi slovom, donjom crtom (`_`), ili znakom dolara (). Ne mogu početi brojem.

```js
// Valid variable names
let age;
let _score;
let $total;

// Invalid variable names
let 1stPlace; // starts with a number
```

Imena promenljivih su osetljiva na veličinu slova, što znači da se reč ``age`` u malim slovima i reč ``Age`` sa velikim ``A`` smatraju različitim promenljivama.

:::interactive_editor

```js
let age = 25;
let Age = 30;
console.log(age); // 25
console.log(Age); // 30
```

:::

Zato je važno pridržavati se dosledne konvencije imenovanja kao što je camelCase. camelCase je gde prva reč ima sva mala slova, a svaka naknadna reč počinje velikim slovom.

Evo primera korišćenja konvencije imenovanja camelCase za varijablu:

```js
let thisIsCamelCase;
let anotherExampleVariable;
let freeCodeCampStudents;
```

Postoje određene ključne reči u JavaScriptu koje ne možete koristiti kao imena promenljivih, kao što su ``let``, ``const``, ``function`` ili ``return``, jer su rezervisane za sam jezik.

Takođe biste trebali izbegavati korišćenje specijalnih znakova kao što su uzvičnici (`!`) ili znaci '@' (`@`), u vašim imenima promenljivih. Najbolje je da zadržite imena promenljivih čitljiva korišćenjem slova, brojeva, donjih crtica ili znakova dolara.

Prateći ove smernice, vaš kod će biti čistiji i upravljiviji kako raste u složenosti.
