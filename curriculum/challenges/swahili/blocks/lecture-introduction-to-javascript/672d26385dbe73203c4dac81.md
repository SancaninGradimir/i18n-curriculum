# --interactive--

JavaScript je moćan programski jezik koji unosi interaktivnost i dinamičko ponašanje na veb sajtove.

Iako su HTML i CSS jezički markup koji se koriste za strukturisanje sadržaja i stilizovanje elemenata na stranici, JavaScript ide dalje od toga omogućavanjem složenije funkcionalnosti, kao što je rukovanje unosom korisnika, animiranje elemenata i čak izgrađivanje punih veb aplikacija.

Na primer, kada kliknete na dugme, pošaljete formular ili pređete mišem preko menija, JavaScript određuje kako stranica funkcioniše.

Evo primera kako ova tri rade zajedno:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

U ovom primeru, koristi se HTML za definisanje sadržaja: naslov (element `h1`) i dugme (element `button`).

CSS se koristi za primenu stilova na naslov, kao što je činjenje teksta zelenim. JavaScript se koristi za prikazivanje upozorenja kada se klikne na dugme.

Zaključno, HTML pruža strukturu, CSS dodaje stilizovanje, a JavaScript omogućava ponašanje, kreirajući kompleтно i interaktivno veb iskustvo.
