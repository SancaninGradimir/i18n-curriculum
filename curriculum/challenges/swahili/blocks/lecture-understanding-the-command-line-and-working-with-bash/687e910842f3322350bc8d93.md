# --description--

Naučićemo se o terminalu i kako se on razlikuje od komandne linije.

Komandna linija je osnovni tekstualni interfejs za unos koji korisniku omogućava da unese "komande", obično u vidu niza znakova, i pošalje ili izvrši ih, obično pritiskom tastera "Enter". Najčešće ćete komandne linijske interfejse videti unutar termina.

Terminal je specijalna aplikacija koja nudi interfejs komandne linije za izvršavanje sistema-nivskih komandi izvan osnovnih operacija čitanja/pisanja. Takođe možete čuti o "emulatorima terminala", koji su u suštini aplikacije koje obuhvataju osnovni interfejs terminala kako bi ponudili dodatne funkcije ili funkcionalnost. Ali za većinu opštih namena, verovatno je sigurno da se na njih možete pozivati i kao "terminali".

Konačno, moraćemo da pričamo i o „shellovima“. Shell je softver koji obuhvata komandnu liniju i tumači vaše unose kao komande, vraćajući izlaz.

Ali kako možete pristupiti terminalu na vašem sistemu?

Pa, ako koristite modernu verziju Windows-a, verovatno ćete imati dve različite opcije. Windows nudi dve različite shell aplikacije iz kutije: Powershell i Command Prompt.

Oba ovih shell-a dolaze sa sopstvenim terminalnim aplikacijama (koje dele ime odgovarajućeg shell-a), tako da možete koristiti meni aplikacija za pokretanje ili Powershell termina ili Command Prompt termina.

Windows takođe nudi relativno novi i modernizovani emulator terminala, koji se adekvatno zove Microsoft Terminal.

Možete instalirati ovu aplikaciju iz Windows Store-a, i onda je možete pokrenuti baš kao što biste pokrenuli svoje Powershell ili Command Prompt terminale. Prednost Microsoft Terminala je u tome što vam omogućava pristup i Powershell i Command Prompt shell-ovima iz iste aplikacije, kao i vašem odgovarajućem Linux shellu ako ste konfigurisali Windows Subsystem za Linux.

MacOS nudi terminal koji je adekvatno nazvan "Terminal". Možete ga pristupiti putem vaše pretrage Spotlight-a ili putem pokretača aplikacija.

MacOS takođe nudi podršku za eksterne terminalne emulatore, kao što su iTerm ili Ghostty. Ove aplikacije možete instalirati sa njihovih početnih stranica.

Na Linuksu, podrazumevani terminal (i kako ga pokrećete) veoma zavisi od distribucije i okruženja radne površine koje koristite. Takođe postoji mnogo podrške za emulatore terminala trećih strana. Na primer, možete postaviti kitty u Arch Linuksu – i možete ga pokrenuti pomoću menadžera aplikacija kao što je wofi.

Linux se, verovatno, može pohvaliti najširim izborom terminalnih aplikacija iz kojih možete da birate. Većina njih se može instalirati putem menadžera paketa vaše distribucije.

Razlika između "terminala", "šela" i "linije komandi" ponekad može delovati prilično pedantički. I u većini slučajeva, pojmovi se generalno mogu koristiti kao zamenski. Ali važno je znati da imaju specifična značenja, i šta su ta značenja.
