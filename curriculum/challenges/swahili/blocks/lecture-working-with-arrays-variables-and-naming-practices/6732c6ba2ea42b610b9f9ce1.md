# --description--

U prethodnim lekcijama, predstavili smo vam svojstvo `length`, koje vraća broj elemenata u nizu. Evo kratkog podsjetnika kako funkcioniše:

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

Moguće je imati nizove sa praznim slotovima. Prazni slotovi su definisani kao slotovi koji ništa ne sadrže. Ovo se razlikuje od niza sa vrijednošću `undefined`. Ovakvi nizovi poznati su kao razrijedjeni nizovi (sparse arrays). Evo primjera:

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

U ovom slučaju, iako imamo samo dva definisana elementa, `1` i `4`, dužina je `4` jer najviši indeks (`3`) plus `1` daje nam dužinu od `4`.

Sada hajde da razgovaramo o tome kako kreirati prazan niz fiksne dužine. Postoji nekoliko načina za to u JavaScriptu, ali jedna uobičajena metoda je korištenje konstruktora `Array()` sa numeričkim argumentom. Konstruktor `Array()` može se koristiti sa ključnom riječi `new` za kreiranje novog niza. Evo primjera:

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

U ovom primjeru, kreiramo novi niz koristeći `Array(5)`. Ovo stvara razrijedjeni niz dužine `5`, gdje su svi slotovi prazni.

Drugi način za kreiranje praznog niza fiksne dužine je korištenje metode `Array.from()` sa argumentom dužine. Za razliku od `new Array(n)`, ova metoda kreira niz zadate dužine u kojem svi elementi postoje i imaju vrijednost `undefined`:

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

Ako želite kreirati niz određene dužine i popuniti ga podrazumevanom vrijednošću, možete koristiti metodu `Array.fill()`:

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

Ovo stvara niz dužine tri i popunjava sve elemente vrijednošću `0`. Napomena: prilikom popunjavanja objektima, svi slotovi referenciraju isti objekat; ako vam trebaju nezavisne kopije, koristite povratnu funkciju (callback) ili `Array.from()` umjesto toga.

Razumijevanje kako dobiti dužinu niza i kreirati nizove fiksne dužine važno je za mnoge programerske zadatke, posebno kada morate inicijalizovati nizove za specifične algoritme ili strukture podataka.
