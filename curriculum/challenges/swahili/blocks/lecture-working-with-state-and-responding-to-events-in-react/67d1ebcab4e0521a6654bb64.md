# --description--

U Reactu, ažuriranje nizova u stanju je prilično jednostavno, ali lako je napraviti grešku, posebno ako dolazite iz vanilog JavaScripta gde možete direktno modifikovati nizove.

U Reactu, stanje se tretira kao nepromenljivo kako bi moglo prepoznati promene i izvršiti odgovarajuće ažuriranja korisničkog interfejsa.

Pogledajmo kako možete ažurirati nizove koji se nalaze u stanju (state) u React-u.

Jedna od najčešćih grešaka prilikom ažuriranja nizova u React stanju je direktno modifikovanje niza korišćenjem metoda kao što su `push()`, `pop()` ili `splice()`. Ove metode mutiraju originalni niz, a React to ne dozvoljava.

React se oslanja na novu referencu niza da bi detektovao promene, pa direktno modifikovanje niza može sprečiti komponentu od toga da se ponovo renderuje kako je očekivano.

Ovo je primer korišćenja metode ``push()`` za dodavanje u niz u stanju (state), što neće raditi:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Ako kliknete na `Add Item` dugme, ništa se ne dešava u korisničkom interfejsu.

Takođe bi moglo biti iskušljivo ukloniti stavke iz niza pomoću metode `pop()`:

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

Da biste ažurirali niz u stanju, ključno je kreirati novi niz, izvršiti svoje operacije i proslediti ga React-u, umesto da mutirate postojeći niz.

Jer je to novi niz, React će znati da je stanje promenjeno i izazvati ponovno renderovanje.

Evo kako možete dodati u niz `items` koristeći operater širenja:

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]` kreira novi niz kopiranjem svih stavki iz postojećeg `items` niza koji je u stanju, zatim dodaje `newItem` na kraj, što inkrementira `id` i broj stavke.

Ako želite da uklonite nešto iz niza, možete koristiti metod `filter()`, koji vraća novi niz nakon filtriranja svega što želite da uklonite:

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

Evo celog koda:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

To su neki uobičajeni načini na koje možete ažurirati niz u stanju.
