# --interactive--

CSS se može primeniti na web stranicu na tri glavna načina: uline, interni ili eksterni.

Svaki metod ima svoj slučaj upotrebe, prednosti i ograničenja, a znati kada koristiti svaki od njih je esencijalno za pisanje čistog, efikasnog i održivog koda.

Razmotrićemo tri tipa CSS-a i kada biste ih trebali koristiti.

Inline CSS se piše direktno unutar HTML elementa koristeći atribut ``style``. On primenjuje stilove na određeni element.

Evo primera korišćenja ugrađenog CSS-a:

:::interactive_editor

```html
<p style="color: green;">This is an inline-styled paragraph.</p>
```

:::

U ovom primeru, koristimo atribut `style` da postavimo tekst paragrafa na `green`.

Inline CSS se generalno koristi za brze, jednokratne stilove ili za nadjačavanje drugih stilova za određeni element.

Međutim, trebalo bi ga izbegavati u većini slučajeva jer može da komplikuje HTML i učini kod teže za održavanje.

U većini slučajeva, bolje je koristiti interni ili eksterni CSS da biste zadržali svoje stilove organizovanim i održivim.

Interni CSS se piše unutar tagova `style` unutar sekcije `head` HTML dokumenta. On primenjuje stilove na celu stranicu i koristan je kada vam treba stilizovati jedan dokument.

Evo primera internog CSS-a:

:::interactive_editor

```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
<body>
  <p>This paragraph is styled using internal CSS.</p>
</body>
```

:::

U gornjem primeru, interni CSS primjenjuje plavi tekst na sve `p` elemente na stranici.

Internal CSS se najbolje koristi kada je potrebno primeniti stilove na određenu stranicu, umesto preko više stranica. Korisno je za sajtove sa jednim stranicom ili kada se stilovi ne moraju ponovo koristiti negde drugde.

Međutim, postoje neke nedostatke, kao što je nepromovisanje ponovne upotrebe na više stranica. Pored toga, kao što je ugrađeni CSS, on mešamo HTML i CSS, čineći kod teže za održavanje u većim projektima.

Eksterni CSS je napisan u zasebnoj `.css` datoteci i povezan sa HTML dokumentom korišćenjem elementa `link` u sekciji `head`.

Omogućava vam da konzistentno stilizujete više stranica i predstavlja preferirani metod u profesionalnom web razvoju.

TEKST:

Evo primera stilizovanja svih pasus elemenata:

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>This paragraph is styled using external CSS.</p>
</body>
```

```css
p {
  color: red;
}
```

:::

U ranijoj lekciji, naučili ste da element `link` ima atribut `rel` koji određuje odnos između trenutnog dokumenta i povezanog resursa, kao što je povezivanje na list stilova ili eksterni resurs.

S druge strane, atribut `href` specificira URL povezanog resursa, ukazujući na to gde bi se resurs trebalo dohvatiti.

Eksterni CSS je idealan za velike projekte gde želite da održite dosledan stil na više stranica.

Podstiče razdvajanje briga tako što dozvoljava da HTML rukuje strukturom, a CSS stilizovanjem, što čini kod održivijim i skalabilnijim.

Razumevanje kada koristiti svaki tip CSS je ključno za efikasan i uspešan razvoj veb aplikacija.

U većini slučajeva, eksterni CSS bi trebalo da bude vaš primarni pristup, posebno za veće i složenije projekte.
