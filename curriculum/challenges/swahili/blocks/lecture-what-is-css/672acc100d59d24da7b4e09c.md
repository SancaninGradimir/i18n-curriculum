# --interactive--

CSS kombinatori se koriste za definisanje odnosa između selektora u CSS-u. Pomažu pri selekciji elemenata na osnovu njihovog odnosa sa drugim elementima, što omogućava preciznije i efikasnije stilizovanje.

Razgovaraćemo o nekim primarnim CSS kombinatorima i njihovim slučajevima upotrebe, počevši od kombinatora potomka (descendant).

Kombinator potomka se koristi za ciljanje elemenata koje pokriva drugi selektor ako su ugnježđeni unutar predak elementa koji odgovara prvom selektoru. Predak može biti roditeljski element ili predak roditelja.

Da bismo bolje razumeli kako ovo funkcioniše, pogledajmo primer.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Relaxing Cat">
  <figcaption>A relaxing cat with a border</figcaption>
</figure>
```

```css
figure img {
  border: 4px solid red;
}
```

:::

U gornjem primeru, koristimo desendant kombinator za selekciju svih elemenata slike unutar `figure` elemenata i primenjujemo `solid red` ivicu na sve četiri strane.

Napomena da se `space` koristi između roditeljskog i detinjastog selektora.

U ovom slučaju, `figure` bi bio roditelj, a `img` bi bio dete.

Ako imate više slika unutar elementa ``figure``, korišćenje desendanta kombinatora bi bio dobar način za primenu čvrste crne ivice oko svake od tih slika.

Drugi tip kombinatora bio bi kombinator deteta.

Kombinator za dete (`>`) u CSS-u se koristi za selekciju elemenata koji su direktna deca definisanog roditeljskog elementa.

Ovaj kombinator cilja samo elemente sa određenim roditeljem, čineći vaše CSS pravilo preciznijim i sprečavajući neželjeno stilizovanje dublje ugnježdene elemente.

Pogledajmo sledeći HTML primer:

:::interactive_editor

```html
<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

:::

U navedenoj HTML strukturi, klasa `container` je primenjena na element `div`.

Unutar ovog kontejnera, postoji direktno dete elementa ``p`` ("Prvi"), praćeno dva dodatna elementa ``div``, svaki od kojih sadrži element ``p`` ("Drugi" i "Treći").

Prvi element ``p`` je direktno dete elementa ``.container``, dok su drugi dva elementa ``p`` ugnježena unutar drugih elemenata ``div``, čineći ih dubljim potomcima.

Da biste primenili stilove samo na direktnog potomka klase `container`, možete koristiti kombinator za potomke ovako:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

```css
.container > p {
  color: blue;
}
```

:::

U gornjem primeru, ciljamo samo direktnog deteta klase `container`. Ovo će dati direktnom detetu boju teksta od `blue`.

Budući da su drugi paragraf elementi ugnježđeni unutar `div` elemenata, ne smatraju se direktnim potomcima klase `container` i neće dobiti plavu boju teksta.

Još jedan uobičajeni kombinator bio bi kombinator sledećeg brata.

Kombinator sledećeg brata (`+`) u CSS-u bira element koji odmah sledi navedeni srodni element. Ovaj kombinator je koristan kada želite primeniti stilove na element koji direktno sledi drugi element, omogućavajući ciljano stilizovanje zasnovano na poziciji elementa u odnosu na njegove srodne elemente.

Pogledajmo sledeći HTML primer:

:::interactive_editor

```html
<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

:::

Ovde imamo `figure` element koji sadrži `img` element praćen `figcaption` elementom. Element `figcaption` je neposredni brat od `img` elementa.

Ako želite da primenite crnu ivicu oko elementa `figcaption`, možete koristiti kombinator sledećeg brata na ovaj način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

```css
img + figcaption {
  border: 4px solid black;
}
```

:::

U ovom primeru, kombinator sledećeg brata (`+`) selektuje element `figcaption` koji odmah sledi nakon elementa `img`. Primena CSS pravila dodaje `4px solid black border` oko elementa `figcaption`.

Još jedan uobičajeni kombinator je kombinator sledećeg brata.

Kombinator naknadnog brata (`~`) u CSS-u selektuje sve braće i sestre određenog elementa koji dolaze posle njega.

Za razliku od kombinatora sledećeg brata, koji cilja samo neposredno sledeće brate, kombinator naknadnog brata (`~`) može da cilja bilo koje braće koja slede navedeni element, nudeći veću fleksibilnost u stilizovanju.

Pogledajmo sledeći HTML primer:

:::interactive_editor

```html
<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

:::

U ovoj HTML strukturi, imamo `h2` element praćen sa četiri paragrafa elementa. Paragraf elementi su braća/sestre `h2` elementa.

Ako želite stilizovati sve paragraf elemente koji dolaze nakon elementa `h2`, onda možete koristiti kombinator naknadnog brata (subsequent-sibling) na ovaj način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

```css
h2 ~ p {
  color: green;
}
```

:::

U ovom primeru, svi paragraf elementi koji slede element `h2` će imati zelenu boju teksta.

Kombinator naknadnog brata (`~`) cilja sve paragraf brate koji se pojavljuju nakon elementa `h2`, bez obzira na to da li su neposredni braća.

Zaključno, razumevanje i korišćenje različitih CSS kombinatora omogućava vam da primenite precizne stilove na vaše HTML elemente, poboljšavajući kontrolu i fleksibilnost vašeg dizajna.

Savladavanjem ovih selektora, možete kreirati sofisticiranije i ciljane stilizacione pravila koja poboljšavaju kako izgled, tako i funkcionalnost vaših veb stranica.

## --text--

Koja CSS pravila će stilizovati sve `span` elemente koji su unutar `div` elemenata, bez obzira na koliko su duboko ugnježđeni?
## --answers--

```css
div > span {
  color: red;
}
```
