# --interactive--

Kada radite sa CSS-om, često naiđete na tri različita tipa ponašanja prikaza za elemente: `inline`, `block`, i `inline-block`.

Svako od ovih svojstava utiče na to kako su elementi pozicionirani i kako oni interaguju sa drugim elementima na stranici.

U ovoj lekciji, fokusiraćemo se na to kako svojstvo `inline-block` radi i kako se razlikuje i od ugrađenih (inline) i od blok nivo elemenata.

Elementi na nivou bloka ponašaju se kao veliki kontejneri ili "blokovi" koji zauzimaju punu širinu svog roditeljskog kontejnera. Uvek počinju na novoj liniji, a njihova visina i širina mogu se podesiti.

In-line elementi zauzimaju samo toliko prostora koliko im je potrebno. Teku unutar okolnog sadržaja i ne prekidaju se na novom redu.

Svojstvo ``inline-block`` je hibrid ova dva ponašanja. Kao *inline* elementi, elementi ``inline-block`` ostaju u toku teksta bez početka na novoj liniji.

Međutim, za razliku od ugrađenih elemenata, možete podesiti širinu i visinu elementa `inline-block`, baš kao što biste to uradili sa blok elementima.

Ukratko, ključna razlika između `inline` i `inline-block` je u tome da elementi `inline` ne mogu imati kontrolisanu veličinu, dok elementi `inline-block` omogućavaju potpunu kontrolu nad dimenzijama i pritom ostaju u liniji sa drugim sadržajem.

Pogledajmo primer.

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Inline-Block</span>
  <span class="inline-block-element element2">Inline-Block</span>
</div>
```

```css
.inline-block-element {
  display: inline-block;
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

U gornjem primeru, imamo `div` sa klasom `container`. Unutar tog `div` elementa, imamo dve `span` elemenata.

Svaki `span` element je postavljen na ``display:inline-block`` i ima postavljenu širinu i visinu.

Elementi tipa inline-block će se pojaviti jedan pored drugog jer se ponašaju kao inline elementi, ali imaju i specificiranu širinu i visinu, što im daje blok-slične svojstva.

Ali, ako uklonite svojstvo `display: inline-block`, ni `height` niti `width` neće biti primenjeni iako ste ga jasno definisali.

Evo revidiranog CSS-a:

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Span element</span>
  <span class="inline-block-element element2">Span element</span>
</div>
```

```css
.inline-block-element {
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

U ovom kodu, uklonili smo svojstvo `display: inline-block;`, ali smo ostavili sve ostalo netaknuto. Ovde, elementi `span` se vraćaju svom podrazumevanom ponašanju kao uline elemente.

Kao rezultat, specificirana širina i visina se ignorišu, a elementi zauzimaju samo prostor potreban za njihov sadržaj.

Razumevanje kako radi ``inline-block`` korisno je jer ga možete koristiti za kreiranje rasporeda koji zahtevaju kontrolu poravnanja i dimenzija unutar kontinualnog teksta.

## --text--

Koja je ključna karakteristika `inline-block` elementa koja ga razlikuje od `inline` elementa?
## --answers--

Elementi tipa inline-block ne mogu biti ugnježđeni unutar blok elemenata.
### --feedback--

Inline-block elements allow setting of height and width, unlike inline elements.

---
