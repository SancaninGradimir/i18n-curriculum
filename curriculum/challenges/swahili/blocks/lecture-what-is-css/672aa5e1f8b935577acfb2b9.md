# --interactive--

CSS, što je skraćenica za Cascading Style Sheets, je ključna komponenta modernog web razvoja. To je jezik za stilove koji se koristi za primenu stila na HTML. Jednostavnije rečeno, ako je HTML struktura web stranice, CSS je ono što joj daje dobar izgled.

TEKST:

Evo primera stilizovanja navigacione trake.

**NAPOMENA**: Ne brinite zbog pokušaja razumevanja CSS koda još uvek. Naučićete kako sve ovo funkcioniše na budućim časovima i radionicama. Ovo je samo da vam dam ideju šta možete uraditi sa malo CSS-a.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

Glavna uloga CSS-a je da razdvoji sadržaj veb stranice od njenog vizuelnog predstavljanja. Ovo razdvajanje omogućava web programerima da kreiraju fleksibilnije i održivije veb sajtove.

Sa CSS-om, možete kontrolisati raspored, boje, fontove i opšti vizuelni izgled veb stranica bez menjanja HTML strukture.

Razmotrimo jednostavnu analogiju. Ako zamislite veb stranicu kao kuću, HTML bi bio temelji i okvir, dok bi CSS bila boja, tapete i dekoracije koje čine kuću vizuelno privlačnom i jedinstvenom.

CSS funkcioniše selektovanjem HTML elemenata i primenjivanjem stilova na njih.

Ovo je primer koji pokazuje neformatiran ``button`` element i stilizovan jedan. Da biste interagovali sa ovim primerom, moraćete da omogućite interaktivni uresnik.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

Ovi stilovi mogu uključivati svojstva kao što su boja, veličina fonta i mnogo više. Promenom ovih svojstava, možete dramatično promeniti izgled veb stranice bez menjanja njenog sadržaja.

Jedan od najmoćnijih aspekata CSS-a je njegova sposobnost kreiranja responsivnih dizajna.

To znači da pomoću CSS-a možete učiniti da vaš sajt izgleda odlično na bilo kom uređaju, bilo da je to desktop računar, tablet ili pametni telefon.

CSS vam omogućava da prilagodite rasporede, veličine fonta i druge vizuelne elemente na osnovu veličine ekrana uređaja koji pregleda vebstranicu.

Pokušajte da omogućite interaktivni uresnik i prilagodite veličinu prozora za pregled kako biste videli kako se raspored prilagođava veličini ekrana.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

Još jedna važna karakteristika CSS-a je njegova kaskadna priroda, od koje potiče i termin „kaskadni“ u njegovom nazivu.

To znači da se stilovi mogu nasleđivati i nadjačavati, omogućavajući hijerarhijsku strukturu stilizovanja. Naučićete više o tome kako ovo funkcioniše u budućim lekcijama.

CSS takođe podržava korišćenje eksternih stilskih listova.

To znači da možete zadržati sve vaše stilizacione pravilo u zasebnoj datoteci, koja se zatim može povezati sa više HTML stranica.

Ova funkcija značajno poboljšava održivost veb sajtova, posebno većih. Umesto da menjate stilove na svakoj pojedinačnoj stranici, možete napraviti promene u jednom CSS fajlu koji će uticati na sve povezane stranice.

U svetu razvoja veb stranica, CSS igra ključnu ulogu u kreiranju vizuelno privlačnih, prilagodljivih i korisnički prijateljskih web sajtova.

Omogućava programerima da pretvore jednostavne HTML dokumente u zanimljiva veb iskustva koja privlače pažnju korisnika i poboljšavaju njihovu interakciju sa veb sadržajem.
