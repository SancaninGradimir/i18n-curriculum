# --interactive--

CSS je odgovoran za stilove veb stranice. Svi ovi stili su sastavljeni od različitih CSS pravila.

CSS pravilo se sastoji od dve glavne komponente: **selektor** i blok **deklaracija**.

Pogledajmo osnovnu sintaksu:

```css
selector {
  property: value;
}
```

Selektor je obrazac korišćen u CSS-u za identifikovanje i ciljanje specifičnih HTML elemenata radi stilizovanja.

Primeri selektora uključuju selektore tipa, selektore klase i selektore ID-ja.

Kružne zagrade koje su deo osnovne sintakse poznate su kao blok deklaracije. Blok deklaracije primjenjuje skup stilova za dati selektor ili selektore.

Unutar bloka deklaracija, imaćete niz deklaracija. Svaka deklaracija se sastoji od svojstva i vrednosti.

Svojstvo je CSS identifikator koji određuje koju karakteristiku se stilizuje. Primer svojstva bi bilo svojstvo `background-color`.

Vrednost bi bila specifična podešavanja primenjena na tu svojstvo. Na primer, ako je svojstvo ``background-color``, vrednost bi mogla biti ``purple``, što postavlja boju pozadine na ljubičastu.

Nakon svakog imena svojstva, morate postaviti dvotačku, a nakon svake vrednosti, trebalo bi da imate zarez-tačku.

Sada kada znate sintaksu za CSS pravilo, pogledajmo primer. Omogućite interaktivni uresnik i kliknite na karticu `styles.css` da biste videli CSS kod.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

U ovom CSS pravilu, tip selektor cilja sve paragraf elemente na stranici.

Unutar bloka deklaracije, postoji jedna deklaracija.

Deklaracija sastoji se od svojstva ``color`` sa vrednošću postavljeno na ``blue``. Ovo će promeniti boju teksta za sve elemente paragrafa u ``blue``.

Ako želite da primenite isti skup stilova na više selektora, možete kreirati listu selektora gde je svaki selektor razdvojen zarezom.

Ovo je primer stilizovanja više selektora (da biste interagovali sa primerom, omogućite interaktivni uresnik):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

U ovoj listi selektora, postoji selektor `id` koji cilja na HTML element sa vrednošću `id` od `title`. Svi `id` selektori moraju početi sa znakom hashtag-a `#`.

Zatim postoji zarez praćen sa `class` selektor koji cilja sve HTML elemente sa `class` vrednošću od `subheading`. Svi klasni selektori moraju početi tačkom `.`

Cela lista selektora će dobiti isto stilizovanje, sa bojom teksta postavljenu na ``navy``.
