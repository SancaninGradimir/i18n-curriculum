# --interactive--

HTML, što je skraćenica za Hypertext Markup Language (Jezik označavanja hiperteksta), je jezik označavanja za kreiranje veb stranica. Kada posetite veb sajt i vidite sadržaj poput pasusa, naslova, linkova, slika i videa; to je HTML.

Evo malog primera koji koristi HTML elemente. Pokušajte da uredite deo teksta u editoru i vidite kako se promene ažuriraju u prozoru za pregled.

:::interactive_editor

```html
<h1>Main heading element</h1>

<p>I am a paragraph element.</p>
```

:::

HTML predstavlja sadržaj i strukturu web stranice korišćenjem elemenata. Većina elemenata ima otvarajući i zatvarajući tag. Ponekad se ti tagovi nazivaju početni i krajnji tagovi. Između tih dva taga nalaziće se sadržaj. Ovaj sadržaj može biti tekst ili drugi HTML elementi.

Ovo je još jedan primer paragrafa elementa. Promenite tekst u editoru da kaže `I love coding!` i vidite rezultate u prozoru za pregled.

:::interactive_editor

```html
<p>I am a paragraph element.</p>
```

:::

Otvarajući i zatvarajući tagovi počinju sa levom uglastom zagradom (``<``) i završavaju se desnom uglastom zagradom (``>``), a ime taga je postavljeno između ovih uglastih zagrada. Iako su imena HTML tagova neosetljiva na veličinu slova, široko prihvaćena konvencija i najbolja praksa je da se pišu malim slovima.

Evo bližeg pogleda samo na tagove za otvaranje i zatvaranje paragrafa:

```html
<p>
```

```html
</p>
```

Ono što razlikuje otvarajući tag od zatvarajućeg taga je kose crta (`/`) postavljena odmah nakon levog uglatog zagrade u zatvarajućem tagu. Neki HTML elementi nemaju zatvarački tag. Oni su poznati kao prazni elementi.

TEKST:

Ovo je primer elementa slike koji je prazan element:

```html
<img>
```

Primećujte da ovaj elementi slike nema zatvarajući tag i da ne sadrži nikakav sadržaj. Elementi koji nemaju sadržaj ne mogu imati sadržaj i imaju samo početni tag.

Ponekad ćete videti prazne elemente koji koriste `/` pre `>` ovako:

```html
<img />
```

Iako mnogi formateri koda poput Prettiera, odlučeće se uključiti `/` u prazne elemente, HTML specifikacija navodi da prisustvo `/` "ne označava početni tag kao samozatvarajući, već je nepotrebno i nema nikakvog efekta".

U razvoju u stvarnom svetu, videćete obe forme, pa je važno da ste upoznati sa obe.

Ako želite da prikažete sliku, moraćete da uključite nekoliko atributa unutar vašeg elementa slike. Atribut je posebna vrednost koja se koristi za prilagođavanje ponašanja za HTML element.

Ovo je primer elementa slike sa atributom `src`. Ažurirajte vrednost atributa `src` na `"https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"`, i videćete da se slika promeni u dve mačke koje mirno spavaju.

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg" />
```

:::

Atribut ``src`` se koristi za određivanje lokacije te slike. Za elemente slika, dobra praksa je uključiti još jedan atribut nazvan ``alt`` atribut. Atribut ``alt`` se koristi za pružanje kratkog, deskriptivnog teksta za slike.

Ovo je primer elementa slike sa atributima `src` i `alt`. Pokušajte da "slomite" sliku ažuriranjem vrednosti `src` na `"https://.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"`. Videćete da se slika nestaje i da je vidljiv samo tekst `alt`.

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

Pa, možda se pitate da li je sam HTML dovoljno za izradu veb stranice. Pa, odgovor je: zavisi. Ako gradite mali praktični projekat koji samo prikazuje tekst i slike, sam HTML bi mogao biti dovoljan. Međutim, ako kreirate moderno profesionalno veb mesto, biće vam potrebno imati HTML, CSS i JavaScript.

HTML je za sadržaj i strukturu. CSS je za stilizovanje. JavaScript služi za dodavanje interaktivnosti vašim veb stranicama. Dobra analogija za ovo je poređenje HTML-a, CSS-a i JavaScript-a sa kompletnom zgradom.

HTML predstavlja blokove, beton i armaturu koji čine zidove. To je temeljac koji čini zgradu snažnom. CSS predstavlja enterijer i eksterijer dizajn koji čini kuću prelepom. JavaScript predstavlja električni i vodeni sistem koji osigurava neprekidni pristup vodi i struji.
