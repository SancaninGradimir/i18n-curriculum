# --interactive--

Element neartikulisanih anotacija, ili kraće rečeno, element `u`, koristi se za predstavljanje ugrađenog teksta na koji je primenjena netekstualna anotacija.

Ovo je primer korišćenja elementa `u` za isticanje različitih pravopisnih grešaka:

:::interactive_editor

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

:::

U primeru, reči `incorrect`, `spelling` i `issues` su pogrešno napisane. Podrazumevano stilizovanje za element `u` je crno podvlačenje ispod teksta.

U HTML4, element `u` se koristio za svrhe stilizovanja. Ali u HTML5, element `u` bi trebalo da se koristi samo za ukazivanje na to da tekst ima primenjano ne-tekstualno anotiranje.

Ako želite da stilizujete deo teksta sa podvakučenjem, trebalo bi da koristite CSS umesto HTML-a.

Element za prekiranje linije, ili kraće rečeno element `s`, treba ga koristiti da predstavlja kada tekst više nije tačan ili relevantan. Evo primera korišćenja elementa `s` za prikaz otkazivanja aktivnosti:

:::interactive_editor

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

:::

U ovom primeru, prva rečenica je precrtana zato što je izlet otkazan zbog vremenskih razloga.

Element `s` nikada ne bi trebalo koristiti samo za prikaz promena u dokumentu. U tom slučaju, odgovarajući elementi bili bi element obrisanog teksta i element umetnutog teksta.

Element `ruby` predstavlja mali tekst prikazan iznad ili ispod glavnog teksta. Obično se koristi za prikaz izgovora istočnoazijskih znakova. Evo primera elementa `ruby` sa stranice MDN web dokumentacije:

:::interactive_editor

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

:::

Element ``rp``, ili element za rezervnu zagradu rubija, koristi se kao rezerva (fallback) za pretraživače koji nemaju podršku za prikazivanje rubijanskih anotacija.

Element ``rt``, ili rubinski tekstualni element, koristi se za indikovanje teksta za rubinsku anotaciju. Ovaj tekst se obično koristi za izgovor, ili detalje prevoda u istočnoazijskoj tipografiji.

Iako se element `ruby` može koristiti za druge vrste anotacija, najčešći slučaj upotrebe je za istočnoazijsku tipografiju.
