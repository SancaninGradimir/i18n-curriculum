# --interactive--

Element za ugrađeni kod se koristi za predstavljanje kratkih fragmenata koda unutar teksta. Uobičajene upotrebe elemenata za kod su tehnički članci i stranice dokumentacije.

Evo primera korišćenja elementa `code` za prikazivanje CSS kodnog isječka:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

U ovom primeru, CSS `color` svojstvo se koristi za postavljanje boje teksta na `blue`. Omotanjem ovog kodnog fragmenta unutar `<code>` tagova, komunicira pregledaču da je tekst deo ugrađenog koda.

Pretraživač će primeniti podrazumevane stilove za sadržaj unutar elementa ``code``. Podrazumevano stilizovanje je monospacen font.

Element `code` je namenjen da predstavlja jednu liniju koda. Ako želite da predstavite više linija koda, moraćete da postavite element `code` unutar elementa teksta sa unapred formatiranim tekstom.

Element za preformatirani tekst koristi se za predstavljanje preformatiranog teksta. Evo primera korišćenja elementa za preformatirani tekst radi prikazivanja CSS deklaracije:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

Kada koristite element `pre`, moraćete biti pažljivi u pogledu razmaka jer će se prikazati tačno kako je napisano unutar HTML dokumenta.

U pretraživaču ćete videti da je kod indentovan nekoliko razmaka udesno. Ako promenite uvlačenje u primeru koda, videćete razliku u uvlačenju na ekranu.

Kada je u pitanju uključivanje kodnih primera unutar vašeg HTML dokumenta, trebalo bi da koristite element `code` za kratke inline primere.

Ako trebate prikazati duže kodne fragmente, onda ćete morati da koristite elemente `pre` i `code`.
