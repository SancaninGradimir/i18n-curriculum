# --description--

Kada pišete Python kod, uobičajeno je naići na greške. Razumevanje ovih grešaka ključno je za brzo i efikasno debagovanje vašeg koda. Ove poruke vam tačno kažu šta je pošlo naopako, ako znate kako da ih pročitate.

Uobičajene greške u Pythonu uključuju `SyntaxError`, `NameError`, `TypeError`, `IndexError`, i `AttributeError`. One nastaju kada Python ne razume vaš kod, ili kada vaša logika ne odgovara podacima sa kojima radite.

Evo primera od `SyntaxError`:

```py
print("Hello, world!"
# SyntaxError: unexpected EOF while parsing
```

Na ovoj liniji nedostaje zatvarajuća zagrada. Python izbacuje ``SyntaxError`` jer kod ne prati pravilne sintaksne pravila.

Evo primera od `NameError`:

```py
print(name)
# NameError: name 'name' is not defined
```

Pokušavate da ispišete varijablu koja još uvek nije definisana. Python generiše ``NameError`` kada ne može da pronađe varijablu po tom imenu.

Evo primera od `TypeError`:

```py
5 + "5"
# TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

Ne možete sabrati ceo broj i niz znakova. Python generiše ``TypeError`` kada pokušate da izvršite operaciju na nekompatibilnim tipovima podataka.

Ovo je primer od `IndexError`:

```py
my_list = [1, 2, 3]
print(my_list[5])
# IndexError: list index out of range
```

Pokušavate da pristupite indeksu koji ne postoji u listi. Python generiše ``IndexError`` kada izađete van granica.

Ovo je primer od `AttributeError`:

```py
num = 42
num.append(5)
# AttributeError: 'int' object has no attribute 'append'
```

Objekat ``int`` nema metod ``append()``. Python generiše ``AttributeError`` kada pokušate da koristite metod ili svojstvo koje ne postoji za taj tip podataka.

Razumevanje uobičajenih poruka o greškama u Pythonu pomaže vam da rešavate probleme brže. Umesto pogađanja, pažljivo pročitajte poruku o grešci. Često vam govori tačno šta je pošlo naopako i gde treba da pogledate.
