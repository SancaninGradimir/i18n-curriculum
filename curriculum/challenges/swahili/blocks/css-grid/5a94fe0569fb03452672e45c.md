# --description--

Možete spojiti ćelije vaše mreže u <dfn>oblast</dfn> i dati toj oblasti jedinstveno ime. To učinite koristeći `grid-template-areas` na kontejneru, kao što je prikazano:

```css
grid-template-areas:
  'header header header'
  'advert content content'
  'advert footer footer';
```

Gornji kod grupiše ćelije mreže u četiri oblasti: `header`, `advert`, `content` i `footer`. Svaka reč predstavlja jednu ćeliju, a svaki par navodnika predstavlja jedan red.

# --instructions--

Izmenite šablon tako da oblast `footer` pokrije ceo donji red. Napomena: Ove oblasti trenutno ne mogu izazvati nikakav vidljivi efekat. Kasnije ćete koristiti element da biste videli kako funkcioniše.

# --hints--

Klasa `container` treba da ima svojstvo `grid-template-areas` kao primer, ali sa oblastima `footer` koje pokrivaju ceo donji red.

```js
assert.match(

  /.container\s*?{[\s\S]*grid-template-areas\s*?:\s*?["|']\s*?header\s*?header\s*?header\s*?["|']\s*?["|']\s*?advert\s*?content\s*?content\s*?["|']\s*?["|']\s*?footer\s*?footer\s*?footer\s*?["|']\s*?;[\s\S]*}/gi
);
```
