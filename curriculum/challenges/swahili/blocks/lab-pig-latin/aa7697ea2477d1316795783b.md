# --description--

Pig Latin je način menjanja engleskih reči praćenjem specifičnih pravila transformacije.

**Cilj:** Ispunite priče korisnika ispod i osigurajte da svi testovi prođu kako biste završili laboratoriju.

**Priče korisnika:**

1. Trebalo bi da kreirate funkciju `translatePigLatin` koja prima jedan niz kao argument.
1. Ako nizski argument počinje sa suglasnikom, vaša funkcija bi trebalo da uzme prvi suglasnik ili skup suglasnika, premesti ga na kraj reči, doda mu `ay` i vrati rezultat.
1. Ako nizski argument počinje samoglasnikom, vaša funkcija bi trebalo da doda `way` na kraj i vrati rezultat.
1. Vaša funkcija bi trebalo da obradi nizske argumente gde prvi samoglasnik dolazi usred reči i vrati odgovarajuće transformisani niz.
1. Ako nizski argument nema samoglasnika, vaša funkcija bi trebalo da doda `ay` na kraj i vrati rezultat.

Napomena: Za kontekst ove laboratorije, samoglasnici su `a`, `e`, `i`, `o`, i `u`. Slovo `y` se ne smatra samoglasnikom.

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
