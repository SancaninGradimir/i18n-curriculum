# --interactive--

Metoda ``.toFixed()`` je ugrađena JavaScript funkcija koja formatira broj koristeći fiksnote notaciju. Posebno je korisna kada morate kontrolisati broj decimalnih mesta broja, posebno za prikaz valutnih vrednosti ili prilikom rada sa preciznim merenjima.

Metoda `.toFixed()` se poziva na broj i prima jedan opcionalni argument, koji je broj cifara koje treba da se pojave iza decimalne tačke. Vraća string reprezentaciju broja sa navedenim brojem decimalnih mesta. Evo osnovnog primera kako radi `.toFixed()`:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

U ovom slučaju, ograničavamo broj decimalnih mesta na dve. Dakle, `3.14159` postaje `3.14`. Važno je napomenuti da `.toFixed()` vraća niz (string), a ne broj. To je zato što je metoda primarno namenjena za formatiranje brojeva radi prikaza, a ne za dalje proračune.

Metoda `.toFixed()` zaokružuje broj na najbližu vrednost koja se može predstaviti sa navedenim brojem decimalnih mesta. Ovo ponašanje zaokruživanja je važno razumeti:

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

Kao što možete da vidite, `.toFixed()` zaokružuje naviše kada je sledeća cifra `5` ili veće, i zaokružuje nadole inače. Ako pozovete `.toFixed()` bez argumenata, podrazumevano je na `0` decimalnih mesta:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

Metoda `.toFixed()` može biti posebno korisna kada radite sa finansijskim proračunima ili prikazivanjem cena:

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

U ovom primeru, `.toFixed(2)` osigurava da je ukupan iznos uvek prikazan sa dve decimale, što je standard za valutu u mnogim zemljama.

Zaključno, metoda `.toFixed()` je moćan alat za formatiranje brojeva u JavaScriptu, posebno kada morate kontrolisati prikaz decimalnih mesta. Iako se koristi prvenstveno za formatiranje izlaza, zapamtite njegovo ponašanje, posebno kada su potrebni precizni proračuni.
