# --description--

OS modul je još jedan standardni modul koji dolazi ugrađen u Node.js.

Omogućava vam da interagujete sa trenutnim operativnim sistemom na kojem radi Node, tako da možete pristupiti vitalnim informacijama kao što su tip operativnog sistema (OS), detalji procesora (CPU), dostupna memorija, ukupna memorija, mrežni interfejsi i još mnogo toga.

Da biste koristili OS modul, importujete ga na ovaj način:

```js
const os = require("os");
```

Modul za operativni sistem pruža nekoliko metoda koje možete koristiti da učinite svoju aplikaciju svesnijom sistema. Pogledajmo neke uobičajene.

Metoda `platform()` dohvaća operativni sistem na kojem je Node trenutno pokrenut:

```js
const os = require("os");
console.log(os.platform()); // darwin
```

Može biti korisno za implementaciju skriptovanja na više platformi:

```js
if (os.platform() === 'win32') {
 // Windows specific code
} else {
 // Non-Windows specific code
}
```

Metoda `arch()` prikazuje niz znakova koji predstavlja CPU arhitekturu za koju je Node.js kompajliran:

```js
const os = require("os");
console.log(os.arch()); // arm64
```

Ovo može biti korisno ako želite da korisnici preuzmu ispravne binare i zavisnosti za određenu arhitekturu.

Ostale moguće vrednosti su `'x64'`, `'arm'`, `'arm64'`, `'ia32'`, `'mips'`, `'ppc'`, i drugi.

`type()` dobija zvanično ime operativnog sistema, tako da možete programski identifikovati operativne sisteme:

```js
const os = require("os");
console.log(os.type()); // Darwin (the core OS for macOS, iOS, and other Apple products)
```

`release()` prikazuje verziju OS kernela sistema, što je osnovni deo operativnog sistema koji upravlja sistemskim resursima i komunikacijom između hardverskih i softverskih komponenti. Ova metoda može biti korisna za praćenje kompatibilnosti između verzija OS kernela i zahteva servera.

Vraća ovo kao niz poput `'20.6.0'`:

```js
const os = require("os");
console.log(os.release()); // 25.0.0
```

`version()` vraća specifičnu verziju operativnog sistema sa više detalja od metode `release()`:

```js
const os = require("os");
console.log(os.version());
// Darwin Kernel Version 25.0.0: Wed Sep 17 21:41:39 PDT 2025;
// root:xnu-12377.1.9~141/RELEASE_ARM64_T8103
```

`cpus()` vraća niz objekata sa detaljima o svakom logičkom CPU jezgru. Ovo može pomoći u praćenju opterećenja CPU-a:

`uptime()` pokazuje koliko vremena je prošlo otkako je sistem pokrenut. Može pomoći u određivanju koliko dugo serveri rade:

```js
const os = require("os");
console.log(os.uptime()); // 23047
```

`totalmem()` i `freemem()` prikazuju ukupnu količinu sistemske memorije u bajtovima i slobodnu sistemsku memoriju u bajtovima, respektivno:

```js
const os = require("os");
console.log(os.totalmem()); // 8589934592 (8 GB)
console.log(os.freemem()); // 93585408 (87 MB)
```

`userInfo()` vraća objekat koji sadrži informacije o trenutnom sistem korisniku:

```js
const os = require("os");
console.log(os.userInfo());

/*

}
*/
```
