# --description--

U JavaScriptu i mnogim drugim programskim jezicima, tačke-zarez pomažu u definisanju naredbi i poboljšavaju čitljivost koda.

U JavaScriptu, tačka-zarez (`;`) se koristi za označavanje kraja naredbe.

Kao što tačka (`.`) označava kraj rečenice na engleskom jeziku, tačka-zarez signalizira kraj izvršne linije koda. To omogućava JavaScript motoru da razlikuje između odvojenih komandi.

Primer:

```js
let variableOne = 5;  // Declare a variable and assign a value
let variableTwo = 10; // Declare another variable and assign a value
```

U ovom kodu, tačke-zarez na kraju svake linije označavaju kraj svake naredbe. Bez njih, JavaScript motor bi mogao imati poteškoća sa interpretacijom gde jedna naredba prestaje i gde druga počinje.

Tačke-zarez se primarno koriste za označavanje kraja naredbe. To pomaže JavaScript motoru da razume razdvajanje pojedinačnih instrukcija, što je važno za pravilno izvršavanje.

```js
let a = 1;   // Statement ends here
let b = 2;   // Another statement starts here
```

Iako JavaScript ima Automatsko ubacivanje tačke-zarez-a (Automatic Semicolon Insertion - ASI) koje može automatski dodati tačke-zarez, eksplicitno njihovo uključivanje poboljšava jasnoću koda i pomaže u sprečavanju grešaka koje mogu proizaći iz neočekivanog ponašanja ASI.

Tačke-zarez se koriste u mnogim programskim jezicima, uključujući C, C++ i Javu.

Tačke-zarez označavaju kraj naredbi ili instrukcija, pomažući kompajleru ili interpreтеру da pravilno parsiraju kod. Kompajler prevodi kod programskog jezika višeg nivoa u mašinski čitljiv kod, što stvara izvršnu datoteku.

Jasnim definisanjem naredbi, tačke-zarez doprinose čitljivosti i održivosti koda. Tačke-zarez pomažu u sprečavanju dvosmislenosti pri izvršavanju koda i osiguravaju da su naredbe pravilno završene.

Tačke-zarez su fundamentalni deo JavaScripta i mnogih drugih programskih jezika.

Oni označavaju kraj naredbi, poboljšavaju čitljivost koda i pomažu u izbegavanju grešaka povezanih sa Automatskim ubacivanjem tačke-zarez-a.

Razumevanjem i pravilnom korišćenjem tačke-zarez-a, možete napisati pouzdaniji i održiviji kod.
