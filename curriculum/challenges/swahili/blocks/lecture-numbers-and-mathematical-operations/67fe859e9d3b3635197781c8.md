# --description--

Celobrojni brojevi i floatovi su primarni numerički tipovi podataka u Pythonu. Njima možete skladištiti numeričke podatke i izvršavati matematičke operacije.

Pogledaćemo šta su ceo brojevi i floatovi, kako se vrši aritmetički proračun sa njima, i na nekoliko metoda koje Python nudi za rad sa oba.

Цели бројеви су цео броеви без децималних запеза, било позитивни нити негативни:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

Evo kako da izvršite operaciju sabiranja sa celim brojevima:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

Evo kako da izvršite oduzimanje sa celobrojima:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

Evo kako da se izvrši operacija množenja sa celim brojevima:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

I ovde je kako da se izvrši operacija deljenja sa celim brojevima:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Division:', div_ints) # Division: 4.666666666666667
```

Flotni brojevi su pozitivni ili negativni brojevi sa decimalnim tačkama, kao što su `3.14`, `-0.5`, ili `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

Evo operacije sabiranja sa floatovima:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

Evo operacije oduzimanja sa floatovima:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

Evo operacije množenja sa floatovima:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

I ovo je operacija deljenja sa floatovima:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

Ako dodate ceo broj i plutajući broj, rezultat se automatski konvertuje u plutajući broj:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

Ovo je tačno i za druge osnovne aritmetičke operacije, kao što su oduzimanje, množenje i deljenje. Ako pomešate cele brojeve i brojeve sa decimalnim zarezom, Python će vratiti rezultat kao float.

Takođe možete izvesti složenije aritmetičke proračune kao što je dobijanje ostatka deljenja dva broja pomoću modulo operatera, celobrojno deljenje i eksponencijalizacija sa celim i float brojevima.

Operator modula (`%`) vraća ostatak kada je vrednost na levoj strani podeljena sa vrednošću na desnoj strani:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

Celobrojno deljenje deli dva broja i vraća najveći ceo broj manji ili jednak rezultatu. To se radi sa operaterom dvostruke kose crte (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

Eksponenciranje podiže broj na stepen drugog i radi se sa operaterom dvostruke zvezdice (`**`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

Ponekad možete primetiti da rezultat operacije koja uključuje floatove ima više decimalnih cifara nego što se očekuje. Na primer, zbir `0.1 + 0.2` je jednak `0.30000000000000004` umesto `0.3`.

Ovo se dešava jer su brojevi pohranjeni u binarnom formatu, a neke frakcije ne mogu biti tačno predstavljene u binaru. Kao rezultat, oni su pohranjeni kao konačne aproksimacije, na isti način kako frakcija `1/3` ne može biti predstavljena sa konačnim brojem cifara u decimalnom sistemu i skraćena je nakon određenog broja svojih beskonačnih cifara (`0.33333...`).

Ovo dovodi do malih grešaka za zaokruživanje.

Python takođe pruža ugrađene funkcije za konvertovanje bilo numeričkih podataka ili stringova u celobrojne ili decimalne brojeve.

Funkcija `float()` vraća broj sa plutajućom zarezom konstruisan od datog broja:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

Funkcija `int()` vraća ceo broj konstruisan od zadatog broja:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

Takođe, možete koristiti iste ugrađene funkcije da konvertujete niz u float ili ceo broj:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

Evo nekih drugih metoda koje Python pruža za rad sa celobrojnim brojevima i brojevima sa decimalnim delovima.

- `round()`: Zaokružuje broj na navedeni broj decimalnih mesta. Podrazumevano, ova funkcija zaokružuje do najbližeg celog broja i vraća ceo broj bez decimalnih mesta:

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: vraća apsolutnu vrednost broja,

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: podiže broj na stepen drugog ili izvodi modularnu eksponencijaciju.

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
