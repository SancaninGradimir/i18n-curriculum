# --interactive--

U modernom JavaScriptu, klase su kao nacrti koje možete definisati u kodu za kreiranje više objekata iste vrste.

Na primer, ako modelujete inventarni sistem, moraćete da kreirate više objekata po tipu proizvoda.

Svaki pojedinačni proizvod će imati sopstvena svojstva i akcije, ali proizvodi iste vrste mogu se kreirati iz istog „nacrta“. Za to se koriste klase.

Klasama se može definisati ili izraz klase ili deklaracija klase.

TEKST:

Evo osnovne sintakse za deklaraciju klase:

```js
class MyClassName {
  // Class Methods
  constructor() { ... }
  method1() { ... }
  method2() { ... }
  ...
}
```

Počinjete tako što ćete napisati ključnu reč ``class`` praćenu imenom klase. Kada imenujete klase, trebalo bi da koristite konvenciju ``PascalCase`` umesto ``camelCase``.

Sadržaj unutar vitičastih zagrada poznat je kao telo klase. Ovo je mesto gde će ići konstruktor i metode.

Konstruktor je specijalna metoda koja se automatski poziva kada se kreira novi objekat iz klase. Obično se koristi za inicijalizovanje svojstava objekta.

U ovom primeru, možete videti `Dog` klasu sa konstruktorom.

```js
class Dog {
  constructor(name) {
    this.name = name;
  }
}
```

Ključna reč `this` se ovde koristi za referenciranje trenutne instance klase `Dog`, omogućavajući vam da dodelite svojstvo `name` objektu koji se kreira. Vi ćete saznati više o ključnoj reči `this` u budućoj lekciji.

Ispod konstruktora, možete imati ono što se naziva metode. Metode su funkcije definisane unutar klase koje obavljaju akcije ili operacije. One se koriste za definisanje ponašanja koja instancama te klase mogu da obave.

Ovo je primer dodavanja metode nazvane `bark` u klasu `Dog`.

```js
class Dog {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
}
```

Kada se metoda `bark` pozove, poruka će biti logovana u konzolu sa stvarnim imenom koje je prosleđeno. Ako kreirate instancu `Dog` sa imenom `"Buddy"`, pozivanje metode `bark()` na toj instanci će ispisati `"Buddy says woof!"` u konzolu.

Kada definišete svoju klasu, možete kreirati instancu te klase sa ključnom reči ``new``, nazivom klase i njenim argumentima.

U ovom primeru, kreiramo novi `Dog` instancu sa imenom `Gino` i zatim je dodeljujemo promenljivoj `dog`.

```js
const dog = new Dog("Gino");
```

Kada napišete ``new Dog("Gino")`` u svoj kod, kreira se novi objekat i konstruktor se automatski poziva sa dato argumentima.

Zatim možete pristupiti ovoj svojstvu koristeći notaciju tačke, kao što vidite u sledećoj liniji koda:

```js
console.log(dog.name);
```

Evo celog primera:

:::interactive_editor

```js
class Dog {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
}

const dog = new Dog("Gino");
console.log(dog.name);
```

:::

Takođe možete pozvati metode. U ovom primeru, pozivamo metodu `bark()` na instanci `dog`.

```js
dog.bark();
```

Evo celog primera sa uključenim pozivom metode:

:::interactive_editor

```js
class Dog {
  constructor(name) {
    this.name = name;
  } 
  bark() {
    console.log(`${this.name} says woof!`);
  }
}

const dog = new Dog("Gino");
console.log(dog.name);
dog.bark();
```

:::

Kao što je ranije pomenuto, možete definisati klase i kao izraz klase. Ovde je klasa anonimna i dodeljena varijabli.

Evo šta raniji primer izgleda kao izraz klase:

```js
const Dog = class {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
};
```

U većini slučajeva, koristićete deklaracije klasa, ali razumevanje izraza klasa je takođe vredno, jer mogu ponuditi veću fleksibilnost, posebno u situacijama dinamičkog programiranja.

Korišćenjem klasa i razumevanjem njihove osnovne sintakse, možete kreirati ponovno upotrebljiv kod, organizovati ga u logičke jedinice i unaprediti modularnost koda.
