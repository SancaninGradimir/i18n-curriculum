# --description--

Funkcije su ponovno upotrebljivi delovi koda koji se izvršavaju kada ih pozovete. Mnogi programski jezici dolaze sa ugrađenim funkcijama koje olakšavaju početak rada. Python nije izuzetak, i već smo pokrili neke ugrađene funkcije kao što je ``print()`` u prethodnim lekcijama.

Još jedna korisna ugrađena funkcija je `input()`, koja vam omogućava da zatražite unos od korisnika:

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

S druge strane, `int()` konvertuje broj, boolean vrednost i numerički string u ceo broj:

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

Takođe možete napisati sopstvene prilagođene funkcije. Za to, koristite ključnu reč ``def``, praćenu imenom koje želite da date svojoj funkciji, par zagrada i dvotačkom. Zatim na novoj liniji pišete kod koji bi vaša funkcija trebalo da izvrši. Kod koji funkcija izvršava takođe se naziva telo funkcije.

Evo primera prilagođene funkcije pod nazivom `hello` koja štampa niz `Hello World` u terminal:

```python
def hello():
    print('Hello World')
```

Da biste pokrenuli funkciju, morate je pozvati sa njenim imenom praćenim parom zagrada:

```python
hello() # Hello World
```

Primećujte uvlačenje pre `print('Hello World')`. Kao što ste možda zapamtili iz prethodnih lekcija, Python se oslanja na uvlačenje da bi utvrdio koje grupe naredbi pripadaju zajedno. Ove grupe naredbi nazivaju se blokovi koda.

Evo još jedne jednostavne funkcije koja štampa zbir dva broja na terminal:

```python
def calculate_sum(a, b):
    print(a + b)
```

Možete videti da naša funkcija, `calculate_sum`, ima `a` i `b` u zagradi, razdvojene zarezom. To se nazivaju parametri. Razmislite o parametrima kao o zamenicama promenljivama koje deluju kao "prostor" za vrednosti koje prosleđujete funkcijama kada ih pozovete.

Da biste koristili parametre, morate proslediti "argumente". Argumenti su vrednosti koje prosleđujete funkciji kada je pozovete.

Evo kako pozvati funkciju ``calculate_sum`` da sabere brojeve ``3`` i ``1``:

```python
calculate_sum(3, 1) # 4
```

Ako pozovete funkciju bez tačnog broja argumenata, dobićete `TypeError`:

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

Funkcije takođe koriste specijalnu ključnu reč ``return`` za izlazak iz funkcije i vraćanje vrednosti. Ako eksplicitno ne koristite ``return``, Python će podrazumevano vratiti ``None``.

Evo primera:

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

Možete videti da funkcija `calculate_sum` ispisuje zbir `a` i `b`, ali ne vraća ništa eksplicitno. Dakle, kada mu dodelimo rezultat u `my_sum`, vrednost je zapravo `None`. Da biste to popravili, možete koristiti ključnu reč `return` da vratite rezultat:

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

Sada, `calculate_sum` vraća sumu od `a` i `b`, što se skladišti u `my_sum`.
