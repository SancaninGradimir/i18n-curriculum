# --description--

U Pythonu, opseg određuje tačku na kojoj možete pristupiti promenljivoj. To je ono što kontroliše trajanje promenljive i kako se ona razrešava u različitim delovima koda.

Da bi se pravilno odredio opseg, Python prati pravilo **LEGB**, što znači sledeće:

- **Lokalni opseg (L)**: Varijable definisane u funkcijama ili klasama.
    
- **Okružujući opseg (E)**: Varijable definisane u okružujućim ili ugnježdene funkcije.
    
- **Globalni opseg (G):** Varijable definisane na najvišem nivou modula ili fajla.
    
- **Ugrađeni opseg (B)**: Rezervisana imena u Pythonu za preddefinisanu funkcije, module, ključne reči i objekte.

Python koristi pravilo LEGB za rešavanje opsega promenljivih u vašem programu. Istražićemo svako od ovih pravila kako biste bolje razumeli proces.

**Lokalni opseg** znači da promenljiva deklarisana unutar funkcije ili klase može se pristupiti samo unutar te funkcije ili klase.

Evo primera:

```python
def my_func():
    my_var = 10
    print(my_var)
```

U ovom slučaju, funkcija `my_func` ima svoj opseg koji ne može biti dostupan izvan funkcije. Pozivanje `my_func` će proizvesti `10`, ali štampanje `my_var` izvan funkcije će dovesti do `NameError`:

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**Okružujući opseg** znači da funkcija koja je ugnježđena unutar druge funkcije može pristupiti varijablama funkcije u kojoj je ugnježđena.

Na primer:

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

U ovom primeru, unutrašnja funkcija, `inner_func`, može slobodno pristupiti varijabli `msg` definisanoj u spoljnoj funkciji, `outer_func`. Međutim, imajte na umu da spoljne funkcije ne mogu pristupiti varijablama definisanim unutar bilo koje ugnježđene funkcije:

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

To je zato što `res` je lokalno ograničeno na `inner_func`. Takođe, primetite da `outer_func` pokušava da ispiše `res` pre nego što se `inner_func` pozove.

Jedno rešenje je da se `res` inicijalizuje kao prazan niz karaktera u obuhvatnom opsegu, koji je unutar `outer_func`. Zatim unutar `inner_func`, napravite `res` ne-lokalnu varijablu sa ključnom rečju `nonlocal`:

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**Globaлни opseg** se odnosi na varijable koje su deklarisane izvan bilo kojih funkcija ili klasa i kojima se može pristupiti bilo gde u programu. Ovde, `my_var` se može pristupiti bilo gde, čak i unutar funkcije u kojoj nije definisan:

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

I ako želite da učinite lokalno opsežnu varijablu definisanu unutar funkcije globalno dostupnom, možete koristiti ključnu reč `global`:

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

Takođe možete koristiti `global` ključu za modifikovanje globalne varijable:

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

Konačno, **ugrađeni opseg** se odnosi na sve ugrađene funkcije, module i ključne reči Pythona, i dostupni su bilo gde u vašem programu:

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
