## Uvod u dinamičko programiranje

- **Definicija**: Dinamičko programiranje je algoritamska tehnika koja rešava složene probleme razbijanjem na jednostavnije podprobleme i skladištenjem rezultata kako bi se izbegle višestruke kalkulacije. 
- **Preklapajući Podproblemi**: Isti manji problemi pojavljuju se više puta prilikom rešavanja većeg problema. Umesto ponovnog izračunavanja ovih podproblema, skladištimo njihova rešenja.
- **Optimalna Struktura**: Optimalno rešenje problema sadrži optimalna rešenja njegovih podproblema. To znači da možemo izgraditi najbolje rešenje kombinovanjem najboljih rešenja manjih delova.

## Rešenja dinamičkog programiranja

- **Memoizacija (Top-down pristup)**: Memoizacija čuva rezultate resursno zahtevnih funkcionalskih poziva i vraća keširani rezultat kada se ponovo pojave isti ulazi.

```js
function climbStairsMemo(n, memo = {}) {
  // If already calculated, return cached value
  if (memo[n] !== undefined) {
    return memo[n];
  }

  // Base cases: 1 way for 1 step, 2 ways for 2 steps
  if (n <= 2) {
    return n;
  }

  // Calculate once and store in memo for future use
  memo[n] =
    climbStairsMemo(n - 1, memo) +
    climbStairsMemo(n - 2, memo);

  return memo[n];
}
```

- **Tabulacija (Pristup odozdo ka gore)**: Tabulacija gradi rešenje od osnove, popunjavajući tabelu rešenjima za podprobleme.

```js
function climbStairsTabulation(n) {
  if (n <= 2) {
    return n;
  }

  // Create an array to store solutions from 0 to n
  const dp = new Array(n + 1).fill(0);

  // Base cases
  dp[1] = 1; // 1 way to reach step 1
  dp[2] = 2; // 2 ways to reach step 2

  // Build up the solution iteratively
  for (let i = 3; i <= n; i++) {
    // Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
    dp[i] = dp[i - 1] + dp[i - 2];
  }

  return dp[n];
}
```

## Primene u stvarnom svetu korišćenjem dinamičkog programiranja

- **Optimizacija ruta**: GPS sistemi koriste algoritme dinamičkog programiranja za pronalaženje najkraćih puteva između lokacija.
- **Obrada teksta**: Proverivači pravopisa i funkcije automatskog dovršavanja često se oslanjaju na dinamičko programiranje za izračunavanje udaljenosti editovanja između reči.
- **Finansijsko modelovanje**: Investicione strategije i optimizacija portfolija često primenjuju tehnike dinamičkog programiranja.
- **Alokacija resursa**: Problem ruksaka i njegove varijante pojavljuju se u raspoređivanju, budžetiranju i upravljanju resursima.

## Kada koristiti dinamičko programiranje

Trebalo bi da razmotrite korišćenje dinamičkog programiranja u sledećim scenarijima:

- Problem se može razbiti na preklapajuće podprobleme.
- Problem poseduje optimalnu podstrukturu.
- Naiovno rekurzivno rešenje bi uključivalo ponovne proračune.
- Morate optimizovati za vremensku složenost na uštrb prostorne složenosti.
