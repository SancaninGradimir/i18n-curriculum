# --interactive--

Destrukturiranje nizova je karakteristika u JavaScriptu koja vam omogućava da izvučete vrednosti iz nizova i dodelite ih varijable na koncizniji i čitljiviji način. Pruža jednostavnu sintaksu za raspakivanje elemenata niza u zasebne varijable.

Ova tehnika je posebno korisna kada radite sa nizovima i funkcijama koje vraćaju više vrednosti. Evo primera korišćenja dekonstrukcije niza:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];

let [first, second, third] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(third);  // "orange"
```

:::

U ovom primeru, imamo niz nazvan `fruits` sa tri elementa. Koristeći dekonstrukciju niza, dodeljujemo prvi element promenljivoj `first`, drugi element `second`, a treći element `third`. Ovo nam omogućava da lako pristupimo pojedinačnim elementima niza bez korišćenja indeksne notacije.

Evo kako bi izgledalo da pristupite svakom od tih elemenata po njihovom indeksu umesto korišćenja razpakovanja nizova:

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const first = fruits[0];
const second = fruits[1];
const third = fruits[2];

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

Destrukturiranje nizova takođe omogućava da preskočite elemente koji vas ne zanimaju korišćenjem zarezaka. Na primer:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow"];
let [firstColor, , thirdColor] = colors;

console.log(firstColor); // "red"
console.log(thirdColor); // "blue"
```

:::

U ovom primeru, preskačemo drugi element niza `colors` koristeći dodatnu zarez. Ovo dodeljuje `red` na `firstColor` i `blue` na `thirdColor`, efektivno ignorisanjem `green`.

Još jedna moćna karakteristika dekonstrukcije nizova je mogućnost korišćenja podrazumevanih vrednosti. Ako niz ima manje elemenata od promenljivih koje pokušavate da dodelite, možete pružiti podrazumevane vrednosti:

:::interactive_editor

```js
let numbers = [1, 2];
let [a, b, c = 3] = numbers;

console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
```

:::

Ovde, dodeljujemo podrazumevanu vrednost `3` na `c` jer niz `numbers` nema treći element.

Sada ćemo razmotriti sintaksu za ostatke (rest syntax), označenu sa tri tačke (`...`). Omogućava vam da uhvatite preostale elemente niza koji nisu dekonstruisani u novi niz. Evo kako to radi:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let [first, second, ...rest] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(rest);   // ["orange", "mango", "kiwi"]
```

:::

U ovom primeru, `first` i `second` hvataju prva dva elementa niza `fruits`, a `rest` hvata sve preostale elemente kao novi niz. Sintaksa za ostatak mora biti poslednji element u obrascu dekonstrukcije.

Destrukturiranje nizova je moćna karakteristika koja može učiniti vaš kod sažetijim i lakšim za čitanje. Posebno je korisno kada radite sa nizovima, i kada morate da izvučete specifične elemente iz niza.
