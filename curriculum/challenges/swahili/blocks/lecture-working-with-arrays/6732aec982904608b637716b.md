# --interactive--

Nizovi u JavaScriptu su dinamički, što znači da možete lako dodati ili ukloniti elemente iz njih. Postoji četiri glavne metode za dodavanje i uklanjanje elemenata sa početka i kraja niza: `push()`, `pop()`, `shift()`, i `unshift()`. Istražićemo svaku od ovih metoda u detalju.

Metoda `push()` se koristi za dodavanje jednog ili više elemenata na kraj niza. Vraćena vrednost za metodu `push()` je nova dužina niza. Evo primera dodavanja novog voća postojećem `fruits` nizu:

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

U ovom primeru, počinjemo sa nizom nazvanim `fruits` koji sadrži dva elementa. Zatim koristimo metod `push()` da dodamo string `orange` na kraj niza.

Možda ste primetili da koristimo `const` kada deklarišemo niz `fruits`. Ali zašto je moguće dodati više elemenata u ovaj `fruits` niz kada je `fruits` konstanta? To je moguće jer deklarisanje niza sa ključnom reči `const` stvara referencu na niz. Iako je sam niz mutabilan i može se modifikovati, ne možete ponovo dodeliti novu vrednost konstanti `fruits`, kao što je ovo:

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

Sledeći metod na koji ćemo pogledati je `pop()` metod. Metod `pop()` uklanja poslednji element iz niza i vraća taj element. Takođe modifikuje originalni niz. Evo kako funkcioniše:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

U ovom primeru, počinjemo sa nizom od tri voća. Metoda `pop()` uklanja poslednji element (`orange`) iz niza i vraća ga. Originalni niz `fruits` je modifikovan i sadrži samo dva elementa.

Metoda ``unshift()`` dodaje jedan ili više elemenata na početak niza i vraća njegovu novu dužinu. Radi slično kao ``push()``, ali modifikuje početak niza umesto kraja. Evo primera:

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

U ovom primeru, koristimo `unshift()` da dodamo broj `1` na početak niza `numbers`. Metoda vraća novu dužinu niza, koja je `3`.

Konačno, metoda `shift()` uklanja prvi element iz niza i vraća taj element. Slično je `pop()`, ali radi na početku niza umesto na kraju. Evo kako to funkcioniše:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

U ovom primeru, počinjemo sa nizom od tri boje. Metoda `shift()` uklanja prvi element (`red`) iz niza i vraća ga. Originalni niz `colors` je modifikovan da sadrži samo dva elementa.

Napomena da iako `push()` i `unshift()` mogu dodati više elemenata odjednom, `pop()` i `shift()` uklanjaju samo jedan element u jednom trenutku.
