# --interactive--

U prethodnoj lekciji, upoznati ste sa radom sa nizovima i pristupanjem različitim elementima u nizovima. Evo podsetnika kako da pristupite drugom elementu u nizu:

:::interactive_editor

```js
const fruits = ["apple", "banana", "cherry"];

console.log(fruits[1]); // "banana"
```

:::

Pošto su nizovi indeksirani od nule, prvi element će biti na indeksu `0`, drugi element je na indeksu `1`, i tako dalje. Važno je napomenuti da ako pokušate pristupiti indeksu koji ne postoji u nizu, JavaScript će vratiti `undefined`.

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]); // undefined
```

:::

U ovom primeru, nema elementa na indeksu ``3`` za niz ``fruits``. Zbog toga će log prikazati ``undefined``. Sada, pogledajmo kako ažurirati elemente u nizu. Možete ažurirati element dodeljivanjem nove vrednosti određenom indeksu.

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[1] = "blueberry";
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

:::

U ovom primeru, zamenili smo `banana` sa `blueberry` na indeksu `1`. Ova metoda vam omogućava da promenite bilo koji element u nizu, sve dok znate njegov indeks. Takođe možete dodati nove elemente u niz dodeljivanjem vrednosti indeksu koji još ne postoji:

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[3] = "date";
console.log(fruits); // ["apple", "banana", "cherry", "date"]
```

:::

Međutim, budite oprezni kada ovo radite. Ako dodelite vrednost indeksu koja je mnogo veća od trenutne dužine niza, kreiraćete nedefinisovane elemente za indekse između, što može dovesti do neočekivanog ponašanja. Kako nastavite da radite sa JavaScript-om, shvatićete da su ove metode pristupa i ažuriranja elemenata niza fundamentalne za mnoge programerske zadatke. Bilo da gradite jednostavnu listu stvari za uraditi (todo list) ili obrađujete složene strukture podataka, ove veštine će biti neprocenjive.
