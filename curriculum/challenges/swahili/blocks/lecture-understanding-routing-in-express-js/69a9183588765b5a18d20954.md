# --description--

U Expressu, objekat `res` (response) koristi se za slanje odgovora nazad klijentu nakon obrade dolaznih zahteva. Express pruža nekoliko ugrađenih metoda za slanje različitih tipova odgovora. Na primer, možete poslati običan tekst, podatke u JSON formatu, preusmeriti korisnika na drugu URL adresu ili prikazati šablon.

Jedan od najčešće korišćenih metoda odgovora je `res.send()`:`

```javascript
app.get("/text", (req, res) => {
  res.send("Hello, World!");
});
```

Metoda `res.send()` šalje odgovor nazad klijentu. Može poslati običan tekst, HTML, JSON ili čak binarne podatke. U ovom primeru, kada klijent poseti `/text`, server odgovara tekstom `Hello, World!`.

Drugi često korišćeni metod odgovora je `res.json()`.

Ovaj metod šalje JSON odgovor klijentu. Posebno je koristan prilikom izgrađivanja API-ja, gde često morate vratiti strukturirane podatke kao što su objekti ili nizovi. Express automatski konvertuje JavaScript objekat u JSON formatirani string:

```javascript
app.get("/user", (req, res) => {
  res.json({ name: "John Doe", age: 30 });
});
```

Ako klijent pošalje zahtev na `/user`, server odgovara sledećim:

```json
{
  "name": "John Doe",
  "age": 30
}
```

Kada trebate kontrolisati HTTP status kod odgovora, možete koristiti `res.status()`. Ova metoda postavlja status kod i obično se povezuje sa drugom metodom odgovora kao što je `res.send()` ili `res.json()`:

```javascript
app.get("/notfound", (req, res) => {
  res.status(404).send("Not Found");
});
```

Ovde, server postavlja kod stanja na `404` i šalje poruku `Not Found` nazad klijentu.

Ako želite da pošaljete korisnika na drugačiji URL, možete koristiti `res.redirect()`:

```javascript
app.get("/oldpage", (req, res) => {
  res.redirect("/newpage");
});
```

Kada klijent poseti `/oldpage`, automatski će biti preusmeren na `/newpage`.

Konačno, `res.render()` se koristi kada radite sa mašinama za šablone kao što su Pug, EJS ili Handlebars. On renderuje šablon prikaza i šalje rezultujući HTML klijentu:

```javascript
app.get("/profile", (req, res) => {
  res.render("profile", { username: "john_doe" });
});
```

U ovom primeru, server renderuje `profile` pogled i prosleđuje varijablu `username` mu, koja se zatim može prikazati u HTML-u.

Ovi metodi odgovora omogućavaju vam da vratite različite tipove podataka u zavisnosti od prirode zahteva. Bez obzira da li šaljete tekst, JSON podatke ili preusmeravate klijenta, Express ima fleksibilan set alata za rukovanje time.
