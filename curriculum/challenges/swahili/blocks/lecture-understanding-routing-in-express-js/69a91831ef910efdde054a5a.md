# --description--

U Express-u, parametri rute su dinamičke vrednosti ugrađene u putanju URL-a. Omogućavaju vašoj aplikaciji da uhvati delove URL-a koji se mogu menjati sa svakim zahtevom. Ovo je korisno za kreiranje dinamičkih stranica, kao što je prikazivanje određenog korisnika, blog objave ili proizvoda na osnovu URL-a.

Evo primera:

```javascript
// Route with a parameter in the path
app.get("/post/:postId", (req, res) => {
  const postId = req.params.postId;
  res.send(`Viewing post with ID: ${postId}`);
});
```

Ovde, ruta sluša GET zahteve na `/post/:postId`.

* `:postId` je parametar rute, zamenski deo u URL-u koji će biti zamenjen stvarnom vrednošću. Na primer, zahtev ka `/post/42` odgovara ovoj ruti, a `42` postaje vrednost za `postId`.
    
* Možete pristupiti parametrima rute koristeći `req.params`. Objekt `req.params` sadrži sve parametre rute, i u ovom slučaju, `req.params.postId` sadrži vrednost iz URL-a, `42`.
    

Konačno, `res.send()` šalje odgovor nazad klijentu. Dakle, ako je URL `/post/42`, klijent vidi `Pregledavanje objave sa ID-om: 42`.

Dakle, kao što vidite, parametri rute čine vaše rute fleksibilnim i dinamičkim bez potrebe za odvojenom rutom za svaku moguću vrednost.

Sada kada razumete kako parametri rute funkcionišu, vratimo se na primer iz poslednje lekcije i vidimo kako možete koristiti dinamičke rute za različite tipove menija:

```javascript
app.get("/menu/:category", (req, res) => {
  const category = req.params.category;
  res.send(`You are viewing the ${category} menu!`);
});
```

I evo objašnjenja kako ovo funkcioniše:

1. Korisnik poseti [http://localhost:3000/menu/drinks](http://localhost:3000/menu/drinks).
    
2. Server poklapa ovaj zahtev sa rutom `/menu/:category` i hvata parametar `category` kao `drinks`.
    
3. Server šalje ovaj odgovor nazad pretraživaču: `Pregledavate meni sa pićima!`
    

Slično tome, ako korisnik poseti [http://localhost:3000/menu/food](http://localhost:3000/menu/food), server odgovara sledećim: `Pregledavate meni sa hranom!`

Parametri rute omogućavaju prikaz dinamičnog sadržaja na osnovu URL-a. Umesto kreiranja odvojene rute za svaku kategoriju, možete koristiti jednu rutu da obradite više kategorija kao što su pića, hrana, deserti i još mnogo toga, samo menjanjem URL-a.
