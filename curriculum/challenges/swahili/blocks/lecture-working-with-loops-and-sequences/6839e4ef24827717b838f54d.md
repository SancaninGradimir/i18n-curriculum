# --description--

Tokom prethodnih lekcija, navikljeni ste na definisanje funkcija korišćenjem ključne reči `def` na ovaj način:

```py
def square(num):
    return num ** 2

print(square(4)) # 16
```

Ali kada je u pitanju rad sa funkcijama višeg reda kao što su `map()` i `filter()`, možete koristiti anonimnu ugrađenu (inline) funkciju. Ovde dolaze lambda funkcije.

Evo kako izgleda funkcija `square()` kada se refaktoriše u lambda funkciju:

```py
lambda num: num ** 2
```

Kao što je ranije pomenuto, lambda funkcije su anonimne, pa ova funkcija više nema ime `square` povezano sa njom. Lambda funkcije su odlične kada ih treba koristiti u funkcijama višeg reda, kao što je ovo:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

U ovom primeru, imamo listu brojeva i želimo da kreiramo novu listu parnih brojeva. Zato prosleđujemo lambda funkciju kao jedan od argumenata funkciji `filter()` da bismo dobili novu listu koja sadrži brojeve `2` i `4`.

Kada radite sa lambda funkcijama, važno je znati najbolje prakse. Na primer, nije dobra praksa dodeljivati lambda funkciju promenljivoj na ovaj način:

```py
numbers = [1, 2, 3, 4, 5]

square = lambda x: x ** 2
squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Ovo poništava svrhu korišćenja anonimnih funkcija. U ovom slučaju, trebalo bi da koristite običnu funkciju, kao što je ovo:

```py
numbers = [1, 2, 3, 4, 5]

def square(num):
    return num ** 2

squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Takođe, trebalo bi da izbegavate kreiranje lambda funkcija koje su teške za čitanje ili nepotrebno komplikovane, kao što je ovo:

```py
result = (lambda x: (x**2 + 2*x - 1) if x > 0 else (x**3 - x + 4))(3)
print(result)  # 14
```

Iako ova funkcija radi kako treba i proizvodi ispravan rezultat od `14`, nije lako za čitanje niti pregledno. U ovom slučaju, bolje je kreirati zasebnu funkciju sa izjavom `if/else`, i zatim pozvati tu funkciju:

```py
def calculate_expression(x):
    if x > 0:
        return x**2 + 2*x - 1
    else:
        return x**3 - x + 4

print(calculate_expression(3))  # 14
```

I regularne funkcije i lambda funkcije imaju svoja mesta upotrebe u Python programima. Ako se bavite pojedinačnim inline izrazima, onda biste mogli razmotriti korišćenje lambda funkcije. U suprotnom, korišćenje regularne funkcije bi bilo najbolje rešenje.
