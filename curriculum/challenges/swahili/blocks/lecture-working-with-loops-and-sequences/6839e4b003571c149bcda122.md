# --description--

U prethodnim lekcijama ste naučili kako da radite sa petljom `for`, koja se koristi za ponavljanje bloka koda određeni broj puta. Evo primera korišćenja petlje `for` da bi se svaki jezik iz liste `languages` ispisa na konzolu:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

Ali šta ako želite da pratite indeks za svaki element? Pa, jedna opcija je kreirati varijablu `index` i inkrementirati je sa `1` za svaku iteraciju petlje, kao što je ovo:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

Iako to funkcioniše, lakši način da se to uradi je korišćenjem funkcije ``enumerate()``. Funkcija ``enumerate()`` prati indeks za iterabilni objekat i vraća enumerator objekat.

Ako prenesemo list `languages` u funkciju `enumerate()` i konvertujemo njenu vraćenu vrednost u listu sa funkcijom `list()`, izgleda ovako:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

Svaki unos u objektu `enumerate` (sada lista) je tupel koji sadrži brojač, praćen vrednošću iz iterabilnog objekta prosleđenog funkciji ``enumerate()``.

Sada, hajde da refaktorišemo primer od ranije kako bismo koristili funkciju `enumerate()`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

Raspakujemo brojač i vrednost za svaki tupel u objektu `enumerate` u varijable nazvane `index` i `language`, respektivno. Na kraju, obe te varijable se koriste u f-stringu koji se štampa na konzolu u svakoj iteraciji petlje.

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

Ovo eliminiše potrebu za ručnim kreiranjem i ažuriranjem varijable `index`.

Funkcija `enumerate()` takođe prihvata opcioni argument `start` koji određuje početnu vrednost za brojanje. Ako se ovaj argument izostavi, brojanje će početi na `0`. Evo primera korišćenja opcionog argumenta `start`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

I ovde je kako će rezultat izgledati u konzoli:

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

Do sada smo iterirali samo preko jedne liste. Ali šta ako treba da iteriraš preko više iterabilnih objekata paralelno? Pa, možeš koristiti funkciju ``zip()`` za to, koja kombinuje liste u parove elemenata i vraća iterator tupela.

Ako prenesemo listu od `developers` i `ids` na funkciju `zip()` i konvertujemo njenu vraćenu vrednost u listu sa funkcijom `list()`, izgleda ovako:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

I ovde je primer korišćenja funkcije `zip()` sa petljom `for` za iteraciju preko `developers` i `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

U ovom primeru, `zip()` spaja dve liste u parove elemenata i vraća iterator tupela. Petlja `for` zatim raspakuje svaki tupel u `name` i `id`. Na kraju, za svako izjava za štampanje (print statement), štampamo svaki `name` i `id` iz lista `ids` i `developers` respektivno. Evo kako izgleda rezultat u konzoli:

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

Funkcije `enumerate()` i `zip()` su veoma moćne, a kada se kombinuju sa petljama, mogu učiniti vaš kod mnogo konciznijim.
