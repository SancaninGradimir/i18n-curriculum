# --description--

Funkcija `range()` se koristi za generisanje sekvence celih brojeva. Evo osnovne sintakse za funkciju `range()`:

```py
range(start, stop, step)
```

Potrebni argument `stop` je ceo broj koji predstavlja krajnju tačku za sekvencu brojeva koja se generiše. Evo primera korišćenja funkcije `range()`:

```py
for num in range(3):
    print(num)
```

Sledeći kod generiše sekvencu brojeva između `0` i `2`. Celobrojka `3` nije uključena jer argument `stop` nije inkluzivan (nije uključujući).

Ako se ne navede argument `start`, podrazumevano je `0`. U suprotnom, možete koristiti opcioni argument `start` da pokrenete sekvencu celih brojeva na ceo broj koji nije `0`. Evo primera generisanja sekvence celih brojeva između `1` i `4`:

```py
for num in range(1, 5):
    print(num)
```

Podrazumevano, sekvenca celih brojeva će se povećavati za `1`. Ali ako želite da promenite taj podrazumevani korak, možete koristiti opcioni argument `step`. Evo primera generisanja sekvence parnih celih brojeva između `2` i `10`:

```py
for num in range(2, 11, 2):
    print(num)
```

Kao što je ranije pomenuto, za funkciju `range()` postoji samo jedan obavezni argument. Ako ne pružite nikakve argumente funkciji `range()`, dobićete `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

Važno je napomenuti da funkcija `range()` prihvata samo cele brojeve kao argumente, a ne float-ove (decimalne brojeve). Zapamtite da su float-ovi brojevi sa decimalnim tačkama poput `3.4`. Ako pokušate da prosledite float-ove kao argumente, dobićete `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

Ako želite da generišete sekvencu celih brojeva u décrošujućem redosledu, možete koristiti negativan ceo broj za argument `step`, na ovaj način:

```py
for num in range(40, 0, -10):
    print(num)
```

Sledeći kod ispisuje brojeve `40`, `30`, `20` i `10` u tom redosledu na konzolu.

Još jedna stvar koju možete uraditi sa funkcijom `range()` je kreiranje liste celih brojeva korišćenjem iste sa konstruktorom `list`. Konstruktor `list` se koristi za konverziju iterabilnog objekta u listu. Evo primera generisanja liste parnih celih brojeva između `2` i `10`:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

Funkcija `range()` je veoma praktičan način za generisanje sekvence celih brojeva u Python-u. Kada savladate ovaj koncept, verovatno ćete se naći da ga mnogo koristite u svojim Python programima.
