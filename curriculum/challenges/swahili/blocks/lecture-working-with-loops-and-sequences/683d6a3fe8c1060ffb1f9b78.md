# --description--

U prethodnoj lekciji ste naučili kako da radite sa tipom podataka tupu. U ovoj lekciji ćete saznati o nekim uobičajenim metodama koje ćete koristiti kada radite sa tupovima.

Prvi metod koji ćemo pokriti je `count()`. Ovaj metod se koristi za određivanje koliko puta se stavka pojavljuje u tupelu. Evo primera kako proveriti koliko puta je niz `Rust` pojavljen u tupelu pod nazivom `programming_languages`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

Pošto `Rust` se pojavljuje dva puta u tupelu, metoda `count()` vraća broj `2`. Ako navedeni element u funkciji `count()` funkcija nije prisutan uopšte u tupelu, vrednost povratka je `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

Ako se nijedan argument ne prosledi u funkciju `count()`, onda Python izbacuje `TypeError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

Sledeći metod koji ćemo pogledati je `index()` metod. Ovaj metod se koristi za pronalaženje indeksa gde je određeni element prisutan u tuplu. Evo primera korišćenja `index()` metode za pronalaženje indeksa za niz (string):

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

Ako navedeni stav ne može biti pronađen, onda Python izbacuje `ValueError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

Još jedna stvar koju možete uraditi sa metodom ``index()`` jeste da prosledite opcione argumente za početni i krajnji indeks. Evo primera prosleđivanja opcionog početnog indeksa:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

U ovom primeru, navodimo gde početi pretraživanje za nizom `Python`. Prosleđivanjem broja `3` kao drugog argumenta funkciji `index()`, navodimo da počnemo pretraživanje na indeksu `3`. Pošto se `Python` pojavljuje dva puta u tupelu, funkcija `index()` će vratiti indeks `5` umesto indeksa `2` zbog korišćenja opcionog argumenta početnog indeksa.

Takođe možete proslediti opcioni indeks za zaustavljanje. Evo modifikovanog primera određivanja početnog i krajnjeg indeksa:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

Sada je rezultat indeks `2` jer započinjemo pretragu na indeksu `2`, i tražimo do, ali ne uključujući, indeks `5`.

Druga često korišćena funkcija koja se koristi sa tupelima je funkcija `sorted()`. U prethodnoj lekciji ste naučili o metodi `sort()` za liste. Pa, funkciju `sorted()` se može koristiti na bilo kom iterabilnom objektu, uključujući tupelima.

Evo primera kreiranja nove liste brojeva korišćenjem funkcije `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

Funkcija `sorted()` uvek će kreirati novu listu sortirane vrednosti. Ovo se razlikuje od metode `sort()`, koja sortira elemente liste na mestu i ne vraća novu listu.

Ako trebate prilagoditi ponašanje sortiranja za iterabilni objekat, možete koristiti opcione argumente `reverse` i `key`. Evo primera korišćenja argumenta `key` za sortiranje elemenata u tuplu po dužini:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

Ako želite da kreirate novu listu vrednosti u obrnutom redosledu, onda možete koristiti argument ``reverse`` na ovaj način:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Tuplovi su uobičajeni tip podataka u Pythonu. Razumevanje načina rada sa njima, zajedno sa nekim korisnim metodama i funkcijama, pomoći će vam da napišete efikasniji kod.
