# --description--

Tokom sledećih nekoliko lekcija, naučićemo o listama, tuplovima i opsezima, koji su tri osnovnih tipa sekvenci korišćeni u Pythonu.

Tip podataka lista je usmerena sekvenca elemenata koja može sastojati se od nizova karaktera, brojeva ili čak drugih lista. Liste su mutabilne i koriste indeksiranje bazirano na nuli, što znači da je prvi element liste na indeksu nula.

Evo osnovne sintakse za listu:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

Da biste pristupili elementu iz liste `cities`, možete se osvrnuti na njegov indeksni broj u sekvenci. Evo primera pristupa prvom elementu liste `cities`:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

Negativno indeksiranje se koristi za pristup elementima počevši od kraja liste, umesto od početka na indeksu `0`. Da biste pristupili poslednjem elementu bilo koje liste, možete koristiti `-1` ovako:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

Drugi način za kreiranje liste je korišćenje ``list()`` konstruktora. Konstruktor ``list()`` se koristi za konverziju iterabilnog objekta u listu, kao što je ovo:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

Iterabil je poseban tip objekta koji se može iterirati po njemu, jedan stavka odjednom. Vi ćete saznati više o petljama u Pythonu kasnije.

Da biste dobili ukupan broj elemenata u listi, možete koristiti funkciju `len()` na ovaj način:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

Ako želite da ažurirate vrednost na određenom indeksu, možete uraditi nešto ovako:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

Pošto su liste mutabilne, možete ažurirati bilo koji element u listi sve dok unesete validan broj indeksa. Ako unesete indeks (bilo pozitivan ili negativan) koji je van granica za listu, tada ćete dobiti `IndexError`:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

Ako želite da uklonite element iz liste, možete koristiti ključnu reč `del` na ovaj način:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

Ponekad je korisno proveriti da li se element nalazi unutar liste. Za to, možete koristiti ključnu reč `in` na ovaj način:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Ponekad je uobičajeno imati liste ugnježđene unutar drugih lista kao ovo:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

U ovom primeru, imamo jednu ugnježdenu listu koja sadrži tri popularna programerska jezika. Da biste pristupili ugnježdenoj listi, moraćete da ga pristupite koristeći indeks `2`, jer su liste indeksoovane od nule:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

Zatim da biste pristupili drugom jeziku iz te ugnježđene liste, moraćete ga pristupiti koristeći indeks `1` na ovaj način:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

Još jedna uobičajena tehnika korišćena sa listama jeste raspakivanje vrednosti.

Raspakivanje vrednosti iz liste je tehnika koja se koristi za dodelu vrednosti iz liste novim varijablama. Evo primera raspakivanja liste `developer` u nove varijable nazvane `name`, `age` i `job`.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

Ovde, `name` ima vrednost `'Alice'`, `age` ima vrednost `34`, i `job` ima vrednost `'Rust Developer'`.

Ako trebate da prikupite bilo koje preostale elemente iz liste, možete koristiti operater zvjezdice (*)(`*`) na ovaj način:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

U ovom primeru, `name` će i dalje imati vrednost `'Alice'`, a `rest` je lista od dva elementa: broj `34` i string `'Rust Developer'`.

Ako broj promenljivih na levoj strani operatera dodeljivanja ne odgovara ukupnom broju stavki u listi, dobićete `ValueError`:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

Poslednji koncept na koji ćemo pogledati je operater sekcije (`:`). Slično stringovima, možete pristupiti delovima liste korišćenjem operatera sekcije ovako:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

U ovom primeru, početni indeks je `1` jer ukazuje na drugi element u listi. Zatim koristimo operater za sečenje praćen završnim indeksom `4`, koji uključuje sve do (ali ne i) element na tom indeksu.

Još jedna stvar koju možete uraditi sa operaterom za sekciju (slice operator) ``:`` je da navedete korak interval koji određuje koliko treba povećati između indeksa. Recimo da imate listu brojeva kao ova:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

Ako želite da izvučete listu samo parnih brojeva, možete koristiti operater sekcionisanja na ovaj način:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

Prvi parni broj je na indeksu `1`, pa će to biti početni indeks. Pošto želimo da prođemo do kraja liste, izostavljamo krajnji indeks. Na kraju, navodićemo `2` za opcijski interval koraka kako bi se inkrementirao samo sa `2` umesto podrazumevanog `1`.

Liste su korisna i fleksibilna struktura podataka koju ćete mnogo koristiti u svojim Python programima. U sledećoj lekciji, naučićete o uobičajenim metodama koje možete koristiti sa listama.
