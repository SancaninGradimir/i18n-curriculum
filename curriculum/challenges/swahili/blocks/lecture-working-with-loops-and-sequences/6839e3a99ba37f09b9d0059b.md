# --description--

Tuple je tip podataka u Pythonu koji se koristi za kreiranje usmerene sekvence vrednosti. Tuple mogu sadržati mešoviti skup tipova podataka, kao što je ovo:

```py
developer = ('Alice', 34, 'Rust Developer')
```

Tuplovi su slični listama, ali dok liste predstavljaju mutabilni tip podataka, tuplovi su imutabilni. To znači da se elementi u tuplu ne mogu promeniti nakon što je kreiran.

Ako pokušate da ažurirate jedan od elemenata u tuplu, dobićete `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

Da biste pristupili elementu iz tupaola, možete koristiti notaciju uglastih zagrada i indeksni broj:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

Ako trebate pristupiti elementima počevši od kraja tupa, možete koristiti negativno indeksiranje. Evo primera korišćenja negativnog indeksa za pristup poslednjem prethodnom elementu u tuplu:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

Ako pokušate da prosledite broj indeksa koji premašuje ili je jednak dužini tupola, onda ćete dobiti `IndexError` ovako:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

Drugi način za kreiranje tupa je korišćenje konstruktora `tuple()` na ovaj način:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

Za konstruktor `tuple()`, možete proslediti različite iterabilne objekte kao što su nizovi (strings), liste i čak druge tuple.

Da biste proverili da li je stavka u tupelu, možete koristiti ključnu reč `in` na ovaj način:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Takođe možete raspakovati stavke iz tupa kao što ste to uradili sa listama:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

U ovom primeru, `name` ima vrednost `'Alice'`, `age` ima vrednost `34`, i `job` ima vrednost `'Rust Developer'`.

Ako trebate prikupiti bilo koje preostale elemente iz tupa, možete koristiti operater zvjezdice (`*`) na ovaj način:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

Ovde, `name` ima vrednost `'Alice'`, i `rest` je lista sastavljena od broja `34` i niza znakova `'Rust Developer'`.

Slično kao sa listom, možete koristiti operater rezanja na tupelu da izvučete deo njega. Evo primera izvlačenja stavki `'pie'` i `'cookies'` u zaseban tupel:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

Zapamtite da prvi broj predstavlja početni indeks za ekstrakciju, dok drugi broj predstavlja krajnji indeks. Ali imajte na umu da stavka na krajnjem indeksu nije uključena u ekstrahovani tupel.

Ako trebate ukloniti stavku iz tupbla, to nije moguće jer su tuplovi nemenjivi. Zato će ovaj primer proizvesti grešku:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

Dakle, kada bi mogao/mogla da koristiš tupel umesto liste?

Ako vam je potrebna dinamička kolekcija elemenata gde možete dodavati, uklanjati i ažurirati elemente, onda biste trebali koristiti listu. Ako znate da radite sa fiksnom i nemenjivoj kolekcijom podataka, onda biste trebali koristiti tupel.
