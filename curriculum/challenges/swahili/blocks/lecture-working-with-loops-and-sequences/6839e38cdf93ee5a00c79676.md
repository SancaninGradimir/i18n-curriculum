# --description--

U prethodnoj lekciji, predstavljen vam je tip podataka lista i naučili ste kako da pristupate elementima iz liste, kao i listnom seccioniranju. U ovoj lekciji, nastavićete da učite o listama i nekim uobičajenim metodama povezanim sa njima kao što su `append()`, `pop()` i `sort()`.

Prvi metod na koji ćemo pogledati je `append()` metod. Koristi se za dodavanje stavke na kraj liste. Evo primera korišćenja `append()` metode za dodavanje broja `6` u listu brojeva:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

Ako želite dodati jednu listu na kraj druge, tako možete koristiti i metod `append()`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

Primećujete kako cela `even_numbers` lista je ugnježđena unutar `numbers` liste.

Ali ako želite da dodate sve pojedinačne brojeve iz liste `even_numbers` na kraj liste `numbers`, onda možete koristiti metodu `extend()`.

Metoda `extend()` je slična metodi `append()`, ali sa `extend()` možete dodati više elemenata iz jedne liste u drugu. Evo primera dodavanja brojeva `6`, `8` i `10` iz jedne liste na kraj liste `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

Kao što vidite, ugnježštena lista je nestala i to je samo lista brojeva.

Da biste umetnuli element na određeni indeks u listu, možete koristiti metod ``insert()``. Ovaj metod prima dva argumenta: indeks gde želite da umetnete novi element i element koji želite da umetnete.

Ovo je primer korišćenja metode `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

Sledeći kod će umetnuti broj `2.5` na indeksu `2` u listi `numbers`.

Ako želite da uklonite element iz liste, možete koristiti metod `remove()`. Metod `remove()` uzima vrednost elementa koji se uklanja kao argument:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

Važno je napomenuti da ova metoda će ukloniti samo prvo pojavljivanje stavke. Ne sve od njih:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

Da biste uklonili element na određenom indeksu iz liste, možete koristiti metod `pop()` ovako:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

Ako ne navedete element za metod ``pop``, onda se uklanja poslednji element.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

Ako treba da ispraznite listu, onda možete koristiti metod `clear()` na ovaj način:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

Sledeći metod na koji ćemo pogledati je metod ``sort()``. Ovaj metod se koristi za sortiranje elemenata na mestu. Evo primera sortiranja nasumične liste brojeva na mestu:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

Za razliku od metode `sort()`, postoji funkcija `sorted()` koja radi za bilo koji iterabilni objekat i vraća novu sortiranu listu umesto da modifikuje originalnu listu. Na primer:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

Kao podsmehnik, iterable je poseban tip objekta po kojem možete iterirati, što vam omogućava pristup svakom elementu pojedinačno. Vi ćete saznati više o tome kako petlje rade u Pythonu u budućoj lekciji.

Oba metoda `sort()` i funkcija `sorted()` prihvataju opcione parametre `key` i `reverse`. Naučićete više o ovim opcionalnim parametrima u budućoj lekciji kada naučite o tupelima. Naučićete ćete više i o drugim ugrađenim funkcijama poput `sorted()` u budućim lekcijama.

Sledeći metod na koji ćemo pogledati je metod ``reverse()``. Ovaj metod će obrnuti listu elemenata na mestu ovako:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

Posledna metoda na koju ćemo pogledati je `index` metoda. Koristi se za pronalaženje prvog indeksa gde se element može pronaći u listi. Evo primera korišćenja `index` metode za pronalaženje jezika `'Java'` u listi `programming_languages`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

Ako se element ne može pronaći, onda Python izbacuje `ValueError`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Postoje još nekoliko metoda za Python liste, ali je ova početna lista metoda dobro mesto za početak.
