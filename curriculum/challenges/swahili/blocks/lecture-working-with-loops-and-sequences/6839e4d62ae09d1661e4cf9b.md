# --description--

Tokom proteklih nekoliko lekcija, naviknut ste na rad sa petljama kao što je ovo:

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

Ovaj primer kreira novu praznu listu nazvanu `even_numbers` i prolazi kroz sekvencu brojeva između `0` i `20`. Unutar petlje, postoji uslov koji proverava da li trenutni broj ima ostatak od `0` kada se deli sa `2`. Ovo se koristi za određivanje da li je broj paran. Ako je uslov `True`, onda se trenutni `num` dodaje na kraj liste `even_numbers`. Konačno, ispisujemo listu `even_numbers` na konzolu.

Iako ovaj kod radi, postoji kraći način za pisanje ovoga koji koristi list comprehension umesto toga. List comprehension vam omogućava da kreirate novu listu u jednoj liniji kombinovanjem petlje i uslova direktno unutar uglastih zagrada. To čini kod kraćim i često lakšim za čitanje.

Evo refaktorisanog primera od ranije, koristeći uglaste zagrade:

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

U ovom refaktorisanom primeru, lista `even_numbers` se kreira korišćenjem samo jedne linije koda. List komprihenzija prolazi kroz brojeve od `0` do `20`, i uključuje samo one koji su deljivi sa `2`. Ovaj pristup je kompaktaniji i eliminiše potrebu za zasebnom petljom i uslovnim blokom.

Pogledajmo još jedan primer da bismo bolje razumeli kako funkcionisanje liste (list comprehension) radi:

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

U ovom primeru, imamo listu brojeva i želimo da kreiramo novu listu tupola koja ukazuje koji su brojevi parni ili neparni. U prvom delu razumevanja liste (list comprehension), koristimo izjavu `if` da proverimo da li je broj ravnomerno deljiv sa `2`. Ako jeste, onda je rezultat tupol tog broja praćen reči `Even`. U suprotnom, rezultat je tupol sa brojem praćen reči `Odd`.

Evo kako izgleda rezultat štampan u konzoli:

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

Drugi način za kreiranje liste koja počinje od postojećeg iterabilnog objekta je funkcija `filter()`. Evo primera kreiranja nove liste koja sadrži samo reči dužih od četiri karaktera:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

Funkcija `filter()` se koristi za selekciju elemenata iz iterabilnog objekta koji zadovoljavaju određeni uslov. Funkcija `filter()` prima funkciju i iterabilni objekat kao argumente. U ovom primeru, prosleđujemo funkciju `is_long_word` u funkciju `filter()` kako bismo proverili da li je trenutni broj reči veći od `4`. Sve reči koje imaju karakteristični broj veći od `4` dodaju se u novu listu i dodeljuju promenljivoj `long_words`.

Pored funkcije `filter()` funkcija, postoji nekoliko drugih funkcija koje su korisne prilikom rada sa listama. Druga funkcija o kojoj treba znati je funkcija `map()`, koja uzima iterabilni objekat i primenjuje funkciju na svaki od njegovih elemenata. Evo primera korišćenja funkcije `map()` za konverziju liste temperatura iz Celzijusa u Farenhajta:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

Kao što funkcija `filter()` radi, `map()` prima funkciju i iterabilni objekat za svoje argumente. Funkcija `to_fahrenheit` uzima temperaturu i konvertuje je iz Celzijusa u Farenhajta.

Poslednja funkcija na koju ćemo pogledati je `sum()` funkcija. Ova funkcija se koristi za dobijanje zbira iz iterabilnog objekta, kao što su lista ili tupel. Evo primera korišćenja `sum()` funkcije:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

Takođe možete proslediti opcioni argument ``start`` koji postavlja početnu vrednost za sumiranje. Evo ažuriranog primera korišćenjem argumenta ``start`` kao pozicionog argumenta:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

Takođe možete izabrati da koristite argument ``start`` kao ključni argument umesto ovoga:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

Obe verzije će proizvesti isti rezultat, ali je argument po ključnoj reči malo eksplicitniji.

Razumevanje lista, kao i druge funkcije poput `map()`, `filter()` i `sum()`, može delovati malo zbunjujuće na početku. Ali sa dovoljno prakse i vremena, početi ćete da se osećate ugodnije kada ih koristite u svojim Python programima.
