# --interactive--

Element ``button`` se koristi za izvršavanje određene akcije kada je aktiviran. Evo primera elementa ``button`` sa tekstom dugmeta ``Start Game``.

```html
<button>Start Game</button>
```

Drugi primeri korišćenja elementa `button` uključuju slanje obrasca, prikazivanje modala ili prebacivanje bočnog menija otvoren/zatvoren. Element `button` ima atribut `type` koji kontroliše ponašanje dugmeta kada je aktivirano. Prva moguća vrednost za atribut `type` bio bi tip `button`. Evo primera korišćenja elementa `button` sa tipom `button` i tekstom `Show Alert`:

```html
<button type="button">Show Alert</button>
```

Podrazumevano, dugme neće ništa uraditi kada se aktivira. Međutim, možete dodati neki JavaScript kod da učinite dugme interaktivnim, kao što je prikazivanje upozorenja u ovom slučaju.

Kliknite na dugme `Show Alert` kako biste videli da se pojavi upozorenje na ekranu. Da biste interagovali sa primerom, moraćete da omogućite interaktivni uresnik.


**NAPOMENA**: Ovaj interaktivan primer koristi JavaScript, ali ne morate brinuti o razumevanju JavaScript koda. Naučićete o JavaScriptu u budućim modulima.

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

Druga moguća vrednost za `type` atribut je `submit` vrednost. Evo primera korišćenja `button` elementa sa `submit` tipom.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

Unutar ovog `form` elementa, postoji `label` i `input` element za e-mail adresu korisnika. Kada korisnik klikne na dugme za slanje, podaci će biti poslati na server i obrađeni. Treća moguća vrednost za atribut `type` je vrednost `reset`. Evo primera `form` elementa sa dugmićima za resetovanje i slanje.

Omogućujte interaktivni uresnik i interagujte sa poljem za unos e-pošte u prozoru pregleda tako što ćete uneti lažnu adresu e-pošte. Zatim kliknite na dugme za resetovanje da biste videli kako vaša e-pošta nestaje iz polja.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

U ovom modifikovanom primeru, elementi `label` i `input` se koriste za prikupljanje e-mail adrese korisnika. Kada korisnik klikne na dugme za resetovanje, obrisaćće sve njihove unosite podatke. Važno je napomenuti da dugmići za resetovanje obično nisu najbolja ideja jer bi mogli dovesti do toga da korisnici slučajno resetuju svoje podatke. Takođe, više dugmadi u vašoj formi može zgužvati korisnički interfejs.

Drugi način za kreiranje dugmadi u HTML-u je korišćenje elementa `input`. Element `input` takođe ima atribut `type` sa mogućim vrednostima `submit`, `reset`, i `button`. Evo primera korišćenja elementa `input` sa `type` postavljenim na `button`:

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

Atribut `value` se koristi za prikaz teksta dugmeta. Dakle, koja je razlika između korišćenja elemenata `input` i `button`? Elementi `input` su prazni elementi (void elements), što znači da ne mogu imati potomke čvorove, kao što je tekst, i mogu imati samo početni tag. S druge strane, element `button` nudi veću fleksibilnost jer možete u njega ugnježdilu (nest) tekst, slike i ikone.
