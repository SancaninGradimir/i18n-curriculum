# --description--

U Pythonu, rečnici su ugrađene strukture podataka koje skladište kolekcije parova ključ-vrednost. Radiše veoma slično stvarnim rečnicima, gde pretražujete reč da biste pronašli njeno odgovarajuće značenje.

Sa Python rečnicima, koristite ključ da pronađete odgovarajuću vrednost. Trebalo bi koristiti rečnike kada morate povezati vrednosti sa jedinstvenim ključevima. Ovo je korisno kada morate brzo pronaći vrednost na osnovu ključa i kada morate predstaviti strukturirane podatke.

Ovo je opšta sintaksa Python rečnika:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

Prvo, pronalazimo varijablu koja sadrži rečnik. Ne morate nužno dodeliti rečnik varijabli, ali je veoma uobičajeno to učiniti kako biste ga zadržali u memoriji i koristili kasnije u kodu.

Zatim sledi vitičaste zagrade, koje se ponekad nazivaju vitičasti uglate zagrade. A unutar vitičastih zagrada postoje parovi ključ-vrednost.

Svaki ključ je povezan sa vrednošću, tako da možete koristiti taj ključ za pristup toj vrednosti. Nakon svake vrednosti, osim poslednje, postoji zarez koji razdvaja različite parove ključ-vrednost. Ključevi moraju biti jedinstveni u rečniku i moraju biti nemenjiv tip podataka. Međutim, vrednosti se mogu ponavljati i mogu biti bilo kog tipa podataka.

Ovde imamo primer rečnika koji skladišti informacije o receptu za Margheritu picu:

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250,
    'toppings': ['mozzarella', 'basil']
}
```

Rečnik je dodeljen promenljivoj ``pizza``. Ima četiri para ključ-vrednost: ``name``, ``price``, ``calories_per_slice`` i ``toppings``.

Druga alternativa bi bilo korišćenje konstruktora `dict()`, koji gradi rečnik iz sekvence parova ključ-vrednost.

Ovo bi bila ekvivalentna sintaksa za naš primer sa pizzom. Prosleđujemo listu tupola kao argument konstruktoru `dict()`. Ovi tupovi sadrže ključ kao prvi element, a vrednost kao drugi element.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

Da biste pristupili vrednosti para ključ-vrednost, možete koristiti ovu sintaksu, poznatu kao notacija uglastih zagrada. To je ime varijable koja drži rečnik, praćeno uglastim zagradama i ključem kojem želite pristupiti unutar uglastih zagrada:

```python
dictionary[key]
```

U našem primeru sa pizzom, ako želite da pristupite vrednosti `name`, napisali biste ime varijable, `pizza`, praćeno uglastim zagradama i ključem, `name`, unutar navodnika:

```python
pizza['name']
```

Ovo će biti jednako:

```python
'Margherita Pizza'
```

Da biste ažurirali vrednost, samo je potrebno dodati operater dodeljivanja, praćen novom vrednošću.

Ako ključ ne postoji u rečniku, kreiraće se nova par ključ-vrednost. U novijim verzijama Pythona, rečnici čuvaju redosled unosa. Ovo je korisno kada morate iterirati kroz reчник:

```python
pizza['name'] = 'Margherita'
```

Sada vrednost ključa `name` je `'Margherita'`:

```python
print(pizza['name']) # 'Margherita'
```

Rečnici takođe imaju korisne metode za izvršavanje uobičajenih operacija.

Metoda ``.get()`` vraća vrednost povezana sa ključem. Slična je notaciji u zagradi koju smo upravo koristili, ali njena prednost je što možete postaviti podrazumevanu vrednost, tako da nećete dobiti grešku ako ključ ne postoji:

```python
dictionary.get(key, default)
```

U ovom primeru, ako ključ `toppings` ne postoji, vratiće prazan spisak, što je podrazumevana vrednost koju ovde prosleđujemo kao drugi argument. Ali ako `toppings` postoji, vratiće tu vrednost:

```python
pizza.get('toppings', []) # ['mozzarella', 'basil']
```

Metode ``.keys()`` i ``.values()`` vraćaju objekat prikaza sa svim ključevima i vrednostima u rečniku, respektivno:

```python
pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

Objekat prikaza je samo način za pregled sadržaja rečnika bez kreiranja zasene kopije podataka.

Metoda `.items()` vraća objekat prikaza sa svim parovima ključ-vrednost u rečniku, uključujući i ključeve i vrednosti:

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

Metoda `.clear()` uklanja sve parove ključ-vrednost iz rečnika:

```python
pizza.clear()
```

Metoda ``.pop()`` uklanja par ključ-vrednost sa ključem koji navedite kao prvi argument i vraća njegovu vrednost. Ako ne postoji ključ, vraća podrazumevanu vrednost koju navedite kao drugi argument. Ako ne postoji ključ i ne prosledite podrazumevanu vrednost, izbacuje se ``KeyError``:

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

U Pythonu 3.7 i novijim verzijama, metoda `.popitem()` uklanja poslednji umetnuti element:

```python
pizza.popitem()
```

I na kraju, metoda `.update()` ažurira parove ključ-vrednost sa parovima ključ-vrednost drugog rečnika. Ako imaju zajedničke ključeve, njihove vrednosti se prepisuju.

U ovom primeru, ažuriramo rečnik `pizza`. Ključ `price` postoji u oba, pa će njegova vrednost biti zamenjena sa `15`.

Međutim, `total_time` je nov, pa će biti dodan u rečnik `pizza` kao novi par ključ-vrednost:

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

Ovo je novi rečnik sa ažuriranim `price` i novim `total_time`. Primećujte kako je cena sada `15`, a `total_time` je novi par ključ-vrednost:

```python
{
    'name': 'Margherita Pizza', 
    'price': 15, 
    'calories_per_slice': 250, 
    'toppings': ['mozzarella', 'basil'], 
    'total_time': 25
}
```

Ovo su neki od najčešće korišćenih metoda rečnika, ali postoji mnogo više. Izbor pravog može biti koristan za efikasno obavljanje složenih operacija.
