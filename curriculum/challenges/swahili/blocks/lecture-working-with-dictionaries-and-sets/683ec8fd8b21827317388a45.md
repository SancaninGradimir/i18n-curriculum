# --description--

Setovi su jedna od ugrađenih struktura podataka Pythona. Jedna je od osnovnih karakteristika setova to što ne čuvaju duplikate vrednosti. Ako pokušate dodati dupliranu vrednost u set, čuvaće se samo jedna od njih.

Skupovi su mutabilni i neuređeni, što znači da njihovi elementi nisu pohranjeni u bilo kom specifičnom redosledu, pa ne možete koristiti indekse ili ključeve za pristup njima. Mogu sadržati samo vrednosti nemutabilnih tipova podataka kao što su brojevi, nizovi znakova i tuplovi. I podržavaju matematičke operacije skupova, uključujući uniju, interseciju, razliku i simetričku razliku.

Da biste definisali skup, samo morate napisati njegove elemente unutar vitičastih zagrada i razdvojiti ih zarezima. Ovo je primer skupa brojeva:

```python
my_set = {1, 2, 3, 4, 5} 
```

Jedna karakteristika rada sa skupovima je da, ako ikada treba da definišete prazan skup, morate koristiti funkciju `set()`. Ako samo napišete puste vitičaste zagrade, kao što je `{}`, Python će automatski kreirati rečnik.

```python
set() # Set
{}    # Dictionary
```

Možete dodati element u skup metodom `.add()`, i proslediti novi element kao argument:

```python
my_set.add(6)
```

U našem primeru, novi skup bi bio:

```python
{1, 2, 3, 4, 5, 6}
```

Ako pokušate dodati element koji je već u skupu, zadržaćće se samo jedan. U ovom slučaju, već imamo broj 5 u skupu:

```python
my_set.add(5)
```

Dakle, skup se neće promeniti:

```python
{1, 2, 3, 4, 5, 6}
```

Da biste uklonili element iz skupa, imate dve opcije. Možete koristiti metod `.remove()` ili metod `.discard()`, i proslediti element koji želite da uklonite kao argument.

Metoda `.remove()` će baciti `KeyError` ako se element ne pronađe, dok metoda `.discard()` to neće:

```python
my_set.remove(4)
my_set.discard(4)
```

Metoda `.clear()` uklanja sve elemente iz skupa:

```python
my_set.clear()
```

Skupovi u Pythonu takođe imaju moćne metode koje obavljaju uobičajene matematičke operacije sa skupovima.

Metode `.issubset()` i `.issuperset()` proveravaju da li je skup podskup ili nadskup drugog skupa, respektivno.

Ovde proveravamo da li je ``your_set`` podskup od ``my_set``, što je ``False`` jer nisu svi elementi u ``your_set`` u ``my_set``.

Takođe proveravamo da li `my_set` je nadskup od `your_set`. Ovo je takođe `False` jer `my_set` nema sve elemente od `your_set`:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

Metoda ``.isdisjoint()`` proverava da li su dva skupa disjunktna, što znači da nemaju zajedničke elemente. U ovom slučaju, to je ``False`` jer ``my_set`` i ``your_set`` zaista imaju zajedničke elemente – 2, 3 i 4:

```python
print(my_set.isdisjoint(your_set)) # False
```

Operator unije `|` vraća novi skup sa svim elementima iz oba skupa:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

Operator preseka `&` vraća novi skup sa samo elementima koje sklopovi imaju u zajedničkom:

```python
my_set & your_set # {2, 3, 4}
```

Operator razlike `-` vraća novi skup sa elementima prvog skupa koji nisu u drugim skupovima. U ovom primeru, brojevi 1 i 5 su u `my_set` ali NIJE u `your_set`:

```python
my_set - your_set # {1, 5}
```

Operater simetrične razlike `^` vraća novi skup sa elementima koji su ili u prvom ili u drugom skupu, ali ne i u oba. U ovom slučaju, 1 i 5 su u `my_set` ali nisu u `your_set`, pa su uključeni. I broj 6 je u `your_set` ali nije u `my_set`, pa je takođe uključen:

```python
my_set ^ your_set # {1, 5, 6}
```

Svaki od ovih operatera takođe ima odgovarajući složeni operater dodeljivanja ako mu dodate znak jednakosti pored njega. Ovi operateri automatski dodeljuju rezultujući skup prvom skupu u izrazu:

```python
|= &= -= ^=
```

Na primer, operater `-=` pronalazi razliku između skupova i ažurira prvi skup tim rezultatom:

```python
my_set -= your_set
```

Nakon ovoga, `my_set` će biti ažurirano na `{1, 5}`:

```python
print(my_set) # {1, 5}
```

Možete proveriti da li je element u skupu ili ne pomoću operatora `in`. Ovde, proveravamo da li je 5 u `my_set`. Rezultat će biti booleanski vrednost `True` ili `False`:

```python
print(5 in my_set)
```

I to su osnove skupova. Oni su veoma korisni kada ne morate da skladištite vrednosti u bilo kom specifičnom redosledu, i kada samo trebate da skladištite jedinstvene vrednosti.
