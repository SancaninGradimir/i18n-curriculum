# --description--

Iako ovaj izraz odgovara `1 hundred dollars`, neće odgovarati `1  hundred  dollars` ili `10 dollars`.

Spam poruke mogu i naći će način da iskoriste nedostatke u vašoj detekciji. Vreme je za poboljšanje vašeg regex-a.

Zamenite prvi doslovni razmak sa izrazom `\s*`. Karakter klasa `\s` odgovara praznom mestu (whitespace), kao što su razmaci, tabulatori i novi redovi. Kvantifikator `*` znači "poklapanje prethodnog karaktera 0 ili više puta".

Zamenite drugi doslovni razmak sa `\s+`. Kvantifikator `+` znači "poklapanje prethodnog karaktera barem jednom".

# --hints--

Your `dollarRegex` must not use literal spaces. 

```js
assert.notMatch(dollarRegex.source, / /);
```

Your `dollarRegex` must allow any number of spaces before the dollar quantity, and one or more spaces after.

```js
let dollarStr = '10hundred dollars'
assert.deepEqual(dollarStr.match(dollarRegex), dollarStr.match(/[0-9]+\s*(hundred|thousand|million|billion)?\s+dollars/i));
dollarStr = '10 hundred   dollars'
assert.deepEqual(dollarStr.match(dollarRegex), dollarStr.match(/[0-9]+\s*(hundred|thousand|million|billion)?\s+dollars/i));
dollarStr = '10    hundred dollars'
assert.deepEqual(dollarStr.match(dollarRegex), dollarStr.match(/[0-9]+\s*(hundred|thousand|million|billion)?\s+dollars/i));
dollarStr = '10 hundreddollars'
assert.notMatch(dollarStr, dollarRegex)
```

const messageInput = document.getElementById("message-input");
const result = document.getElementById("result-message");
const checkMessageButton = document.getElementById("check-message-btn");

const helpRegex = /please help|assist me/i;
--fcc-editable-region--
const dollarRegex = /[0-9]+ (hundred|thousand|million|billion)? dollars/i;
--fcc-editable-region--

const denyList = [helpRegex, dollarRegex];

const isSpam = (msg) => denyList.some((regex) => regex.test(msg));

checkMessageButton.addEventListener("click", () => {
  if (messageInput.value === "") {
    alert("Please enter a message.");
    return;
  }

  result.textContent = isSpam(messageInput.value)
    ? "Oh no! This looks like a spam message."
    : "This message does not seem to contain any spam.";
  messageInput.value = "";
});
