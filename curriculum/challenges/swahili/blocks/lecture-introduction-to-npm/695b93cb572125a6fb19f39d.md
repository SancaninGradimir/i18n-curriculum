# --description--

Držanje vaših zavisnosti ažurnim—i znati kako instalirati određenu verziju—važno je za održavanje stabilnog projekta. Pomaže vam da izbegnete neočekivane greške i daje vam više kontrole kada se objave nove verzije.

Da biste instalirali određenu verziju paketa, koristite ovaj format:

```bash
npm install package@version-number
```

Na primer, ako vaš projekat zahteva stariju verziju Expressa, rečeno `4.21.2`, možete ga instalirati na ovaj način:

```bash
npm install express@4.21.2
```

Ako ne uključite broj verzije, npm će automatski instalirati najnoviju dostupnu verziju.

Ako niste sigurni koje verzije postoje, možete ih proveriti ovom komandom:

```bash
npm view package-name versions
```

Na primer:

```bash
npm view express versions
```

Da biste ažurirali pakete, možete ih ažurirati pojedinačno ili sve odjednom. Pre nego što to učinite, korisno je prvo videti koji su zastareli:

```bash
npm outdated
```

Ovo će vam pokazati koje zavisnosti imaju dostupnije novije verzije, i trebalo bi da vidite izlaz sličan ovom:

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-spec-version  % npm outdated
Package  Current  Wanted  Latest  Location              Depended by
chalk      5.5.0   5.5.0   5.6.2  node_modules/chalk    install-spec-version
cleave     1.0.1   1.0.1   1.0.2  node_modules/cleave   install-spec-version
express   4.21.2  4.21.2   5.2.1  node_modules/express  install-spec-version
```

Nakon što proverite koje zavisnosti su zastarele, možete početi da ih ažurirate.

Da biste ažurirali jednu zavisnost, koristite sledeće:

```bash
npm update package-name
```

Ovo ažurira paket samo unutar opsega verzija dozvoljenog vašom fajlom ``package.json``. Na primer, da biste ažurirali Express, možete koristiti sledeće:

```bash
npm update express
```

Ako želite da preskočite opseg verzija i instalirate najnoviju verziju paketa, koristite sledeću komandu:

```bash
npm install package-name@latest
```

Da biste ažurirali sve zavisnosti koje spadaju u dozvoljene opsege verzija, pokrenite:

```bash
npm update
```

Međutim, imajte na umu da `npm update` ne nadograđuje paket na novu glavnu verziju ako vaš `package.json` to ne dozvoljava. Glavne verzije često sadrže promene koje razbijaju (breaking changes), pa npm neće primeniti to automatski.

Ako zaista želite najnoviju glavnu verziju, morate pokrenuti:

```bash
npm install package-name@latest
```

Ovo prisiljava npm da instalira najnovije izdanje, bez obzira na opseg verzija u vašem `package.json`.
