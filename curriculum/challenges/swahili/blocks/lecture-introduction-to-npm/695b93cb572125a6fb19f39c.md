# --description--

U prethodnoj lekciji, naučili ste šta su zavisnosti i kako da izaberete prave za vaš projekat.

U ovoj lekciji, naučićete kako da instalirate i uklonite zavisnosti koristeći npm.

Da biste instalirali zavisnost, koristite komandu `npm install` na ovaj način:

```bash
npm install dependencyName
```

Tako tako možete skratiti ``install`` na ``i``, kao što je ovo:

```bash
npm i dependencyName
```

Nakon instaliranja paketa, npm dodaje novi `dependencies` deo u vaš `package.json` fajl (ako već nije tu) i navodi instalirani paket. Na primer, ovako izgleda kada instalirate `chalk` paket:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "chalk": "^5.6.2"
  }
}
```

Kada instalirate zavisnost, pojavljuju se dve nove stvari u korenskom folderu vašeg projekta:

* `node_modules/`: fascikla koja sadrži stvarni kod za pakete koje ste instalirali.
    
* `package-lock.json`: fajl koji skladišti detaljne informacije o tačnim verzijama tih paketa, uključujući njihove sopstvene zavisnosti. (Učenje ćete više o ovom fajlu u kasnijoj lekciji.)

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/how-can-you-install-and-remove-dependencies-1.png" alt="">

Većina projekata zahteva više od jedne zavisnosti. Umesto da ih instalirate po jedan, npm vam omogućava da instalirate više paketa odjednom unosom liste imena paketa razdvojenih razmakom:

```bash
npm install express lodash bcrypt
```

Kao i ranije, kada instalirate više zavisnosti odjednom, npm ažurira vaše `package.json` i `package-lock.json` fajlove, kao i vaš `node_modules`, da ih uključi. Evo kako bi izgledao `package.json`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  }
}
```

Da biste instalirali `devDependency`, dodajte zastavicu `--save-dev` na komandu `install`. Takođe možete koristiti kraću `-D` verziju:

```bash
npm i nodemon -D
```

Nakon pokretanja ove komande, npm ažurira vaš fajl `package.json` i dodaje paket pod sekcijom `devDependencies`. Evo primera kako to izgleda:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  },
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Kada pokrenete komandu `npm install`, npm će prikazati poruku koja vam govori da li je instalacija uspešna ili ne. Poruka o uspehu obično izgleda otprilike ovako:


To uninstall or remove a package, use the `npm uninstall` command:

```bash
npm uninstall dependencyName
```

Na primer, da uklonite `chalk` iz vašeg projekta, pokrenuli biste ovu komandu:

```bash
npm uninstall chalk
```

Takođe možete deinstalirati više paketa odjednom navođenjem ih sa razmakima između svakog imena paketa:

```bash
npm uninstall express lodash bcrypt
```

Takođe ćete dobiti poruku koja pokazuje da li je komanda za deinstalaciju uspela ili nije:

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm uninstall chalk

added 1 package, removed 1 package, and audited 101 packages in 1s

20 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

I vaš `package.json` fajl će biti ažuriran automatski:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Da biste deinstalirali `devDependency`, možete jednostavno pokrenuti:

```bash
npm uninstall nodemon
```

Napomena da ne treba vam zastavica ``-D`` prilikom deinstalacije. npm će automatski ukloniti paket iz odgovarajućeg dela, ``devDependencies`` u ovom slučaju.

Nakon deinstalacije `devDependency`, vaš `package.json` biće ponovo ažuriran:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC"
}
```
