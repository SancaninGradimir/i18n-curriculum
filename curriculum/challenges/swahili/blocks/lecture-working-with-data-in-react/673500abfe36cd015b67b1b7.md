# --description--

Uslovno renderovanje u React-u omogućava vam da kreirate dinamički korisnički interfejsi. Koristi se za prikaz različitog sadržaja na osnovu određenih uslova ili stanja unutar vaše aplikacije.

Najčešći pristupi korišćenju uslovnog renderovanja uključuju korišćenje naredbi `if`, ternarnog (`?:`) operatera i logičkog AND (`&&`) operatera.

Najjednostavniji oblik uslovnog renderovanja koristi naredbu `if`. Evo primera:

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  }
  return <h1>Please sign in</h1>;
}
```

U ovom primeru, komponenta `Greeting` proverava prop `isLoggedIn`. Ako je `true`, vraća poruku dobrodošlice, inače traži od korisnika da se prijavi.

Evo primera korišćenjem komponente `Greeting` unutar roditeljske komponente `App`:

```jsx
function App() {
  return (
    <div className="App">
      <Greeting isLoggedIn={false} />
    </div>
  );
}
```

Za jednostavnije uslove, ternarni operater (`?:`) se često koristi direktno unutar JSX-a. Omogućava ugrađeno uslovno renderovanje, što vaš kod može učiniti sažetnijim:

```jsx
function Greeting({ isLoggedIn }) {
  return <h1>{isLoggedIn ? "Welcome back!" : "Please sign in."}</h1>;
}
```

Ovaj kod postiže isti rezultat kao prethodni primer, ali u kompaktnijem obliku. Ternarni operater proverava `isLoggedIn` i renderuje odgovarajuću poruku.

Drugi uobičajeni obrazac za uslovno renderovanje je korišćenje logičkog AND (`&&`) operatera. Ovo je posebno korisno kada želite da renderujete nešto, ili ništa, na osnovu uslova:

```jsx
function Notification({ message }) {
  return (
    <div>
      {message && <p>{message}</p>}
    </div>
  );
}
```

U ovom primeru, paragrafski element sa porukom renderuje se samo ako je prop `message` *truthy* (istinit). Ako je `message` *falsy* (nistan) – što znači da je prazan niz znakova, `null` ili `undefined`, ništa se ne renderuje na ekran.

Savladavanjem ovih tehnika uslovnog renderovanja, možete izgraditi interaktivnije i korisnički prijateljske aplikacije koje se prilagođavaju promenljivim podacima i interakcijama korisnika.
