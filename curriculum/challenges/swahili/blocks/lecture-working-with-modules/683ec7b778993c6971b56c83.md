# --description--

U razvoju softvera, biblioteka je kao alatnik za programere.

Umesto da morate sami implementirati svaki deo koda od nule, biblioteka vam daje unapred napisani i ponovno upotrebljiv kod, kao što su funkcije, klase i strukture podataka koje možete koristiti u svojim projektima.

Python ima ekstensivnu standardnu biblioteku sa mnogo različitih ugrađenih modula. Oni su sve standardizovana, dobro proverena rešenja za mnoge probleme i zadatke sa kojima ćete se svakodnevno susreti kao programer, kao što su:

* Interakcija sa operativnim sistemom.
    
* Rad sa fajlovima.
    
* Mreže.
    
* Rad sa datumom i vremenom.
    
* Izvršavanje matematičkih operacija.
    
* Korišćenje redovnih izraza.
    
* Testiranje i debagovanje vašeg koda.
    
* I mnogo više!

Neki primeri popularnih ugrađenih modula su `math`, `random`, `re` (skraćeno od „izrazi s pravilima“), i `datetime`.

Modul `math` ima korisne funkcije za obavljanje složenijih matematičkih operacija.

Modul `random` je koristan za generisanje nasumičnih brojeva.

Modul `re` se koristi za rad sa regularnim izrazima.

I `datetime` modul je koristan za rad sa datumima i vremenima u Pythonu.

Ali kako možete pristupiti varijablama, konstantama, funkcijama i klasama definisanim u ovim ugrađenim modulima?

Koristite import naredbu. Ove naredbe vam omogućavaju da uvezete module u vaš Python skript. Import naredbe se obično pišu na vrhu fajla. Takođe, možete ih prilagoditi svojim potrebama. Prvo, koristite naredbu ``import``, praćenu imenom modula:

```python
import module_name
```

Recimo da želite uvesti modul `math`. U tom slučaju, napisali biste ovo na vrhu vaše datoteke:

```python
import math
```

Zatim, ako morate pozvati funkciju iz tog modula u vašem Python skriptu, koristili biste tačkastu notaciju, sa imenom modula praćenim imenom funkcije:

```python
module_name.function_name() 
```

Na primer, da biste dobili kvadratni koren iz 36, napisali biste `math` praćen tačkom i zatim `sqrt`, skraćenicu za kvadratni koren, a unutar zagrade biste prosledili sve potrebne argumente. U ovom slučaju, moramo samo da prosledimo broj iz kojeg želimo kvadratni koren:

```python
math.sqrt(36)
```

Ovo je najosnovnija verzija izjave za uvoz, ali postoje i druge alternative.

Ako trebate uvesti modul sa drugim imenom (poznat i kao "aljas"), možete koristiti ovu sintaksu, sa `as` praćenim aljasem na kraju izjava za uvoz:

```python
import module_name as module_alias
```

Ovo se često koristi za skraćivanje dugih imena modula ili za izbegavanje sukoba imena.

Na primer, da biste se osvrnuli na modul `math` kao `m` u vašem kodu, možete mu dodeliti alias, ovako:

```python
import math as m 
```

Zatim, možete pristupiti elementima modula koristeći alias:

```python
m.sqrt(36)
```

Ali ponekad ne morate da uvezete sve iz modula. Možda vam treba samo jedna ili dve specifične funkcije ili klase. Python ima upravo ono što vam je potrebno u tom slučaju.

Sada izjava za uvoz počinje sa ``from``, praćena imenom modula, a zatim ključnom rečju ``import`` praćenom imenom elemenata koje želite da uvezete:

```python
from module_name import name1, name2
```

Zatim, možete koristiti ova imena bez prefiksa modula u vašem Python skriptu.

Ako želite da dodelite aliasima ovim imenima, možete to učiniti korišćenjem ključne reči ``as`` nakon svake, praćeno aliasom koji želite da koristite:

```python
from module_name import name1 as alias1, name2 as alias2
```

Recimo da želite da uvezete samo funkcije radijana, sinusa i kosinusa iz modula `math`. Npisali biste:

```python
from math import radians, sin, cos
```

Sada možete pozvati ove funkcije direktno u svoj kod, bez modula `math` kao prefiksa.

Ovde imamo detaljniji primer:

Da bismo pronašli sin i kosinus određenog ugla, prvobitno izraženog u stepenima, možemo pozvati funkciju `radians` da ga konvertuje u radijane, a zatim pozvati sin i kosinus funkcije, prosleđujući ugao u radijanima:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

Primećujete kako pozivamo funkcije direktno, bez imena modula kao prefiksa. To je zato što smo uvezli funkcije sa ovom alternativnom sintaksom.

Ovo je korisno, ali može doći do konflikata imena ako već imate funkcije ili varijable sa istim imenom definisane u samom Python skriptu. Dakle, to je nešto što treba imati na umu kada birate koji tip naredbe za uvoz (import statement) želite da koristite.

I na kraju, nalazimo ovu izjavu za uvoz koja se završava sa zvezdicom. Zvezdica govori Pythonu da želite da uvezete sve iz tog modula, ali želite da ga uvezete tako da ne morate koristiti ime modula kao prefiks:

```python
from module_name import *
```

Na primer, ako ovo uradite dok uvozite modul `math`, moći ćete da pozovete bilo koju funkciju definisanu u tom modulu bez navođenja imena modula kao prefiksa. Evo nekoliko primera:

```python
from math import *
print(sqrt(36))  # 6.0
print(pow(5, 2)) # 25.0
print(exp(1))    # 2.718281828459045
```

Međutim, ovo je generalno nepreporučljivo jer može dovesti do kolizija prostora imena i otežati znati odakle dolaze određena imena.

Izjave za uvoz funkcionišu tačno isto za funkcije, klase, konstante, varijable i sve ostale elemente definisane u modulu.

Evo primera konstante iz `math` modula, broj `pi`:

```python
import math
print(math.pi)
```

I ovde je primer klase iz `datetime` modula. Kreiramo objekat datuma koji predstavlja 15. jula 1959. godine. Zatim, dodeljujemo taj `date` objekat promenljivoj i pristupamo danu, mesecu i godini pojedinačno koristeći notaciju tačke:

```python
import datetime
birthday = datetime.date(1959, 7, 15)
print(birthday.day)    # 15
print(birthday.month)  # 7
print(birthday.year)   # 1959
```

Možete pronaći više informacija o sadržaju modula u zvaničnoj Python dokumentaciji za taj modul.

Odlično. Sada kada znate više o modulima, trebalo bi da znate i za ovaj veoma važan idiom u Python skriptama, jer su vrlo blisko povezani:

```python
if __name__ == '__main__': 
    # Code
```

`__name__` je specijalna ugrađena promenljiva u Pythonu.

Kada se Python fajl izvrši direktno, Python postavlja vrednost ove varijable na niz (string) `"__main__"`.

Ali ako se Python fajl uvozi kao modul u drugi Python skript, vrednost varijable `__name__` postavljena je na ime tog modula (obično naziv fajla bez ekstenzije `.py`).

Ovo je zašto ćete često naići na ovu uslovnu proveru u Python skriptama. Sadrži kod koji želite da pokrenete **samo ako** je Python skripta pokrenuta kao glavni program:

```python
if __name__ == '__main__': 
    # Code
```

Ali ako je skripta uvezena kao modul, kod unutar tog bloka se ne izvršava.

Ovo je korisno jer omogućava Python skriptama da imaju dve svrhe. Mogu se pokrenuti direktno kako bi izvršili svoju glavnu logiku, ili mogu biti uvezeni u drugi modul bez izvršavanja njihove glavne logike.
