# --interactive--

U JavaScriptu, konstruktor je specijalni tip funkcije koji se koristi za kreiranje i inicijalizovanje objekata. Poziva se sa ključnom rečju `new` i može da inicijalizuje svojstva i metode na novokreiranom objektu.

U ovoj lekciji, pogledaćemo kako da radimo sa konstruktorom `Object()`. Konstruktor `Object()` kreira novi prazan objekat. Evo primera:

```js
new Object()
```

Kada pozovete ``new Object()``, on vraća novi objekat koji se može koristiti za skladištenje vrednosti.

Konstruktor ``Object()`` se može koristiti sa ili bez ključne reči ``new``. Kada se pozove kao funkcija bez ``new``, ponaša se drugačije zavisno od tipa vrednosti koja mu je prosleđena. Evo primera korišćenja konstruktora ``Object()`` bez ključne reči ``new``:

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

Kao što možete videti u drugom `console.log`, `numObj` je objekat. Ovo se dešava jer smo koristili `Object()` konstruktor da pretvorimo taj unos broja u objekat.

Šta se dešava ako pokušamo da prosledimo ``null`` ili ``undefined`` konstruktoru ``Object()``?

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

Pa, rezultat će biti prazan objekat. Drugi slučaj upotrebe za konstruktor ``Object()`` je kada radite sa vrednošću nepoznatog tipa i morate osigurati da je objekat. Pogledajmo sledeći primer:

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

U ovom primeru, imamo funkciju nazvanu `toObject`. Drugi uslov će proveriti da li je vrednost tip objekta i vratiće vrednost ako je uslov `true`. Ovaj uslov će proveriti objekte, kao i nizove, jer su nizovi posebni tipovi objekata.

Ako nisu tačni nijedan od uslova, funkcija vraća `Object(value)`, što konvertuje ulaz u objekat. Ovo radi za vrednosti kao što su brojevi, stringovi i booleani

Većinom vremena nećete koristiti konstruktor `Object()` za kreiranje novih objekata jer ćete umesto toga koristiti sintaksu doslovnog objekta (npr., `const objectLiteral = { name: "Beau" }`). Ali i dalje je dobro razumeti osnove rada sa konstruktorom Object.
