# --interactive--

Postoji nekoliko načina za uklanjanje svojstava iz objekta, pri čemu je operator ``delete`` najjednostavniji i najčešće korišćeni metod.

Kada koristite ``delete``, uklanja iz objekta odabljenu svojstvo. Evo primera kako koristiti operater ``delete``:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

U ovom primeru, počinjemo sa a `person` objektom koji ima tri svojstva: `name`, `age`, i `job`. Zatim, koristimo operater `delete` da uklonimo svojstvo `job`. Nakon brisanja, objekat `person` više nema svojstva `job`.

Drugi način za uklanjanje svojstava je korišćenje dekonstrukcionog dodeljivanja sa rest parametrima. Ovaj pristup zapravo ne briše svojstvo, već kreira novi objekat bez navedenih svojstava:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

U ovom primeru, koristimo dekonstrukciju da izvučemo `job` i `city` iz objekta `person`, i prikupimo preostala svojstva u novi objekat nazvan `remainingProperties`. Ovo kreira novi objekat bez svojstava `job` i `city`.

Razumevanje kako ukloniti svojstva iz objekata je važna veština u programiranju na JavaScriptu. Omogućava vam da dinamički manipulišete objektima i očistite nepotpune podatke.
