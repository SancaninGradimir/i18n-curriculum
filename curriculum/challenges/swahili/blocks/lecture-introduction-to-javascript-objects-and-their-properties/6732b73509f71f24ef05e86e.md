# --interactive--

Kada radite sa JavaScript-om, često ćete naići na složene strukture podataka koje uključuju ugnježđene objekte i nizove unutar objekata. Ove strukture mogu predstavljati bogate, hijerarhijske podatke, ali zahtevaju i jasno razumevanje kako pristupiti i manipulisati podacima unutar njih. Istražimo kako efikasno navigirati ovim ugnježđenim strukturama.

Pristup svojstvima iz ugnježdjenih objekata podrazumeva korišćenje tačkasto ili uglasto notacije, slično pristupu svojstvima iz jednostavnih objekata. Međutim, moraćete da lančujete ove akcesore kako biste se spustili u ugnježdenu strukturu.

Na primer, razmotrimo ugnježđeni objekat koji predstavlja osobu sa kontakt informacijama:

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

Da biste pristupili poslovnom broju telefona `Alice`, povezali biste pristupe svojstava na ovaj način:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

Takođe možete koristiti oznaku sa uglastim zagradama, što je posebno korisno kada imena svojstava sadrže razmake ili specijalne znakove, ili kada koristite varijable za pristup svojstvima:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

Sada ćemo pogledati kako možemo pristupiti podacima gde jedan od svojstava objekta ima vrednost niza. Evo modifikovanog objekta ``person`` koji uključuje niz adresa:

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

Evo primera kako pristupiti gradu radne adrese za `Alice`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

U ovom primeru, `person.addresses` se odnosi na niz adresa. Da bismo pristupili drugoj adresi u tom nizu, koristimo notaciju uglastih zagrada i indeks `1`. Zatim, koristimo notaciju tačke za pristup `city` iz tog objekta adrese.

Razumevanje kako pristupiti svojstvima u ugnježdenim objektima i nizovima je esencijalno kada radite sa kompleksnim strukturalima podataka. U budućim radionicama i laboratorijama, imaćete priliku da vežbate rad sa ovim tipovima struktura podataka.
