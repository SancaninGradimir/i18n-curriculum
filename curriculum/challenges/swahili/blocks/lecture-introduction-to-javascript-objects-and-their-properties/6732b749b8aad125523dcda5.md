# --interactive--

U JavaScriptu, funkcije i metode objekata su oba načina za enkapsulaciju ponovno upotrebljivog koda, ali imaju neke ključne razlike u tome kako su definisane, korišćene i kontekstu u kojem operišu. Razumevanje ovih razlika je ključno za pisanje efikasnog i organizovanog JavaScript koda.

Kao što ste naučili u ranijim modulima, funkcije su ponovno upotrebljivi blokovi koda koji obavljaju određeni zadatak:

:::interactive_editor

```js
function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alice")); // "Hello, Alice!"
```

:::

Metode objekata, s druge strane, su funkcije koje su povezane sa objektom. Definisane su kao svojstva objekta i mogu pristupiti i manipulisati podacima tog objekta. Evo primera objekta sa metodom:

:::interactive_editor

```js
const person = {
    name: "Bob",
    age: 30,
    sayHello: function() {
        return "Hello, my name is " + this.name;
    }
};

console.log(person.sayHello()); // "Hello, my name is Bob"
```

:::

U ovom primeru, `sayHello` je metoda objekta `person`. Ključna reč `this` omogućava metodi `sayHello` pristup svojstvima objekta pod nazivom `person`. Naučićete više o ključnoj reči `this` u budućim časovima.

Razlika između funkcija i metoda je u tome kako se pozivaju. Funkcije se pozivaju po imenu, dok se metode pozivaju korišćenjem dot notacije na objektu kojem pripadaju. Na primer, mi pozivamo funkciju `greet` kao `greet("Alice")`, ali mi pozivamo metodu `sayHello` kao `person.sayHello()`.

Još jedna važna razlika je kontekst u kojem operišu. Redovne funkcije imaju svoj opseg, ali nemaju ugrađenu referencu na bilo koji određeni objekat. Metode, međutim, su vezane za svoj objekat i mogu pristupiti njegovim svojstvima i drugim metodama koristeći ključnu reč `this`.

Važno je napomenuti da metode pomažu u organizovanju koda u logičke objekte, dok se funkcije koriste za opštiji, ponovno upotrebljiv kod.

Razumevanje razlike između funkcija i metoda objekata je važno kako napredujete u svom JavaScript putovanju. Iako mogu delovati slično na prvi pogled, prepoznavanje kada koristiti svaku pomoći će vam da napišete organizovaniji, efikasniji i bolji kod.
