# --interactive--

U JavaScriptu, objekat je fundamentalna struktura podataka koja vam omogućava skladištenje i organizovanje povezanih podataka i funkcionalnosti.

Možete razmišljati o objektu kao kontejneru koji sadrži različite komadi informacije, baš kao što ormarić za fajlove drži različite fascikle i dokumente.

Ovi komadi informacija zovu se svojstva, i sastoje se od imena (ili ključa) i vrednosti.

```js
const exampleObject = {
  propertyName: value,
}
```

Objekti su neverovatno svestrani i čine osnovu JavaScript-a. Zapravo, skoro sve u JavaScript-u je objekat ili se može tretirati kao takav. Ovo uključuje nizove, funkcije, pa čak i primitivne tipove podataka poput stringova i brojeva kada se koriste na određene načine.

Ova objekto-centrična priroda JavaScripta je jedan od razloga zašto je tako fleksibilan i moćan jezik. Pogledajmo kako možete kreirati objekat:

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
```

U ovom primeru, kreirali smo objekat nazvan `person` sa tri svojstva: `name`, `age`, i `city`. Svako svojstvo ima ime i vrednost, razdvojeno dvotačkom.

Sada, hajde da istražimo kako možete pristupiti ovim svojstvima. Postoje dva glavna načina za pristup svojstvima objekta u JavaScriptu: tačka notacija i zagrada notacija.

Notacija tačke je najčešći i najjednostavniji način za pristup svojstvima objekta. Evo osnovne sintakse za notaciju tačke:

```js
objectName.propertyName
```

Evo kako biste koristili dot notaciju sa našim objektom `person`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
```

:::

Dot notacija je sažeta i laka za čitanje, što je čini preferiranim izborom kada znate tačno ime svojstva kojem želite da pristupite, a to ime je validni JavaScript identifikator (što znači da ne počinje brojem i ne sadrži specijalne znakove niti razmake).

Bracket notacija, s druge strane, omogućava vam pristup svojstvima objekta korišćenjem stringa unutar uglastih zagrada. Evo kako biste koristili bracket notaciju:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person["name"]); // Alice
console.log(person["age"]); //  30
```

:::

Notacija uglastih zagrada je fleksibilnija od notacije tačke jer vam omogućava da koristite imena svojstava koja nisu važeći JavaScript identifikatori. Na primer, ako imate ime svojstva sa razmacima ili koje počinje brojem, morali biste da koristite notaciju uglastih zagrada:

:::interactive_editor

```js
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
```

:::

Još jedna prednost notacije sa uglastim zagradama je u tome što vam omogućava da koristite varijable za dinamički pristup svojstvima:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "Wonderland"
};

let propertyName = "city";
console.log(person[propertyName]); // Wonderland
```

:::

Ova fleksibilnost čini notaciju uglastih zagrada posebno korisnom kada ne znate tačno ime svojstva u trenutku pisanja koda, ili kada radite sa imenima svojstava koja dolaze iz unosa korisnika ili nekog drugog dinamičkog izvora.

Vredi napomenuti da su objekti u JavaScriptu neverovatno moćni i svestrani. Oni mogu da sadrže ne samo jednostavne vrednosti kao što su nizovi znakova i brojevi, već i nizove ili druge objekte.

Razumevanje objekata i načina rada sa njima je ključno u JavaScriptu jer se intenzivno koriste širom jezika, kao i u mnogim JavaScript bibliotekama i okvirima.

Dok nastavite da učite i radite sa JavaScript-om, shvatićete da savladavanje objekata otvara svet mogućnosti za kreiranje kompleksnih i moćnih aplikacija.
