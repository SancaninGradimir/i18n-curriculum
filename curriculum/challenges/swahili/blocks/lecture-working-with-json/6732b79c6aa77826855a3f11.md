# --interactive--

Postoje dva moćna metoda u JavaScriptu za rukovanje JSON podacima: `JSON.parse()` i `JSON.stringify()`. Ovi metodi se često koriste za konverziju između JSON stringova i JavaScript objekata.

`JSON.stringify()` koristi se za konverziju JavaScript objekta u JSON niz (string). Ovo je korisno kada želite da sačuvate ili prenesete podatke u format koji se lako deli ili prenosi između sistema.

Evo kako možete koristiti metodu `JSON.stringify()`:

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

Metoda ``JSON.stringify()`` takođe prihvata opcionalni parametar nazvan *replacer*, koji može biti funkcija ili niz. Evo primera korišćenja niza za opcionalni parametar *replacer*:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

U ovom primeru, imamo `developerObj` sa četiri svojstva. Kada koristimo metod `JSON.stringify()`, možemo proslediti niz za drugi parametar i specificirati koja svojstva želimo da se stringifikuju. Rezultat će biti stringifikovani objekat koji sadrži samo svojstva `firstName` i `country`.

Još jedan opcioni parametar za metod ``JSON.stringify()`` bio bi spajajući parametar. Ovo vam omogućava da kontrolišete razmak za stringifikovani rezultat:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

Većinom vremena nećete koristiti nijedan od ovih opcionalnih parametara za metod ``JSON.stringify()``, ali je i dalje korisno biti svestan njih.

Drugi metod koji ćete mnogo koristiti u programiranju je metod `JSON.parse()`. `JSON.parse()` konvertuje JSON string nazad u JavaScript objekat. Ovo je korisno kada povlačite JSON podatke sa veb servera ili iz `localStorage` i morate da manipulišete podacima u vašoj aplikaciji. Vi ćete više naučiti o `localStorage` u budućoj lekciji.

Evo primera kako da radite sa metodom `JSON.parse()`:

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

Ovo vam omogućava da radite sa podacima u vašem programu kao sa normalnim JavaScript objektom, što olakšava manipulaciju i korišćenje.

U budućim modulima, nastavićemo da učimo više o tome kako raditi sa JSON-om i metodama `JSON.parse()` i `JSON.stringify()`.
