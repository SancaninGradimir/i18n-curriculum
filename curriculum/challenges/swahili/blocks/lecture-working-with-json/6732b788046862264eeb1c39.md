# --description--

Šta je JSON i kako pristupiti vrednostima koristeći zagradnu i tačkastu notaciju?

JSON je skraćenica za JavaScript Object Notation. To je lagan, tekstualni format podataka koji se često koristi za razmenu podataka između servera i veb aplikacije.

Jedan od razloga zašto je JSON tako popularan u web razvoju jeste što je istovremeno i mašinski parsabilan i čitljiv za ljude.

Pošto je JSON nezavisan od jezika, možete lako poslati podatke u JSON formatu iz Java aplikacije u Python aplikaciju, ili iz JavaScript aplikacije u C# aplikaciju.

JSON podržava mnogo tipova podataka, uključujući objekte, nizove, stringove, boolove, null i brojeve.

Evo primera JSON objekta:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

Kao što vidite, JSON koristi parove ključ-vrednost za skladištenje informacija i svaki par je razdvojen zarezom. Svaki ključ mora biti obavnut u dvostruke navodnike, inače ćete dobiti grešku.

Da biste pristupili podacima iz JSON objekta, možete koristiti tačkastu ili uglast zapis (notaciju). U ovom primeru, koristimo tačkastu notaciju za pristup `age` iz JSON objekta:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

Ovaj konkretan primer koristi ono što je poznato kao ``import`` deklaracija, koja uvozi JSON objekat u ovaj fajl kako bismo imali pristup njemu. Vi ćete saznati više o ``import`` deklaraciji u budućoj lekciji.

Takođe možete koristiti notaciju uglastih zagrada za pristup informacijama iz JSON objekata. Evo primera pristupa nizu `list of courses`:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

Korišćenje notacije uglastih zagrada je posebno korisno ovde jer ključ sadrži više reči razdvojene razmakom. Ako pokušamo da koristimo notaciju tačaka, rezultiraće greška.

Ukratko, JSON je svestrani format koji može da skladišti mnogo tipova podataka, uključujući nizove i ugnježđene objekte. Koristeći tačkastu notaciju ili zagradnu notaciju, možete lako pristupiti vrednostima sačuvanim unutar JSON objekta.
