# --interactive--

HTML entitet, ili referenca karaktera, je skup znakova koji se koriste za predstavljanje rezervisanog karaktera u HTML-u.

Recimo da želite prikazati tekst `This is an <img/> element` na ekranu. Ako koristite kod trenutno u editoru, neće prikazati željeni rezultat. Čak i ako dodate atribute `src` i `alt` primeru, prikazujeće sliku usred paragrafa. Nije željeni rezultat. Da biste interagovali sa primerom, moraćete da omogućite interaktivni editor.


:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

Kada HTML parser vidi znak manjeg od (`<`) praćen imenom HTML taga, tumači to kao HTML element. Zato ne dobijate željeni rezultat od `This is an <img/> element` na ekranu.

Da biste rešili ovaj problem, možete koristiti HTML entitete. Evo ažuriranog primera koji koristi ispravne HTML entitete za znak manje od (`<`) i veće od (`>`). Sada bi trebalo da vidite `This is an <img/> element` na ekranu.

Omogući interaktivni uresnik i pokušaj dodati ``&lt;p&gt;learning is fun&lt;/p&gt;`` ispod paragrafa elemenata. Trebalo bi da vidite ``<p>learning is fun</p>`` na ekranu.

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

Ovi tipovi referenci karaktera poznati su kao nazvane reference karaktera. Nazivne reference počinju sa znakom ampersanda (`&`) i završavaju se tačkom-zarezom (`;`). Korišćenjem nazivne reference karaktera, HTML parsir neće ovo zbuniti sa stvarnim HTML elementom.

Drugi tip karakter referencije bio bi decimalna numerička referenca. Ova karakter referenca počinje znakom ampersanda i heš simbola (`#`), praćen jednim ili više decimalnih cifara, praćen semikolonom.

Evo primera korišćenja decimalne numeričke reference za simbol "manje od".

Omogući interaktivni uresnik i promenite kod da biste videli različite simbole. Možete koristiti `&#169;` za simbol autorskih prava i `&#174;` za simbol registrovanog žigova.

:::interactive_editor

```html
&#60;
```

:::

Poslednji tip referenciranja karaktera bio bi heksadecimalna numerička referenca. Ova karakter referenca počinje znakom ampersanda, znakom haša i slovom `x`. Zatim ga prate jedan ili više ASCII heksadecimalnih cifara i završava se develestotkom tačke-zarez.

Ovo je primer korišćenja heksadecimalne numeričke reference za simbol manje od.

Omogući interaktivni uresnik i promeni kod da vidiš različite simbole. Možeš koristiti `&#x20AC;` za evrov simbol i `&#x03A9;` za grčko veliko slovo Omega.

:::interactive_editor

```html
&#x3C;
```

:::
