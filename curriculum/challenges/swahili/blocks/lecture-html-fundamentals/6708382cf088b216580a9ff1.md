# --interactive--

Atribut `id` dodaje jedinstveni identifikator na HTML element.

Ovo je primer `h1` elementa sa `id` od `title`.

Ispod elementa `h1`, dodajte element `h2` sa `id` podešenim na `"subtitle"`. Možete napisati bilo koji tekst koji vam odgovara za `h2`, a promene ćete videti u prozoru za pregled. Da biste interagovali sa primerom, moraćete da omogućite interaktivni editor.

:::interactive_editor

```html
<h1 id="title">Movie Review Page</h1>

```

:::

Možete se osvrnuti na ime `id` od `title` unutar vašeg JavaScript-a ili CSS-a. Evo CSS primera koji se odnosi na `id` od `title` kako bi promenio tekst `color` u `red`.

**NAPOMENA**: Neki CSS je dostavljen za vas u ovom interaktivnom primeru. Ne brinite zbog pokušaja razumevanja CSS koda jer ćete o tome naučiti više u budućim časovima. Ali ako želite da vidite kako se boja teksta menja u plavu, omogućite interaktivni editor, kliknite na karticu ``styles.css`` i promenite ``color: red;`` u ``color: blue;``.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Review page Example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">Movie Review Page</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

Hash simbol (""`#`"""), ispred `title`, govori računaru da želite da ciljate `id` sa tom vrednošću. Ime `id` ne bi trebalo da se koristi više od jednom i uvek treba da bude jedinstveno. Još jedna stvar na koju treba obratiti pažnju u vezi sa vrednostima `id`, jeste da ne smeju sadržati razmake. Evo primera primene reči `main` i `heading` na vrednost atributa `id`:

```html
<h1 id="main heading">Main heading</h1>
```

Pretraživači će ovaj prostor videti kao deo `id`, što će dovesti do neželjenih problema kada je u pitanju stilizovanje i skriptovanje. `id` vrednosti atributa bi trebalo da sadrže samo slova, cifre, donje linije i crtice.

Za razliku od atributa `id`, vrednost atributa `class` ne mora biti jedinstvena i može sadržati razmake.

Ovo je primer primene klase nazvane `box` na element `div`.

```html
<div class="box"></div>
```

Ako želite dodati više imena klasama nekom elementu, možete to učiniti razdvajanjem imena razmakom. Evo ažuriranog primera primene više klasa na element ``div``.

```html
<div class="box red-box"></div>
```

Evo još jednog primera primene više klasama na više `div` elemenata.

**NAPOMENA**: Neki CSS je dostavljen za vas u ovom interaktivnom primeru. Ne brinite zbog pokušaja razumevanja CSS koda jer ćete više o tome naučiti u budućim lekcijama. Ali ako želite da promenite boju prvog i trećeg kutija, omogućite interaktivni uresnik i kliknite na karticu ``styles.css`` i promenite ``background-color: red;`` u ``background-color: black;``.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Colored boxes example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
  </body>
</html>
```

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

:::

Dakle, da rezimiramo, kada bi trebalo koristiti `id` naspram `class`? Klase se najbolje koriste kada želite primeniti skup stilova na mnogo elemenata. Ako želite da ciljate određeni element, najbolje je koristiti `id` jer te vrednosti moraju biti jedinstvene.

## --text--

Kada bi trebalo da koristiš `id` naspram `class`?
## --answers--

Koristite ``class`` kada želite jedinstveni identifikator koji bi trebalo da se primeni samo na jedan element.
### --feedback--

Zapamtite da je ime `id` jedinstveno i trebalo bi se koristiti samo jednom po HTML dokumentu.
---

Use an `id` when you need a unique identifier for a specific element, and use a `class` when you want to apply styles or behavior to multiple elements.

---

Use an `id` when you want to style multiple elements consistently across different parts of your webpage.

### --feedback--
