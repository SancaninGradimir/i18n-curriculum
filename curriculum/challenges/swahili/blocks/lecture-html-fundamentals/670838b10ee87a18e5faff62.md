# --interactive--

Element ``script`` se koristi za ugradnju izvršljivog koda. Većina programera će ovo koristiti za izvršavanje JavaScript koda. JavaScript se koristi za dodavanje interaktivnosti vašim veb stranicama. Uobičajeni primeri korišćenja JavaScripta uključuju interaktivne igre, galerije slika (image sliders) i dinamične forme koje validiraju unos korisnika u realnom vremenu.

Ovo je primer korišćenja elementa `script` u HTML dokumentu. Uklonite `//` ispred `alert("Welcome to freeCodeCamp");` i trebalo bi da vidite pojavno obaveštenje. Da biste videli predgleđene verzije, moraćete da omogućite interaktivni uresnik.

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

Iako tehnički možete napisati sav svoj JavaScript kod unutar tagova ``script``, smatra se najboljom praksom povezivanje na eksternu JavaScript datoteku. Evo primera korišćenja elementa ``script`` za povezivanje na eksternu JavaScript datoteku:

```html
<script src="path-to-javascript-file.js"></script>
```

Atribut ``src`` se ovde koristi za određivanje lokacije tog eksternog JavaScript fajla. ``src`` znači "izvor". Razlog zašto se ne preporučuje postavljanje celog vašeg JavaScript-a unutar HTML dokumenta je zbog razdvajanja briga (separation of concerns). Razdvajanje briga je princip dizajna gde razdvajate svoje programe u različite sekcije i tako da svaka sekcija obrađuje zasebnu brigu. U ovom slučaju, želimo da odvojimo naš JavaScript kod od našeg HTML koda.
