# --description--

Komponente su građevni blokovi React aplikacija koji omogućavaju programerima da razbijaju složene korisničke interfejse na manje, upravljive delove, što olakšava razvoj i održavanje velikih skala aplikacija.

Dva tipa komponenti u Reactu su funkcionalne i komponente bazirane na klasama. U modernom Reactu, programeri će koristiti funkcionalne komponente i svi primeri koje danas pregledamo biće funkcionalne komponente.

Na višoj razini, možete razmišljati o komponentama kao da su JavaScript funkcije koje vraćaju elemente koji opisuju korisnički interfejs.

Ovaj korisnički interfejs je opisan korišćenjem JSX-a, sintaktičke ekstenzije za JavaScript koja izgleda slično HTML-u, ali vam omogućava da pišete elemente korisničkog interfejsa na deklarativniji način.

Pogledajmo primer React komponente:

```jsx
function Greeting() {
  const name = "John"
  {/* The result will be Hello John*/}
  return <h1 className="title">Hello {name}</h1>;
}
```

U ovom primeru, definisali smo komponentu nazvanu `Greeting`. Zagrade `{}` unutar tagova `h1` omogućavaju nam ugradnju JavaScript izraza, što nam dozvoljava pristup promenljivoj `name` unutar elementa `h1`.

Takođe primenjujemo `className` koji se zove `title` na element `h1`.

Ali zašto koristimo ``className`` umesto ``class``, kao sa običnim HTML elementima?

Pa, ovo je zato što u JavaScriptu, `class` je rezervirano ime. Zato, moramo koristiti `className` umesto toga.

Takođe koristimo komentar u JSX-u koji pokazuje kakav će biti rezultat. Važno je napomenuti da možete koristiti standardnu sintaksu komentara kao što je ovo, ali mora biti obavijen uglastim zagradama kako bi funkcionisalo:

```jsx
{/* Block Comments */}
```

Još jedna stvar koju ste možda primetili je da koristimo veliko slovo na početku imena komponente. Ali zašto ne možemo da koristimo samo mala slova?

Ovo je zato što React tretira komponente sa velikim slovom kao prilagođene komponente, dok su elementi sa malim slovima smatrani ugrađenim HTML elementima.

Kada React naiđe na tag malim slovima, kao što su `<div>` ili `<span>`, pretpostavlja da je to standardni HTML element. Međutim, ako ime komponente počinje velikim slovom, React će ga tretirati kao komponentu definisanu od strane korisnika i prikazati je shodno tome. Ova razlika pomaže React-u da razlikuje između ugrađenih HTML tagova i komponenti koje kreirate.

Da bismo koristili ovaj `Greeting` komponent u našoj aplikaciji, napisali bismo nešto ovako:

```jsx
<Greeting />
```

Ovo bi prikazalo element `h1` sa tekstom `Hello John` na stranici. Ali pogledajte pažljivije sintaksu ovde. Kada koristimo komponentu, završava se sa naprednim kosim crtama i zatim znakom veće od.

Kada radite sa JSX-om, sve oznake i korišćenja komponenti moraju biti eksplicitno zatvorene. Dakle, ako komponenta ili oznaka nema potomke, morate je eksplicitno zatvoriti kao što je prikazano ovde:

```jsx
<Greeting /> {/* /> is required */}
```

Do sada smo gledali samo kako da renderujemo jedan `h1` element. Ali zapravo možete renderovati više elemenata.

Pogledajmo sledeći primer koda ovde:

```jsx
function Greeting() {
  const name = "John";
  {/* This will throw an error */}
  return <h1>Hello {name}</h1> 
  <p>Nice to meet you.</p>
}
```

Pokušavamo da dodamo još jednu rečenicu sa ``Nice to meet you``, ali se ne prikazuje pravilno na stranici.

Umesto toga, čini se da postoji poruka o grešci. Poruka o grešci kaže `Adjacent JSX elements must be wrapped in an enclosing tag.`

Razlog zašto dobijate tu poruku o grešci je zato što više elemenata braće treba da bude umotano u roditeljski element. Iako biste mogli da umotate elemente `h1` i `p` u jednostavan `div`, postoji još jedan način da se greška utišati.

React fragmeti se koriste za grupisanje elemenata zajedno. Evo kako će izgledati revidirani primer:

```jsx
function Greeting() {
  const name = "John";
  return (
    <Fragment>
      <h1>Hello {name}</h1>
      <p>Nice to meet you.</p>
    </Fragment>
  );
}
```

Takođe možete izabrati da koristite prazne JSX oznake koje mogu služiti kao skraćeni zapis za fragmente:

```jsx
function Greeting() {
  const name = "John";
  return (
    <>
      <h1>Hello {name}</h1>
      <p>Nice to meet you.</p>
    </>
  );
}
```

U budućim časovima, nastavićemo da učimo više o tome kako raditi sa komponentama i JSX-om. Ali za sada, stekli ste čvrst uvod u izradu korisničkih interfejsa sa komponentama, postavljajući snažan temelj za ono što sledi.
