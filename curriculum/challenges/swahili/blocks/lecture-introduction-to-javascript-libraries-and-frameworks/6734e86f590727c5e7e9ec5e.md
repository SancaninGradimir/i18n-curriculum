# --description--

React je jedna od najpopularnijih JavaScript biblioteka za izgradnju korisničkih interfejsa i veb aplikacija. Originalno razvijena i održavana od strane Facebook-a, React je stekla ogromnu popularnost u veb razvoju zbog svoje efikasnosti, fleksibilnosti i karakteristika.

Osnovni koncept u Reactu je kreiranje ponovo upotrebljivih UI komponenti. Ove komponente, kao što su dugmad, kartice i avatarnih komponenti, mogu se lako ponoviti širom vaše aplikacije. Možete ugnježđivati ove komponente jedna unutar druge kako biste izgradili složenije, dinamičke interfejse.

Jedna od ključnih prednosti Reacta je to što ovi prilagođeni komponente mogu da se ažuriraju i prikažu nezavisno kako se podaci menjaju.

Za razliku od tradicionalnog JavaScript-a, koji zahteva direktnu manipulaciju DOM-om (Document Object Model), React koristi virtuelni DOM, što poboljšava performanse i efikasnost. Naučićete više o virtuelnom DOM-u i kako funkcioniše u narednim lekcijama.

Pored ponovo upotrebljivih komponenti, React takođe nudi moćan način za upravljanje stanjem vaše aplikacije. Stanje se odnosi na podatke koji određuju kako komponenta renderuje i ponaša se.

Sa React-om, možete lako pratiti i ažurirati stanje komponenti, osiguravajući da korisnički interfejs (UI) odražava najnovije podatke. Naučićete više o radu sa stanjem u budućim lekcijama.

Iako postoji mnogo biblioteka unutar JavaScript ekosistema, freeCodeCamp će se uglavnom fokusirati na učenje Reacta zbog njegove široke upotrebe i potražnje u industriji.

Naše sledeće lekcije će se detaljnije baviti izgradnjom prilagođenih komponenti, upravljanjem stanjem i više.
