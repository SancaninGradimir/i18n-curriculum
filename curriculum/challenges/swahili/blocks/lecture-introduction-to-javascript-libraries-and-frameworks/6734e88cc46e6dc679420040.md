# --description--

Za razliku od rada sa manjim vanilla HTML, CSS i JavaScript projektima, pokretanje novog React projekta uključuje nekoliko dodatnih koraka kako bi se osiguralo da sve radi ispravno. Umesto pokušaja samostalnog podešavanja svih potrebnih konfiguracija, postoje alati koji to mogu učiniti za vas.

Jedan od najpopularnijih alata za postavljanje projekata je Vite. Vite, što znači "brz" na francuskom, build alat (alat za izgradnju) koji teži pružanju bržijeg iskustva razvoja za moderne web projekte. Vite se može koristiti sa React-om, kao i sa drugim bibliotekama i frameworkovima poput Vue ili Svelte. Možete ga čak koristiti i sa vanilla JavaScript projektima.

Da biste kreirali novi projekat sa Vite-om, moraćete da koristite komandnu liniju (command line). Ako koristite Windows mašinu, možete koristiti Command Prompt ili Windows PowerShell. Ako koristite Mac, možete koristiti aplikaciju Terminal.

U slučaju da nemate instaliran Node.js, koji je potreban za rad npm-a, evo linka ka zvaničnoj veb stranici Node.js: https://nodejs.org. Pratite uputstva za instalaciju, a zatim se vratite ovde da nastavite sa instalacijom React-a.

Kada otvorite komandnu liniju, možete koristiti sledeću komandu:

```bash
npm create vite@latest my-react-app -- --template react
```

Ova komanda kreira novi React projekat pod imenom `my-react-app` koristeći Vite-ov React šablon (template).

Zatim možete otvoriti novi projekat i videti React "boilerplate" kod koji vam je Vite pružio.

Odlična stvar kod Vite-a jeste da će pružiti samo fajlove koji su apsolutno neophodni za početak rada na vašem React projektu.

Da biste zapravo pokrenuli projekat u trenutnom stanju, moraćete da instalirate zavisnosti (dependencies) koristeći sledeće komande u komandnoj liniji:

```bash
cd my-react-app
npm install
```

Komanda `cd my-react-app` osigurava da pređete u ispravan direktorijum gde je vaš projekat smešten.

Komanda `npm install` se koristi za instaliranje zavisnosti neophodnih za pravilno pokretanje vašeg React projekta. Zavisnosti su biblioteke ili paketi koje vaš React projekat zahteva kako bi funkcionisao ispravno, kao što su sam React, ReactDOM, ili drugi treće strane paketi koji pružaju dodatnu funkcionalnost.

Pokretanjem `npm install` čita se fajl `package.json` u direktorijumu vašeg projekta i instalira sve zavisnosti navedene tamo, omogućavajući vam da izgradite i pokrenete svoju React aplikaciju bez propuštanja bilo kojih potrebnih komponenti.

Fajl `package.json` je ključni konfiguracioni fajl u projektima koji sadrži metapodatke o vašem projektu, uključujući njegovo ime, verziju i zavisnosti. Takođe definiše skripte, informacije o licenciranju i druge postavke koje pomažu u upravljanju projektom i njegovim zavisnostima.

Nakon instalacije zavisnosti, trebalo bi da primetite novi folder u vašem projektu pod nazivom `node_modules`.

Folder `node_modules` je mesto gde su smešteni svi paketi i biblioteke koje vaš projekat zahteva. Ovaj folder sadrži stvarni kod za zavisnosti navedene u vašem fajlu `package.json`, uključujući kako direktne zavisnosti vašeg projekta, tako i sve zavisnosti tih zavisnosti.

Da biste pokrenuli svoj projekat, pokrenite komandu `npm run dev` i otvorite novi pregledački tab na adresi `http://localhost:5173/`.

Trebalo bi da vidite početni šablon koji vam je Vite pružio.

Ako želite da zaustavite lokalni server, pritisnite `CTRL + C` u komandnoj liniji.

Da biste zapravo videli kod ovog početnog šablona, možete otići u svoj projekat unutar foldera `src` i trebalo bi da vidite fajl `App.jsx`.

Odatle možete početi da menjate fajl, sačuvati svoje promene i videti nove promene prikazane u pregledaču.

I sa tim ste spremni da započnete izgradnju vaše React aplikacije!
