# --interactive--

Kada radite sa nizovima znakova u JavaScriptu, postoji mnogo slučajeva kada možda morate proveriti da li niz sadrži određeni podnizak, što je manji deo tog niza.

Na primer, možda želite da proverite da li unos korisnika sadrži određenu reč ili karakter pre izvršavanja neke akcije. Jedan način za postizanje toga je korišćenje metode ``includes()``.

Metoda `includes()` se koristi za proveru da li niz sadrži određeni podniz. Ako je podniz pronađen unutar niza, metoda vraća `true` inače, vraća `false`.

Evo osnovne sintakse:

```js
string.includes(searchValue);
```

Za sintaksu, ``searchValue`` je podstring koji želite tražiti unutar niza. I evo primera:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

U ovom primeru, reč `awesome` se nalazi unutar stringa `JavaScript is awesome!`, pa metoda `includes()` vraća `true`.

Važno je napomenuti da metoda ``includes()`` je osetljiva na veličinu slova. To znači da je potreban tačan podudarnost karaktera, uključujući njihovu veličinu slova.

Na primer:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

Pošto `Awesome` (sa velikim slovom `A`) ne odgovara `awesome` (sa malim slovom `a`), rezultat je `false`.

Takođe možete koristiti metod ``includes()`` za proveru podstringa koji počinje na određenom indeksu u nizu, pružanjem drugog parametra:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

Ovde, pretraga podstringa `JavaScript` počinje od 7. pozicije u nizu karaktera, osiguravajući da preskoči bilo koje znakove pre ove pozicije.

Metoda `includes()` vraća samo rezultat `true` ili `false`. Ne pruža informacije o tome gde je podstring lokovan u nizu ili koliko puta se pojavljuje. Ako vam je potreban taj nivo detalja, druge metode, kao što je metoda `indexOf()`, mogu biti prikladnije.
