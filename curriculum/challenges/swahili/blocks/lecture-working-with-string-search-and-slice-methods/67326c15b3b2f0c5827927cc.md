# --interactive--

Kada radite sa nizovima karaktera u JavaScriptu, često morate da izvučete deo ili podniz iz većeg niza.

Na primer, možda želite da izvučete deo reči, specifičnu sekvencu karaktera ili samo fragment rečenice.

JavaScript pruža nekoliko metoda za ovaj zadatak, jedna od najčešće korišćenih je metoda `slice()`.

Metoda `slice()` omogućava vam da izvučete deo niza karaktera i vraća novi niz karaktera, bez modifikovanja originalnog niza. Prihvata dva parametra: početni indeks i opcioni krajnji indeks.

Evo osnovne sintakse:

```js
string.slice(startIndex, endIndex);
```

`startIndex` je pozicija gde ekstrakcija počinje. `endIndex` je mesto gde ekstrakcija se završava. Ako nije navedeno, `slice()` izvlači do kraja niza karaktera.

Pogledajmo jednostavan primer izvlačenja dela niza karaktera:

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

U ovom primeru, `slice(0, 5)` izvlači karaktere počevši od indeksa `0` do, ali ne uključujući indeks `5`. Kao rezultat, reč `Hello` je izvučena.

Ako izostavite drugi parametar, `slice()` će izdvojiti sve od početnog indeksa do kraja niza:

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

Ovde, `slice(7)` izvlači niz počevši od indeksa `7` do kraja niza, što rezultira sa `world!`.

Takođe možete koristiti negativne brojeve kao indekse. Kada koristite negativan broj, on broji unazad od kraja niza:

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

U ovom slučaju, `slice(-4)` izvlači poslednja četiri karaktera iz niza, dajući nam `fun!`.

Recimo da želite da izvučete sekciju iz sredine niza. Možete navesti i početne i krajnje indekse kako biste precizno kontrolisali koji deo niza želite:

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

Ovde, `slice(7, 17)` izvlači podstring koji počinje na indeksu 7 i završava se tačno pre indeksa `17`, što je reč `JavaScript`.

Metoda ``slice()`` je moćan alat za izvlačenje delova niza u JavaScriptu.

Vi navedite početne i krajnje indekse, a metoda vraća novi niz znakova koji sadrži izvučeni deo.

Sa opcijama za pozitivne, negativne i izostavljene indekse, možete ga prilagoditi različitim situacijama bez menjanja originalnog niza karaktera.
