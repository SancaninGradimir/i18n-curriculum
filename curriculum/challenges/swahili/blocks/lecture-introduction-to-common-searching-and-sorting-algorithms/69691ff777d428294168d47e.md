# --description--

Paradigma razdvajanja i osvajanja u računarskoj nauci je tehnika za rekurentno razlaganje problema na manje podprobleme. Jedan od ključnih aspekata ove tehnike je rekurzija, koja se dešava kada funkcija poziva samu sebe više puta dok ne bude dostupan bazni slučaj. U ovoj lekciji, pogledaćemo algoritam spajanja (merge sort) da bismo bolje razumeli kako tehnika razdvajanja i osvajanja funkcioniše.

Recimo da imamo ovu listu brojeva:

```md
42 37 53 17
```

Cilj je da se ta lista sortira od najmanjeg do najvećeg koristeći algoritam spajanja. Prvi korak je da se ta lista podeli na pola:

```md
42 37 | 53 17
```

Zatim moramo pogledati levu stranu liste:

```md
42 37
```

Uzimamo tu podlistu i ponovo je delimo na pola dok svaka podlista ne sadrži samo jedan element:

```md
42 | 37
```

Lista koja sadrži samo jedan element je po defaultu sortirana. Zatim moramo spojiti svaku od tih podlista sa jednim elementom u sortiranu listu:

```md
37 42
```

Zatim pratićemo isti proces za desnu stranu originalne liste:

```javascript
// right side of original list
53 17

// divide the list in half
53 | 17

// merge the lists in sorted order
17 53
```

Sada kada su oba dela originalne liste sortirana, spojićemo ta dva dela zajedno i sortirati elemente:

```markdown
17 37 42 53
```

Evo kako algoritam izgleda u kodu:

```javascript
function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }

    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));

    const sorted = [];
    let i = 0;
    let j = 0;

    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            sorted.push(left[i]);
            i += 1;
        } else {
            sorted.push(right[j]);
            j += 1;
        }
    }

    return sorted.concat(left.slice(i)).concat(right.slice(j));
}
```

Vremenska složenost za algoritam spajanja bi bila `O(n log n)` jer se lista kontinuirano deli na pola (`log n`) a zatim spaja zajedno (`O(n)`). Za razliku od drugih algoritama za sortiranje, poput 'bubble sort'-a, algoritam spajanja nije sortirana na mestu i ima složenost prostora od `O(n)`.
