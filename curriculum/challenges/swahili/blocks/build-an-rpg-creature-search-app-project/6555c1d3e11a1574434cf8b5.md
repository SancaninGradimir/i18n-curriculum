title: Naučite kako da kreirate veb stranicu koristeći HTML i CSS
description: # --description--
Želite li da naučite kako da kreirate lep veb sajt? Počnite od osnova jezika HTML i CSS.
# --/description--

instructions: # --instructions--
Ovim zadatkom, naučićete osnove HTML-a i CSS-a koristeći stvarne primere. Obavezno pokušajte sve što ste naučili!
# --/instructions--

notes: # --notes--
Ne brinite ako ne znate nijedan jezik pre početka. Svako počinje negde.
# --/notes--

hints: # --hints--
Zapamtite da koristite `<p>` tag za pasus i `<h1>` za najveći naslov.
# --/hints--

## --text--
Molimo vas da koristite HTML za kreiranje strukture ove stranice. Uverite se da imate zaglavlje, glavni deo i podnožje.
## --answers--
```html
<!DOCTYPE html>
<html>
<head>
    <title>Moja Stranica</title>
</head>
<body >

    <h1>Dobrodošli na Moj Sajt</h1>

    <p>Ovo je prvi pasus.</p>

    <div id="main-content">
        <h2>Glavni Deo</h2>
        <p>Ovde ćete postaviti najvažniji sadržaj vašeg sajta.</p>
    </div >

    <footer>
        <p>&copy; 2024 Moj Sajt</p>
    </footer>

</body >
</html>
```
