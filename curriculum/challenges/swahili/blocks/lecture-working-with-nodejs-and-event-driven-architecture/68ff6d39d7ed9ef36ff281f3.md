# --description--

Sada ste spremni da napravite prve korake u svet Node.js-a instaliranjem na vaš računar. Ali prvo, pre nego što instalirate Node.js, preporučujemo instaliranje alata pod nazivom NVM (Node Version Manager). Kao što njegovo ime sugeriše, NVM je alat za upravljanje više verzija Node.js-a na istom računaru.

Podešavanje ovog alata od samog početka pomoći će vam u budućnosti jer možda ćete morati da prebacujete između različitih verzija Node.js-a za različite projekte. Sa NVM-om, možete ovo vrlo lako uraditi.

Hajde da vidimo kako možete instalirati NVM na macOS/Linux i Windows, i kako ga koristiti za instaliranje Node.js-a.

## Kako instalirati NVM na macOS/Linuxu

Na macOS-u i Linuxu, možete instalirati NVM direktno putem shell skripte.

Ali za macOS, primetite da zvanična dokumentacija za NVM pominje da „morate ručno instalirati alate za komandnu liniju Xcode-a pre pokretanja skripte za instalaciju, inače će propasti.“

Na macOS-u, možete instalirati alate za komandnu liniju Xcode-a pokretanjem ove komande u vašem terminalu:

```bash
xcode-select --install
```

Nakon što je instalacija završena, možete proveriti broj verzije i verifikovati instalaciju ovom komandom:

```bash
xcode-select --version
```

Zatim je vreme za instalaciju NVM-a. Možete preuzeti i izvršiti NVM instalacioni skript pokretanjem jedne od ovih komandi:

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

```bash
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

Da biste dobili najnoviju verziju NVM-a u trenutku kada čitate ovaj članak, molimo vas da pogledate zvaničnu dokumentaciju NVM-a: `https://github.com/nvm-sh/nvm?tab=readme-ov-file#install--update-script`. Tamo ćete pronaći ove komande sa najnovijim brojem verzije.

Možda ćete morati da ponovo pokrenete svoj terminal kada se proces završi kako biste učitali novu komandu i novu konfiguraciju.

Zatim, pokrenite ovu komandu da potvrdite da je NVM ispravno instaliran i da proverite vašu trenutnu verziju:

```bash
nvm -v
```

Trebalo bi da vidite svoju verziju NVM u izlazu.

## Kako instalirati Node.js sa NVM-om na macOS/Linuxu

Sada kada imate NVM instaliran na macOS/Linuxu, možete instalirati Node.js.

Da biste instalirali najnoviju verziju sa dugoročnom podrškom (LTS) za Node.js, pokrenite ovu komandu:

```bash
nvm install --lts
```

Primećujte zastavicu `--lts`. Ova zastavica ukazuje da želite instalirati trenutnu verziju Dugotrajne podrške (LTS) za Node.js. Ova verzija se obično preporučuje jer daje prioritet stabilnosti, pouzdanosti i bezbednosti umesto novih ili eksperimentalnih funkcija. Temeljno je testirano i spremno za produkciju. Takođe je zagarantovano da ima duže periode podrške sa fokusom na popravke grešaka (bug fix) i sigurnosne zakrpe (security patches).

Alternativno, možete instalirati određenu verziju Node.js-a navođenjem broja verzije u komandi:

```bash
nvm install <version>
```

Na primer, evo kako možete instalirati Node.js v20:

```bash
nvm install 20
```

Kada završite instaliranje verzije Node.js, možete preći na tu verziju ovom komandom:

```bash
nvm use <version>
```

Na primer, evo kako da pređete na verziju 20:

```bash
nvm use 20
```

Konačno, možete proveriti trenutnu verziju Node.js-a ovom komandom:

```bash
node -v
```

## Kako instalirati NVM na Windows-u

Sada ćemo videti kako možete instalirati i koristiti NVM na Windows-u.

Zvanične verzije NVM podržavaju samo Windows u specifičnim slučajevima. Ako imate Windows Subsystem for Linux (WSL) podešen na vašem uređaju, možete pratiti vodič za macOS/Linux iznad da biste instalirali i koristili NVM.

Međutim, za sam Windows, postoji nekoliko alternativa. Jedna od njih je `nvm-windows`, alat za upravljanje verzijama Node.js za Windows. Ovo je potpuno zaseban projekat od zvaničnog NVM projekta. Ali svrha ovih alata je ista — da vam pomogne u radu sa više verzija Node.js na istom uređaju.

Pre instaliranja `nvm-windows`, trebalo bi da znate da zvanični repozitorij preporučuje deinstaliranje bilo koje prethodne Node instalacije pre instaliranja NVM za Windows. Ako je ovo prvi put da instalirate Node.js, trebalo bi vam biti u redu. Ako već imate instalirane druge verzije Node.js-a, preporučuje se da ih prvo deinstalirate.

Za instalaciju `nvm-windows`:

- Idi na sekciju za verzije (Releases) zvaničnog repozitorijuma: `https://github.com/coreybutler/nvm-windows/releases`.
- Preuzmi najnoviji instalater.
- Prati uputstva.

Nakon završetka instalacije, možda ćete morati da ponovo pokrenete svoj terminal ako komanda ``nvm`` nije odmah dostupna.

Možete proveriti da li je instalacija uspešna i videti vašu trenutnu verziju od `nvm-windows` pokretanjem ove komande:

```powershell
nvm -v
```

## Kako instalirati Node.js korišćenjem NVM-a za Windows

Pošto ste instalirali `nvm-windows` na Windows-u, spremni ste da instalirate Node.js.

Da biste instalirali najnoviju verziju sa dugoročnom podrškom (LTS) za Node.js, pokrenite ovu komandu:

```powershell
nvm install lts
```

Verzija za dugoročnu podršku (LTS) je obično preporučena, ali možete instalirati različitu verziju navođenjem njenog broja verzije u komandi:

```powershell
nvm install <version>
```

Na primer:

```powershell
nvm install 20
```

Kada je instalacija završena, možete preći na tu verziju Node.js-a ovom komandom:

```powershell
nvm use <version>
```

Na primer:

```powershell
nvm use 20
```

Takođe možete proveriti trenutnu verziju Node.js-a ovom komandom:

```bash
node -v
```

## Osnovne komande

Sada kada znate kako da instalirate NVM i Node.js, pogledajmo neke osnovne komande koje će vam biti korisne prilikom rada sa Node.js-om i NVM-om.

Takođe, iako će sve ove komande raditi za NVM instaliran na macOS/Linuxu, uključujući i WSL, mogu raditi drugačije ili možda uopšte neće raditi za `nvm-windows`. Ostavićemo alternativne komande za `nvm-windows` kad god je to moguće.

### Kako podesiti politiku izvršavanja u Powershelu

Ako koristite Windows i pratite sa Powershell-om i `nvm-windows`, možda ćete morati da prilagodite politiku izvršavanja Powershella. Podrazumevana politika izvršavanja za Powershell je `Restricted`, što je sigurnosna mera koja sprečava pokretanje skripti.

Prvo, pokrenite Powershell kao administratora desnim klikom na Powershell, a zatim klikom na „Run as Administrator“.

Zatim, da biste videli trenutnu politiku izvršavanja trenutne Powershell sesije, pokrenite sledeću komandu:

```powershell
Get-ExecutionPolicy
```

Konačno, da biste postavili PowerShell politiku izvršavanja na `RemoteSigned`, što omogućava pokretanje lokalnih skripti i `npm`, koristite sledeću komandu:

```powershell
Set-ExecutionPolicy RemoteSigned
```

Nakon što pokrenete komandu iznad, možete zatvoriti administratorski prozor Powershela i otvoriti običan Prozor Powershela.

### Liste svih verzija

Da biste naveli sve instalirane verzije Node.js-a i proverili koja je trenutno korišćena, izvršite ovu komandu:

```bash
nvm ls
```

### Postavljanje podrazumevane verzije Node.js-a (Samo NVM na macOS/Linuxu)

Ova komanda postavlja podrazumevanu verziju Node.js-a koja će se koristiti za nove terminalne sesije:

```bash
nvm alias default <version>
```

Na primer:

```bash
nvm alias default 22.20.0
```

Takođe možete kreirati prilagođene aliase da dodelite specifičnije ime verziji:

```bash
nvm alias <new_name> <version>
```

Na primer:

```bash
nvm alias new-api 18.20.8
```

Nakon podešavanja aliasa, možete koristiti ovo novo ime da pređete na tu verziju:

```bash
nvm use <new_name>
```

Na primer:

```bash
nvm use new-api
```

### Uklanjanje verzije Node.js-a

To remove a version of Node.js, run this command:

```bash
nvm uninstall <version>
```

Na primer:

```bash
nvm uninstall 20.19.5
```

Sa NVM-om za macOS/Linux, možete ukloniti verziju Node.js pokretanjem komande `uninstall` sa glavnom verzijom kao što je `22` ili `20`. Primer:

```bash
nvm uninstall 20
```

Ako imate više verzija za tu glavnu verziju instalirano kao `20.14.0` i `20.12.2`, najnovija verzija će biti uklonjena prva. U ovom slučaju, pokretanje `nvm uninstall 20` će ukloniti `20.14.0` prvo.

### Proverite verziju vašeg Node.js-a

Da biste proverili trenutnu verziju Node.js-a, možete koristiti sledeće:

```bash
node -v
```

### Proverite verziju vašeg npm-a

`npm` je menadžer paketa za Node.js. Možete ga koristiti za instaliranje, objavljivanje i upravljanje softverskim paketima. Automatski se instalira kada instalirate Node.js.

To check your current version of `npm`, run this command:

```bash
npm -v
```

### Pokreni JavaScript fajl

Ova komanda pokreće JavaScript fajl koji navedite u trenutno aktivnom Node.js okruženju:

```bash
node <file.js>
```

Na primer:

```bash
node app.js
```

### Kreiranje projekta sa Node.js-om

Ova komanda stvara novi Node.js projekat stvaranjem fajla `package.json` koji prati njegove detalje i zavisnosti.

```bash
npm init
```

Kada pokrenete komandu `npm init`, vodiće vas kroz interaktivni CLI vodič za podešavanje fajla `package.json`. Postaviće vam pitanja o imenu projekta, opisu, verziji itd. Ako želite da počnete sa podrazumevanim fajlom `package.json`, možete koristiti komandu `npm init -y` ili `npm init --yes`.

### Instaliranje paketa

Da biste instalirali određeni paket, pokrenite:

```bash
npm install <package>
```

Na primer:

```bash
npm install express
```

### Instalirajte zavisnosti
