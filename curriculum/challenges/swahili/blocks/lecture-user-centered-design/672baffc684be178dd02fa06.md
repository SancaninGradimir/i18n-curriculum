# --interactive--

**NAPOMENA**: Neki interaktivni primeri mogu koristiti CSS koji još niste naučili. Ne brinite zbog pokušaja razumevanja celog koda. Cilj primera je da vam pokažu predgleđaje ovih dizajnerskih koncepata kako biste bolje razumeli kako stvari funkcionišu. Da biste videli predgleđaje, moraćete da omogućite interaktivni editor.

Komponente kartica su veoma uobičajeno pojavljivanje u e-trgovini, društvenim mrežama i vestnim sajtovima. One se koriste za prikaz informacija na strukturisan način. Kada dizajnirate svoje kartice, važno je razumeti najbolje prakse kako bi korisnici lako razumeli informacije koje pokušavate da prenesete.

Prvo razmatranje za dizajn kartica treba da bude jednostavnost. Ne želite da vaše kartice budu vizuelno pretrpane ili prikazuju previše informacija. Na primer, ako je dizajn kartica vizuelno pretrpan, biće previše informacija koje korisnik ne može da obradi odjednom.

Evo primera preopterećenog dizajna kartice:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Manje informacija i dobar razmak između stavki na kartici olakšavaju korisniku obradu informacija, a takođe omogućavaju prikazivanje više kartica na stranici.

Evo primera jednostavnog dizajna kartice:

Još nešto što treba uzeti u obzir je gde korisnik može da klikne na kartici. Neki dizajni kartica će imati jedno dugme, čineći očiglednim gde korisnik može da klikne. Ostali dizajni kartica će omogućiti da je cela kartica klikabilna. Kada korisnik pređe mišem preko bilo kog dela kartice, kartica će promeniti boju ili imati senkasti efekat kako bi ukazalo da je kartica klikabilna. Koji god dizajn izaberete, mora biti dosledan širom vašeg sajta i lak za razumevanje korisniku.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

Drugo razmatranje je korišćenje medija na vašim karticama. Izbor visokokvalitetnih medija može značajno poboljšati korisničko iskustvo. Ako koristite slike ili video snimke za, recimo, karticu proizvoda, što je veći kvalitet, to će korisnik biti zainteresovaniji za taj proizvod. Ali ako koristite loš kvalitet medija, korisnik možda neće verovati u kvalitet vaših proizvoda i usluga.

Jedna od poslednjih stvari koje treba uzeti u obzir je korišćenje hijerarhije boja. Želite da se uverite da su najvažnije informacije na kartici najistaknutije. Možete koristiti jarke boje za važne elemente kao što je dugme za poziv na akciju (call-to-action), i svetle boje za manje važne stavke na kartici.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {

  text-align: center;
}

  max-width: 100%;
  height: auto;
```
