# --interactive--

**NAPOMENA**: Neki interaktivni primeri mogu koristiti CSS i JavaScript koji još niste naučili. Ne brinite zbog pokušaja razumevanja celog koda. Cilj primera je da vam pokažu predgleđaje ovih dizajnerskih koncepata kako biste bolje razumeli kako stvari funkcionišu. Da biste videli predgleđaje, moraćete da omogućite interaktivni uresnik.

Indikacija napretka je način da se korisnicima pokaže koliko su daleko u procesu. Može se koristiti u formularima, procesima registracije i podešavanja. Cilj je pomoći korisnicima da razumeju gde su u procesu i koliko još moraju da urade.

Na primer, možete koristiti traku za indikaciju napretka da pokažete korisnicima šta im je ostalo da urade prilikom popunjavanja formulara. Ne želite stvoriti situaciju u kojoj korisnik mora da popuni dugotrajan formular, a ne zna koliko je još koraka koje treba da završi. Transparentnost je ključna kako bi korisnik znao da li ima dovoljno vremena da sedne i popuni formular ili da će morati da se vrati kasnije.

```js
const form = document.getElementById('multiStepForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const progressText = form.querySelector('.progress-text');
const totalSteps = steps.length;

let currentStep = 0;

function updateProgress() {
  const percent = ((currentStep + 1) / totalSteps) * 100;
  progressBar.style.width = percent + '%';
  progressText.textContent = `Step ${currentStep + 1} of ${totalSteps}`;
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

showStep(currentStep);

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted!');
});
```

Kada dizajnirate sekciju za indikaciju napretka, postoji nekoliko najboljih praksi koje treba imati na umu. Prvo razmatranje je da ga održite jednostavnim. Ne želite da preopteretite korisnika sa previše informacija, jer će se iznervirati i napustiti sajt.

Drugo razmatranje je omogućiti povratak na prethodne korake. Ovo je važno jer korisnici možda žele da se vrate i provere svoje prethodne odgovore ili da izvrše izmene.

Druga razmatranost je da se sekcija za indikaciju napretka učini lako pronađenom. Ako korisnik ne može da ga pronađe, neće znati koliko je daleko u procesu.

Poslednja razmatranost je imati jasne naslove sekcija, procenate ili korake. Ako samo imate traku napretka bez konteksta, korisnik neće znati šta to znači.

```js
const form = document.getElementById('progressForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const currentStepSpan = document.getElementById('currentStep');
const totalStepsSpan = document.getElementById('totalSteps');
const percentageSpan = document.getElementById('percentage');

const totalSteps = steps.length;
let currentStep = 0;

totalStepsSpan.textContent = totalSteps;

function updateProgress() {
  const percent = Math.round(((currentStep + 1) / totalSteps) * 100);
  progressBar.style.width = percent + '%';
  currentStepSpan.textContent = currentStep + 1;
  percentageSpan.textContent = percent + '%';

  // Update ARIA attribute
  form.querySelector('.progress-container').setAttribute('aria-valuenow', percent);
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted! Thanks!');
});
```
