# --interactive--

**NAPOMENA**: Neki interaktivni primeri mogu koristiti CSS koji još niste naučili. Ne brinite zbog pokušaja razumevanja celog koda. Cilj primera je da vam pokažu predgleđaje ovih dizajnerskih koncepata kako biste bolje razumeli kako stvari funkcionišu. Da biste videli predgleđaje, moraćete da omogućite interaktivni editor.

Kada je reč o web dizajnu, postoji mnogo vrsta navigacionih pomagala koja možete koristiti. Primeri uključuju gornje navigacione trake, sajdbarove i podnožja. Ali ako je vaš sajt na složenijoj strani sa dubljim nivoima navigacije, možda biste želeli da razmotrite korišćenje kručnjaka (breadcrumbs).

Krumpe za hleba su pomoćnik navigacije koji korisniku pokazuje gde se nalazi u hijerarhiji sajta. Evo primera kako izgledaju krumpe za hleba za maket sajt elektronike:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home / Electronics / Phones / Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Na većini veb sajtova, "breadcrumbs" su prikazani na vrhu stranice, pokazujući korisniku put kojim je stigao do trenutne stranice. Počevši od `Homepage`, korisnik je navigirao do kategorije `Electronics`, zatim do kategorije `Phones`, i na kraju do proizvoda `Smartphone XYZ`. Verovatno ste interagovali sa "breadcrumbs"-ima na veb sajtu dok ste tražili proizvod ili određeni deo informacija.

Korišćenje putanja (breadcrumbs) je korisno jer može pomoći korisnicima da razumeju gde se nalaze u hijerarhiji sajta i kako da navigiraju nazad na prethodne stranice. Ovo je posebno korisno kada je korisnik došao sa rezultata pretrage ili eksternog linka i mora da razume kontekst stranice na kojoj se nalazi.

Kada je u pitanju dizajniranje "breadcrumb-ova" (navigacionih putanja), postoji nekoliko stvari koje treba imati na umu. Prvo je odlučiti koji će razdvojnik biti. Razdvojnik je karakter koji odvaja različite nivoe hijerarhije. Uobičajeni razdvojnici uključuju znak veće od (`>`), znake navodnika sa desne strane (`»`) i napredni kosu crtu (`/`).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Drugo razmatranje je postavljanje krumpekova (breadcrumb-a). Krumpekovi su obično postavljeni na vrh stranice, bilo iznad ili ispod glavne navigacione trake. Korisnici ne bi trebalo da se muče da pronađu krumpekove, pa osigurajte da su vidljivi i lako dostupni.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<nav class="main-nav">
  <a href="#">Home</a>
  <a href="#">Electronics</a>
  <a href="#">Phones</a>
</nav>
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.main-nav {
  background-color: #333;
  padding: 1rem;
}

.main-nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Drugo razmatranje je veličina kručnjaka. Želite da se uverite da su kručnjaci dovoljno veliki da se lako čitaju, ali ne i preveliki pa da zauzimaju previše prostora na stranici. Zapamtite, kručnjaci nisu namenjeni kao primarno navigaciono sredstvo, već kao sekundarna navigaciona pomoć. Na sajtovima gde ima mnogo informacija na stranici, korisnici mogu lako da vide gde se nalaze u hijerarhiji i da se vrate na prethodne stranice koristeći kručnjake.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</p>
```

```css
.breadcrumbs {
  font-size: 1rem;
  color: #555;
  margin: 1rem 0;
}   

p {
  font-size: 1.2rem;
  line-height: 1.6;
}
```

:::

Dakle, kada biste trebali koristiti breadcrumbs na vašoj stranici? Ako je vaša stranica jednostavna i ima plitku hijerarhiju, breadcrumbs možda neće biti neophodni. Plitka hijerarhija znači da stranica ima samo nekoliko nivoa navigacije, pa breadcrumbs možda neće dodati mnogo vrednosti. Tipično ćete videti breadcrumbs na sajtovima za e-trgovinu, vestnim sajtovima i drugim sajtovima sa dubokom hijerarhijom, kao što su sajtovi tehničke dokumentacije.
