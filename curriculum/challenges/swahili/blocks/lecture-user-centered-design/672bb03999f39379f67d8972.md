# --interactive--

**NAPOMENA**: Neki interaktivni primeri mogu koristiti CSS koji još niste naučili. Ne brinite zbog pokušaja razumevanja celog koda. Cilj primera je da vam pokažu predgleđaje ovih dizajnerskih koncepata kako biste bolje razumeli kako stvari funkcionišu. Da biste videli predgleđaje, moraćete da omogućite interaktivni editor.

Progresivno otkrivanje je dizajnerski obrazac koji se koristi za prikazivanje korisnicima samo relevantnog sadržaja na osnovu njihove trenutne aktivnosti i skrivanje ostatka. To se radi kako bi se smanjio kognitivni teret i učinilo korisničko iskustvo intuitivnijim.

:::interactive_editor

```html
<div class="progressive-disclosure">
  <button id="show-details-btn" aria-expanded="false" aria-controls="extra-content">
    Show More Details
  </button>
  <div id="extra-content" class="hidden" tabindex="-1">
    <p>
      Here are additional details that are revealed only when you want to see them.
      This keeps the interface clean and reduces cognitive load.

</div>
```

```js
const btn = document.getElementById('show-details-btn');
const content = document.getElementById('extra-content');

btn.addEventListener('click', () => {
  const isHidden = content.classList.toggle('hidden');
  btn.setAttribute('aria-expanded', !isHidden);
  if (!isHidden) {
    content.focus();
  }
});
```

Na primer, kada koristite Googleovu stranicu za pretragu, većina vaših pretraga će biti jednostavna. Zato ima smisla imati jednostavan korisnički interfejs sa vrlo malo opcija. Ali za one retke prilike kada ćete trebati napredniju pretragu, Google nudi opciju napredne pretrage koja će prikazati više opcija.

Opciju naprednog pretraživanja Googlea možete pronaći u meniju podešavanja na početnoj stranici Google-a. Ili možete otići na `google.com/advanced_search`.

Drugi primer progresivnog otkrivanja bi bio dugme "Više detalja" na proizvodima na sajtu za e-trgovinu. Kada posetite sajt poput Amazona, videćete listu slika proizvoda u različitim kategorijama. Ovo omogućava korisnicima da kupuju sa lakoćom i da se ne preopterete previše informacijama.

Ako je korisnik zainteresovan za određeni proizvod, može kliknuti na sliku proizvoda da bi otkrio više detalja o njemu. Prikazivanje svih informacija odjednom može biti preopterećujuće za korisnika. Stoga, korišćenjem progresivnog otkrivanja, korisnik može sam odabrati da vidi više informacija kada je spreman.

```js
document.querySelectorAll('.details-btn').forEach(button => {
  button.addEventListener('click', () => {
    const product = button.closest('.product');
    const details = product.querySelector('.product-details');
    const isHidden = details.classList.toggle('hidden');
    product.setAttribute('aria-expanded', !isHidden);
    if (!isHidden) {
      details.focus();
      button.textContent = 'Hide details';
    } else {
      button.textContent = 'More details';
    }
  });
});
```
