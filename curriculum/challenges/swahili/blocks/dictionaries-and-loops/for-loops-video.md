# --description--

U ovom videu, naučićete kako koristiti `for` petlje za iteraciju kroz sekvence kao što su liste i nizovi.

# --questions--

## --text--

Koja od sledećih opcija je ispravan način za iteriranje kroz svaki slovo u nizu karaktera `"Giraffe Academy"`?

## --answers--

```python
for (letter) in "Giraffe Academy":
    print(letter)
```

---

```python
for key, letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter, value in "Giraffe Academy":
    print(letter)
```
