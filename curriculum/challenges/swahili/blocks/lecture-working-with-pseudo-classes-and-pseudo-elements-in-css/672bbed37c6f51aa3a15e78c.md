# --interactive--

Jedan od najzanimljivijih aspekata CSS-a je korišćenje pseudo-elemenata. U ovom kontekstu, "pseudo" znači "ne stvarni", pa su pseudo-elementi virtuelni ili sintetički elementi koji ne odgovaraju direktno nijednom stvarnom HTML elementu. Omogućavaju vam da stilizujete specifične delove elementa ili umetnete sadržaj bez dodavanja dodatnog HTML-a.

Da biste primenili pseudo-element, priložite ga selektoru originalnog elementa koristeći dvostruki kolon (`::`). Napomena da selektor može biti bilo kog tipa, kao što je selektor klase ili ID-ja. Evo kako izgleda osnovna sintaksa pseudo-elemenata:

```css
selector::pseudo-element {
  property: value;
}
```

Ovaj dvostruki dvotačka je ono što razlikuje pseuoelemente od pseuklasova, koji koriste jedan dvotačka.

Pseudo-elementi vam omogućavaju da stilizujete specifične delove sadržaja elementa ili da umetnete sadržaj pre ili posle njega, ali ne mogu da postoje nezavisno. Element kojem je pseudo-element vezan naziva se njegov izvorni element.

Početi ćemo gledanjem primera za `::before` i `::after` pseudo-elemente. Kako njihova imena sugerišu, `::before` vam omogućava da umetnete sadržaj odmah pre sadržaja elementa dok `::after` vam omogućava da ga umetnete posle njega.

Ovo je primer elementa dugmeta. U CSS-u, koristićemo apsolutno pozicioniranje i pseudo-element `::before` da dodamo zvezdicu pre teksta dugmeta `Learn More`. Vi ćete naučiti više o apsolutnom pozicioniranju u budućim lekcijama.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

Svojstvo ``content`` se koristi za predstavljanje sadržaja koji želite dodati pre teksta dugmeta. U ovom primeru, dodajemo zvezdicu. Primeći ćete da ne samo da možete umetnuti sadržaj, već ga i stilizovati. Evo primera pseudo-elementa ``::after`` sa istim dugmetom:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

U `::after` pseudoelementu, svojstvo `transition` se koristi za animiranje promena tokom 0.3 sekunde sa efektom ublažavanja (easing effect), stvarajući glatku i postepenu transformaciju umesto nagle. Naučićete više o svojstvu `transition` u budućim lekcijama.

Takođe možete dodeliti pseudo-klasu sadržaju koji umetnete u drugi sadržaj pomoću pseudo-elemenata `::before` i `::after`. Na primer, stanje prelaska mišem (hover) za sadržaj. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

Sa `transform: translateX(2px)` u stanju prelaska mišem, sadržaj se pomera udesno pomoću `2px` svaki put kada korisnik pređe mišem preko dugmeta. Sama svojstvo tranzicije (transition) u `::after` osigurava da proces traje `0.3s`.

To je ono što svojstvo ``transform`` radi – omogućava vam da rotirate, nagnete, skalirate ili translirate element u određenom pravcu. Vi ćete više o tome naučiti u budućim časovima.

U sledećem primeru, posmatraćemo pseudo-element `::first-letter`. Pseudo-element `::first-letter` cilja na prvo slovo sadržaja elementa, omogućavajući vam da ga stilizujete. Evo primera nekog pasusnog teksta. Ako želimo da stilizujemo prvo slovo, možemo koristiti pseudo-element `::first-letter` na ovaj način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

U poslednjem primeru, posmatraćemo pseudo-element `::marker`, koji vam omogućava da izaberete oznaku, kuglicu ili numerisanje stavki liste za stilizovanje. Pseudo-element `::marker` nudi način za poboljšanje identiteta brenda vaše veb stranice prilagođavanjem oznaka liste kako bi odgovarale vašoj paleti boja.

Evo primera neuređenog spiska i uređenog spiska. Da biste promenili boju i veličinu markera stavke spiska, možete koristiti pseudoelement `::marker` na ovaj način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

U ovoj lekciji pokrili smo samo nekoliko pseudo-elemenata. Ali postoji mnogo više sličnih kao što su `::placeholder`, `::spelling-error` i `::selection` koje vas podstičem da istražite sami.
