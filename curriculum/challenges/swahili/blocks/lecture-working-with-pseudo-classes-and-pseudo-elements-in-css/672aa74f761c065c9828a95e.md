# --interactive--

Pseudo-klase su specijalne CSS ključne reči koje vam omogućavaju da izaberete element na osnovu njegovog specifičnog stanja ili položaja. Stanje ili položaj elementa mogu uključivati:

- Kada je aktivno.
- Kada ga miš prelazi.
- Kada je prvi dete roditelja.
- Kada je poslednje dete roditelja.
- Kada je link posetao.
- Kada je onemogućeno.

Da biste koristili pseudoklasu, dodajete ga selektoru koristeći dvotačku (:) praćenu nazivom pseudoklase:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

Pogledaćemo kako možete koristiti pseudo-klasu da predstavite svako od stanja i pozicija koje smo već pomenuli.

Pseudo-klasa ``:active`` omogućava vam da izaberete aktivno stanje elementa, kao što je klikatanje na dugme:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

Pseudo-klasa ``:hover`` definiše stanje prelaska mišem preko elementa. Primer bi bilo prebacivanje mišem preko dugmeta ili veze:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

Pseudo-klasa ``:first-child`` bira element koji je prvi potomak svog roditeljskog elementa. Evo primera ciljanja prvog paragraf elementa u ``div`` kontejneru:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Prvi paragrafski element u tom ``div`` će dobiti pozadinsku boju ``lightcoral`` i ``padding`` od ``0.4rem`` na svih četiri strane.

Pseudo-klasa ``:last-child`` cilja poslednji element koji je dete svog roditelja. Evo primera ciljanja poslednjeg paragraf elementa u kontejneru ``div`` element:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

`:visited` pseudo-klasa vam omogućava da stilizujete link koji je korisnik već posetao:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

Pseudo-klasa ``:disabled`` vam omogućava da stilizujete interaktivni element u onemogućenom režimu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

Kao što vidite, pseudo-klase vam omogućavaju stilizovanje elemenata na osnovu interakcija korisnika, kao što je naletavanje miša ili poseta linku. Ovo čini vaš sajt interaktivnijim.

Pored pseudoklasova već pomenutih, postoje i drugi kao što su:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`, i više.
