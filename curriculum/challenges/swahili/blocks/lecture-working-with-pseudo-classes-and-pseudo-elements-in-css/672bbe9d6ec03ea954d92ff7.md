# --interactive--

Izgled i ponašanje elemenata za unos važni su prilikom izgrađivanja web formulara. Forma sa poljima za unos koja reaguju na akcije korisnika mnogo doprinosi poboljšanju korisničkog iskustva, a ne bi trebalo da izaziva zbunjenost ili frustraciju.

CSS pruža nekoliko pseudo-klasa za ulaz koji mogu učiniti vaše obrasce intuitivnijim i prijatnijima za korisnika. Pogledajmo ove pseudo-klase u detalju.

Pseudo-klasa ``:focus`` omogućava vam da ciljate element za unos kada korisnik odabere ili klikne na njega, privlačeći pažnju na aktivno polje za unos. Ovo pomaže korisnicima da lako identifikuju gde trenutno kucaju:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <input type="text" />
</form>
```

```css
input:focus {
  border: 2px solid crimson;
  outline: none;
}
```

:::

Kao što ime ukazuje, pseudo-klasa `:hover` se aktivira kada korisnik postavi kursor preko elementa. On pruža vizuelnu povratnu informaciju pre nego što korisnik interaguje sa unosom, efektivno ih upozoravajući da je unos spreman za akciju.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <input type="text" />
</form>
```

```css
input:hover {
  background-color: orange;
}
```

:::

U gornjem primeru, pozadina elemenata za unos menja se u narandžastu boju kada korisnik pređe mišem preko njega, što im govori da je polje spremno za interakciju.

Pseudo-klasa ``:checked`` vam omogućava da stilizujete radio dugme ili prokupljicu drugačije kada ga korisnik izabere. Na ovaj način, korisnik može lako da vidi koju opciju je izabrao.

Evo primera sa poljem za čekiranje koje služi za saglasnost s uslovima na veb sajtu.

**NAPOMENA**: Neki od CSS-a u ovom primeru koristi svojstva koja još nisu pokrivena. Ovo je samo da vam dam ideju kako kreirati prilagođeni okvir za čekiranje (checkbox). Naučićete kako sve ovo funkcioniše u budućim časovima i radionicama.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label>
  Agree <input class="checkbox" type="checkbox" />
  </label>
</form>
```

```css
.checkbox {
  appearance: none;
  width: 18px;
  height: 18px;
  border: 2px solid #ccc;
  border-radius: 4px;
  display: inline-block;
  position: relative;
  cursor: pointer;
  transition: all 0.25s ease;
  vertical-align: middle; 
}

.checkbox:hover {
  border-color: #888;
}

.checkbox:checked {
  background-color: #4caf50;
  border-color: #4caf50;
}

.checkbox:checked::after {
  content: "";
  position: absolute;
  left: 4px;
  top: 0px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}

.checkbox:focus {
  outline: 2px solid #90caf9;
  outline-offset: 2px;
}

```

:::

`:required` cilja elemente unosa koji imaju atribut `required`. To signalizira korisniku da mora popuniti polje kako bi poslao obrazac.

Here is an example with a form that has some required input fields:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form action="#">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required />

  <label for="phone">Phone Number:</label>
  <input type="tel" id="phone" name="phone" />

  <button type="submit">Submit</button>
</form>
```

```css
input:required {
  border: 2px solid orange;
}
```

:::

Prilikom validacije formulara, možete koristiti pseudoklas `:valid` za stilizovanje polja za unos koja ispunjavaju kriterijume validacije, i `:invalid` za stilizovanje polja za unos koja ne ispunjavaju kriterijume. Obično ćete koristiti zelenu boju za validni unos i crvenu boju za nevažeći unos.

Here is an example using the `:valid` pseudo-class:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />
</form> 
```

```css
input:valid {
  border-color: green;
}
```

:::

Ovo je primer korišćenja pseudo-klase `:invalid`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />
</form>
```

```css
input:invalid {
  border-color: crimson;
}
```

:::

Pseudo-klasa `:disabled` vam omogućava da izaberete i stilizujete elemente unosa koji su onemogućeni. Ovi elementi imaju atribut `disabled`, koji korisnicima sprečava interakciju sa njima. Kada je unos onemogućen, ne može se kliknuti, fokusirati ili editovati.

Ovo je primer oznake i onemogućenog polja za unos.

U CSS-u, ciljamo na taj onemogućen unos i dajemo mu pozadinsku boju od `lightgray` i menjamo pokazivač kursora na `not-allowed`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label for="name">Name:</label>
  <input class="text-input" type="text" id="name" name="name" disabled />
</form>
```

```css
.text-input:disabled {
  background-color: lightgray;
  cursor: not-allowed;
}
```

:::

Vrednost CSS svojstva ``cursor: not-allowed;`` menja izgled pokazivača kako bi ukazala na to da akcija nije dozvoljena.

Kada se primeni na element, prikazuje kursor sa simbolom kružni-kosa linija (obično krug sa dijagonalnom linijom kroz njega) kako bi signalizirao korisnicima da element nije interaktivan ili ga ne može kliknuti niti sa njim interagovati.

Evo još nekoliko primera ulaznih pseudo-klasa i šta one rade:

- `:autofill`: primenjuje stilove na polja za unos koja pregledač automatski popunjava sa sačuvanim podacima.
- `:optional`: stilizuje elemente za unos koji nisu obavezni i mogu ostati prazni.
- `:in-range` i `:out-of-range`: stilizuju elemente na osnovu toga da li su njihove vrednosti unutar ili izvan navedenih ograničenja opsega.
