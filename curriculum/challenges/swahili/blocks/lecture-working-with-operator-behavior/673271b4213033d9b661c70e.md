# --interactive--

U JavaScriptu, svi aritmetički operateri imaju formu složenog dodeljivanja. Operatori složenog dodeljivanja pružaju koncizan skraćeni zapis za operaciju na promenljivoj, praćenu skladištenjem rezultata u tu istu promenljivu. Oni kombinuju operaciju i dodeljivanje u kraći oblik poput `x += y`, što je ekvivalentno pisanju `x = x + y` ali bez ponavljanja imena promenljive. Umesto da se piše nešto ovako:

:::interactive_editor

```js
let num = 5;
num = num + 2;

console.log(num); // 7
```

:::

Možete napisati nešto ovako:

:::interactive_editor

```js
let num = 5;
num += 2;

console.log(num); // 7
```

:::

Primećujete kako `num += 2` kombinuje i korake sabiranja i dodeljivanja u jedan. Ovo štedi vreme i smanjuje nered u vašem kodu. Hajde da zaronimo dublje u najčešće kompozitne operatore dodeljivanja u JavaScriptu.

Kao što ste već videli, operater ``+=`` omogućava dodavanje vrednosti postojećoj promenljivoj. Poznat je kao operater dodeljivanja sabiranja. Operater dodeljivanja sabiranja uzima trenutnu vrednost promenljive, dodaje joj navedeni broj, a zatim rezultat vraća nazad u promenljivu:

:::interactive_editor

```js
let total = 10;
total += 5;

console.log(total); // 15
```

:::

Kao što možda pogađate, postoji operator dodeljivanja oduzimanja označen sa `-=`. Operator dodeljivanja oduzimanja oduzima navedenu vrednost od trenutne vrednosti varijable i dodeljuje novu vrednost nazad varijabli:

:::interactive_editor

```js
let score = 20;
score -= 7;

console.log(score); // 13
```

:::

Ako niste koristili operater oduzimanja za dodelu, uradili biste nešto slično ovome:

:::interactive_editor

```js
let score = 20;
score = score - 7;

console.log(score); // 13
```

:::

Operater za množenje dodeljivanja je predstavljen sa `*=`. On množi trenutnu vrednost varijable sa navrhanim brojem i ponovo je dodeljuje toj varijabli:

:::interactive_editor

```js
let points = 5;
points *= 3;

console.log(points); // 15
```

:::

Napokon, postoji operator dodeljivanja deljenja označen sa ``/=``. Kao i drugi pre njega, omogućava vam da podelite trenutnu vrednost varijable sa brojem koji navedete, a zatim dodelite rezultat nazad varijabli:

:::interactive_editor

```js
let balance = 100;
balance /= 4;

console.log(balance); // 25
```

:::

Zapamtite da postoji operater složene dodeljuje za svaki operater u JavaScriptu. Dakle, osim četiri već pomenuta, imamo i:

- Operator dodeljivanja ostatka (`%=`), koji deli varijablu sa navedenim brojem i dodeljuje ostatak varijabli.

- Operater dodeljivanja eksponenta (`**=`), koji podiže varijablu na stepen navedenog broja i ponovo joj dodeljuje rezultat.

- Operater dodeljivanja bitovnog AND-a (`&=`), koji izvodi operaciju bitovnog AND-a sa navrhanim brojem i ponovo dodeljuje rezultat varijabli.

- Operater dodeljivanja bitnog ILI (OR) (`|=`), koji izvršava operaciju bitnog ILI (OR) sa navedenim brojem i ponovo dodeljuje rezultat promenljivoj.
