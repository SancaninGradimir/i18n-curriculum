# --interactive--

Opseg u programiranju se odnosi na vidljivost i dostupnost promenljivih u različitim delovima vašeg koda. On određuje gde se promenljive mogu pristupiti ili modifikovati. U JavaScriptu, razumevanje opsega je ključno za pisanje čistog, efikasnog i bezgrešnog koda. Postoje tri glavna tipa opsega: globalni opseg, lokalni opseg i blok opseg.

Global opseg je najspoljašnji opseg u JavaScript programu. Varijable deklarisane u globalnom opsegu dostupne su bilo gde u vašem kodu, uključujući i unutar funkcija i blokova. Ove varijable se često nazivaju globalnim varijablama. Iako mogu biti praktične, globalne varijable bi trebalo koristiti retko jer mogu dovesti do konflikata imena i otežati održavanje vašeg koda. Evo primera globalne varijable:

:::interactive_editor

```js
let globalVar = "I'm a global variable";

function printGlobalVar() {
    console.log(globalVar);
}

printGlobalVar(); // "I'm a global variable"
```

:::

U ovom primeru, `globalVar` je deklarisan u globalnom opsegu i može se pristupiti unutar funkcije `printGlobalVar`.

Lokalni opseg, s druge strane, odnosi se na varijable koje su dostupne samo unutar funkcije. Evo primera lokalnog opsega:

:::interactive_editor

```js
function greet() {
    let message = "Hello, local scope!";
    console.log(message);
}

greet(); // "Hello, local scope!"
// console.log(message); // This will throw an error
```

:::

U ovom kodu, `message` je lokalna promenljiva unutar funkcije `greet`. Može se koristiti unutar funkcije, ali pokušaj pristupa izvan funkcije rezultiraće greškom.

Blok opseg je koncept uveden sa ključnim rečima `let` i `const` u ES6. Blok je bilo koji deo koda unutar vitičastih zagrada, `{}`, kao što su u `if` izjava, `for` petlje ili `while` petlje. Koncept petljiće će biti naučen u sledećoj lekciji.

Varijable deklarisane sa `let` ili `const` unutar bloka dostupne su samo unutar tog bloka. Evo primera opsega bloka:

```js
if (true) {
    let blockVar = "I'm in a block";
    console.log(blockVar); // "I'm in a block"
}
console.log(blockVar); // This will throw an error
```

U ovom primeru, `blockVar` je dostupan samo unutar bloka `if`. Pokušaj pristupa van bloka rezultiraće greškom. Razumevanje ovih različitih tipova opsega je esencijalno za upravljanje dostupnošću promenljivih i izbegavanje neželjenih nuspojava u vašem kodu.

Globalne varijable bi trebalo koristiti štedljivo, jer mogu dovesti do konflikata imena i učiniti vaš kod teže za održavanje. Lokalne varijable pomažu u izolovanju različitih delova vašeg koda, što je posebno korisno u većim programima. Blok opsega (scoping) sa `let` i `const` pruža još finiju kontrolu nad dostupnošću varijabli, pomažući u sprečavanju grešaka i čineći vaš kod predvidljivijim. Savladavanje ovih osnovnih koncepata globalnog, lokalnog i blok opsega pružiće čvrste osnove za razumevanje naprednijih tema.
