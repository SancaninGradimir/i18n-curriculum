# --interactive--

Funkcije su ponovo upotrebljivi delovi koda koji obavljaju određeni zadatak ili izračunavaju vrednost. Zamislite funkcije kao mašinu koja prima neki ulaz, vrši na njemu neke operacije, a zatim proizvodi izlaz. Evo primera deklarisanja funkcije:

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

U ovom primeru, deklarisali smo funkciju nazvanu `greet`. Unutar te funkcije, imamo deo koji loguje poruku `Hello, Jessica!`. Ako pokrenemo ovaj kod, nećemo videti da se poruka pojavljuje u konzoli. Ovo je zato što moramo pozvati funkciju.

Poziv funkcije, ili invocacija, dešava se kada zapravo koristimo ili izvršavamo funkciju. Da biste pozvali funkciju, moraćete da navedete ime funkcije praćeno skupom zagrada:

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

Sada će poruka od `Hello, Jessica!` biti logovana u konzolu. Ali šta ako želimo da poruka kaže `Hello, Nick!` ili `Hello, Anna!`? Ne želimo da pišemo novu funkciju svaki put kada pozdravimo različit korisnik. Umesto toga, možemo kreirati ponovo upotrebljivu funkciju koja koristi parametre i argumente funkcije.

Parametri deluju kao zamenske vrednosti za vrednosti koje će biti prosleđene funkciji kada se ona pozove. Omogućavaju funkcijama da prihvate ulaz i rade sa tim ulazom. Argumenti su stvarne vrednosti prosleđene funkciji kada se ona pozove. Evo ažurirane verzije funkcije `greet` koja koristi parametre i argumente:

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

`name` služi kao parametar, dok stringovi `Alice` i `Nick` služe kao argumenti. Sada imamo ponovno upotrebljivu funkciju koja se može koristiti desetine puta kroz naš kod sa različitim argumentima.

Kada funkcija završi izvršavanje, uvek će vratiti vrednost. Podrazumevano, povratna vrednost će biti `undefined`. Evo primera:

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

Ako vaša funkcija mora da vrati određenu vrednost, onda ćete morati da koristite naredbu ``return``. Evo primera korišćenja naredbe ``return`` za vraćanje zbira dve vrednosti:

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

Često ćete koristiti izjavu `return`, jer možete koristiti tu vrednost koja je bila generisana funkcijom kasnije u vašem kodu.

Do sada smo radili sa funkcijama sa imenom, ali možete kreirati i ono što se zove anonimna funkcija. Anonimna funkcija je funkcija bez imena koja se može dodeliti promenljivoj na ovaj način:

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

U ovom primeru, imamo varijablu `const` nazvanu `sum` i dodeljujemo joj anonimnu funkciju koja vraća zbir `num1` i `num2`. Zatim smo u stanju da pozovemo `sum` i prosledimo brojeve `3` i `4` kako bismo dobili rezultat `7`.

Funkcije podržavaju podrazumevane parametre, omogućavajući vam da postavite podrazumevne vrednosti za parametre. Ove podrazumevane vrednosti se koriste ako je funkcija pozvana bez argumenta za taj parametar. Evo primera:

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

U ovom primeru, ako nije dostavljen argument za `name`, podrazumeva vrednost `Guest`.

Ukratko, funkcije vam omogućavaju da pišete ponovno upotrebljiv i organizovan kod. One mogu primiti ulaze (parametre), izvršavati akcije i vraćati izlaze.
