# --interactive--

U CSS-u, postoji nekoliko načina za dodavanje ivica oko slika, svaki nudeći različite opcije stilizovanja i nivoe kontrole.

Istražimo neke od najčešćih i najsvestranijih metoda.

Najjednostavniji način za dodavanje ivice slici je korišćenje svojstva ``border``. Ovo svojstvo je skraćeni zapis koji vam omogućava da postavite širinu, stil i boju ivice odjednom.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

Ovo CSS pravilo dodaje solidnu crvenu ivicu širine od 2 piksela oko svih elemenata `img`. Možete podesiti širinu, stil (kao što su `dashed`, `dotted`, ili `double`) i boju kako bi odgovaralo vašim dizajnerskim potrebama.

Ako vam je potreban veći kontrola nad pojedinačnim stranama granice, možete koristiti specifična svojstva ivica za svaku stranu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

Ovo vam omogućava da kreirate jedinstvene stilove ivica za svaku stranu slike.

Drugi način za kreiranje efekta ivice je korišćenje svojstva `outline`. Iako je sličan border-u, outline ne utiče na dimenzije ili raspored elementa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

Ovo stvara zlatni okvir oko slike bez menjanja njene veličine ili pozicije.

Ako želite da kreirate zaobljene uglove na vašoj ivici, možete koristiti svojstvo `border-radius` u kombinaciji sa border:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

Zapamtite, ove tehnike se mogu kombinovati i prilagoditi kako bi se stvorili jedinstveni efekti ivica. Izbor metode zavisi od vaših specifičnih zahteva dizajna i nivoa složenosti koji vam je potreban.
