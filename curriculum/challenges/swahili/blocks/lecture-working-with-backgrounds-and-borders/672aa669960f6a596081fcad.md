# --interactive--

Kada radite sa pozadinskim slikama u CSS-u, imate nekoliko svojstava na raspolaganju da kontrolišete kako su ove slike prikazane.

Glavne karakteristike na koje ćemo se fokusirati su `background-size`, `background-repeat`, `background-position` i `background-attachment`.

Prvo pogledajmo svojstvo `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

CSS iznad postavlja pozadinsku sliku mačke za element `body`.

Ako želite da postavite veličinu za pozadinsku sliku, možete koristiti svojstvo ``background-size``.

Možete koristiti ``contain`` da skalirate sliku što je moguće veće bez rezanja ili istezanja.

Evo primera sa `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

Postavljamo ``min-height`` na ``100px`` kako bi bila vidljiva pozadinska slika i da raspored zadrži osnovnu visinu, osiguravajući da dizajn izgleda uravnoteženo i vizuelno privlačno čak i sa minimalnim sadržajem.

Ako promenimo svojstvo `background-size` da koristi vrednost `cover`, onda će pozadinska slika skalirati da pokrije ceo element `body` uz održavanje njegovog aspektnog odnosa.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

U prethodnim primerima, verovatno ste primetili da pozadinska slika se kontinuirano ponavlja.

Podrazumevano, pozadinske slike se ponavljaju i horizontalno i vertikalno kako bi popunile ceo element. Međutim, možete kontrolisati ovo ponašanje.

Možete koristiti svojstvo ``background-repeat`` sa vrednošću postavljenu na ``no-repeat``.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Sa `background-size` podešenim na `contain` i `background-repeat` podešenim na `no-repeat`, slika više neće ponavljati na ekranu.

Ako želite da ponovite pozadinsku sliku horizontalno, možete koristiti vrednost ``repeat-x`` za svojstvo ``background-repeat``.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

Ako želite da ponovite pozadinsku sliku vertikalno, možete koristiti vrednost `repeat-y` za svojstvo `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

Za pozicioniranje pozadinske slike na ekranu, možete koristiti svojstvo `background-position`.

Svojstvo `background-position` vam omogućava da postavite gde u elementu će se pojaviti pozadinska slika. Možete koristiti ključne reči kao što su `top`, `bottom`, `left`, `right`, i `center`, ili specifične vrednosti u pikselima ili procentima.

Evo primera korišćenja `center` i `top` za `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

Ovaj CSS kod pozicionira pozadinsku sliku u centru elementa po horizontalnoj i na vrhu po vertikalnoj osi.

Konačno, `background-attachment` određuje da li se pozadinska slika skrolova sa sadržajem ili ostaje fiksirana kada se stranica skrološe.

Glavne vrednosti su `scroll` (podrazumevano), gde se pozadinska slika skroluje sa sadržajem, i `fixed`, gde pozadinska slika ostaje na istoj poziciji na ekranu.

Evo primera korišćenjem vrednosti `fixed` za svojstvo `background-attachment`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

Ovaj CSS kod postavlja pozadinsku sliku da ostane fiksirana na mestu čak i kada se stranica skrola.

Ako želite da kombinujete nekoliko svojstava u jednu liniju, možete to učiniti korišćenjem skraćene (shorthand) svojstva `background`.

Evo primera:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Navedeni primer kombinuje `background-image`, `background-position`, i `background-attachment` u jednu liniju.

Savladavanjem ovih svojstava, dobićete veliku kontrolu nad načinom prikaza slika u pozadini u vašem web dizajnu, što omogućava vizuelno privlačnije i funkcionalnije rasporede.
