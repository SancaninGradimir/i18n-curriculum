# --interactive--

Pozadinski gradijent u CSS-u je gladak prelaz između dve ili više boja koji se može primeniti na pozadinu elementa. Gradijenti vam omogućavaju da kreirate vizuelno privlačne pozadine bez potrebe za slikama.

Postoje dve glavne vrste gradijenata u CSS-u: linearni gradijenti i radijalni gradijenti.

Linearni gradijent prelazi boje duž prave linije. Možete definisati smer i uključene boje.

Evo osnovne sintakse:

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

U ovom primeru, koristimo `background` CSS svojstvo sa vrednošću `linear gradient`.

Direkcija određuje smer gradijenta. Može biti ugao (kao što je `45deg`), ključna reč (kao što su `to right`, `to bottom`), ili strana/ugao.

`color-stop` određuje boje i pozicije gde se dešavaju prelazi gradijenta.

Da bismo bolje razumeli kako linijski gradijenti funkcionišu, pogledajmo sledeći primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

Ovaj CSS kreira linearni gradijent koji prelazi sa `red` na `left` do `yellow` na `right`. Gradijent je primenjen na element sa visinom od `40%` od visine prozora. Naučićete više o `vh` jedinicama u budućoj lekciji.

Pravac `to right` znači da je gradijent horizontalno od leve ka desnoj strani.

Drugi tip gradijenta bio bi `radial` gradijent.

Radijalni gradijent prelazi boje koje zrače od izvora (obično centra) ka spolja u kružnom ili eliptičnom obliku.

Evo osnovne sintakse:

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

U sintaksi, `shape` definiše oblik gradijenta koji može biti `circle` ili `ellipse`.

`size` određuje veličinu završnog oblika gradijenta koji bi mogao biti `closest-side`, `closest-corner`, `farthest-side` ili `farthest-corner`.

`position` određuje poziciju centra gradijenta, što se može specificirati korišćenjem ključnih reči (kao što su `center`, `top left`, `bottom right`) ili preciznim vrednostima (kao što su `50% 50%`, `10px 20px`).

Konačno, color stopovi predstavljaju listu boja kroz koje prođe gradijent. Svaki color stop opcionalno može uključiti vrednost pozicije (procenat ili dužina) koja ukazuje gde bi boja trebalo da bude postavljena.

Primer bi bio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

Ovaj CSS kreira cirkularni radijalni gradijent centriran u elementu. Počinje sa `red` u centru, prelazi na `yellow` pri `50%` od radijusa, i završava se sa `green`.

Ključna reč ``closest-side`` omogućava da se završni oblik gradijenta uklopi na najbližu stranu elementa. Gradijent se primenjuje na element sa visinom od ``60%`` visine prozora.

Razumevanje načina rada sa CSS gradijentima može značajno poboljšati vaše dizajne pružanjem vizuelno privlačnih pozadina bez potrebe za slikama.

Sa opcijama kao što su linearni gradijenti za glatke tranzicije i radijalni gradijenti za kružne efekte, oni nude kako fleksibilnost tako i kreativnost u web dizajnu.
