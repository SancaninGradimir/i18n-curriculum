# --description--

CRUD je akronim koji znači Kreiranje, Čitanje, Ažuriranje i Brisanje. Ovo su četiri operacije trajnog skladištenja.

Trajno skladište se odnosi na čuvanje podataka na način koji ih čini dostupnim čak i nakon što se napajanje isključi ili uređaj ponovo pokrene.

Razumevanje kako funkcioniše osnovni rad CRUD je ključno za veb razvoj jer čini temelj za rad sa bazama podataka i izradu aplikacija gde korisnici mogu dodati, pregledati, modifikovati i obrisati podatke.

Sada ćemo detaljno pogledati svaki deo CRUD-a.

- Kreiranje se odnosi na proces kreiranja novih podataka. Na primer, u veb aplikaciji, ovo može biti kada korisnik doda novi post na blog.

- Čitanje je operacija u kojoj se podaci preuzimaju iz baze podataka. Na primer, kada posetite blog objavu ili pregledate svoj profil na veb-sajtu, izvršavate operaciju čitanja kako biste dohvatili i prikazali podatke sačuvanima u bazi podataka.

- Ažuriranje podrazumeva modifikovanje postojećih podataka u bazi podataka. Primer bi bilo uređivanje blog objave ili ažuriranje vaših profilnih informacija.

- Brisanje je operacija koja uklanja podatke iz baze podataka. Na primer, kada obrišete blog objavu ili nalog, izvršavate operaciju brisanja.

CRUD se koristi prilikom rada sa bazama podataka, korisničkim interfejsom (UI) i RESTful API-jima. RESTful API-ji su skup konvencija za izradu veb servisa koji omogućavaju klijentu da interaguje sa bazom podataka ili sistemskim 'back-endom' izvršavanjem CRUD operacija putem standardnih HTTP metoda.

HTTP znači Hypertext Transfer Protocol i čini osnovu komunikacije podataka na vebu. Postoje HTTP metode koje definišu akcije koje se mogu izvršiti na resursima preko veba. Uobičajene metode su `GET`, `POST`, `PUT`, `PATCH`, `DELETE`.

Naučićete više o RESTful API-jima i HTTP u budućim lekcijama, ali evo kratkog pregleda kako se CRUD mapira na različite HTTP metode.

- `POST` se koristi za kreiranje novog resursa.

- `GET` se koristi za dohvaćanje ili čitanje podataka.

- `PUT` se koristi za ažuriranje resursa zamjenom celog sadržaja.

- `PATCH` se koristi za delimično ažuriranje resursa.

- `DELETE` koristi se za uklanjanje resursa.

Evo primera kako se CRUD operacije mogu predstaviti u kodu koristeći jednostavan niz u JavaScriptu:

```js
let items = [];

// Create
function createItem(item) {
  items.push(item);
}

// Read
function readItems() {
  return items;
}

// Update
function updateItem(index, newItem) {
  items[index] = newItem;
}

// Delete
function deleteItem(index) {
  items.splice(index, 1);
}

// Example Usage
createItem('Book');
console.log(readItems()); // ['Book']
updateItem(0, 'Magazine');
console.log(readItems()); // ['Magazine']
deleteItem(0);
console.log(readItems()); // []
```

U ovom primeru:

- Kreiramo stavku guranjem je u niz.

- Čitamo stavke vraćanjem niza.

- Ažuriramo stavku modifikovanjem elementa niza na zadatom indeksu.

- Brisemo stavku uklanjanjem je iz niza koristeći `splice()`.

Ovo je osnovna reprezentacija kako CRUD operacije funkcionišu na konceptualnom nivou.
