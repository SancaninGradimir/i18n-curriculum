# --description--

JavaScript je jezik dinamički tipizovan, što znači da ne morate specificirati tip podataka varijable kada je deklarišete. Umesto toga, tip se određuje na osnovu vrednosti dodeljene varijabli dok program radi. Ovo vam omogućava da promenite tip varijable tokom celog programa.

Pogledajmo primer:

```js
let example = "Hello";
example = 42;
```

U ovom primeru, imamo varijablu nazvanu `example` sa tipom podataka string. Ali zatim ažuriramo vrednost da bude broj umesto toga.

Fleksibilnost dinamičkog tipiziranja čini JavaScript-ov jezik opuštenijim i lakšim za rad pri brkom skriptiranju, ali takođe može uvesti greške koje je teže uhvatiti, posebno kako vaš program raste.

U statički tipizovanim jezicima kao što su C# ili C++, morate deklarisati tip podataka varijable kada je kreirate, i taj tip se ne može promeniti.

Na primer, ako deklariš varijablu kao `integer`, možeš joj dodeliti samo celobrojne vrednosti. Ako pokušaš da joj dodeliš drugi tip, program će baciti grešku.

Evo primera na jeziku C#:

```csharp
int data = 42; // data must always be an integer
data = "Hello"; // This would cause an error in C#
```

Razlika između dinamičkog i statičkog tipiziranja leži u fleksibilnosti naspram sigurnosti vašeg koda. Dinamički tipizovani jezici nude fleksibilnost, ali po cenu potencijalnih grešaka tokom izvršavanja (runtime).

Statički tipizovani jezici nameću stroža pravila koja mogu sprečiti određene greške, ali zahtevaju više prethodnog deklarisanja i nude manju fleksibilnost u promeni tipova.

Evo još jednog primera kreiranja varijable sa tipom postavljenim na `number`, a zatim menjanja tog tipa da kasnije bude tip `string`:

```js
let data = 100;  // Initially a number
data = "New data";  // Dynamically changes to a string
```

U jeziku sa statičkim tipizacijom, ovaj tip promene ne bi bio dozvoljen jer je tip podataka fiksiran.

Zaključno, dinamičko tipiziranje JavaScript-a dozvoljava promenljivama da slobodno menjaju tipove, što nudi fleksibilnost, ali može dovesti do neočekivanih grešaka tokom izvršavanja.

Jezici sa statičkim tipiziranjem poput Jave zahtevaju da unapred definišete tipove promenljivih, što pomaže u hvatanju grešaka pre pokretanja programa, ali nude manju fleksibilnost.
