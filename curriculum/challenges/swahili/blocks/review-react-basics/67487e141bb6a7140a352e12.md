## JavaScript Biblioteke i Okvirci

JavaScript biblioteke i framework-ovi nude brka rešenja za uobičajene probleme i ubrzavaju razvoj pružanjem predizgrađenog koda.
Biblioteke su generalno više fokusirane na pružanje rešenja za specifične zadatke, kao što je manipulisanje DOM-a, rukovanje događajima ili upravljanje AJAX zahtevima. 
Neki primeri JavaScript biblioteka su jQuery i React.
Framework-ovi, s druge strane, pružaju definisanu strukturu za izgradnju aplikacija. Često dolaze sa setom pravila i konvencija koje programeri moraju pratiti.
Primeri framework-ova uključuju Angular i Next.js, meta framework za React.
**Jednonoćne aplikacije** (Single Page Applications - SPAs) su web aplikacije koje učitavaju jednu HTML stranicu i dinamički ažuriraju tu stranicu kako korisnik interaguje sa aplikacijom bez ponovnog učitavanja cele stranice.
SPAs koriste JavaScript za upravljanje stanjem aplikacije i prikaz sadržaja (render). Ovo se često radi korišćenjem framework-ova koji pružaju odlične alate za izgradnju složenih korisničkih interfejsa.
Neki problemi vezani za SPAs uključuju:

- Čitači ekrana imaju poteškoća sa dinamički ažuriranim sadržajem.
- URL se ne menja kada korisnik navigira unutar aplikacije, što može otežati označavanje (bookmarking), povratak na prethodnu stranicu ili deljenje specifičnih stranica.
- Početno vreme učitavanja može biti sporo ako je aplikacija velika, jer su svi resursi potrebni za učitavanje unapred.

## React

- React je popularna JavaScript biblioteka za izgradnju interfejsa korisnika i veb aplikacija.
- Osnovni koncept React-a je kreiranje ponovo upotrebljivih UI komponenti koje mogu da se ažuriraju i prikažu nezavisno kako se podaci menjaju.
- React omogućava programerima da opišu kako bi interfejs korisnika (UI) trebalo da izgleda na osnovu stanja aplikacije. Zatim, kada se podaci ili stanje promene, React ažurira i prikazuje prave komponente.

## Komponente Reacta

- Komponente su građevni blokovi React aplikacija koji omogućavaju razvojnim programerima da razbiju kompleksne korisničke interfejse na manje, upravljive delove.
- UI se opisuje korišćenjem JSX-a, proširenja sintakse JavaScripta, koje omogućava razvojnim programerima da pišu kod sličan HTML-u unutar JavaScripta.
- Komponente su po suštini JS funkcije ili klase koje vraćaju deo korisničkog interfejsa.
