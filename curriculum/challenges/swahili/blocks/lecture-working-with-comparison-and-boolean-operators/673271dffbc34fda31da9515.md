# --interactive--

Operatori poređenja omogućavaju da uporedite dve vrednosti i vrate rezultat `true` ili `false`. Zatim možete koristiti rezultat za donošenje odluke ili kontrolisanje protoka vašeg programa. Poređenja koristite u izjavama `if`, petljama i mnogim drugim situacijama gde morate doneti odluke na osnovu određenih uslova. Hajde da se upoznamo sa najčešćim operatorima poređenja i vidimo kako funkcionišu.

Operator za veće od, predstavljen desnim uglastim zagrade (`>`), proverava da li je vrednost na levoj strani veća od one na desnoj:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

Operator veće ili jednako, predstavljen znakom veće kose crte i znakiem jednakosti (`>=`), proverava da li je vrednost na levoj strani ili veća ili jednaka vrednosti na desnoj strani:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

Operator manjeg od, predstavljen levim uglastim zagrade (`<`) radi slično kao `>`, ali obrnuto. Proverava da li je vrednost na levoj strani manja od one na desnoj strani:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

Operator "manje ili jednako", predstavljen sa levim uglastim zagrade i znakom jednakosti (`<=`) proverava da li je vrednost na levoj strani manja ili jednaka onoj na desnoj strani:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
