# --description--

Nasleđivanje je sledeći ključni koncept u objektno usmerjenom programiranju (OOP) koji ćemo pokriti.

Pogledajmo bliže ovaj koncept i kako vam omogućava pisanje ponovo upotrebljivog koda.

Sa nasleđem, podklasa (ili detijska klasa) može koristiti atribute i metode osnovne klase (ili roditeljske klase). Ovo vam omogućava ponovna upotreba koda, kreiranje jasnih hijerarhija klasa i prilagođavanje ponašanja bez prepisivanja svega. Možete prilagoditi proširivanjem postojećih metoda ili preklapanjem u detijskoj klasi.

Evo osnovne sintakse za nasleđivanje:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

Da bi klasa `Child` nasledila od klase `Parent`, moraš proslediti Roditelja Detlji.

Ovaj stil se naziva jednaka nasleđivanje, jer klasa detinjstvo nasleđuje od tačno jedne roditeljske klase.

Evo primera:

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound
print(jack.bark)  # woof! woof!! woof!!!
```

Možete videti da možemo ponovo koristiti atribut ``self.name`` i metodu ``sound()`` od roditeljske klase ``Animal`` u ćerkolskoj klasi ``Dog``.

Prepisujemo metod `sound()` iz roditeljske klase `Animal` u detinjoj klasi `Dog`, kako bismo omogućili da `sound()` koristi klasnu varijablu `bark`:

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound.'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Override sound() to use bark class variable
    def sound(self):
        return f'{self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack barks woof! woof!! woof!!!
```

Ako želite da sačuvate vrednost povratka od `sound()` i kasnije dodate varijablu klase 'bark', možete proširiti `sound()` korišćenjem funkcije `super()`:

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Call Animal.sound(), then append bark
    def sound(self):
        base = super().sound()
        return f'{base}, then {self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound, then Jack barks woof! woof!! woof!!!
```

U ovom primeru, `base` je rezultat pozivanja metode `sound()` iz klase `Animal`, i zatim mu dodajemo specifičan zvuk klase `Dog`. Na ovaj način, možete proširiti funkcionalnost roditeljske klase `Animal` zadržavajući pritom njeno originalno ponašanje.

Postoji takođe višestruko nasleđivanje, gde se podklasa može naslediti od više od jedne roditeljske klase.

Evo osnovne sintakse višestrukog nasleđivanja:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

Jednostavan način za demonstraciju višestrukog nasleđivanja je sa žabom, koja može i da hoda po zemlji i da pliva u vodi:

```py
class Walker:
    def walk(self):
        return 'I can walk on land'

class Swimmer:
    def swim(self):
        return 'I can swim in water'

# Amphibian inherits from both Walker and Swimmer
class Amphibian(Walker, Swimmer):
    def __init__(self, name):
        self.name = name

    def introduce(self):
        return f"I'm {self.name} the frog. {self.walk()} and {self.swim()}."

frog = Amphibian('Freddy')
print(frog.introduce())
# Output: I'm Freddy the frog. I can walk on land and I can swim in water.
```
