# --interactive--

**NAPOMENA**: Neki interaktivni primeri mogu koristiti CSS koji još niste naučili. Ne brinite zbog pokušaja razumevanja celog koda. Cilj primera je da vam pokažu predgleđaje ovih dizajnerskih koncepata kako biste bolje razumeli kako stvari funkcionišu.

Prvo, možda ćete morati da se upoznate sa kontrastom. Kontrast je razlika između dve boje – ili koliko je lako uočiti njihovu razliku.

Boje sa većim kontrastom biće vizuelno različenije, dok će boje sa nižim kontrastom biti vizuelno sličnije. Na primer, crna i bela imaju veoma visok odnos kontrasta. Dok svetlo plava i svetlo ljubičasta imaju manji kontrast.

:::interactive_editor

```html
<style>
  .example {
    padding: 20px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .high-contrast {
    background-color: black;
    color: white;
  }

  .low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
  }
</style>

<div class="example high-contrast">
  <div class="label">High Contrast</div>
  This text has high contrast (black background and white text).
</div>

<div class="example low-contrast">
  <div class="label">Low Contrast</div>
  This text has low contrast (light blue background and light purple text).
</div>
```

:::

Naravno, u ovim primerima razlika može biti jasnija zbog rasporeda. Ali šta je sa primenom ovih boja na tekst? Možete imati crni tekst visokog kontrasta na beloj pozadini. I ljubičasti tekst niskog kontrasta na plavoj pozadini.

:::interactive_editor

```html
<style>
  .text-example {
    padding: 10px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .text-high-contrast {
    background-color: white;
    color: black;
  }

  .text-low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
    display: block;
  }
</style>

<div class="text-example text-high-contrast">
  <span class="label">High Contrast Text</span>
  This is high contrast text: black text on a white background.
</div>

<div class="text-example text-low-contrast">
  <span class="label">Low Contrast Text</span>
  This is low contrast text: purple text on a light blue background.
</div>
```

:::

Ali kako da odredite kakav je "dovoljan" kontrast? Ne možete se osloniti samo na ono kako vam tekst izgleda, jer će svaki korisnik imati drugačije iskustvo. Srećom, smernice za pristupačnost web sadržaja, ili WCAG, pružaju standard za ovo.

Tekst sa kontrastnim odnosom od 4,5:1 smatra se AA standardom, što je apsolutni minimum koji biste trebali pratiti da bi bio dostupan većini korisnika. Tekst sa kontrastnim odnosom od 7:1 smatra se AAA standardom i obezbeđuje najbolji nivo pristupačnosti.

:::interactive_editor

```html
<style>
  .contrast-example {
    padding: 15px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .aa-contrast {
    background-color: #ffffff; 
    color: #4b4b4b; 
  }

  .aaa-contrast {
    background-color: #ffffff; 
    color: #1a1a1a; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }
</style>

<div class="contrast-example aa-contrast">
  <span class="label">AA Standard (Contrast Ratio ~4.5:1)</span>
  This text meets the minimum accessibility standard for contrast.
</div>

<div class="contrast-example aaa-contrast">
  <span class="label">AAA Standard (Contrast Ratio ~7:1)</span>
  This text meets the highest accessibility standard for contrast.
</div>
```

:::

Postoji nekoliko veb sajtova koji mogu proveriti odnos kontrasta između dve boje, ali većina pregledača će vam dozvoliti da ovo uradite direktno u alatima za programere na vašem sajtu.

Pogledajmo još nekoliko primera kontrastnog odnosa kako bismo bolje razumeli koncept.

Ovo je primer crnog teksta na beloj pozadini koji ima kontrastno ommerenje od 21:1.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This is black text on a white background, which has the highest contrast ratio of 21:1.
</div>
```

:::

Ovaj primer više od zadovoljava AAA standard. Sada, hajde da pogledamo ljubičasti tekst na plavom pozadinjama.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .purple-on-blue {
    background-color: #0000cc;
    color: #800080; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="purple-on-blue">
  <span class="label">Purple on Blue (Lower Contrast)</span>
  This doesn't meet the AA standard.
</div>
```

:::

Ovaj primer ima odnos kontrasta od 1,48:1 koji čak ni ne ispunjava AA standard.

Kako možete da ovo popravite? Pa, postoje tri aspekta koji utiču na odnos kontrasta.

Prvi je ton (hue), koji predstavlja opšti tip boje, kao što su crvena, plava, narandžasta. Prebacimo se na ton koji je dalje od plave. U ovom slučaju, koristićemo crvenu.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .red-on-blue {
    background-color: #0000cc; 
    color: #ff0000; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="red-on-blue">
  <span class="label">Red on Blue (Higher Contrast Hue Shift)</span>
  This doesn't meet accessibility standards.
</div>
```

:::

Nažalost, ova promena je vratila odnos kontrasta na 1.49:1, što znači da morate promeniti zasićenje ili "iznos" prisutne boje. Povećajmo količinu crvene boje u tekstu. To nas približava cilju, sa odnosom kontrasta od 3.54:1.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .low-sat-red-on-blue {
    background-color: #0000cc; 
    color: #b23333; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .high-sat-red-on-blue {
    background-color: #0000cc; 
    color: #ff4d4d;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="low-sat-red-on-blue">
  <span class="label">Low Saturation Red on Blue (Contrast ~1.49:1)</span>
  This red has low saturation, resulting in a poor contrast ratio.
</div>

<div class="high-sat-red-on-blue">
  <span class="label">Higher Saturation Red on Blue (Contrast ~3.54:1)</span>
  Increasing the saturation of red improves contrast but it’s still below AA standard.
</div>
```

:::

Međutim, potencijalno možete doći još bliže menjanjem poslednje vrednosti, koja je svetlina. Svetlina predstavlja koliko belog je prisutno u boji.

Smanjićemo svetlinu crvene boje. Sada postoji mnogo tamnija crvena boja na pozadini svetlo plave boje, što povećava kontrastni odnos na 10.34:1.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .blue-bg {
    background-color: #add8e6; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .low-sat-red-on-blue {
    background-color: #0000cc; 
    color: #b23333;
    margin-bottom: 20px;
  }

  .high-sat-red-on-blue {
    background-color: #0000cc; 
    color: #ff4d4d; 
    margin-bottom: 20px;
  }
  
  .dark-red-on-light-blue {
    background-color: #add8e6; 
    color: #8b0000;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="low-sat-red-on-blue blue-bg">
  <span class="label">Low Saturation Red on Medium Blue (Contrast ~1.49:1)</span>
  This red has low saturation, resulting in a poor contrast ratio.
</div>

<div class="high-sat-red-on-blue blue-bg">
  <span class="label">Higher Saturation Red on Medium Blue (Contrast ~3.54:1)</span>
  Increasing the saturation of red improves contrast but it’s still below AA standard.
</div>

<div class="dark-red-on-light-blue">
  <span class="label">Darker Red on Light Blue (Contrast ~10.34:1)</span>
  Decreasing the lightness of the red increases the contrast ratio significantly.
</div>
```

:::
