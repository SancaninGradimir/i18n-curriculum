# --interactive--

**NAPOMENA**: Neki interaktivni primeri mogu koristiti CSS koji još niste naučili. Ne brinite zbog pokušaja razumevanja celog koda. Cilj primera je da vam pokažu predgleđaje ovih dizajnerskih koncepata kako biste bolje razumeli kako stvari funkcionišu.

Dodavanje slika na vaše veb-sajtove je odličan način za angažovanje vaših korisnika i povećanje vizuelne privlačnosti sajta. Ali postoji nekoliko stvari koje treba uzeti u obzir kada radite sa slikama u svojim dizajnima.

Prva stvar koju treba uzeti u obzir je kreiranje responsivnih slika. Responsivne slike su slike koje se skaliraju kako bi odgovarale veličini ekrana na kojem se gledaju. Ovo je važno jer osigurava da vaše slike izgledaju dobro na svim uređajima, od desktop računara do mobilnih telefona.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #fefefe;
    color: #333;
    text-align: center;
  }

  img {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
  }

  p {
    font-size: 16px;
    max-width: 600px;
    margin: 20px auto;
    line-height: 1.6;
  }
</style>

<h1>Responsive Cat Image</h1>

<img 
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
  alt="Two cats peacefully sleeping together."
/>

<p>
  This image automatically scales based on the screen size. Whether you're viewing on a desktop or a mobile phone,
  it adjusts its size without losing proportions, making the design clean and user-friendly on all devices.
</p>
```

:::

Još jedna stvar na koju treba uzeti u obzir je rezolucija za slike. Slike višeg kvaliteta sa boljom rezolucijom imaju više piksela po inču. Pikseli su mali kvadrati koji čine sliku.

Pikseli po inču, ili PPI, je broj piksela u jednom inču slike. Što je PPI viši, to je kvalitet slike bolji.

Želite da se uverite da su vaše slike visokog kvaliteta i da izgledaju dobro na svim uređajima. To znači da biste trebali koristiti slike visoke rezolucije koje su optimizovane za veb.

Još nešto što treba uzeti u obzir je veličina vaših slika i kako se uklapaju u razmake u rasporedu. Želite da budete sigurni da su vaše slike prave veličine i da nisu ni prevelike ni premale.

Korišćenje velikih slika koje su namenjene da stanu u manje prostore u dizajnu može usporiti vaš veb sajt i otežati korisnicima učitavanje sajta. Želite da se uverite da su vaše slike prave veličine i optimizovane za veb.

Što se tiče pozicioniranja slika, želite da razmislite o balansu, hijerarhiji i poravnanju kako biste pomogli osigurati da su vaše slike optimizovane za web.

Balans je raspodela vizuelne težine u dizajnu. Želite da se osigurate dobar balans između teksta i slika na sajtu kako bi stvorio harmoničan dizajn.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    margin: 0;
    padding: 40px 20px;
    background-color: #f9f9f9;
    color: #333;
  }

  .container {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    justify-content: space-between;
    gap: 30px;
    max-width: 1000px;
    margin: 0 auto;
  }

  .text {
    flex: 1 1 400px;
  }

  .text h2 {
    font-size: 28px;
    margin-bottom: 10px;
  }

  .text p {
    font-size: 16px;
    line-height: 1.6;
  }

  .image {
    flex: 1 1 400px;
  }

  .image img {
    width: 100%;
    height: auto;
    border-radius: 8px;
  }
</style>

<div class="container">
  <div class="text">
    <h2>Balanced Layout</h2>
    <p>
      Balance is essential in web design. By evenly distributing visual weight—such as pairing this block of text
      with a complementary image—you create a layout that feels calm, structured, and easy to navigate.
    </p>
  </div>

  <div class="image">
    <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two cats peacefully sleeping together.">
  </div>
</div>
```

:::

Hijerarhija je redosled kojim se elementi pregledaju na stranici. Želite da se osigurate da su slike koje odgovaraju važnom sadržaju postavljene više od slika koje nisu tako važne.

Poravnanje je postavljanje elemenata u odnosu jedan prema drugom. Želite da se uverite da su vaše slike poravnane sa tekstom i drugim elementima na vašem sajtu kako bi stvorile koherentan dizajn.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 40px 20px;
    background-color: #ffffff;
    color: #222;
    max-width: 900px;
    margin: 0 auto;
  }

  .aligned-content {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    gap: 20px;
  }

  .aligned-content img {
    width: 300px;
    height: auto;
    border-radius: 8px;
    flex-shrink: 0;
  }

  .aligned-text {
    flex: 1;
    min-width: 250px;
  }

  .aligned-text h2 {
    font-size: 24px;
    margin-bottom: 10px;
  }

  .aligned-text p {
    font-size: 16px;
    line-height: 1.6;
  }
</style>

<div class="aligned-content">
  <img 
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
    alt="Two cats peacefully sleeping together."
  >

  <div class="aligned-text">
    <h2>Aligned Elements</h2>
    <p>
      In this example, the image is aligned horizontally next to the text block. This creates a visually cohesive layout, 
      where both elements feel like they belong together. Proper alignment like this improves readability and reinforces 
      structure in your design.
    </p>
  </div>
</div>
```

:::

Poslednja stvar koju treba uzeti u obzir je pristupačnost slika. Želite da se uverite da su vaše slike dostupne svim korisnicima, uključujući one sa smanjenim vidnim sposobnostima. To znači da biste trebali koristiti `alt` tekst za vaše slike kako bi čitači ekrana mogli da pročitao tekst korisnicima sa smanjenim vidnim sposobnostima.
