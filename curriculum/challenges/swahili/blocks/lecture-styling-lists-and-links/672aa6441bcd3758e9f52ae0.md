# --interactive--

Margine i visina linije su neophodni za razmak liste stavki radi poboljšanja čitljivosti i vizuelne privlačnosti.

Prvo, hajde da počnemo sa razmakljivanjem stavki liste korišćenjem margina!

Margine se mogu koristiti za kreiranje razmaka između stavki liste primenjivanjem svojstava margine na elemente ``li``. Ova metoda vam omogućava da kontrolišete razmak izvan svake stavke liste, efektivno povećavanjem ili smanjenjem praznine između njih.

Pogledajmo primer neuređenog spiska sa tri stavke spiska.

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

Podrazumevano, HTML neće primeniti toliko mnogo razmaka između stavki liste.

Da biste primenili određeni razmak na dno svakog stavke liste, možete koristiti svojstvo ``margin-bottom`` ovako:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

U ovom primeru, `40px` margine će biti primenjena na dno svakog elementa liste unutar neuređenog spiska.

Drugi način za razdvajanje stavki liste je korištenje svojstva `line-height`.

Svojstvo `line-height` prilagođava vertikalni razmak između redova teksta unutar jedne stavke liste.

Iako primarno utiče na razmak između redova teksta unutar svake stavke, takođe može indirektno da utiče i na ukupni razmak između elemenata liste ako te elemente sadrže samo jedan red teksta.

Ako stavke liste imaju više redova teksta, `line-height` će uticati na razmak između tih linija, ali ne prilagođava direktno razmak između samih odvojenih stavki liste.

Da biste kontrolisali razmak između pojedinačnih stavki liste, koristili biste `margin` ili `padding` svojstva umesto toga.

Koristeći istu neuređenu listu od ranije, evo primera primene `line-height` na stavke liste:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

U ovom primeru, `line-height: 2;` postavlja visinu linije na dvostruku veličinu fonta, stvarajući više vertikalnog prostora unutar svakog stavke liste.

S obzirom na CSS pravilo `li { margin-bottom: 15px; }`, koji uticaj ovo ima na stavke liste?

Dodaje `15px` razmak unutar svake stavke liste.

Ovo pravilo utiče na prostor izvan stavke liste, a ne unutar.
