# --interactive--

Podrazumevani stilovi linkova igraju ključnu ulogu u poboljšanju upotrebljivosti i pristupačnosti na vebu.

Ovi stilovi, obično plavi za neposete linkove i ljubičasti za posete linkove, postali su standard na koji se korisnici naviknu da očekuju i oslanjaju prilikom navigacije po veb sajtovima.

Glavna svrha podrazumevanih stilova za linkove je pružanje jasnih vizuelnih indikatora koji korisnicima pomažu da razlikuju interaktivne i neinteraktivne elemente na veb stranici.

Ova razlika je fundamentalna za kreiranje intuitivnog i korisnički prijateljskog iskustva pregledavanja.

Razmotrimo osnovne podrazumevane stilove za linkove:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: underline;
}

a:visited {
  color: purple;
}
```

:::

Ovi stilovi služe nekoliko važnih funkcija.

Prvo, plava boja za neposeta linkove se ističe na pozadini većine boja i teksta, što čini linkove lako prepoznatljivima. Ovaj kontrast je ključan za korisnike kako bi brzo skenirali stranicu i pronašli navigacione elemente ili važne informacije.

Podvlačenje dodatno naglašava da je tekst klikabilan, pružajući dodatni vizuelni signal. Ovo je posebno korisno za korisnike koji mogu imati problem sa vidom boja ili im je teško razlikovati boje.

Promena boje za posete linkovima (obično u ljubičasto) pomaže korisnicima da prate gde su bili. Ova funkcija je neprocenjiva za navigaciju velikim sajtovima ili sprovođenje istraživanja, jer sprečava korisnike od slučajnog ponovnog posećivanja istih stranica.

Razmotrite ovaj HTML primer:

:::interactive_editor

```html
<p>Learn more about <a href="https://www.example.com/cats">cats</a> and <a href="https://www.example.com/dogs">dogs</a>.</p>
```

:::

Bez bilo kakvog prilagođenog CSS-a, većina pregledača će prikazati ove linkove u plavoj boji sa podvlačenjem. Nakon klika na jedan od linkova, njegova boja će se promeniti u ljubičastu, pružajući korisniku trenutnu povratnu informaciju o njegovoj istoriji pretraživanja.

Iako je uobičajeno da dizajneri modifikuju ove podrazumevane stilove kako bi odgovarali estetici veb-sajta, ključno je održati osnovna načela iza njih.

Ako odlučite da promenite podrazumevane stilove, uverite se da su linkovi i dalje jasno razlikovljivi od običnog teksta, postoji vidljiva razlika između posetađenih i neposetađenih linkova, i da izabrane boje pružaju dovoljan kontrast sa pozadinom.

Na primer, možete koristiti prilagođeni stil kao ovaj:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: none;
  border-bottom: 1px solid blue;
}

a:visited {
  color: purple;
  border-bottom: 1px solid purple;
}
```

:::

Ovo zadržava plavo i ljubičasto shema boja, istovremeno zamenjujući podvlačenje donjom ivicom radi modernijeg izgleda.

Takođe je važno uzeti u obzir različite stanja linkova. Pored podrazumevanog i poseta stanja, linkovi obično imaju stanje prelaska mišem (hover) i aktivno stanje (active):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:hover {
  color: red;
}

a:active {
  color: darkorange;
}
```

:::

Ova stanja pružaju trenutnu povratnu informaciju korisnicima dok interaguju sa linkovima, poboljšavajući ukupnu upotrebljivost sajta.

Zaključno, iako je moguće prilagoditi stilove linkova, principi iza podrazumevanih stilova bi trebalo da se zadrže.

Oni igraju ključnu ulogu u kreiranju upotrebljivog i pristupačnog veb iskustva, pomažući korisnicima da efikasno navigiraju i interaguju sa sadržajem.

Uvek prioritetizujte jasnoću i korisničko iskustvo prilikom dizajniranja stilova za linkove za vaše veb-sajtove.
