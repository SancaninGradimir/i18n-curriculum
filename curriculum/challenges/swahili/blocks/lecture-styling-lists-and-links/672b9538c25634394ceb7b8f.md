# --interactive--

Postoje različiti stanja linka, uključujući `link`, `visited`, `hover`, `focus`, i `active`. Ova stanja su važna za pomoć korisnicima u prepoznavanju linkova i pružanje jasne povratne informacije nakon interakcija, što poboljšava kako upotrebljivost tako i pristupačnost.

Stilovanje ovih različitih stanja linkova je ključno za upotrebljivost i pristupačnost, jer pruža vizuelne signale o trenutnom stanju linka. Ovo pomaže korisnicima da razumeju koje su linkove posetili, sa kojim linkom interaguju, i šta će se desiti kada kliknu.

Za korisnike sa smanjenim vidom ili kognitivnim teškoćama, ovi različiti stilovi mogu učiniti navigaciju mnogo lakšom i intuitivnijom.

Takođe, jasna stanja linkova poboljšavaju ukupno korisničko iskustvo pružanjem trenutne povratne informacije o interakcijama korisnika, smanjujući zbunjenost i unapređujući navigaciju sajta.

Ova stanja mogu se stilizovati koristeći nešto nazvano `pseudo-classes` u CSS-u.

Pseudo-klasa je ključna reč dodata selektoru koja određuje poseban status izabranog elementa.

Na primer, `:hover` može promeniti boju dugmeta kada korisnikov pokazivač pređe preko njega, dok `:visited` može promeniti boju veze koja je već poseta.

Pseudo-klase vam omogućavaju da stilizujete elemente na osnovu njihovog stanja ili interakcije korisnika sa njima, bez potrebe za dodatnim označavanjem u vašem HTML-u.

Sintaksa za `pseudo-class` izgleda otprilike ovako gde `A` je selektor i `:B` je `pseudo-class`:

```css
A:B {
  property: value;
}
```

Da bismo bolje razumeli kako stilizovati različita stanja linkova, hajde da pogledamo neke primere.

Pseudo-klasa "`:link`" stilizuje neposetaovane linkove, što ukazuje da su klikabilni.

Evo primera ciljanja ankora elementa i korišćenja pseudoklase `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

Gornji primer će promeniti podrazumevanu plavu boju veze na `red` kada nije posetana.

`:visited` stilizuju linkove koji su već posetili ili kliknuli, pomažući korisnicima da praćenje kojih linkova su prethodno kliknuli. Evo primera korišćenja `:visited` pseudo-klase:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

Ovaj kod će obojiti link ka `green` kada ga kliknete.

`:hover` menja stil veze kada korisnik pređe mišem preko nje, pružajući vizuelni signal da je veza interaktivna. Evo primera korišćenja pseudo-klase `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

Ovaj kod će obojiti linkove za `green` kada se mišem pređe preko njih.

`:focus` dodaje stilove oko veze kada je fokusirana, kao što je prilikom navigacije tastaturom, ili poboljšanja pristupačnosti.

Evo primera korišćenjem svojstva `outline` da bi se primenila čvrsta narandžasta kontura kada je link fokusiran.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` menja stilove linka dok se link klikta, pružajući korisniku trenutnu povratnu informaciju da mu je akcija registrovana. Evo primera korišćenja pseudo-klase `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

Ovaj kod primer će odmah obojiti link ka `pink` kada se klikne na njega.
