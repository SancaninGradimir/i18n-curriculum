# --interactive--

Destrukturiranje objekata je moćna funkcija u JavaScriptu koja vam omogućava da izvučete vrednosti iz objekata i dodelite ih promenljivama na koncizniji i čitljiviji način.

To deo je ES6 (ECMAScript 2015) specifikacije i postao je neophodan alat za mnoge JavaScript programere.

Destrukturiranje može pojednostaviti vaš kod, posebno kada radite sa kompleksnim objektima ili kada vam je potrebno izvući više vrednosti odjednom.

U svojoj srži, dekonstrukcija objekata se tiče raspakivanja vrednosti iz objekata u zasežne varijable. Umesto pristupa svojstvima objekta jedno po jedno, možete izdvojiti više svojstava u jednoj naredbi. Ovo može učiniti vaš kod čistijim i efikasnijim.

Počajmo sa primerom da ilustrujemo kako objektnu dekonstrukciju radi:

:::interactive_editor

```js
const person = { name: "Alice", age: 30, city: "New York" };

const { name, age } = person;

console.log(name); // Alice
console.log(age);  // 30
```

:::

U ovom primeru, izvlačimo svojstva `name` i `age` iz objekta `person` i dodeljujemo ih promenljivama sa istim imenima.

Jedan od moćnih aspekata dekonstrukcije objekata je što možete izvučene vrednosti dodeliti promenljivama sa različitim imenima. Ovo je posebno korisno kada radite sa objektima koji imaju imena svojstava koja mogu da kolidiraju sa postojećim promenljivama ili kada želite da koristite drugo ime:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };

let { name: personName, age: personAge } = person;

console.log(personName); // Alice
console.log(personAge); //  30
```

:::

U ovom slučaju, ekstrahujemo svojstvo `name` i dodeljujemo ga promenljivoj nazvanoj `personName`, i radimo isto sa `age` i `personAge`.

Destrukturiranje objekata takođe omogućava postavljanje podrazumevanih vrednosti. Ako svojstvo ne postoji u objektu koji dekonstruišete, možete navesti rezervnu vrednost:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };
let { name, age, country = "Unknown" } = person;

console.log(country); // Unknown
```

:::

Ovde, pošto ``country`` ne postoji u našem ``person`` objektu, dobija podrazumevanu vrednost ``Unknown``.

Još jedan uobičajen slučaj je destrukturiranje ugnježdene objekte. Možete destrukturirati svojstva ugnježdena unutar drugih objekata korišćenjem drugog skupa vitičastih zagrada:

:::interactive_editor

```js
const recipe = {
  name: "Chocolate Cake",
  ingredients: {
    flour: "2 cups",
    sugar: "1 cup"
  }
};

// Extract `flour` from `ingredients`
const { ingredients: { flour } } = recipe;

console.log(flour); // "2 cups"
```

:::

TEKST:

Ovo je ekvivalentno direktnom pristupu svojstvu:

```js
const flour = recipe.ingredients.flour;
console.log(flour); // "2 cups"
```

Sada, hajde da pričamo o skraćeno notaciji u dekonstrukciji objekata. Kada kreirate objekte, posebno kada imena svojstava odgovaraju imenima promenljivih, možete koristiti skraćenu sintaksu:

:::interactive_editor

```js
let name = "Bob";
let age = 25;

let person = { name, age };

console.log(person); // { name: "Bob", age: 25 }
```

:::

Gornji kod uzima svojstva koja imaju isto ime kao naše varijable i dodeljuje im vrednosti tih varijabli.

Ova skraćena notacija je posebno korisna kada vraćate objekte iz funkcija ili kreirate objekte sa više svojstava:

:::interactive_editor

```js
function createPerson(name, age) {
  return { name, age };
}

let person = createPerson("Charlie", 35);
console.log(person); // { name: "Charlie", age: 35 }
```

:::

Destrukturiranje objekata i skraćena notacija objekata su moćne karakteristike koje mogu učiniti vaš kod sažetijim i lakšim za čitanje.

Oni su posebno korisni kada radite sa složenim strukturama podataka, ili kada morate proslediti više parametara funkcijama.

Kako nastavite da radite sa JavaScript-om, naići ćete na mnogo situacija gde ove tehnike mogu pojednostaviti vaš kod i učiniti ga izražajnijim.
