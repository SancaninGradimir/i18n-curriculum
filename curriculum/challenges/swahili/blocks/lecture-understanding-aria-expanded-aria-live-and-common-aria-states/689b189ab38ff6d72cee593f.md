# --interactive--

Ranije ste naučili kako semantički označavanje i ARIA atributi čine sadržaj pristupačnim. U aplikacijama iz stvarnog sveta, sadržaj retko je statičan, a stranice se često dinamički ažuriraju putem JavaScripta. Kada se ovo dogodi, važno je osigurati da su te promene takođe odražene u HTML-u. To omogućava čitačima ekrana da tačno prenesu ažurirani status korisnicima. Bez toga, ljudi koji se oslanjaju na asistivne alate možda nikada neće znati da se nešto promenilo, ili bi mogli dobiti zastarele ili pogrešne informacije.

Pogledajmo u uobičajen scenarij gde JavaScript menja stanje elementa, i kako možete da održite svoj HTML pristupačnim ažuriranjem ARIA atributa.

Pretpostavimo da imate dugme koje otvara i zatvaraju padajući meni:

:::interactive_editor

```html
<button id="menuButton" aria-expanded="false" aria-controls="menuList">
  Menu
</button>

<ul id="menuList" hidden>
  <li>List item 1</li>
  <li>List item 2</li>
</ul>
<script src="index.js"></script>
```

```js
const button = document.getElementById("menuButton");
const menu = document.getElementById("menuList");

button.addEventListener("click", () => {
  const expanded = button.getAttribute("aria-expanded") === "true";
  button.setAttribute("aria-expanded", String(!expanded));
  menu.hidden = expanded;
});
```

:::

HTML postavlja dugme koje kontroliše padajući meni. Dugme koristi `aria-expanded` da pokaže da li je meni otvoren, i `aria-controls` da ga poveže sa menijem referenciranjem menijevog `id`.

JavaScript sluša klikove na dugme za meni. Kada se klikne, prebacuje vrednost `aria-expanded` i ažurira svojstvo `hidden` na meniju. Ovo održava vidljivost menija i informacije o pristupačnosti usklađenima, tako da i korisnici koji vide i korisnici čitača ekrana znaju da li je meni otvoren ili zatvoren.

Naučićete više o tim ARIA atributima u narednim lekcijama. Za sada, zapamtite samo da kada vaš JavaScript promeni stanje elementa, takođe biste trebali ažurirati relevante ARIA atribute kako bi asistivne tehnologije tačno odražavale ono što je zaista na ekranu.
