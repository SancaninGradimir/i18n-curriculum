# --description--

String je sekvenca znakova okruženih ili jednostrukim ili dvostrukim navodnicima. U nekim programskim jezicima, znakovi okruženi jednostrukim navodnicima tretiraju se drugačije od znakova okruženih dvostrukim navodnicima, ali u Pythonu, oni se tretiraju jednako. Zbog toga možete koristiti oba kada radite sa stringovima. Evo nekih primera stringova:

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

Ako vam treba višelinijski niz, možete koristiti trostruke navodne ili jednostruke navodnike:

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

Ako vaš niz sadrži ili pojedinačne ili dvostruke navodne znakove, onda imate dve opcije:

- Koristite suprotnu vrstu navodnika. To je, ako vaš niz sadrži pojedinačne navodnike, koristite dvostruke navodnike za obmotavanje niza, i obrnuto:

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- Escapirajte pojedinačni ili dvostruki navodnik u stringu pomoću povratnika (`\`). Ovim metodom, možete koristiti pojedinačne ili dvostruke navodnike za obmotavanje samog niza:

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

Ponekad je možda potrebno proveriti da li niz sadrži jedan ili više karaktera. Za to, Python pruža operater `in`, koji vraća booleanski tip podataka (boolean) koji određuje da li karakter ili karakteri postoje u nizu ili ne.

Evo nekih primera:

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

Sada, pogledajmo kako možete dobiti dužinu niza karaktera i raditi sa pojedinačnim karakterima u nizu, proces koji se zove **indeksiranje**. Da biste dobili dužinu niza karaktera, možete koristiti ugrađenu funkciju `len()`. Evo primera:

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

Svaki karakter u nizu ima poziciju nazvanu indeks. Indeks je baziran na nuli, što znači da je indeks prvog karaktera niza `0`, indeks drugog karaktera `1`, i tako dalje. Da biste pristupili karakteru po njegovom indeksu, koristite uglaste zagrade (`[]`) sa indeksom karaktera kojem želite da pristupite unutra. Evo nekoliko primera:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

Negativno indeksiranje je takođe dozvoljeno, pa možete dobiti poslednji karakter bilo kog niza sa `-1`, predposlednji karakter sa `-2`, i tako dalje:

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

Mnogi drugi programski jezici široko grupišu tipove podataka kao primitivne ili referentne tipove. Primitivni tipovi su jednostavni i nemenjivi, što znači da se ne mogu promeniti nakon deklarisanja. Referentni tipovi mogu sadržati više vrednosti i jesu ili menjivi ili nemenjivi. Ali Python ne crta jasnu granicu između te dve grupe. Umesto toga, svi podaci se tretiraju kao objekti, a neki objekti su nemenjivi dok su drugi menjivi.

Imutabilni tipovi podataka ne mogu biti modifikovani niti izmenjeni nakon što su deklarisani. Možete usmeriti njihove varijable na nešto novo, što se naziva ponovno dodeljivanje (reassignment), ali ne možete promeniti sam originalni objekat dodavanjem, uklanjanjem ili zamjenjivanjem bilo kojeg njegovog elementa.

Stringovi su nemenjivi tipovi podataka u Pythonu. To znači da možete ponovo dodeliti različiti string promenljivoj:

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

Ali direktna modifikacija stringa nije dozvoljena:

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Primeri drugih nemenjivih tipova podataka u Pythonu su ceo broj, float, boolean, tuple i opseg. Naučićete se svakog od ovih tipova u narednim lekcijama.
