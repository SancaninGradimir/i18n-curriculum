# --description--

Python pruža niz ugrađenih metoda koje čine rad sa stringovima izuzetno jednostavnim. Oni uključuju, ali nisu ograničeni na sledeće:

- `upper()`: Vraća novi niz znakova sa svim karakterima konvertovanim u VELIKI NASLONEK.

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: Vraća novi niz znakova sa svim karakterima konvertovanim u mala slova.

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: Vraća novi niz znakova sa uklonjenim navedenim vodećim i zakašnjelim karakterima. Ako nije prosleđen nijedan argument, uklanja vodeći i zakašnji razmak.

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: Vraća novi string sa svim pojavama `old` zamenjenim sa `new`.

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: Razdvaja string po navedenom razdvojnom znaku u listu stringova. Ako nije naveden razdvojni znak, razdvaja po razmacima.

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: Spaja elemente iterabilnog objekta u niz (string) sa separatorom.

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: Vraća boolean vrednost koja ukazuje da li niz počinje sa navedenim prefiksom.

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: Vraća boolean vrednost koja ukazuje da li niz završava sa navedenim sufiksom.

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: Vraća indeks prvog pojavljivanja `substring`, ili `-1` ako ne pronađe nijedan.

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: Vraća broj puta koliko se podstring pojavljuje u nizu karaktera.

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: Vraća novi niz znakova sa prvim karakterom velikim slovom, a ostalim malim.

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: Vraća `True` ako su sva slova u nizu velika i `False` ako nisu.

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: Vraća `True` ako su sva slova u nizu mala i `False` ako nisu.

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: Vraća novi niz čiji je prvo slovo svake reči kapitalizovano.

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
