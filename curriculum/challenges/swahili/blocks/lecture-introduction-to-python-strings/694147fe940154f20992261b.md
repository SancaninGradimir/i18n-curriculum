# --description--

U prethodnoj lekciji, naučili ste kako se svaki karakter u nizu može identifikovati po svom indeksu (počevši od nule), i pristupiti mu korišćenjem notacije sa uglastim zagradama:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**Slicanje stringova** vam omogućava da izvučete deo stringa ili radite samo sa određenim delom njega. Evo osnovne sintakse:

```python
string[start:stop]
```

Ako želite da izvučete znakove od određenog indeksa do drugog, samo razdvojite indekse `start` i `stop` dvotačkom:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

Napomena da indeks `stop` nije inkluzivan, pa je `[1:4]` upravo izvukao znakove od indeksa `1`, i do, ali ne uključujući, karakter na indeksu `4`.

Takođe možete izostaviti indekse `start` i `stop`, a Python će podrazumevano koristiti `0` ili kraj niza, respektivno. Na primer, ovo je šta se dešava ako izostavite indeks `start`:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

Ovo izvlači sve od indeksa `0` do (ali ne uključujući) karakter na indeksu `7`. I ovde je šta se dešava ako izostavite `stop` indeks:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

Ovo izvlači sve od karaktera na indeksu `8` do kraja niza.

Note that slicing a string does not modify the original string:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

Takođe možete izostaviti oba indeksa `start` i `stop`, što će izvući ceo string:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

Osim od `start` i `stop` indeksa, postoji i opcioni `step` parametar, koji se koristi za određivanje inkrementa između svakog indeksa u segmentu.

Evo sintakse za to:

```python
string[start:stop:step]
```

U primeru ispod, sekvenciranje počinje na indeksu `0`, zaustavlja se pre `11`, i izvlači svaki drugi karakter:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

Koristan trik koji možete uraditi sa parametrom `step` je da obrnete niz postavljanjem koraka na `-1`, i ostavljanjem `start` i `stop` praznim:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
