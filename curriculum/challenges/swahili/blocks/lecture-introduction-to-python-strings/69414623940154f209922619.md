# --description--

Kada radiš sa stringovima, spajanje različitih delova teksta zajedno je uobičajena operacija s kojom ćeš često naići.

U Pythonu, možete kombinovati više stringova zajedno sa plus (+) operatorom `(`+`)`. Ovaj proces se naziva **konkatenacija stringova**. Evo kako da spojite dva stringa sa plus (+) operatorom:

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

Međutim, napomenite da ovo funkcioniše samo sa nizovima (stringovima). Ako pokušate da konkatenišete string sa brojem, dobićete `TypeError`:

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

Ovo se dešava zato što Python ne konvertuje automatski druge tipove podataka, kao što su integeri, u stringove kada ih konkatenirate. Python zahteva da svi elementi budu stringovi pre nego što ih može konkatenirati. Da biste to popravili, možete konvertovati broj u string pomoću ugrađene funkcije `str()`, koja vraća string reprezentaciju datog objekta bez modifikovanja originalnog objekta:

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

Takođe možete koristiti operater proširene dodeljivanja za konkatenaciju. Ovo je predstavljeno znakom plus i jednakosti (`+=`), i obavlja kako konkatenaciju tako i dodeljivanje u jednom koraku. Evo ga u akciji:

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

Proces ubacivanja varijabli i izraza u niz se naziva **interpolacija niza**. Python ima kategoriju niza nazvanu **f-stringovi** (skraćeno od *formatted string literals*), koja vam omogućava da rukujete interpolacijom sa kompaktnim i čitljivim sintaksom.

F-stringovi počinju sa `f` (ili malim, ili velikim slovom) pre navodnika, i omogućavaju vam da ugradite varijable ili izraze unutar polja za zamenu označena vitičnim zagradama (`{}`). Evo primera:

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

Primećujete kako ne morate da konvertujete tipove koji nisu stringovi pomoću funkcije ``str()``. U gornjem primeru, vrednost promenljivih ``age``, ``num1`` i ``num2`` konvertuje se "ispod haube" u string tokom procesa interpolacije.
