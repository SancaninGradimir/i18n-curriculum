# --description--

Imena varijabli mogu biti napisana po jednom od sledećih formata:

- `camelCase`
- `PascalCase`
- `snake_case`
- `kebab-case`

U gore navedenim formatima, reči su razdvojene donjom crtom (`_`), koso ravni linije (`-`) ili nova reč počinje velikim slovom.

Da biste konvertovali u SCREAMING_SNAKE_CASE:

- Sve slova učinite velikim
- Razdvojite reči donjom crtom (`_`)

```py
import re
def to_screaming_snake_case(variable_name):
    temp = re.sub(r'[-_]+', ' ', variable_name)
    temp = re.sub(r'([a-z0-9])([A-Z])', r'\1 \2', temp)
    words = temp.strip().split()
    return '_'.join(words).upper()
```
