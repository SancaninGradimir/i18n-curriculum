# --description--

Kada vam je dat niz brojeva i ciljni broj, vratite prvi podskup sa dve ili više brojeva čiji zbir odgovara ciljnom broju.

- Prvi podskup je onaj čiji elementi imaju najmanji mogući broj elemenata, uzimajući u obzir elemente u prvom redosledu.
- Svaki broj u nizu podataka može se koristiti samo jednom.
- Ako ne postoji validan podskup, vratite `"Sum not found"`.

Vratite odgovarajuće brojeve kao niz podataka u redosledu kako su pojavili u originalnom nizu podataka.

```py
def find_sum(arr, target):
    def search(start, remaining, current):
        if remaining == 0 and len(current) >= 2:
            return current
        if start >= len(arr):
            return None
        for i in range(start, len(arr)):
            result = search(i + 1, remaining - arr[i], current + [arr[i]])
            if result is not None:
                return result
        return None

    return search(0, target, []) or "Sum not found"
```
