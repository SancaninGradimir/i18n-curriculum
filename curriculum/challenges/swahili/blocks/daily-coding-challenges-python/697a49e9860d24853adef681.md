# --description--

Ako vam je dat 2D niz podataka pobednika medalja, vratite brojač medalja za svaku zemlju u formatu CSV niza karaktera.

- U datom nizu podataka, svaki podniz predstavlja jedan događaj: `[gold_winner, silver_winner, bronze_winner]`

- Vraćeni CSV niz karaktera treba da ima sledeći format, gde je prvi red zaglavlja:

```sh
Country,Gold,Silver,Bronze,Total
country_name,gold_count,silver_count,bronze_count,total_medals
```

- Razdvojite nove linije korišćenjem karaktera nove linije (`"\n"`).
- Ne uključujte razmake pored zareza ili na kraju redova.
- Sortirajte vraćeni CSV niz karaktera po broju zlatnih medalja, od najvećeg ka najmanjem. Ako dve zemlje imaju isti broj zlatnih medalja, sortirajte ih po abecednom redu.

Na primer, ako dobijete:

```js
[
  ["USA", "CAN", "NOR"],
  ["NOR", "USA", "CAN"],
  ["USA", "NOR", "SWE"]
]
```

Povratna vrednost (Return):

```sh
"Country,Gold,Silver,Bronze,Total\nUSA,2,1,0,3\nNOR,1,1,1,3\nCAN,0,1,1,2\nSWE,0,0,1,1"
```

Šta izgleda ovako kada se ispiše:

```sh
Country,Gold,Silver,Bronze,Total
USA,2,1,0,3
NOR,1,1,1,3
CAN,0,1,1,2
SWE,0,0,1,1
```

```py
def count_medals(winners):
    counts = {}

    for gold, silver, bronze in winners:
        for country, medal_type in [(gold, "Gold"), (silver, "Silver"), (bronze, "Bronze")]:
            if country not in counts:
                counts[country] = {"Gold": 0, "Silver": 0, "Bronze": 0}
            counts[country][medal_type] += 1

    countries = [
        {
            "Country": country,
            "Gold": medals["Gold"],
            "Silver": medals["Silver"],
            "Bronze": medals["Bronze"],
            "Total": medals["Gold"] + medals["Silver"] + medals["Bronze"]
        }
        for country, medals in counts.items()
    ]

    countries.sort(key=lambda c: (-c["Gold"], c["Country"]))

    lines = ["Country,Gold,Silver,Bronze,Total"]
    for c in countries:
        lines.append(f'{c["Country"]},{c["Gold"]},{c["Silver"]},{c["Bronze"]},{c["Total"]}')

    return "\n".join(lines)
```
