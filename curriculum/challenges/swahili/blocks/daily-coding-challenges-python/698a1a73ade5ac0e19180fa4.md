# --description--

Dato je matric (niz nizova) celih brojeva. Premestite sve vrednosti u matricom za zadatu količinu.

- Pozitivan pomeraj pomera vrednosti udesno.
- Negativan pomeraj pomera vrednosti ulevo.

Vrednosti se moraju ciklički prebacivati:

- Otvažite matric kao jednu sekvencu vrednosti.
- Kada vrednost prođe kraj reda, nastavlja na početak sledećeg reda.
- Kada vrednost prođe poslednju poziciju u matrici, vraća se na prvu poziciju.
- Ista pravila važe i za pomeraj ulevo.

Na primer, ako dobijete:

```js
[
  [1, 2, 3],
  [4, 5, 6]
]
```

i pomeraj od `1`, premestite sve brojeve jedan korak udesno:

```js
[
  [6, 1, 2],
  [3, 4, 5]
]
```

```py
def shift_matrix(matrix, shift):
    rows = len(matrix)
    cols = len(matrix[0])
    flat = [num for row in matrix for num in row]
    n = len(flat)
    s = shift % n
    shifted = flat[-s:] + flat[:-s] if s != 0 else flat[:]
    result = []
    for i in range(rows):
        result.append(shifted[i*cols:(i+1)*cols])

    return result
```
