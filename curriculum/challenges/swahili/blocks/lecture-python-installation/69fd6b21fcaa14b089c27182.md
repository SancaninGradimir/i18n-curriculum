# --description--

U prethodnoj lekciji ste naučili kako da instalirate Python na svoj lokalni uređaj. U ovoj lekciji, naučićete kako da pokrenete Python kod lokalno.

Pre nego što počnete da gradite svoje Python skripte lokalno, moraćete da instalirate urednik koda ili IDE. IDE je skraćenica za Integrated Development Environment (Integrisano razvojno okruženje) i ima funkcije koje uključuju uređivač koda, alate za testiranje, terminal i još mnogo toga.

Jedan popularan uređivač koda koji koriste mnogi programeri je VS Code. Posetite `https://code.visualstudio.com/download` da preuzmete VS Code, koji podržava Mac, Windows i Linux okruženja. Ovo je uređivač koji će se koristiti u ovoj lekciji.

Ostale uobičajene opcije za razvoj Pythona uključuju sledeće:

- PyCharm
- Spyder

Kada preuzmete svoj uređivač, moraćete ga otvoriti. Nakon što je otvoren, moraćete da otvorite fasciklu (folder). Možete kliknuti na opciju menija `File` u gornjem levom uglu i kliknuti na `Open Folder`. Odatle, možete izabrati ili kreirati fasciklu za svoj projekat. Za dodatnu vežbanje van lekcije, možda želite fasciklu pod nazivom `python-projects`.

Kada postavite svoju fasciklu, možete otvoriti istraživač (explorer), koji je bočna traka na levoj strani u VS Code-u. Kliknite na ikonu `New File`, koja izgleda kao komad papira sa plus ikonom iznad njega. Zatim kreirajte fajl pod nazivom `main.py` i pritisnite Enter. `.py` je ekstenzija fajla koja signalizira da je ovo Python fajl.

Otvorite svoj `main.py` fajl i ukucajte sledeći kod:

```py
print("Hello, world!")
```

Da biste pokrenuli ovaj kod, imate nekoliko opcija. Prva opcija je kliknuti na dugme za pokretanje (run button), koje izgleda kao taster za reprodukciju i nalazi se u gornjem desnom uglu. To će otvoriti terminal i pokrenuti vaš Python skript tamo. Trebalo bi da vidite tekst `"Hello, world!"`.

Druga opcija je da otvorite terminal i ručno pokrenete program koristeći:

```bash
python main.py
```

Morate pokrenuti ovu komandu iz fascikle gde je `main.py` sačuvan. Na primer, ako se `main.py` nalazi unutar fascikle pod nazivom `python-projects`, prvo koristite sledeće u terminalu:

```bash
cd python-projects
```

Zatim pokrenite:

```bash
python main.py
```

U nekim okruženjima, posebno na macOS i Linux sistemima gde su instalirani i Python 2 i Python 3, možda ćete morati da koristite `python3` umesto `python`. Ovo je uobičajeno kada komanda `python` ne postoji ili ukazuje na stariju verziju Pythona.

```bash
python3 main.py
```

Kako napredujete kroz kurs, trebalo bi da pokušate sa primerima iz lekcija lokalno i da vidite rezultate. Takođe biste trebali da pokušate da sami kreirate primere kako biste testirali ono što ste naučili iz lekcija.
