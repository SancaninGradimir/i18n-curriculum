# --description--

U prethodnoj lekciji ste naučili kako da pokrenete Python skripte lokalno. Ali mogu postojati situacije kada ne želite da kreirate pune Python programe i samo vam je potrebno testirati neki Python kod. Ovde dolazi Python interaktivan shell.

Ali prvo, hajde da pregledamo šta je terminal.

Terminal je tekstualni interfejs koji vam omogućava da komandama pišete i interagujete sa svojim računarom. Svaki operativni sistem dolazi sa podrazumevanom aplikacijom za terminal. Na macOS-u, možete koristiti aplikaciju Terminal. Na Windowsu, možete koristiti Command Prompt ili PowerShell. Na Linuksu, svako okruće za radnu površinu ima svoju podrazumevanu aplikaciju za terminal, kao što su GNOME Terminal ili Konsole.

Otvorite terminalnu aplikaciju i ukucajte u `python` i pritisnite `Enter`. Ovo će pokrenuti interaktivnu Python šelju. Interaktivna šelja je program koji vam omogućava da ukucate komande jednu po jednu i vidite rezultate.

Kada započnete novu sesiju, možda ćete prvobitno videti ovakav tip izlaza:

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

Simbol `>>>` znači da Python čeka da unesete komandu. Pokušajte da ispišete `"Hello, world!"` u terminal:

```bash
print("Hello, world!")
```

TEKST:

Trebalo bi da vidite tekst kako se pojavljuje ovako:

```bash
>>> print("Hello, world!")
Hello, world!
```

Nakon što se tekst ispruša, Python se vraća na sledeće:

```bash
>>>
```

Ovo znači da možete uneti drugu komandu.

Python interpresor prati ono što je poznato kao ciklus Čitaj, Procцени, Ispiši, Ponovi. Ili REPL skraćeno. Kad ukucate komande, interpresor će pročitati unos, proceniti ga, ispisati rezultat i vratiti se da prikaže `>>>` kako biste mogli uneti više komandi.

Šta se dešava ako pokušate da ukucate nevažeći komandni red kao ovaj?

```bash
>>> something random
```

Pa, Python interpreser će i dalje pratiti isti REPL proces. U ovom slučaju, dobićete poruku o grešci:

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

Ako želite napustiti interaktivnu šelju, možete ukucati `exit()` ili pritisnuti `Ctrl + D` za (Mac/Linux) ili `Ctrl + Z + Enter` za (Windows).

Korišćenje interaktivne šelje Pythona odlično je za male eksperimente sa kodom ili osnovnu istraživanje. Dok prolazite kroz preostale lekcije u kursu, možete ukucati neke od novih metoda koje ste naučili unutar interaktivne šelje.

Ako planirate da radite sa dužim programima ili radite preko više fajlova, onda je bolje raditi u uređivaču koda ili IDE-u.
