# --description--

Prethodni izazovi su se doticali specificnih HTML elemenata i njihove upotrebe. Ipak, postoji nekoliko elemenata koji daju celokupnu strukturu stranici i treba da budu ukljuceni u svaki HTML dokument.

Na samom vrhu dokumenta treba da kazete pregledacu koju verziju HTML-a koristi stranica. HTML je jezik koji se razvija i redovno se azurira. Moderni pregledaci podrzavaju najnoviju verziju, HTML5. Ipak, starije web stranice mogu koristiti ranije verzije jezika.

Ovu informaciju pregledacu saopstavate dodavanjem `<!DOCTYPE ...>` oznake u prvi red, gde je `...` verzija HTML-a. Za HTML5 koristi se `<!DOCTYPE html>`.

`!` i veliko `DOCTYPE` su bitni, posebno za starije pregledace. `html` moze biti malim ili velikim slovima.

Zatim ceo HTML kod treba da bude obavijen `html` tagovima. Otvarajuci `<html>` tag ide odmah ispod linije `<!DOCTYPE html>`, a zatvarajuci `</html>` tag ide na kraj stranice.

Evo primera strukture stranice. Vas HTML kod ide izmedju dva `html` taga.

```html
<!DOCTYPE html>
<html>

</html>
```

# --instructions--

Dodajte `DOCTYPE` oznaku za HTML5 na vrh praznog HTML dokumenta u editoru. Ispod nje dodajte otvarajuci i zatvarajuci `html` tag, koji treba da obuhvataju `h1` element. Naslov moze sadrzati bilo koji tekst.

# --hints--

Vas kod treba da sadrzi `<!DOCTYPE html>` oznaku.

```js
assert.match(code,/<!DOCTYPE\s+?html\s*?>/gi);
```

Treba da postoji jedan `html` element.

```js
assert.lengthOf(document.querySelectorAll('html'), 1);
```

`html` tagovi treba da obuhvataju jedan `h1` element.

```js
assert.match(code,/<html>\s*?<h1>\s*?.*?\s*?<\/h1>\s*?<\/html>/gi);
```
